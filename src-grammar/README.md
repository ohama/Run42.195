## About This Grammar Guide

이 문법 가이드는 Raymond Murphy의 "English Grammar in Use" 교재 구조를 따릅니다.
각 유닛은 소설 속 실제 예문으로 문법을 설명합니다.

## Each Unit Contains

1. **문법 설명** - 영어/한국어 설명
2. **소설 속 예문** - 3-5개 예문 + 한국어 번역
3. **비교 표** - 유사 문법 비교
4. **연습 문제** - 빈칸 채우기, 문장 변환
5. **암기 문장** - Key Sentences

## How to Use

1. 왼쪽 목차에서 학습할 유닛 선택
2. 문법 설명 읽기
3. 소설 예문으로 실제 사용법 확인
4. 연습 문제 풀기
5. 암기 문장 복습
