# 달리는 일기 (Run42.195)

14살 소년이 버스 멀미로 시작해 풀 마라톤을 완주하기까지, 1년간의 성장 일기.

## 소개

제주도에 사는 중학교 1학년 김도윤. 매일 만원 버스에서 멀미와 싸우던 그가 우연히 걸어서 등교하게 되면서 시작되는 이야기. 걷기에서 달리기로, 달리기에서 마라톤으로. 1년 동안 104편의 일기를 통해 소년의 변화를 따라갑니다.

## 형식

- **장르**: 청소년 성장 소설 (일기체)
- **분량**: 104편 (주 2회, 월/목)
- **기간**: 2024년 3월 ~ 2025년 3월 (1년)
- **시점**: 1인칭 (김도윤)

## 구성

| Part | 기간 | 내용 |
|------|------|------|
| 1 | 3-4월 | 걷기의 시작 - 버스에서 도보 통학으로 |
| 2 | 5-8월 | 뛰기의 시작 - 걷기에서 달리기로 |
| 3 | 9-12월 | 마라톤을 향해 - 본격적인 훈련 |
| 4 | 1-3월 | 42.195km - 첫 풀 마라톤 도전 |

## 주요 인물

- **김도윤** (14세) - 운동을 싫어하던 평범한 중학생
- **강민서** - 도윤의 친구, 관찰력이 좋음
- **고영철 할아버지** - 감귤밭 농부, 도윤의 멘토
- **엄마** - 걱정하지만 응원하는 어머니
