# Short Stories — 문법 암기용 단편

소설 "제주 소년의 마라톤 도전기"의 장면을 바탕으로,
여러 문법 유닛을 하나의 이야기로 엮어 자연스럽게 암기할 수 있도록 만든 단편 모음입니다.

## 활용법

1. 이야기를 처음부터 끝까지 읽으며 **굵은 글씨** 문법에 집중
2. 각 파트 끝의 📝 해설로 문법 포인트 확인
3. Grammar Map으로 유닛별 복습
4. Key Sentences 암기
