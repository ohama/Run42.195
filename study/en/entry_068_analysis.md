# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Running-Related Everyday Phrases

### "I ran 17km today."

**Original Korean:** "오늘 17km를 달렸다."

**Variations by Context:**

| Context | Expression | Nuance |
|---------|------------|--------|
| Casual report | "I ran 17km today." | 단순 보고 |
| Proud | "I ran 17km today!" | 자긍심 |
| Surprised at self | "I actually ran 17km today." | 자신도 놀람 |
| Explanation | "Today I ran 17km." | 강조 |
| Past accomplishment | "I did 17km." | 축약 (일상) |

**Practice Dialogue:**
```
A: What did you do this morning?
B: I ran 17km.
A: Seriously? That's amazing!
B: Yeah, I was tired afterward, but I made it.
```

---

### "My last long run before the race."

**Original Korean:** "마지막 장거리 뛰기"

**Grammar Point:** Appositive noun phrase (동격 표현)

**Structure:** [Achievement], [purpose/context].

**Similar Expressions:**

| Expression | Context |
|------------|---------|
| "My last long run before the race." | 최종 연습 |
| "My final practice before the exam." | 시험 직전 |
| "My last chance to impress them." | 마지막 기회 |
| "My biggest challenge so far." | 가장 큰 도전 |

**Pronunciation Tip:**
- "Last" = /læst/ (ㅐ 발음, 혀를 펴서)
- "Race" = /reɪs/ (에이 소리)

### "Once I got past 15km, it was really hard."

**Original Korean:** "15km를 넘으니까 정말 힘들었다."

**Grammar Point:** Temporal clause with "once"

**Structure:** Once + subject + past tense, + result

**"Once" Patterns:**

| Example | Meaning |
|---------|---------|
| "Once I got past 15km, it was hard." | 15km 이상부터는 힘들었다 |
| "Once the race started, I felt calm." | 경주 시작하니 진정됐다 |
| "Once I saw the finish line, I had energy." | 결승선 보니 힘이 났다 |

**Similar Temporal Connectors:**

| Connector | Example |
|-----------|---------|

| After | "After 15km, things got difficult." |
| When | "When I hit 15km, my legs felt heavy." |

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: Announcing Achievement + Surprise Reaction

**Pattern:**
```
A: I ran 17km this morning.
B: Seriously? That's crazy.
A: It was tough. Once I got past 15km, it was really hard.
B: But you did it.
A: Yeah.
```

**Why This Works:**
- Direct statement
- Skeptical response shows significance
- Honest difficulty admission
- Sympathetic acknowledgment
- Brief confirmation

**Variations:**

| A's Statement | B's Reaction | A's Response |
|---------------|--------------|-------------|
| "I ran a marathon." | "That's insane!" | "It was the hardest thing I've done." |
| "I ran 20km." | "No way!" | "Yeah, I was surprised too." |
| "I ran my fastest 10k." | "That's amazing!" | "Thanks, I trained hard for it." |

---

## Pattern 2: Downplaying Difficulty, Then Admitting

**Pattern:**
```
A: How was your run?
B: It was okay.
A: Really? You don't look okay.
B: [Sighs] Honestly, it was exhausting. My legs were shaking afterward.
```

**Why This Works:**
- Initial minimization (pride/not wanting to complain)
- Physical evidence contradicts
- Honest admission follows
- Validates difficulty

---

## Pattern 3: Explaining Difficulty with Timeline

**Pattern from Text:**
```
"It was tough. Once I got past 15km, it was really hard."
```

**General Structure:**
"[Achievement] was [emotion]. Once I [difficulty point], it was [intensity]."

| Achievement | Emotion | Difficulty | Intensity |
|------------|---------|------------|-----------|
| "The presentation" | "scary" | "started talking" | "okay" |
| "The hike" | "good" | "hit the altitude" | "really hard" |
| "The interview" | "fine" | "got asked about failure" | "really tough" |

---

## Pattern 4: Coach's Encouragement Structure

**Pattern:**
```
Coach: "You completed 17km. You'll be able to do it in the race too."
Coach: "You just need to run 4 more kilometers."
```

**Logic Flow:**
1. Acknowledge what was accomplished
2. Connect to future goal
3. Quantify remaining challenge
4. Simplify/minimize remaining distance

**Practice Creating:**
```
A (Coach): "You did [big achievement]. You'll be able to [future goal] too."
A: "You just need to [remaining challenge]."
```

---

## Pattern 5: Family Pride Moment

**Pattern from Text:**
```
Dad: "17km? That's amazing."
Me: "It was hard. But I made it."

```

**Three Different Responses:**
- Dad: Amazement (감탄)

- Mom: Caution/concern (우려)

**Dialogue Practice:**
```

Sister: "How was it?"

```

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Reporting to Coach After Long Run

```
Me: Coach, I ran 17km today.
Coach: Oh, nice work. Was it hard?
Me: Yeah. Once I got past 15km, it was really tough.
Coach: Of course. That's uncharted territory for you.
Me: But I did it.
Coach: You'll be able to do it in the race too. You just need to run 4 more kilometers.
Me: I'm confident about it now.
Coach: Next week, start cutting back the distance.
Me: Yes, sir. I'll stick to the plan.
Coach: You need to let your body rest. Condition is everything.
Me: I understand.
```

**Key Patterns:**
- Achievement statement
- Coach validation + recognition of difficulty
- Honest feedback about struggle
- Encouragement connecting to future goal
- Specific next steps
- Student commitment

---

### Scenario 2: Sharing Achievement with Friend

```
Me: I ran 17km this morning.
Minseo: Seriously? That's crazy.
Me: It was tough. Once I got past 15km, it was really hard.
Minseo: But you did it.
Me: Yeah. It was my last long run before the race.
Minseo: When's the race?
Me: November 24th.
Minseo: So you've got like three weeks left.
Me: Yeah. Now I need to cut back on distance and work on my condition.
Minseo: Got it. I'll cheer you on.
Me: Thanks. That means a lot.
Minseo: You're going to crush it.
```

**Key Patterns:**
- Direct announcement of achievement
- Friend's surprise response
- Honest admission of difficulty
- Time/date information
- Training plan discussion
- Mutual support

---

### Scenario 3: Family Discussion About Training

```
Me: I ran 17km today.
Dad: 17km? That's amazing.
Me: It was hard. But I made it.
Mom: Don't push too hard. You can't get injured before the race.
Me: I know. I'm cutting back on distance from now on.
Dad: How much longer until the race?
Me: Three weeks. November 24th.
Mom: That's enough time to recover and prepare properly.
Me: I'm ready.
Dad: I'm proud of you.
```

**Key Patterns:**
- Father's admiration
- Mother's practical concern
- My reassurance about training plan
- Timeline confirmation
- Readiness statement
- Parental pride expression
