# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Morning & Running Start Expressions

### "Clear skies. Cool air."

**Original Korean:** "맑은 날씨. 시원한 바람."

**Analysis:**
- Fragment 형식으로 날씨 묘사
- 직관적이고 감정 표현

**Variations by Formality:**
| Casual | Standard | Formal |
|--------|----------|--------|
| "Clear skies. Cool air." | "The skies were clear and the air was cool." | "There were clear skies and cooling atmospheric conditions." |
| "Beautiful day." | "It was a beautiful day." | "The meteorological conditions were favorable." |
| "Nice weather." | "The weather was nice." | "Weather conditions were pleasant." |

**Weather Description Patterns:**
```
Fragment format (casual): "Clear skies. Cool air. Perfect day."
Full sentence (standard): "It was a clear day with cool air."
Poetic (literary): "Beneath clear skies and cool morning air..."
```

---

### "Just pulling back on the distance."

**Original Korean:** "거리를 줄여서"

**Grammar Point:** "Pull back" = 물러나다, 줄이다

**Variations:**
| Expression | Nuance |
|------------|--------|
| "pulling back on the distance" | 거리 줄이기 (점진적) |
| "cutting back" | 줄이기 (더 직접적) |
| "reducing the distance" | 거리 감소 (중립) |
| "easing off" | 조금 덜하기 (부드러움) |
| "backing off" | 물러나기 |

**Training Terminology Context:**
```
"I was pulling back on the distance." (거리를 줄였다)
"I'm cutting back on my workouts." (운동량을 줄였다)
"The coach said to ease off for a week." (코치가 일주일간 덜하라고 했다)
```

---

### "So much lighter than when I was doing 17km."

**Original Korean:** "17km를 했을 때보다 훨씬 가볍다"

**Grammar Point:** Comparative + "than" structure

**Pattern:**
```
X is [adjective] than when I was doing Y.
```

**Examples:**
| Comparison | Full Sentence |
|------------|---------------|
| 더 가볍다 | "I feel so much lighter than when I was doing 17km." |
| 더 쉽다 | "This is so much easier than when I was training hard." |
| 덜 피곤하다 | "I'm so much less tired than last month." |
| 더 행복하다 | "I'm so much happier than when I had that job." |

**Recovery Language:**
- "I feel lighter." (가벼워졌어)
- "My legs feel fresher." (다리가 더 신선해)
- "The fatigue is dropping off." (피로가 빠져나가고 있어)

## 1.2 Casual Dialogue Responses

### "Hey, you look good."

**Original Korean:** "어라, 넌 좋아 보여."

**What "Look Good" Means Here:**
- 눈에 띄게 나아 보임
- 더 건강해 보임
- 피곤해 보이지 않음

**"Look" + Adjective Patterns:**
| Expression | Meaning |
|------------|---------|
| "You look good." | 넌 좋아 보여 (건강/잘 지냄) |
| "You look tired." | 넌 피곤해 보여 |
| "You look happy." | 넌 행복해 보여 |
| "You look different." | 넌 달라 보여 |
| "You look amazing." | 넌 멋져 보여 |
| "You look exhausted." | 넌 너무 피곤해 보여 |

**Responding to Compliments:**
```
"You look good."
"Huh? Really?"
"Yeah, you look less tired than last week."
"Oh, maybe because I'm running shorter distances now."
"Seems like it's working."
```

---

### "Huh? Really?"

**Original Korean:** "응? 진짜?"

**Analysis:** 의외성/확인

**Nuance Spectrum:**
| Response | Tone | Meaning |
|----------|------|---------|
| "Huh?" | 당황 | 뭐라고? |
| "Really?" | 의외 | 정말? |
| "Seriously?" | 믿기 어려움 | 진짜? |
| "No way?" | 매우 놀람 | 말도 안 되네 |

**How to Use:**
```
A: "You look good."
B: "Huh? Really?" (당황한 확인)
A: "Yeah, you look less tired."
B: "Oh!" (이해함)
```

---

### "Seems like it's working."

**Original Korean:** "효과가 있는 것 같은데"

**Grammar Point:** "It + seems like" for observation

**Pattern:**
```
It seems like + statement.
Seems like + noun phrase.
```

**Variations:**
| Expression | Meaning |
|------------|---------|
| "Seems like it's working." | 효과가 있는 것 같아 |
| "It seems like a good idea." | 좋은 생각인 것 같아 |
| "Seems like progress." | 진척되는 것 같아 |
| "It seems to be helping." | 도움이 되는 것 같아 |

**Affirmation Examples:**
```
"How's the new training plan?"
"Seems like it's working. I feel better."
"Great, keep it up then."
```

## 1.4 Coach's Instructions

### "If you want to run your best at the race, you need to rest now."

**Original Korean:** "레이스에서 최고의 컨디션으로 달리려면, 지금은 쉬어야 해"

**Grammar Point:** Conditional + necessity

**Pattern:**
```
If you want + goal, you need to + action.
```

**Coach-Like Statements:**
| Goal | Action | Full Sentence |
|------|--------|---------------|
| 경기 잘하기 | 쉬기 | "If you want to do well at the race, you need to rest." |
| 건강 유지 | 먹기 | "If you want to stay healthy, you need to eat right." |
| 목표 달성 | 노력하기 | "If you want to achieve your goal, you need to work hard." |
| 부상 피하기 | 스트레칭하기 | "If you want to avoid injury, you need to stretch." |

---

**Original Korean:** "그것만? 괜찮아?"

**Questioning Small Numbers:**

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "Is this what [experience] feels like?"

**Structure:** Is this what + noun phrase + feels like?

**From Text:** "Is this what conditioning adjustment feels like?"

**Examples:**
| Question | Context |
|----------|---------|
| "Is this what success feels like?" | 성공 경험 |
| "Is this what freedom feels like?" | 자유로움 |
| "Is this what adulthood feels like?" | 성인이 되는 느낌 |
| "Is this what depression feels like?" | 우울증 |
| "Is this what love feels like?" | 사랑하는 감정 |

**Nuance:** 새로운 경험에 대해 확인하는 질문

---

## Pattern 2: Coach Affirmation + Explanation

**Structure:** [Affirmation] + "That's + noun." + Explanation.

**From Text:**
```
Coach: "That's conditioning adjustment. The fatigue is dropping off."
```

**Pattern:**
```
A: "My body feels light."
Coach: "That's [technical term]. [Explanation]."
```

**Examples:**
```
Student: "I feel less sore."
Coach: "That's recovery. Your muscles are adapting."

Runner: "I have more energy."
Coach: "That's overcompensation. Your body's adjusting."
```

---

## Pattern 3: Conditional Advice (If... then...)

**Structure:** If + goal, you need/have to + action.

**From Text:** "If you want to run your best at the race, you need to rest now."

**Coach Patterns:**
| Goal | Action | Full Sentence |
|------|--------|---------------|
| 경기 잘하기 | 쉬기 | "If you want to run well, you need rest." |
| 부상 피하기 | 스트레칭 | "If you want to avoid injury, you need to stretch." |
| 체력 유지 | 먹기 | "If you want to maintain fitness, you need to eat." |
| 집중하기 | 충분한 수면 | "If you want to focus, you need sleep." |

---

## Pattern 4: Reassurance About Small Numbers

**Structure:** [Surprise about small number] → Coach reassurance

**From Text:**
```
Athlete: "That little? That's okay?"
Coach: "Yeah. You have to give your body a break. You've got all the fitness built up by now."
```

**Dialogue Pattern:**
```
A: [Expresses doubt about reduced training]
B: [Explains recovery principle]
A: [Accepts explanation]
```

---

## Pattern 5: Progress Reflection

**Structure:** [Activity] + [Result observation]

**From Text:**
```
"12km adjustment run complete.
Body feels light.
Fatigue dropping away.
Doing what the coach said."
```

**Note-Taking Style:**
- Fragment sentences

## 7.3 Conversational Hedging

### "Maybe because I'm running shorter distances now."

**Original Korean:** "그래서일지도 모르니까"

**Hedging Words:**
| Word | Function |
|------|----------|
| maybe | 불확실성 |
| I think | 의견임을 나타냄 |
| kind of | 부분적 |
| sort of | 어느 정도 |
| I guess | 추측 |

**Natural Explanation Pattern:**
```
"You look better."
"Maybe because I'm running shorter distances."
"Seems like it's working."
```

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Coach Meeting - Adjustment Run Discussion

```
Athlete: "Coach, my body feels so light these days."
Coach: "That's conditioning adjustment. The fatigue is dropping off."
Athlete: "Oh."
Coach: "If you want to run your best at the race, you need to rest now."
Athlete: "Okay."
Coach: "During race week, just do about 3km."
Athlete: "That little? That's okay?"
Coach: "Yeah. You have to give your body a break. You've got all the fitness built up by now."
Athlete: "Got it."
```

**Key Learning Points:**
- Coach uses technical terms
- Athlete asks clarifying questions
- Coach provides reassurance
- Athlete shows acceptance

---

### Scenario 2: Friend Noticing Improvement

```
Minseo: "Hey, you look good."
Doyun: "Huh? Really?"
Minseo: "Yeah. You look less tired than last week."
Doyun: "Oh, maybe because I'm running shorter distances now."
Minseo: "Seems like it's working."
Doyun: "Yeah. Just doing what the coach told me to."
Minseo: "Smart. The race is coming up soon, right?"
Doyun: "Thirteen days. Can't wait."
```

**Observation Skills:**
- Physical changes visible
- Friends notice improvement
- Explanation of training changes

---

### Scenario 3: Training Explanation

```
Friend A: "Why are you running less?"
Friend B: "I'm in the adjustment phase. The race is coming."
Friend A: "Adjustment phase?"
Friend B: "Yeah, you reduce mileage to recover. It's what my coach said."
Friend A: "Does it feel different?"
Friend B: "Yeah, my body feels lighter. Less fatigue dropping away."
Friend A: "That's smart. You'll be ready for the race."
Friend B: "That's the goal. Peak condition."
```

## 10.4 Discussion Questions
