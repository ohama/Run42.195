# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Weather-Related Expressions

### "Just a drizzle."

**Original Korean:** "이슬비일 뿐이야"

**Analysis:**
- 영어에서도 짧고 단호한 표현
- 빗의 강도를 정확하게 표현
- 상황을 긍정적으로 평가하는 뉘앙스

**Variations by Intensity:**
| Expression | Intensity | Korean |
|------------|-----------|--------|
| "Drizzle" | ★☆☆ | 이슬비, 가랑비 |
| "Light rain" | ★☆☆ | 약한 빗소리 |
| "Steady rain" | ★★☆ | 꾸준한 빗소리 |
| "Heavy rain" | ★★★ | 많은 빗소리 |
| "Pouring" / "Downpour" | ★★★★ | 억수같은 빗소리 |
| "It's raining cats and dogs" | ★★★★★ | 퍼붓는 빗소리 (관용구) |

**Pronunciation Tip:**
- "Drizzle" = /ˈdrɪzəl/ (두 음절, 첫음절에 강세)
- "Drizzle" vs "Drizzly" (명사 vs 형용사)

**Practice Dialogue:**
```
Mom: It's raining again today?
You: It's just a drizzle. I can run in this.
Mom: Are you sure?
You: Yeah. I looked it up, and light rain is fine.
Mom: Alright, but be careful.
```

### "It picks up"

**Original Korean:** "빗이 세지다"

**Grammar Point:** Phrasal verb with weather conditions

**Structure:** Weather condition + picks up = (날씨가) 강해지다

**Similar Weather Phrasal Verbs:**

| Phrasal Verb | Meaning | Example |
|--------------|---------|---------|
| "pick up" | 강해지다/심해지다 | "If the rain picks up, go inside." |
| "let up" | 약해지다/그치다 | "The rain finally let up." |
| "clear up" | 개다/맑아지다 | "The sky should clear up by noon." |
| "cloud over" | 흐려지다 | "The sun clouded over." |
| "cool down" | 시원해지다 | "The temperature should cool down." |
| "warm up" | 따뜻해지다 | "It'll warm up by afternoon." |

**Examples:**
```
"Stop right away if it picks up." (빨라지면 멈춰)
"The wind is picking up." (바람이 불기 시작했다)
"The traffic should pick up around 5pm." (5시쯤 교통량이 늘겠지)
```

## 1.2 Affirmative/Permission Expressions

### "Alright, this should be okay."

**Original Korean:** "좋아, 이 정도면 괜찮겠네"

**Grammar Point:** "This should be" = 아마 괜찮을 거야

**"Should" for Prediction:**

| Usage | Meaning | Example |
|-------|---------|---------|
| "This should be okay." | 아마 괜찮을 거야 | Mom assessing weather |
| "We should arrive on time." | 우리 시간에 도착할 거야 | Prediction with confidence |
| "This should work." | 이게 작동할 거야 | Expecting success |
| "Everything should be fine." | 다 괜찮을 거야 | Reassurance |

**Nuance Difference:**

| Expression | Certainty Level | Usage |
|-----------|----------------|-------|
| "This will be okay." | ★★★★★ | 확실함 |
| "This should be okay." | ★★★★ | 거의 확실함 |
| "This might be okay." | ★★☆ | 약간의 의심 |
| "This could be okay." | ★★☆ | 가능성만 있음 |

**Practice Dialogue:**
```
Child: Can I go outside? It's just a drizzle.
Parent: Alright, this should be okay. But call me if it gets worse.
Child: Got it. I'll be careful.
```

---

### "Got it."

**Original Korean:** "알았다"

**Usage:** 지시, 경고를 받아들임

**Different Contexts for "Got it":**

| Context | Nuance | Example |
|---------|--------|---------|
| 지시 수락 | 이해함 | "Don't touch that." / "Got it." |
| 정보 이해 | 알았다 | "The meeting is at 3pm." / "Got it." |
| 경고 수락 | 주의하겠다 | "Watch out for the ice." / "Got it." |
| 농담 이해 | 웃음 | "Why did the chicken cross the road?" / "Got it, very funny." |

**Similar Acknowledgment Expressions:**

| Expression | Formality | Tone |
|------------|-----------|------|
| "Got it." | Casual | 친화적 |
| "Understood." | Standard | 중립 |
| "I understand." | Formal | 격식체 |
| "Copy that." | Very casual | 군인식/기계적 |
| "Roger that." | Slang | 매우 캐주얼 |
| "Sure." | Casual | 동의 |
| "Okay." | Casual | 동의 |

## 1.3 Reflective/Introspective Expressions

### "I looked it up"

**Original Korean:** "알아봤어"

**Grammar Point:** Phrasal verb - research/information seeking

**"Look up" Meanings:**

| Context | Meaning | Example |
|---------|---------|---------|
| 사전에서 찾기 | 단어/정보 찾다 | "I looked up the word in the dictionary." |
| 인터넷 검색 | 조사하다 | "I looked up information about running in rain." |
| 주소 찾기 | 위치 확인 | "I looked up the restaurant's address." |
| 존경하다 | 우러러보다 | "She looks up to her father." (다른 구조) |

**Similar Research Expressions:**

| Verb/Phrase | Meaning | Example |
|-------------|---------|---------|
| "look it up" | 알아보다 | "I'll look it up later." |
| "check it out" | 확인해보다 | "Let me check that out." |
| "search for" | 검색하다 | "I searched for information." |
| "find out" | 알아내다 | "I found out the answer." |
| "research" | 조사하다 | "I researched the topic." |
| "verify" | 확인하다 | "Let me verify that first." |

**Practice Dialogue:**
```
Mom: How do you know it's safe to run in the rain?
You: I looked it up. Light rain is actually fine.
Mom: What did you find out?
You: That people run in rain all the time. Just not in storms.
```

# PART 2: PHYSICAL SENSATIONS (신체 감각 표현)

## 2.1 Rain & Moisture Sensations

### "The rain touched my face. Damp. But cool."

**Original Korean:** "빗이 얼굴에 닿았다. 촉촉하다. 하지만 시원했다."

**Sensory Sequence:** Simple, vivid descriptions

**Why This Works:**
- 한 문장씩 감각을 분리해서 강조
- 반대의 감정을 나열 (damp지만 cool)
- 신체 감각에 집중

**Rain-Related Tactile Vocabulary:**

| Expression | Feeling | Context |
|------------|---------|---------|
| "The rain touched my face." | 빗이 닿다 | 부드러운 시작 |
| "Damp" | 촉촉한/젖은 | 약간 젖음 |
| "Cool" | 시원한 | 온도 낮음 |
| "Refreshing" | 상큼한 | 긍정적 느낌 |
| "Soaked to the bone" | 뼛속까지 젖음 | 완전히 젖음 |
| "Dripping" | 뚝뚝 떨어짐 | 물이 흘러내림 |
| "Wet through" | 흠뻑 젖음 | 완전 젖음 (영국식) |

**Body + Weather Sensations:**

| Sensation | Expression |
|-----------|------------|
| 얼굴에 빗 | "The rain touched my face." |
| 옷이 무거워짐 | "Clothes grew heavy." |
| 머리카락 젖음 | "Hair stuck to my forehead." |
| 몸 전체 젖음 | "My whole body was getting wet." |
| 차가움 | "Cool / refreshing" |
| 개운함 | "It was refreshing." |

### "My whole body was getting wet. Hair stuck to my forehead. Clothes grew heavy."

**Original Korean:** "온몸이 젖어가고 있었다. 머리가 이마에 붙어 있었다. 옷이 무거워졌다."

**Style:** Fragment sequence for cumulative effect

**Physical Changes During Rain Running:**

| Stage | Expression | Korean |
|-------|------------|--------|
| 초기 | "The rain touched my face." | 빗이 닿다 |
| 몸 젖음 | "My whole body was getting wet." | 온몸이 젖다 |
| 머리 젖음 | "Hair stuck to my forehead." | 머리가 이마에 붙다 |
| 옷 무거워짐 | "Clothes grew heavy." | 옷이 무거워지다 |
| 극도 | "Completely drenched." | 완전히 흠뻑 젖다 |

**"Grew" vs "Got" vs "Became":**

| Verb | Nuance | Example |
|------|--------|---------|
| "grew" | 천천히 변함 | "Clothes grew heavy." |
| "got" | 점차 변함 (casual) | "My clothes got heavy." |
| "became" | 갑자기/격식체 | "The clothes became heavy." |

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "I can run in this."

**Structure:** I + can + verb + in + condition

**Meaning:** 이 상황에서도 할 수 있다

**Decision/Capability Patterns:**

| Example | Situation |
|---------|-----------|
| "I can run in this." | 빗속에서도 달릴 수 있다 |
| "I can manage this." | 이것 정도는 할 수 있어 |
| "I can handle it." | 나는 할 수 있어 |
| "We can do this." | 우리는 할 수 있어 |

**"It's just" + noun Structure:**

| Example | Meaning |
|---------|---------|
| "It's just a drizzle." | 이슬비일 뿐이야 |
| "It's just a little rain." | 약한 빗소리일 뿐이야 |
| "It's just in my head." | 내 생각일 뿐이야 |

---

## Pattern 2: "I'm running in the rain. Other people are ... But I'm running."

**Structure:** Contrast through parallel structure

**Meaning:** 다른 사람과 자신의 선택 비교

**Contrast Patterns:**

| Structure | Effect |
|-----------|--------|
| "I'm running. Other people are ... But I'm running." | 자신의 독특함 강조 |
| "Most people do X. But I do Y." | 자신의 차이점 표현 |
| "They chose X. I chose Y." | 선택의 차이 |

---

## Pattern 3: "Not because someone told me to. Because I wanted to."

**Structure:** "Not because X. Because Y."

**Meaning:** 자신의 의지를 강조

**Motivation Clarification:**

| Pattern | Meaning |
|---------|---------|
| "Not because X. Because Y." | 진정한 동기는 Y임을 강조 |
| "I didn't do it for X. I did it for Y." | 동기 구분 |
| "It wasn't X. It was Y." | 오해 해결 |

**Examples:**
```
"Not because I was forced. Because I wanted to."
"Not because it's easy. Because it matters."
"Not because it's fun. Because I'm determined."
```

---

## Pattern 4: "Getting soaked is fine. Being uncomfortable is fine. Because I'm doing what I want."

**Structure:** Action + "is fine" + reason

**Meaning:** 불편함도 기꺼이 감수할 수 있다

**Acceptance/Resignation Patterns:**

| Pattern | Tone |
|---------|------|
| "X is fine. Y is fine. Because Z." | 수용적/긍정적 |
| "X is okay. Y is okay. As long as Z." | 조건부 수용 |
| "I don't mind X or Y. As long as Z." | 별 상관 없음 |

---

## 10.1 Dialogue Practice

### Scenario 1: Deciding to Run in the Rain

```
Mom: It's raining again today. Are you sure you want to run?

Mom: Really? Okay, but if it picks up, stop right away.
You: Got it. I'll be careful.

You: Thanks, Mom. I'll be back soon.
```

```
Friend: Did you really run in the rain? That's insane!

Friend: Didn't you get cold?

```

```
Friend: You're crazy, but I kind of admire that.
You: [laughs] Thanks, I think?
```
