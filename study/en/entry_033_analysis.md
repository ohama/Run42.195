# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Suggesting & Offering Activities

### "Hey, what if I started running too?"

**Original Korean:** "야, 나도 달리기 시작하면 어떨까?"

**Grammar Point:** "What if" pattern for suggestions/possibilities

**Structure:** "What if + subject + past tense?" = 만약 ~하면 어떨까?

**Uses:**
- 새로운 생각/제안 제시
- 상대의 경험을 따라하고 싶을 때
- 호기심이나 가능성 표현

**Similar Suggestion Patterns:**

| Expression | Nuance | When to Use |
|------------|--------|------------|
| "What if I started running too?" | 호기심/가능성 | 새로운 아이디어 떠올랐을 때 |
| "Maybe I should try it." | 생각해보기 | 신중하게 고려할 때 |
| "I could give it a shot." | 시도 의욕 | 도전해볼 준비가 있을 때 |
| "What's stopping me?" | 반발심 | 장애물 극복하려 할 때 |
| "Want to do it together?" | 동반자 찾기 | 혼자가 아니고 싶을 때 |

**Practice Dialogue:**
```
A: You always look so happy when you run.
B: Yeah, it makes me feel good.
A: Hey, what if I started running too?
B: Really? That'd be awesome!
A: Yeah, but I'm not sure I could do it alone.
B: Then we'll do it together.
```

### "Watching you run lately, it looks pretty good."

**Original Korean:** "요즘 너 달리는 거 보니까 좋아 보이네."

**"Looks + adjective" pattern:** 겉보기에, ~해 보이다

**Analysis:**
- "Looks" = 겉으로 보기에는
- 주관적 인상/관찰 표현
- 칭찬/긍정적 평가 함축

**Similar Observation Phrases:**

| Expression | Nuance | Example |
|------------|--------|---------|
| "It looks pretty good." | 긍정적 인상 | "Your new haircut looks pretty good." |
| "It looks like fun." | 재미있어 보임 | "That activity looks like fun." |
| "You look happy." | 행복해 보임 | "You look happy when you run." |
| "It seems rewarding." | 보람있어 보임 | "Running seems rewarding for you." |
| "I can tell you enjoy it." | 즐김을 앎 | 직접 관찰한 결과 |

**Pronunciation Tip:**
- "Looks" = /lʊks/ (ʊ는 책 book의 o 발음)

**Practice Dialogue:**
```
A: I've been running every morning now.
B: Yeah, I've noticed. It looks pretty good.
A: You really think so?
B: Definitely. You're actually smiling more.
A: Maybe I should keep going then.
B: You should. It's clearly working for you.
```

## 1.2 Responding to Suggestions

### "Seriously?"

**Original Korean:** "진짜?"

**Meaning Spectrum:** 놀라움, 의심, 확인

**Uses of "Seriously?"**

| Context | Meaning | Tone |
|---------|---------|------|
| 긍정적 놀라움 | 진짜? (좋게) | 기대감 ↑ |
| 의심 | 정말? (믿기지 않음) | 회의적 |
| 확인 | 진짜야? | 중립 |
| 비판 | 진지해? | 비난적 |

**Similar Surprise/Confirmation Expressions:**

| Expression | Intensity | Usage |
|------------|-----------|-------|
| "Really?" | ★★☆ | 일반적 놀라움 |
| "Seriously?" | ★★★ | 강한 놀라움/의심 |
| "You're joking!" | ★★★★ | 매우 믿기 어려움 |
| "No way!" | ★★★★★ | 최고 수준 놀라움 |
| "Are you kidding?" | ★★★★ | 진짜냐고 재차 확인 |

**Intonation Matters:**
```
"Seriously?" (올라가는 음조) = 정말이야? (놀라움)
"Seriously." (내려가는 음조) = 진짜 그래. (확신)
```

**Practice Dialogue:**
```
A: I'm thinking about taking up running.
B: Seriously?
A: Yeah, I'm tired of just sitting around.
B: That's awesome! I'll help you.
A: Really? You mean it?
B: Of course. When do we start?
```

### "So did you."

**Original Korean:** "너도 했잖아."

**Grammar Point:** Elliptical response (생략 표현)

**Full Form:** "So did you [hate working out]."

**Why Ellipsis Works:**
- 이미 언급한 내용 생략
- 자연스럽고 캐주얼한 대화 흐름
- 상대의 주장을 인정하면서도 되받아침

**"So did I" vs "Me too" vs "Same"**

| Expression | Formality | When to Use |
|------------|-----------|------------|
| "So did I." | Standard | 상대 말에 대응 |
| "Me too." | Casual | 친구/캐주얼한 상황 |
| "Same." | Very casual | 매우 친한 친구들 |
| "I did too." | Standard | 더 명확한 표현 |
| "Yeah, I hated it too." | Descriptive | 감정까지 포함 |

**Structure & Examples:**

```
Statement: "I hate running."
Response: "So did I." = "I hated it too."

Statement: "I love ice cream."
Response: "So do I." = "I love ice cream too."

Statement: "I've been to Jeju."
Response: "So have I." = "I've been to Jeju too."
```

**Practice Dialogue:**
```
A: I hate exercise, honestly.
B: So did I. But things change.
A: Yeah, I guess people can surprise you.
B: Exactly. You proved that.
```

## 1.3 Expressing Uncertainty & Encouragement

### "But I'm scared to do it alone."

**Original Korean:** "근데 혼자 하기는 무서워."

**"Scared to do" Pattern:** ~하기가 무섭다, 두렵다

**Structure:** "I'm scared to" + verb = ~하기가 무섭다

**Courage Vocabulary:**

| Expression | Level of Fear | Meaning |
|------------|---------------|---------|
| "I'm nervous." | ★☆☆ | 약간 긴장 |
| "I'm hesitant." | ★★☆ | 망설여짐 |
| "I'm scared." | ★★★ | 두려움 |
| "I'm terrified." | ★★★★ | 매우 겁남 |
| "I'm petrified." | ★★★★★ | 극도의 공포 |

**Similar Expressions:**

| Expression | Nuance |
|------------|--------|
| "I'm scared to do it alone." | 혼자라서 무섭다 |
| "I'm not confident enough." | 자신감 부족 |
| "I don't think I can do it alone." | 능력 의심 |
| "I need someone's support." | 지원이 필요 |
| "I'd feel better with a friend." | 함께하고 싶음 |

**Practice Dialogue:**
```
A: Why don't you try this class?
B: I'm scared to do it alone.
A: How about I come with you?
B: Really? You would?
A: Of course. We can learn together.
B: That would be amazing. Thanks!
```

---

### "I think I could do it if we did it together."

**Original Korean:** "우리가 함께 하면 할 수 있을 것 같아."

**"I think I could do it if..." Pattern:** 조건부 긍정

**Structure:** "I think I could + verb + if + condition"

**Meaning:** 혼자로는 어렵지만, 조건(함께함)이 있으면 가능하다고 생각

**Similar Conditional Encouragement:**

| Expression | Meaning |
|------------|---------|
| "I could do it if we did it together." | 함께면 할 수 있어 |
| "I'd be more confident with your help." | 도움이 있으면 자신감이 생겨 |
| "Maybe I could try if you join me." | 너도 함께하면 시도해볼 수 있어 |
| "It would be easier with you." | 너랑이면 쉬울 것 같아 |

**Psychological Principle:**
한국인 학습자에게 중요: 혼자서 모든 것을 해내려는 개인주의적 문화 vs. 함께 하는 동료(buddy) 문화 이해

**Practice Dialogue:**
```
A: I want to start exercising but I'm nervous.
B: What if we did it together?
A: You'd really do that?
B: Yeah. I think it'd be more fun that way.
A: I think I could do it if we did it together.
B: Then let's start this weekend.
```

## 1.4 Making Plans

### "Want to run together?"

**Original Korean:** "같이 달릴래?"

**Grammar Point:** Casual yes/no question

**Structure:** "Want to" + verb = ~할래? (캐주얼한 제안)

**Compare Formality Levels:**

| Expression | Formality | Use Case |
|------------|-----------|----------|
| "Want to run together?" | Very casual | 친구/동료 |
| "Would you like to run together?" | Formal | 존댓말/격식 |
| "Can we run together?" | Neutral | 중간 |
| "Do you want to run with me?" | Standard | 일반 |
| "Running together—interested?" | Casual/Fragment | 매우 친한 친구 |

**"Want to" vs "Would you like to":**

```
"Want to grab lunch?" (친구에게 - 캐주얼)
"Would you like to join us for lunch?" (처음 만난 사람 - 존댓말)
"Wanna run?" (매우 캐주얼, 축약)
```

**Responding to "Want to...?"**

| Positive | Negative | Uncertain |
|----------|----------|-----------|
| "Yeah, sure!" | "Nah, I'm good." | "Maybe, I'll let you know." |
| "Definitely!" | "Can't today." | "I'll think about it." |
| "Sounds great!" | "Not really." | "I'm not sure." |

### "Meet up at the park near my house."

**Original Korean:** "우리 집 근처 공원에서 만나자."

**"Meet up" Phrasal Verb:** 만나다 (특정 장소에서 약속하여)

**Meeting-Related Phrasal Verbs:**

| Phrasal Verb | Meaning | Example |
|--------------|---------|---------|
| meet up | 만나다 | "Let's meet up at the park." |
| meet at | (장소에서) 만나다 | "Meet at the entrance." |
| get together | 모이다 | "We should get together soon." |
| hang out | 함께 시간 보내다 | "Want to hang out this weekend?" |
| catch up | 따라가며 만나다/소식 나누다 | "Let's catch up next week." |

**Making Specific Plans:**

| Element | Expression | Example |
|---------|------------|---------|
| 장소 | "Meet at the park." | "공원에서 만나" |
| 시간 | "Saturday morning." | "토요일 아침" |
| 거리/거리 | "Try about 2km first." | "처음에 2km 정도 해봐" |
| 준비물 | "Bring water." | "물 가져와" |

**Practice Dialogue:**
```
A: So when can we run?
B: How about Saturday morning?
A: Perfect. Where should we meet?
B: The park near my house?
A: Sounds good. What time?
B: How about 6 AM?
A: That works for me.
```

### "Try about 2km first."

**Original Korean:** "처음에 2km 정도 해봐."

**"Try about [distance]" Pattern:** 처음부터 이 정도를 시도해보다

**Structure:** "Try + quantity + first" = 먼저 ~만큼 해보다

**Running Distance Context:**

| Distance | Level | Description |
|----------|-------|-------------|
| 500m | Beginner | 겨우 반 킬로미터 |
| 1-2km | Beginner-Light | 가볍게 시작하는 거리 |
| 5km | Intermediate | 일반적인 조깅 |
| 10km | Advanced | 습관적 장거리 |
| 21km (Half-marathon) | Very Advanced | 반마라톤 |
| 42km (Marathon) | Elite | 마라톤 |

**Beginner-Friendly Expressions:**

| Expression | Meaning |
|------------|---------|
| "Try about 2km first." | 처음에 2km 정도 해봐 |
| "Start small." | 작게 시작해 |
| "Take it easy." | 천천히 해 |
| "Don't push yourself too hard." | 너무 무리하지 마 |
| "You can work up to longer distances." | 점점 늘려갈 수 있어 |

**Practice Dialogue:**
```
A: How far should we run?
B: Since it's your first time, try about 2km.
A: That sounds reasonable.
B: Yeah, we can do longer later once you get stronger.
A: I like that approach. Not too much pressure.
```

# PART 2: EMOTIONAL EXPRESSIONS & ENCOURAGEMENT (감정 표현과 격려)

## 2.1 Surprise & Delight

### "Minseo's face lit up."

**Original Korean:** "민서 얼굴이 환해졌다."

**Verb:** "Light up" = 밝아지다, 환해지다, 행복이 드러나다

**"Light up" in Different Contexts:**

| Subject | Example | Meaning |
|---------|---------|---------|
| Face | "His face lit up." | 얼굴이 밝아졌다 (기쁨) |
| Eyes | "Her eyes lit up." | 눈이 빛났다 (기대감) |
| Room | "A lamp lit up the room." | 방을 밝혔다 (물리적) |
| Person | "You lit up my life." | 삶을 밝혔다 (은유) |

**Physical Expressions of Joy:**

| Expression | Meaning | Body Language |
|------------|---------|----------------|
| "lit up" | 환해짐 | 얼굴이 밝아짐, 미소 |
| "beamed" | 활짝 웃음 | 귀까지 올라오는 미소 |
| "grinned" | 웃음 | 넓은 미소 |
| "smiled brightly" | 밝게 웃음 | 눈까지 웃음 |
| "radiated joy" | 기쁨이 넘침 | 온 몸에서 행복이 드러남 |

**Practice Context:**
```
When someone receives unexpected good news:
- "When I told him, his face just lit up."
- "She beamed at the surprise."
- "He couldn't help but grin."
```

### "I was looking forward to it. Weirdly."

**Original Korean:** "기대가 돼서. 이상하게."

**"Looking forward to" Pattern:** ~을 기대하다

**Grammar:**
- "look forward to" + gerund (V-ing)
- 미래의 좋은 일을 기다리는 감정

**"Weirdly" = 이상하게, 신기하게**

**Similar Expressions:**

| Expression | Meaning | Context |
|------------|---------|---------|
| "I was looking forward to it." | 기대됐어 | 긍정적 기대 |
| "I could hardly wait." | 기다릴 수 없었어 | 더 강한 기대 |
| "I was excited." | 신났어 | 흥분된 감정 |
| "I was anticipating it." | 예상하고 있었어 | 더 공식적 |

**"Weirdly" in Emotional Context:**

```
"I was looking forward to it. Weirdly."
= 기대가 됐어. 이상하게.
= 혼자라고 생각했는데 함께한다니, 이상하게도 기대가 된다.
```

**Why "Weirdly" Here:**
- 자신도 예상 못한 감정
- 혼자 달리는 걸 좋아했는데 함께하는 게 기대됨
- 심리적 변화/성장을 표현

**Practice Dialogue:**
```
A: Are you excited about running with Minseo?
B: Yeah, kind of.
A: Just kind of?
B: Well, I was looking forward to it. Weirdly. I usually prefer doing things alone.
A: That's good, though. It means you're opening up.
B: I guess you're right.
```

## 2.2 Encouragement & Support

### "It's hard for everyone at first."

**Original Korean:** "누구나 처음엔 힘들어."

**Grammar:** "It's + adjective + for + noun + adverb/clause"

**Meaning:** 모두가 같은 어려움을 경험한다는 위로

**Empathetic Encouragement Patterns:**

| Pattern | Example | Effect |
|---------|---------|--------|
| "It's hard for everyone." | 모두 어렵다 (정상화) | 혼자가 아니라는 위로 |
| "It was for me too." | 나도 그랬다 (공감) | 상대의 경험 인정 |
| "You'll get better." | 나아질 거다 (희망) | 미래 긍정 |
| "I believe in you." | 너를 믿어 (신뢰) | 직접적 지지 |
| "You've got this." | 할 수 있어 (자신감) | 능력 인정 |

**Comparing Difficulty:**

| Expression | Meaning | Tone |
|------------|---------|------|
| "It's hard for everyone." | 모두가 힘들어 | 위로하는 |
| "Everyone struggles at first." | 모두 처음엔 어려워 | 정상화 |
| "Most people find it difficult." | 대부분 어렵다고 해 | 객관적 |
| "Don't worry, you'll improve." | 나아질 거야 | 희망적 |

**Practice Dialogue:**
```
A: I'm worried I won't be able to do this.
B: Hey, it's hard for everyone at first.
A: Really?
B: Yeah, it was for me too. I could barely do 500m.
A: Now you can do 7km!
B: Exactly. You'll get there too.
```

### "I could barely do 500m in the beginning."

**Original Korean:** "처음엔 500m도 힘들었어."

**"Barely" = 겨우, ~할 정도, 거의 ~할 수 없을 정도**

**"Could barely" Structure:** 겨우 ~할 수 있었다 (매우 어려웠다)

**Examples:**

| Expression | Meaning |
|------------|---------|
| "I could barely run 500m." | 500m도 겨우 뛸 수 있었어 |
| "I could barely walk." | 겨우 걸을 수 있었어 |
| "I could barely stay awake." | 깨어있기 힘들었어 |
| "I could barely understand." | 겨우 이해할 수 있었어 |

**Progress Narrative:**

```
Beginning: "I could barely do 500m."
Progress: "Now I can do 5km."
Mastery: "Now I run 7km every morning."
```

**This is Powerful Encouragement Because:**
- 구체적인 숫자로 과정을 보여줌
- 자신도 초심자였음을 인정
- 현재 능력을 칭찬
- 상대에게 희망을 줌

**Practice Dialogue:**
```
A: Can you really do 7km?
B: Yeah, but I started small.
A: How small?
B: I could barely do 500m in the beginning.
A: And now you're doing 7km!
B: Exactly. You'll surprise yourself.
```

---

### "If that's true, then I can definitely do it!"

**Original Korean:** "그게 진짜면 난 충분히 할 수 있을 것 같은데!"

**Grammar:** "If + condition, then + conclusion" = 만약 ~이면, ~할 수 있겠다

**Logical Reasoning in Conversation:**

| Pattern | Meaning |
|---------|---------|
| "If that's true, then I can do it." | 만약 그게 진짜면, 난 할 수 있어 |
| "Then I can definitely do it!" | 그럼 난 확실히 할 수 있어! |
| "So if you did it, I can too." | 그러면 너도 할 수 있으면 나도 해 |

**"Definitely" Intensity:**

| Expression | Certainty Level |
|------------|-----------------|
| "I can do it." | ★★☆ 보통 |
| "I can probably do it." | ★★☆ 가능성 |
| "I can definitely do it!" | ★★★★★ 확실함 |
| "I can absolutely do it!" | ★★★★★ 절대적 확실 |
| "There's no doubt I can do it." | ★★★★★ 의심의 여지 없음 |

**Psychology of Self-Talk:**
상대의 구체적인 경험(500m→7km)을 듣고 자신도 할 수 있다고 자신감을 얻음

## 2.3 Gratitude & Appreciation

### "Thanks, Minseo."

**Original Korean:** "고마워, 민서."

**Simple Gratitude:**

| Expression | Formality | Use |
|------------|-----------|-----|
| "Thanks." | Very casual | 친구/캐주얼 |
| "Thank you." | Standard | 일반 |
| "Thanks a lot." | Warm | 더 진심 있게 |
| "I really appreciate it." | Formal | 격식 있게 |
| "I can't thank you enough." | Sincere | 매우 감사함 |

**When "Thanks" is Most Powerful:**
- 일기장에 혼자 쓸 때 (개인적이고 진심)
- 짧지만 깊은 감정 표현
- 마지막에 나와서 임팩트 있음

**Context in the Diary:**
```
"Minseo asked me to run with him.
...
Thanks, Minseo."
```

- 처음 친구이면서도 이렇게까지 신뢰
- 고마움이 깊음 (단순 도움이 아니라 인생 변화)

# PART 3: DESCRIBING SITUATIONS & RELATIONSHIPS (상황 및 관계 묘사)

## 3.1 Describing Changes & Growth

### "Running alone was nice. But running together would be different."

**Original Korean:** "혼자 달리기도 좋았는데. 함께 달리면 다르겠지."

**Comparison Structure:** "X was nice. But Y would be different."

**Nuance:**
- 과거(혼자 달리기)를 긍정하면서
- 미래(함께 달리기)의 새로운 가능성을 제시
- 변화에 열린 마음

**Similar Contrastive Patterns:**

| Pattern | Example |
|---------|---------|
| "X was good. But Y would be better." | 이건 좋지만 저게 더 좋을 것 같아 |
| "I liked doing it alone. Now I want to try with others." | 혼자 하는 게 좋았지만, 이제 남과 함께 해보고 싶어 |
| "Solo is fine. Group activity has its appeal." | 혼자도 괜찮지만, 함께의 매력이 있어 |

**"Would be different" - Why This Phrasing?**

```
"Running together WILL be different." = 확실하다 (단언)
"Running together WOULD be different." = 가능성 (추측, 미지수)
"Running together IS different." = 이미 경험했다 (현실)
```

**The narrator uses "would" because:**
- 아직 경험하지 못함 (토요일 아침이 아직 안 됨)
- 긍정적 예상이지만 확실하지 않음
- 신기한/다른 경험이 될 거라는 느낌

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "What if I started...?"

**Structure:** "What if" + I + past tense = 만약 ~하면?

**Uses:**
- 새로운 가능성 제시
- 상대의 경험을 따라하고 싶을 때
- 호기심이나 제안

| Example | Context |
|---------|---------|
| "What if I started running?" | 새 활동 시작 |
| "What if I quit my job?" | 인생 결정 |
| "What if we traveled together?" | 함께 하는 것 제안 |

---

## Pattern 2: "But [person] had a point."

**Structure:** "But [person] had a point." = 그 사람 말이 맞았어

**Meaning:** 상대의 주장을 인정함

| Example | Situation |
|---------|-----------|
| "But Minseo had a point." | 민서 주장 인정 |
| "He had a point." | 그 말도 맞다 |
| "That's a fair point." | 합리적인 지적 |
| "I see your point." | 너의 뜻 이해 |

**Full Context from Text:**
```
Doyun: "But you hate working out."
Minseo: "So did you."
Doyun: "..."
Doyun (thinking): "He had a point."
```

---

## Pattern 3: "Want to [verb] together?"

**Structure:** "Want to" + verb + "together?" = ~할래?

**Offers/Suggestions:**

| Pattern | Example |
|---------|---------|
| "Want to run together?" | 함께 달릴래? |
| "Want to eat together?" | 함께 먹을래? |
| "Want to come with me?" | 나랑 가볼래? |

**Responding:**
```
Positive: "Yeah!" / "Sure!" / "Definitely!"
Uncertain: "Maybe" / "I'll think about it"
Negative: "Nah" / "Can't"
```

---

## Pattern 4: "Sure. But... [condition]"

**Structure:** Agreement + BUT + limitation/condition

| Example | Meaning |
|---------|---------|
| "Sure. But I head out at 5:30." | 좋지. 근데 난 5시반에 출발해 |
| "Sure. But I can't on weekends." | 좋지. 근데 주말에 못 해 |

**Purpose:** 동의하면서도 현실적 제약 제시

---

## Pattern 5: "The Answer came out before I realized it."

**Original Korean:** "생각하기도 전에 대답이 나왔다."

**Spontaneous Response Pattern:**

| Expression | Meaning |
|------------|---------|
| "I said it before I could think." | 생각하기 전에 말했어 |
| "The answer came out automatically." | 자동으로 대답이 나왔어 |

| "It just came out." | 그냥 나왔어 |

**Psychology:**
진짜 원하는 것은 생각보다 행동이 빠르다는 의미

---

## Pattern 6: "Can you...? / You okay with...?"

**Making Accommodations/Checking Comfort:**

| Pattern | Example |
|---------|---------|

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Encouraging a Friend to Try Something New

```
A: I want to start running, but I don't think I can do it alone.
B: Hey, what if I started with you?
A: Seriously? You'd do that?
B: Yeah. Watching you lately, it looks pretty good. I want to try.
A: But you hate exercise.
B: So did you. But people change, right?
A: I guess you're right.
B: So what do you say? Want to run together?
A: Uh... sure! That'd be amazing.
B: Perfect. When are you free?
A: How about Saturday morning?
B: That works for me. Let's do it!
```

### Scenario 2: Making Plans for a New Activity

```
A: I'm really looking forward to running with you.
B: Me too! It's going to be fun.
A: Since it's your first time, we can take it easy.
B: How far should we go?
A: Try about 2km first. That's a good starting distance.
B: That sounds reasonable. Where should we meet?
A: The park near my house?
B: Perfect. What time?
A: Saturday morning, say 6 AM?
B: I can do that. What should I wear?
A: Just workout clothes. Bring water too.
B: Got it! Can't wait for Saturday.
```

### Scenario 3: Reflecting on Friendship

```
A: I'm grateful you asked me to run with you.
B: Why's that?
A: Because now I feel like I have a teammate, not just a friend.
B: I like being your teammate.
A: You know, back in March when I first missed the bus, you were the first to talk to me.
B: I remember. You seemed like you needed a friend.
A: And you've been listening to all my running stories and cheering me on.
B: That's what teammates do!
A: I know. Thanks for being there.
```
