# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Race Preparation & Training

### "Last run before the race."

**Original Korean:** "경주 전 마지막 운동"

**Grammar Point:** Noun phrase for emphasis (완전한 문장이 아닌 강조 표현)

**Structure:** [Adjective/Noun] + noun + before/after + event

**More Examples:**

| Event | Phrase | Full Sentence |
|-------|--------|---------------|
| the exam | "last study session before the exam" | "I did my last study session before the exam." |
| the wedding | "last fitting before the wedding" | "I had my last fitting before the wedding." |
| the interview | "last practice before the interview" | "I did my last practice before the interview." |
| the performance | "last rehearsal before the performance" | "We did our last rehearsal before the performance." |
| the trip | "last packing session before the trip" | "I finished my last packing session before the trip." |

**Why this matters in context:**
짧고 강렬한 표현으로, 중요한 순간이 다가왔음을 강조.

---

**Original Korean:** "겨우 5km, 편한 속도로"

**Pattern:** "[Amount/Distance], [intensity/manner]."

|------------|-----------|--------|

| comfortable pace | ★☆☆ | 편한 속도 (대화 가능) |
| moderate pace | ★★☆ | 중간 정도 속도 |
| steady pace | ★★☆ | 일정한 속도 |
| brisk pace | ★★★ | 빠른 걸음 |
| race pace | ★★★★ | 경주 속도 |
| hard pace | ★★★★ | 힘든 속도 |
| sprint pace | ★★★★★ | 전력 질주 |

**Talking About Running Distances:**

| Expression | Korean |
|------------|--------|
| "Just 5km" | "겨우 5km만" |
| "A quick 5km" | "빠르게 5km" |
| "A recovery 5km" | "회복 주 5km" |
| "Easy 5km" | "편한 5km" |

**Dialogue Practice:**
```

```

## 1.3 Encouragement & Support

### "Go do well. I'm rooting for you."

**Original Korean:** "잘 해라. 응원할게"

**"Rooting for" = ~을 응원하다 (열정적으로)**

**"Go do well" = 잘해라 (격려)**

**Support Expressions:**

| Expression | Korean | Formality |
|------------|--------|-----------|
| "I'm rooting for you." | "응원할게" | Warm, casual |
| "I'm cheering for you." | "응원할게" | Warm, casual |
| "Go for it!" | "해봐!" | Very casual, excited |
| "You can do it!" | "해낼 수 있어!" | Encouraging |
| "Good luck!" | "화이팅!" | Standard |
| "Break a leg!" | "화이팅!" (공연용) | Theatrical |
| "I believe in you." | "너를 믿어" | Sincere, close |

**Rooting for vs. Cheering for:**
- "Rooting for" = 더 깊은 감정, 개인적 관계
- "Cheering for" = 활발한 응원
- 이 문맥에서 Grandpa의 따뜻한 응원 표현

**When to Use:**

```
"I'm rooting for you on the big day."
"Everyone's rooting for you."
"I was rooting for you the whole time!"
"Go do well—I'm rooting for you."
```

# PART 2: PHYSICAL SENSATIONS (신체 감각 표현)

## 2.1 Pre-Race Physical State

### "Ready."

**Original Korean:** "준비됐다" / "괜찮다"

**Single Word as Complete Thought:**
짧은 한 단어가 자신감을 전달하는 효과

**"Ready" Spectrum:**

| Expression | Confidence Level | Korean |
|------------|-----------------|--------|
| "I guess." | ★☆☆ | 그런 것 같은데 |
| "I think so." | ★★☆ | 그런 것 같아 |
| "Yeah." | ★★★ | 응 |
| "Ready." | ★★★★ | 준비됐다 |
| "I'm ready." | ★★★★ | 준비 완료 |
| "Absolutely ready." | ★★★★★ | 완전 준비됐다 |

**Shorter Versions (Fragment speech):**
```
"Ready?"
"Ready."
"No problems."
"Got it."
"Ready to go."
```

---

### "No problems."

**Original Korean:** "이상 없음" / "문제 없음"

**Status Check Short Responses:**

| Question | Response | Meaning |
|----------|----------|---------|
| "How do you feel?" | "No problems." | 문제 없어 |
| "Any concerns?" | "No problems." | 걱정 없어 |
| "Everything okay?" | "No problems." | 다 괜찮아 |
| "Ready for Sunday?" | "No problems." | 준비 완료 |
| "Any pain?" | "No problems." | 통증 없어 |

**This fragment style:**
- 자신감 표현
- 효율적 의사소통
- 청자와의 신뢰 관계 암시

## 2.2 Race Day Anxiety & Confidence

### "I'm nervous."

**Original Korean:** "긴장했다"

**Nervousness Spectrum:**

| Expression | Intensity | Korean |
|------------|-----------|--------|
| "I'm a little excited." | ★☆☆ | 좀 설레 |
| "I'm eager." | ★★☆ | 간절해 |
| "I'm nervous." | ★★★ | 긴장했다 |
| "I'm very nervous." | ★★★★ | 매우 긴장했다 |
| "I'm terrified." | ★★★★★ | 무섭다 |

**Nervousness Expressions for Competitions:**

| Expression | Context | Tone |
|------------|---------|------|
| "nervous" | 일반적 긴장 | Neutral |
| "anxious" | 걱정 섞인 긴장 | Worried |
| "jittery" | 초초함 | Agitated |
| "on edge" | 예민함 | Tense |
| "butterflies in my stomach" | 설렘/긴장 섞임 | Mixed emotions |

**Practice:**
```
A: How are you feeling before the race?
B: I'm nervous. But not scared.
A: That's normal. You've trained hard.
B: I know. I'm ready.
```

---

### "But I'm not scared."

**Original Korean:** "하지만 두렵지는 않아"

**Nervous vs. Scared - Important Distinction:**

| Emotion | Korean | Nuance |
|---------|--------|--------|
| nervous | 긴장되다 | 불확실성 앞에서 자극받음 |
| scared | 두렵다 | 나쁜 결과 예상 |
| worried | 걱정되다 | 무언가 잘못될까봐 |
| excited | 설레다 | 긍정적 기대 |

**The Contrast Pattern:**

```
"I'm [emotion], but not [stronger emotion]."

"I'm nervous, but not scared."
"I'm anxious, but not hopeless."
"I'm tired, but not broken."
"I'm frustrated, but not defeated."
```

**Why this matters:**
자신이 감정을 통제할 수 있다는 신뢰 표현

---

**Confidence Building Statement:**

**"I've trained for this" Pattern:**

| Situation | Statement |

**Alternative Expressions of Readiness:**

| "I've been building to this." | "이 순간을 위해 준비했다" |

---

### "Eight months of training."

**Original Korean:** "8개월간의 훈련"

**Time + Duration Expression:**

**"[Number] months/weeks/days of [activity]" Pattern:**

| Duration | Activity | Full Sentence |
|----------|----------|---------------|
| 8 months | training | "Eight months of training." |
| 3 years | hard work | "Three years of hard work." |
| 6 weeks | preparation | "Six weeks of preparation." |
| 100 days | practice | "One hundred days of practice." |

**Why Mentioning Duration Matters:**

In competitive contexts, mentioning how long you've prepared shows:
- 노력의 양
- 결정의 진지성
- 성공 가능성

**Dialogue:**
```
A: You're ready for the race?
B: Eight months of training. Yeah, I'm ready.
A: That's a lot of work.
B: It was. But I can do it.
```

# PART 3: RACE PREPARATION & ADVICE (경기 준비 & 조언)

## 3.1 Coach's Pre-Race Instructions

### "Remember a few things for the race."

**Original Korean:** "경주 때 기억할 몇 가지가 있다"

**"Remember" + Instructions Pattern:**

**Structure:** "Remember [noun/clause] for [situation]"

**Examples:**

| Situation | Reminder | Full Sentence |
|-----------|----------|---------------|
| 운동 | breathing techniques | "Remember your breathing techniques for the race." |
| 시험 | time management | "Remember time management for the exam." |
| 발표 | eye contact | "Remember to make eye contact for your presentation." |
| 면접 | confidence | "Remember to show confidence for the interview." |
| 여행 | passport | "Remember your passport for the trip." |

**Coach-Athlete Communication:**

```
"Remember a few things..."
"Keep a few things in mind..."
"Don't forget..."
"Make sure you..."
"One thing to remember..."
"The key things are..."
```

### "First, the first 5km—go slow."

**Original Korean:** "첫째, 처음 5km는 천천히 가라"

| Number | Korean | English Pattern |
|--------|--------|-----------------|
| First | 첫째 | "First, [instruction]" |
| Second | 둘째 | "Second, [instruction]" |
| Third | 셋째 | "Third, [instruction]" |

**Pacing Strategy Language:**

| Strategy | Korean | Context |
|----------|--------|---------|

| Distance | Korean | What Happens |
|----------|--------|---------------|

## 3.2 The Most Important Instruction

### "Enjoy it. It's your first half marathon."

**Original Korean:** "즐겨라. 처음 반마라톤이니까"

**"Enjoy it" for Events:**

**When to emphasize enjoyment:**
- 처음 경험
- 경쟁 아닌 참가
- 장거리 완주 후
- 성취감 중심

**Tone Shift from Coach:**
앞의 모든 조언 (기술적)에서 지혜로운 조언(정신적)으로 전환

**Life Advice in Athletic Context:**

| Statement | Meaning | When |
|-----------|---------|------|

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "I feel great." + "No problems." (Simple Positive Assessment)

**Structure:** Short positive statements in sequence

**Uses:** Pre-event readiness check

```
"How do you feel?"

"Ready for Sunday?"
"Ready. Everything feels good."

"Any concerns?"
"None. I'm prepared."
```

---

## Pattern 2: "First, [instruction]... Second, [instruction]... Third, [instruction]"

**Structure:** Numbered advice format

**Full Example from Text:**
```
"First, the first 5km—go slow. Don't get excited and sprint.
Second, from 10km on, maintain your pace. Don't push too hard.
Third, after 15km it's going to hurt. Don't give up then."
```

**Why Effective:**
- 체계적 정보 전달
- 기억하기 쉬움
- 신뢰감 형성

**Practice:**
```
Coach: Three things to remember for the exam.
Student: Okay, tell me.
Coach: First, read carefully. Second, manage your time. Third, double-check your answers.
Student: Got it.
```

---

## Pattern 3: "The most important thing [is]..."

**Structure:** Emphasis before main point

**From Text:**
```
Coach: "And the most important thing."
You: "What?"
Coach: "Enjoy it."
```

**Why This Works:**
- 호기심 유발 (What? - expectation)
- 감정적 클라이맥스
- 기술적 조언에서 정신적 조언으로 전환

**Similar Emphasis Patterns:**

```
"Here's the thing..."
"The key is..."
"Don't forget..."
"Most importantly..."
"At the end of the day..."
```

---

## Pattern 4: "I'm [emotion], but I'm not [stronger emotion]"

**Structure:** Acknowledging one feeling while denying worse

**From Text:** "I'm nervous. But I'm not scared."

**More Examples:**

```
"I'm tired, but I'm not broken."
"I'm worried, but I'm not hopeless."
"I'm excited, but I'm not reckless."
"I'm afraid, but I'm not defeated."
```

**Why This Pattern Works:**
- 감정의 단계를 인정
- 통제 능력 표현
- 긍정적 마음가짐 유지

---

## Pattern 5: "I've [past action] for this"

**Structure:** Past preparation + current readiness

**Variations:**
