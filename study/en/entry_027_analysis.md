# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Weather-Related Comments

### "Rain. The rainy season has begun."

**Original Korean:** "비. 장마가 시작됐다."

**Analysis:**
- Fragment opening for dramatic effect
- Simple, direct statement
- 날씨 변화를 객관적으로 기술하면서도 개인의 감정 내포

**Variations by Formality:**
| Casual | Standard | Formal |
|--------|----------|--------|
| "Rain. Great." | "It's raining." | "Precipitation has commenced." |
| "Raining again." | "The rain has started." | "Inclement weather conditions." |
| "Great, more rain." | "Rain's here." | "Meteorological patterns indicate..." |

**Practice Dialogue:**
```
A: How's the weather?
B: Rain. The rainy season has begun.
A: For how long?
B: About two weeks of nonstop rain.
A: You're not going to run?
B: Can't. Too dangerous.
```

## 1.2 Expressing Desires & Wishes

### "I wanted to run."

**Original Korean:** "뛰고 싶었다"

**Grammar Point:** Past tense of desire

**Desire Spectrum:**
| Intensity | Expression | Meaning |
|-----------|------------|---------|
| Casual | "I kinda wanted to run." | 약간 하고 싶었어 |
| Normal | "I wanted to run." | 하고 싶었어 |
| Strong | "I really wanted to run." | 정말 하고 싶었어 |
| Desperate | "I was dying to run." | 간절히 하고 싶었어 |
| Physical | "My body was restless." | 몸이 안절부절못했어 |

**Alternative Phrasings:**
- "I felt like running." (뛰고 싶은 기분이 들었어)
- "I was itching to run." (뛰고 싶어 안달이 났어)
- "Nothing else crossed my mind." (뛰는 것만 생각났어)
- "The urge wouldn't leave me." (하고 싶은 마음이 떨어지지 않았어)

**Practice Dialogue:**
```

```

## 1.3 Problem-Solving Language

### "But they said not in heavy rain. Too slippery, too dangerous."

**Original Korean:** "근데 무거운 비에는 위험하대. 너무 미끄럽고."

**Expressing Limitations:**
| Expression | Tone |
|------------|------|

| "But experts warn against heavy rain." | 더 격식적 |

| "Heavy rain is too dangerous." | 직접 평가 |
| "The risk is too high when it's pouring." | 위험도 강조 |

**Danger Vocabulary:**
| Term | When to use |
|------|------------|
| dangerous | 일반적 위험 |
| risky | 위험성 있음 |
| slippery | 미끄러움 |
| unsafe | 안전하지 않음 |
| hazardous | 공식적 위험 표현 |

### "Light rain is okay, though."

**Original Korean:** "가벼운 비는 괜찮대."

**"Though" for Softening**

**Grammar:** Concessive conjunction - 예외/반대 조건 제시

**Examples with "Though":**
| Context | Example |
|---------|---------|
| 예외 | "I hate running. Light rain is okay, though." |
| 대비 | "He's strict. He's fair, though." |
| 놀라움 | "She looks tired. She says she's fine, though." |

**Similar Softening Words:**
- "Light rain is okay, though." (가벼운 비는 괜찮지만)
- "Light rain might work." (가벼운 비면 괜찮을 수도)
- "Light rain is doable." (가벼운 비는 가능해)
- "Light rain isn't impossible." (가벼운 비가 불가능하진 않아)

**Light vs. Heavy Rain Expressions:**
```
Light: "a light drizzle", "sprinkle", "mist"
Medium: "steady rain", "moderate rain"
Heavy: "heavy rain", "pouring", "downpour"
```

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "This used to be [normal]. Now [different]."

**Structure:** This + used to + verb + . Now + present tense

**From Text:** "This used to be normal. Now I can barely stand it."

**Used to Pattern - Contrast:**

| Past | Present | Full Sentence |
|------|---------|---------------|
| normal | unbearable | "This used to be normal. Now it's unbearable." |
| easy | difficult | "That used to be easy. Now it's so hard." |
| enjoyable | painful | "It used to be fun. Now it hurts." |
| simple | complicated | "It was simple. Now everything's complex." |

**Life Change Through Habits:**
```
"I used to hate the bus. Now I miss running."
"This commute used to feel long. Now I feel cooped up."
"I used to not care about weather. Now it ruins everything."
```

**Emphasizing Personal Growth:**
This pattern shows how much has changed since March. The bus that was bearable is now intolerable. Running has become essential.

---

**Structure:** [Activity] has become + [state]

**Habit Formation Language:**

## Pattern 3: "What if [situation]? [Response]."

**Structure:** What if + subject + verb? [Solution-oriented response]

**From Text:** "What if it rains again tomorrow?"

| What If | Response | Pattern |
|---------|----------|---------|
| "What if it rains?" | "I'll run in light rain." | 해결책 제시 |
| "What if it's too wet?" | "I'll wait for it to ease up." | 대안 제시 |
| "What if I get sick?" | "Then I'll rest." | 수용 |

**Worry Transformation Pattern:**
```
"What if it rains again tomorrow?" (걱정)
→ "I checked the forecast." (정보 수집)
→ "I looked it up." (조사)

**Decision-Making Process:**

## 10.1 Dialogue Practice

```
A: How do you plan to run during rainy season?

A: Have you told your mom?
B: Yeah. She said only in light rain.

```

```
