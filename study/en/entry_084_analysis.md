# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Weather and Time-Related Expressions

### "Clear skies, below freezing."

**Original Korean:** "맑음, 영하."

**Analysis:**
- Fragment sentences used to paint a quick weather picture
- 날씨를 간단하고 시각적으로 묘사하는 방법
- Journal/diary style: complete thoughts not necessary

**Variations by Context:**

| Simple | Detailed | Poetic |
|--------|----------|--------|
| "Clear." | "The sky was clear." | "Clear skies stretched overhead." |
| "Freezing." | "It was below freezing." | "Below freezing, the world turned crisp." |
| "Cold and clear." | "It was cold and clear out." | "Crisp, clear morning with frost on the grass." |

**Weather Fragment Examples:**
```
"Sunny, 75 degrees."
"Overcast, humid."
"Rainy, miserable."
"Bright blue sky, not a cloud."
```

**Practice Dialogue:**
```
A: What's the weather like?
B: Clear skies, below freezing.
A: Sounds perfect for a run.
B: That's what I thought.
```

---

### "Winter."

**Original Korean:** "겨울."

**Analysis:**
- Single word expresses entire season's significance
- 한 단어로 그 시절의 의미를 전달
- Implies dormancy, waiting, natural rest

**Seasonal Reflections (Single Words):**

| Season | Implication | Example |
|--------|-------------|---------|
| "Spring." | 새시작, 성장 | "Spring. Time to start fresh." |
| "Summer." | 활기, 열정 | "Summer. The best time of year." |
| "Fall." | 변화, 수확 | "Fall. Everything changes." |
| "Winter." | 휴식, 기다림 | "Winter. Everything sleeps." |

**Extended Context:**
```
"Winter. The field was empty. It'll turn green again when spring comes."
```

**Alternative Expressions:**
- "Wintertime." (더 formal)
- "The cold months." (좀 더 설명적)
- "Hibernation season." (유머)

## 1.2 Casual Greetings and Messages

### "What're you doing?"

**Original Korean:** "뭐 해?"

**Analysis:**
- Contraction: "What are" → "What're"
- Casual, friendly greeting/conversation opener
- 친한 사람들 사이의 자연스러운 인사

**Casual Greeting Patterns:**

| Expression | Formality | Context |
|------------|-----------|---------|
| "What're you doing?" | Very casual | 친한 친구 |
| "What are you up to?" | Casual | 친구들 사이 |
| "What're you doing right now?" | Casual | 지금 바로 |
| "What've you been doing?" | Casual | 최근에 뭐 했어? |
| "What are you doing?" | Neutral | 좀 더 formal |

**Texting Variations:**
- "Whatchu doing?" (very casual, texting)
- "Wyd?" (texting acronym, extremely casual)
- "Up to anything?" (casual)

**Response Examples:**
```
A: What're you doing?
B: Just got back from a run.
A: Nice. I was thinking about running tomorrow.
B: Let's go together.
```

### "Just got back from a run."

**Original Korean:** "방금 달리고 왔어."

**Grammar Point:** Present perfect for very recent past

**Structure:** "Just + past participle" or "Just + came back + from + activity"

**Similar Time-Recent Expressions:**

| Expression | Timeframe |
|------------|-----------|
| "Just got back." | 방금 (1-5분) |
| "I just finished." | 방금 끝냈어 |
| "I just got home." | 방금 집 왔어 |
| "I only just arrived." | 겨우 도착했어 |
| "I'm just back." | 방금 돌아왔어 |

**In Messaging Context:**
```
A: What're you doing?
B: Just got back from a run.
A: How was it?
B: Good. 10km. Felt nice and easy.
```

### "ㅋㅋㅋ Even the day after Christmas?"

**Original Korean:** "ㅋㅋㅋ 크리스마스 다음날에도?"

**Analysis:**
- "ㅋㅋㅋ" = Korean laughing emoji (Internet culture)
- In English, translate as "Haha" or just emotional reaction
- Question expresses surprise/admiration

**Expressing Surprise/Amazement:**

| Expression | Tone | Intensity |
|------------|------|-----------|
| "Seriously?" | Questioning | ★★☆ |
| "Even today?" | Surprised | ★★☆ |
| "Really? Today?" | Very surprised | ★★★ |
| "You're running today?" | Amazed | ★★★ |
| "Even on Christmas?" | Impressed | ★★★★ |

**Translation of Korean Internet Laughing:**

| Korean | English | Context |
|--------|---------|---------|
| ㅋㅋㅋ | Haha / Lol | 가벼운 웃음 |
| ㅋㅋㅋㅋ | HAHA / Lol | 더 많이 웃음 |
| ㅋㅋㅋㅋㅋ | HAHAHA / LOL | 많이 웃음 |

**Natural English Response:**
```
"ㅋㅋㅋ Even the day after Christmas?"
= "Haha, you're running on the day after Christmas? That's crazy!"
```

|------------|--------|

## 1.3 Planning and Agreement Expressions

### "Wanna run together? Tomorrow."

**Original Korean:** "함께 달릴래? 내일."

**Analysis:**
- Fragment "Tomorrow" adds emphasis
- "Wanna" = "Want to" (very casual)
- Clear, direct proposal

**Casual Proposal Patterns:**

| Formal | Casual | Very Casual |
|--------|--------|------------|
| "Would you like to run together?" | "Want to run together?" | "Wanna run together?" |
| "Could we run together tomorrow?" | "Can we run tomorrow?" | "Let's run tomorrow?" |
| "Would tomorrow work for you?" | "How about tomorrow?" | "Tomorrow work?" |

**"Wanna" Pronunciation:** /ˈwɑnə/
- Very casual, spoken English
- In writing, use "want to" for formal contexts

**Making Plans Dialogue:**
```
A: Wanna run together? Tomorrow.
B: Sure. Where?
A: The coastal road. 10km.
B: OK.
```

## 1.5 Motivation and Determination

### "Gotta keep pushing."

**Original Korean:** "계속 밀어붙여야 한다."

**Grammar Point:** "Gotta" = "Got to" = "Have to" (very casual)

**Structure:** Gotta + base verb (no "to")

**Obligation Expressions (Casual):**

| Expression | Formality | Meaning |
|------------|-----------|---------|
| "Gotta keep pushing." | Very casual | 계속 밀어붙여야 해 |
| "I have to keep pushing." | Standard | 계속 밀어붙여야 해 |
| "I've got to keep pushing." | Standard | 계속 밀어붙여야 해 |
| "I need to keep pushing." | Standard | 밀어붙여야 해 |
| "I should keep pushing." | Softer | 밀어붙였으면 좋겠어 |

**"Pushing" in Running Context:**

| Expression | Meaning |
|------------|---------|
| "Keep pushing." | 계속 노력해 |
| "Push harder." | 더 노력해 |
| "Give it one more push." | 한 번 더 밀어붙여 |
| "Don't push too hard." | 너무 무리하지 마 |

**Motivation Dialogue:**
```
A: 66 days until the marathon.
B: That's a lot of training ahead.
A: Yeah. Gotta keep pushing.
B: You can do it.
```

**"Speak up" Meanings:**

| Context | Meaning | Example |
|---------|---------|---------|
| 먼저 말하다 | "The kid spoke up." | "아이가 먼저 말을 걸었다" |
| 목소리 높이다 | "Speak up! I can't hear." | "더 크게 말해! 안 들려" |
| 의견 표현 | "Don't be afraid to speak up." | "의견 말하기를 두려워하지 마" |

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: Fragment + Time Expression

**Structure:** Activity + comma/period + time marker

**From Text Examples:**
```
"Ran 10km, nice and easy."
"58 minutes."
"Not bad."
```

**Usage:** Emphasize specific details in diary/casual writing

**Examples:**
```
"Did the coastal road, 10km, easy pace, 58 minutes."
"Perfect weather, cold, crisp, clear skies."
"Christmas morning, quiet, peaceful, family time."
```

---

## Pattern 2: Conditional Future Reflection

**Structure:** "Until [event/time]. Until [more specific]. [number] days."

**From Text:**
```
"Until next March.
Until the race.
66 days.
Gotta keep pushing."
```

**Countdown Pattern:**
- Remote goal (next March)
- Specific goal (the race)
- Exact number (66 days)
- Motivation statement

**Similar Patterns:**
```
"Until graduation.
Until I move out.
2 years, 3 months, 14 days.
Can't wait."
```

---

## Pattern 3: Observation + Reflection

**Structure:** Observation + Seasonal marker + Future promise

**From Text:**
```
"The field was empty.
Winter.
It'll turn green again when spring comes."
```

**Cyclical Thinking Pattern:**
- Present state (empty)
- Current season (winter)
- Future change (spring returning)
- Implied: Things always change

---

## Pattern 4: Minimal Affirmation

**Structure:** "Sure. [Question word]?"

**From Text:**
```
"Sure. Where?"
```

**Agreement + Immediate Action Pattern:**
- Agreement (Sure)
- Request for specifics (Where?)

**Similar Patterns:**
```
A: Wanna get dinner?
B: Sure. When?
A: 6pm?
B: Works for me.
```

---

## Pattern 5: Goal-Setting Statement

**Structure:** Time expression + motivation expression

**From Text:**
```
"66 days.
Gotta keep pushing."
```

**Motivation Combination:**
- Concrete number
- Action verb
- Forward momentum

**Similar Patterns:**
```
"300 days until the marathon. Have to train hard."
"One month to go. Keep focused."
"Only 50 days. Never give up."
```

# PART 7: TEEN/CASUAL SPEECH FEATURES (구어체 특징)

## 7.1 Fragment Sentences (불완전 문장)

**Examples from Entry:**
```
"Clear skies, below freezing."
"Winter."
"The field was empty."
"58 minutes."
"Not bad."
"Evening."
```

**Why Fragments Work:**
- Rhythm and pace control
- Emphasis through brevity
- Natural thought pattern
- Diary/journal convention
- Creates vivid impressions

**Comparison:**

| Full Sentence | Fragment | Effect |
|---------------|----------|--------|
| "The weather was clear and below freezing." | "Clear skies, below freezing." | More punchy, vivid |
| "I ran 10 kilometers and it took 58 minutes." | "Ran 10km. 58 minutes." | More energetic, direct |
| "The season is winter." | "Winter." | More atmospheric, poetic |

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Making Running Plans

```
A: Got a text from you!
B: Yeah, I was thinking about running tomorrow.
A: What're you thinking? Distance-wise?
B: The coastal road. 10km. Easy pace.
A: Sounds good. What time?
B: Maybe 6am? Before it gets busy.
A: Perfect. I'll see you then.
B: Cool. Let's do it.
```

### Scenario 2: Reflecting on the Year

```
A: Can you believe 2024 is almost done?
B: I know, right? It flew by.
A: What would you call it? Your year?
B: Honestly? The year of running. Changed everything.
A: That's cool. What's next?
B: March 22nd. The marathon. That's what I'm focused on now.
A: 66 days? That's coming up fast.
B: Yeah. Gotta keep pushing.
```

### Scenario 3: Messaging a Friend

```
A: What're you doing?
B: Just got back from a run.
A: Even the day after Christmas?
B: Yeah! Ran 10km, easy pace.
A: Seriously, that's crazy.
B: Haha. Wanna run together tomorrow?
A: Sure. Where?
B: The coastal road. 10km.
A: ㅇㅋ. See you then.
```

Write a diary entry about your year. What defined it? What's your goal for the next year?

```
- What you're grateful for
- Your goal for next year
- How many days/weeks until your goal
- Your determination/motivation statement
```

Write a paragraph describing your favorite place to run or exercise.

```
Include:
- Location details (coastal road, field, park)
- Distance/time typically involved
- Weather/season description
- How you feel running there
- Who you run with (if anyone)
- What makes it special

```

```
- When you'll return to routine
- How you felt coming back
```
