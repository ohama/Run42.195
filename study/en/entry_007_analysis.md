# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Weather & Morning Observations

### "Rain. It's been falling since morning."

**Original Korean:** "비. 아침부터 내리고 있다."

**Analysis:**
- Fragment sentence for dramatic effect
- Present perfect continuous for ongoing action
- Sets melancholic mood immediately

**Grammar Point:** Present Perfect Continuous for ongoing situations

**Structure:** It's been + verb-ing + time expression

**More Examples:**

| Time Expression | Full Sentence | Korean |
|-----------------|---------------|--------|
| since morning | "It's been raining since morning." | 아침부터 비가 내리고 있다 |
| all day | "It's been raining all day." | 하루 종일 비가 오고 있다 |
| for hours | "It's been raining for hours." | 몇 시간째 비가 오고 있다 |
| since yesterday | "It's been raining since yesterday." | 어제부터 비가 오고 있다 |
| nonstop | "It's been raining nonstop." | 쉬지 않고 비가 오고 있다 |

**Variations:**

| Casual | Standard | Formal |
|--------|----------|--------|
| "Been raining since morning." | "It's been raining since morning." | "The rain has continued since this morning." |
| "Still raining." | "It's still raining." | "The rain persists." |
| "Won't stop raining." | "It won't stop raining." | "The precipitation continues unabated." |

**Practice Dialogue:**
```
A: Is it still raining outside?
B: Yeah, it's been falling since morning.
A: Great. I didn't bring an umbrella.
B: Me neither. We're stuck here.
```

### "As soon as I opened my eyes, I looked at the window."

**Original Korean:** "눈을 뜨자마자 창문을 봤다."

**Grammar Point:** "As soon as" for immediate sequence

**Structure:** As soon as + subject + past tense, subject + past tense

**More Examples:**

| Action 1 | Action 2 | Full Sentence |
|----------|----------|---------------|
| opened my eyes | looked at window | "As soon as I opened my eyes, I looked at the window." |
| woke up | checked my phone | "As soon as I woke up, I checked my phone." |
| heard the alarm | turned it off | "As soon as I heard the alarm, I turned it off." |
| got home | took off my shoes | "As soon as I got home, I took off my shoes." |
| finished class | ran to the bus stop | "As soon as class finished, I ran to the bus stop." |

**Alternative Structures:**

| Expression | Meaning | Example |
|------------|---------|---------|
| "As soon as" | ~하자마자 | "As soon as I saw him, I knew." |
| "The moment" | ~하는 순간 | "The moment I saw him, I knew." |
| "Right when" | ~할 때 바로 | "Right when I saw him, I knew." |
| "Immediately after" | ~한 직후 | "Immediately after seeing him, I knew." |
| "No sooner had...than" | ~하자마자 (격식) | "No sooner had I seen him than I knew." |

**Common Mistakes:**
❌ "As soon as I opened my eyes, I was looking at the window." (진행형 불필요)
✅ "As soon as I opened my eyes, I looked at the window."

## 1.2 Expressions of Disappointment

### "I was going to walk today, but it rained."

**Original Korean:** "오늘 걸어가려고 했는데, 비가 왔다."

**Grammar Point:** "was going to" for cancelled plans

**Structure:** Subject + was/were going to + verb, but + reason

**More Examples:**

| Plan | Reason | Full Sentence |
|------|--------|---------------|
| walk | rain | "I was going to walk, but it rained." |
| go out | tired | "I was going to go out, but I was too tired." |
| call you | forgot | "I was going to call you, but I forgot." |
| exercise | injury | "I was going to exercise, but I hurt my knee." |
| study | friends came | "I was going to study, but my friends came over." |

**Alternative Structures:**

| Structure | Example |
|-----------|---------|
| "I had planned to..., but..." | "I had planned to walk, but it rained." |
| "I meant to..., but..." | "I meant to walk, but it rained." |
| "I intended to..., but..." | "I intended to walk, but it rained." |
| "I was supposed to..., but..." | "I was supposed to walk, but it rained." |

**Expressing Regret:**
```
"I was going to walk today. But it rained."
"I wish I had walked earlier."
"If only it hadn't rained."
"Too bad it rained."
```

### "Can't help it when it rains."

**Original Korean:** "비 오면 어쩔 수 없지."

**"Can't help it" Expression:**

**Meaning:** 어쩔 수 없다, 방법이 없다

**Variations:**

| Expression | Context | Korean |
|------------|---------|--------|
| "Can't help it." | 일반적 | 어쩔 수 없지 |
| "Can't be helped." | 일반적 (수동) | 어쩔 수 없는 거지 |
| "Nothing I can do about it." | 강조 | 내가 할 수 있는 게 없어 |
| "It is what it is." | 수용 | 그런 거지 뭐 |
| "That's just how it is." | 수용 | 원래 그런 거야 |
| "Oh well." | 체념 | 뭐 어쩔 수 없지 |

**With "When" Clauses:**
```
"Can't help it when it rains." (비 오면)
"Can't help it when you're sick." (아프면)
"Can't help it when traffic is bad." (교통이 막히면)
"Can't help it when things go wrong." (일이 잘못되면)
```

**Practice Dialogue:**
```
A: Sorry I couldn't make it yesterday.
B: What happened?
A: It was pouring. Can't help it when it rains.
B: Yeah, I get it. These things happen.
```

## 1.3 Routine Expressions

### "I had to take the bus."

**Original Korean:** "버스를 타야 했다."

**Grammar Point:** "had to" for past necessity/obligation

**Structure:** Subject + had to + verb

**Obligation Expressions (Past):**

| Expression | Strength | Example |
|------------|----------|---------|
| "I had to" | ★★★★★ | "I had to take the bus." (필수) |
| "I needed to" | ★★★★☆ | "I needed to take the bus." (필요) |
| "I was supposed to" | ★★★☆☆ | "I was supposed to take the bus." (예정) |
| "I ended up having to" | ★★★★☆ | "I ended up having to take the bus." (결국) |

**Contrastive Usage:**
```
"I wanted to walk, but I had to take the bus."
"I was going to walk, but I had to take the bus instead."

```

**Original Korean:** "민서가 이미 와 있었다."

**"Already" Placement:**

## 2.3 Sensory Experiences

**Original Korean:** "버스에서 못 했던 숨."

| Denied Experience | Where | Full Sentence |
|-------------------|-------|---------------|

| sleep | plane | "The sleep I couldn't get on the plane." |

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "I was going to [verb], but [obstacle]."

**Structure:** Subject + was going to + verb, but + reason/obstacle

**Uses:** 취소된 계획, 실현되지 못한 의도

| Plan | Obstacle | Full Sentence |
|------|----------|---------------|
| walk | rained | "I was going to walk, but it rained." |
| go outside | too cold | "I was going to go outside, but it was too cold." |
| call her | phone died | "I was going to call her, but my phone died." |
| stay longer | had to leave | "I was going to stay longer, but I had to leave." |
| cook | too tired | "I was going to cook, but I was too tired." |

**Practice Sentences:**
```
"I was going to walk today, but it rained."
"I was going to exercise, but my back hurt."
"I was going to study, but I fell asleep."
"I was going to buy it, but it was sold out."
```

## Pattern 3: "Before, [past situation]. But now it's different."

**Structure:** Before, [past habit/feeling] + But now + [current state]

**Uses:** 변화 표현, 성장/발전 묘사

**From Text:** "Before, when it rained, I'd just think 'oh, it's raining.' Neither good nor bad. But now it's different."

| Before | Now | Topic |
|--------|-----|-------|
| just thought "oh" | hate it | 비에 대한 감정 |
| didn't care | really want it | 걷기 |
| was scared | enjoy it | 새로운 활동 |
| hated mornings | look forward to them | 아침 |

**Full Examples:**
```
"Before, I used to sleep in. But now I wake up early."
"Before, I didn't like coffee. But now I can't live without it."
"Before, I was afraid of dogs. But now I have two."
"Before, exercise was a chore. But now I actually enjoy it."
```

## Pattern 5: "I don't know if I [verb]. Maybe I just [verb]."

**Structure:** I don't know if I [positive action] + Maybe I just [alternative reason]

**Uses:** 자기 감정/동기 분석, 불확실성 표현

**From Text:** "I don't know if I like it. Maybe I just hate the bus."

| Positive | Alternative | Full Sentence |
|----------|-------------|---------------|
| like it | hate the alternative | "I don't know if I like it. Maybe I just hate the bus." |
| love him | need company | "I don't know if I love him. Maybe I just need company." |
| enjoy work | fear unemployment | "I don't know if I enjoy work. Maybe I just fear unemployment." |
| like running | want to escape | "I don't know if I like running. Maybe I just want to escape." |

```

# PART 10: EXTENDED PRACTICE (연습)

## 10.1 Dialogue Practice

### Scenario 1: Cancelled Plans Due to Rain

```
A: Hey, I thought you were going to walk today.
B: I was going to, but look outside.
A: Oh no, it's pouring.
B: Yeah. It's been coming down since morning.
A: That sucks. You were really looking forward to it.
B: Can't help it when it rains. Nothing I can do about it.
A: Maybe tomorrow?
B: I hope so. The forecast says it'll clear up in the afternoon.
A: Fingers crossed.
```

### Scenario 2: Discussing Motion Sickness

```
A: You don't look so good. Are you okay?
B: Not really. Feeling a bit nauseous.
A: Is it the bus?
B: Yeah. My stomach started churning as soon as we got on.
A: Want a peppermint candy? It might help.
B: Thanks. [takes candy] This helps a little.
A: The bus is always worse when it's packed.
B: And stuffier. The windows are shut because of the rain.
A: Just a few more stops. Hang in there.
B: I'm trying. I just need some fresh air.
```

### Scenario 3: Reflecting on Change

```
A: So you really like walking now?
B: I don't know if I like it. Maybe I just hate the bus.
A: Isn't that the same thing?
B: I'm not sure. It feels different somehow.
A: How so?
B: Before, when it rained, I'd just think "oh, it's raining."
A: And now?
B: Now I actually hate it. Because I can't walk.
A: Wow. When did that happen?
B: I don't know. When did walking become this important to me?
A: That's actually kind of cool. You found something you care about.
```

### Scenario 4: Weather Forecast Discussion

```
A: Did you check the forecast for tomorrow?
B: Yeah, cloudy in the morning, clearing up in the afternoon.
A: So no rain?
B: It might not rain in the morning. Fingers crossed.
A: You really want to walk, huh?
B: You have no idea. Today was rough.
A: The bus?
B: Yeah. All those things came back. The nausea, the smell...
A: Things you'd forgotten.
B: Exactly. I need to walk tomorrow.
```

2. "We go outside." → "We ___ ___ ___ go outside, but..."

Answers: 1. was going to  2. were going to  3. was going to
```

### Change to Ongoing Action (has been + -ing)
```
1. "It rains since morning." → "It ___ ___ ___ since morning."
2. "She works all day." → "She ___ ___ ___ all day."
3. "They wait for hours." → "They ___ ___ ___ for hours."

Answers: 1. has been raining  2. has been working  3. have been waiting
```
