# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Problem Description & Realization

### "My sneakers don't fit right."

**Original Korean:** "운동화가 발에 안 맞아."

**Analysis:**
- 직설적인 문제 표현
- "fit right" = 제대로 맞다, 편하다
- 더 일반적인 표현: "These shoes don't fit well." / "My shoes are uncomfortable."

**Variations by Context:**

| Situation | Expression |
|-----------|-----------|
| 기본 | "My shoes don't fit right." |
| 더 강함 | "My shoes are killing me." |
| 약간 불편 | "My shoes are a bit tight." |
| 매우 불편 | "These shoes are torture." |
| 구체적 | "My toes are pinched." |

**Synonyms:**
- "These shoes don't fit properly."
- "My shoes are uncomfortable."
- "My feet hurt in these shoes."
- "They're too tight."

**Dialogue Practice:**
```
A: Why are you limping?
B: My sneakers don't fit right.
A: You should get new ones.
B: Yeah, I'm thinking about it.
```

**Grammar Point:** Negation for clarification (아니라 = not... but)

**Pattern:** Not X—Y (더 구체적인 정정)

**Body Parts - Foot Anatomy:**

| English | Korean | Location |

## 1.2 Visiting the Nurse's Office

### "The nurse said: 'Did this happen while running?'"

**Original Korean:** "간호사 선생님이 말했다"

**Analysis:** 직접 인용 사용

**Variations of Enquiry about Injuries:**

| Expression | Context |
|------------|---------|
| "What happened?" | 일반적 |
| "How did you hurt yourself?" | 상상 가능한 상황 |
| "Did you fall?" | 낙상 의심 |
| "Did this happen while running?" | 특정 활동 확인 |
| "When did this start?" | 시간 확인 |

**Dialogue in Medical Context:**
```
Nurse: What's the problem?
Student: My foot got scraped while running.
Nurse: Let me see. How bad does it hurt?
Student: It's pretty painful when I run.
Nurse: You need proper running shoes.
```

|-----------|---------|-----------|
| "It seems like..." | 그런 것 같다 | 추측 |
| "Looks like..." | 보인다 | 시각적 |

| "I think..." | 생각한다 | 확신 없음 |

- "This appears to be an infection."
- "It seems like you've twisted your ankle."

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "I didn't know that."

**Structure:** I didn't know + object clause

**From Text:** "I didn't know that. I thought all sneakers were the same."

| Example | Realization |
|---------|------------|
| "I didn't know that." | 몰랐어 |
| "I didn't realize that." | 깨닫지 못했어 |
| "Really? I had no idea." | 진짜? 몰랐어 |
| "That's news to me." | 처음 듣는데 |

**Knowledge Discovery:**
```
A: Running shoes are different from regular sneakers.
B: I didn't know that. What's the difference?
A: Better cushioning and shock absorption.
B: Oh, that makes sense.
```

---

## Pattern 2: "So those are a separate thing."

**Structure:** So + demonstrative + copula + category

**Meaning:** 그러면 그건 별개인 거네 (새로운 카테고리 인식)

| Context | Example |
|---------|---------|
| 새로운 종류 | "So those are a separate thing." |
| 다른 것이라고 확인 | "So that's different from what I thought." |
| 별도 설명 필요 | "So I need to understand this differently." |

**Understanding/Realization Patterns:**
- "Oh, so it's a different thing?"
- "So they're not the same?"
- "I see, it's its own category."
- "So I've been doing it wrong?"

---

## Pattern 3: "Shoes specifically for running."

**Structure:** Nouns + specifically for + noun (더 구체적 설명)

| Example | Meaning |
|---------|---------|
| "Shoes specifically for running" | 달리기 전용 신발 |
| "Clothes designed for athletes" | 운동선수용 옷 |
| "Equipment made for beginners" | 초보자용 장비 |
| "Shoes built for comfort" | 편함을 위해 만든 신발 |

**Specific Purpose Expressions:**
- "These are running shoes, not casual sneakers."
- "They're designed specifically for distance running."
- "This equipment is made for professional use."
- "This is a beginner's version."

## Pattern 4: "If you're running a lot, you should wear proper running shoes."

**Structure:** If + present continuous, you should + verb

**Advice Pattern based on Behavior:**

- "If you're committed to running, get the right equipment."
- "If this matters to you, invest in it."

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Explaining an Injury

```
Friend: What happened to your foot?
You: I scraped it while running today.
Friend: Ouch, that looks bad. From running?
You: Yeah, my shoes don't fit right, and I got blisters.
Friend: You should get better shoes.
You: I know. The nurse told me the same thing.
Friend: Are you going to?
You: Yeah, my mom and I are going shopping this weekend.
```

### Scenario 2: Discussing Shoe Types

```
You: I didn't know there were different types of running shoes.
Friend: Yeah, running shoes are specially designed for distance.
You: What's the difference from regular sneakers?
Friend: Better cushioning, better support, better shock absorption.
You: That makes sense. How much do they cost?
Friend: Anywhere from 50,000 to 200,000 won.
You: That's expensive!
Friend: Yeah, but it's worth it if you're running seriously.
```

### Scenario 3: Asking for Financial Help

```
You: Mom, can I get running shoes?
Mom: Running shoes? Why?
You: When I run, my feet get injured. The nurse said I need proper shoes.
Mom: How much are they?
You: The cheaper ones are around 50,000 won.
Mom: That's a lot! Can you pay half from your allowance?
You: Really? You'd help?
Mom: If you're serious about running, yes. But you have to promise to keep running.
You: I promise! Thank you!
```

### Scenario 4: Researching Online

```
A: What are you doing?
B: Looking up running shoes. I need to find a good pair.
A: How much are they?
B: The prices are all over the place—50,000 to 200,000 won.
A: That's a huge difference!
B: Yeah, depends on the brand. Some are cheap, some are luxury.
A: Which brand does your dad recommend?
B: He likes Nike, but they're expensive.
A: Maybe look for sales?
B: Good idea. Let me check.
```

## 10.4 True/False Statements (with Corrections)

```
1. Doyun's feet hurt because he ran too fast.
   FALSE: His feet hurt because his shoes don't fit properly.

2. The nurse said he needs to wear different socks.
   FALSE: The nurse said he needs proper running shoes.

   FALSE: Running shoes are specially designed with better cushioning.

4. Doyun's mom refused to help pay for the shoes.
   FALSE: His mom agreed to pay half if he pays the other half.

5. The cheapest running shoes cost 200,000 won.
   FALSE: Some cheap ones cost around 50,000 won; expensive ones are 200,000+.
```
