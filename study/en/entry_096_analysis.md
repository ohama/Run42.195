## PART 4: PHRASAL VERBS (구동사)

### 4.1 Common Phrasal Verbs in the Text

| Phrasal Verb | Meaning | Korean | Example | Usage |
|---|---|---|---|---|
| **Ran into** | Encountered unexpectedly | (우연히) 만나다 | "Ran into my teacher" | 우연한 만남 표현 |
| **Got back to** | Returned to; resumed | ~로 돌아가다 | "Got back to my room" | 장소 복귀 |
| **Looked back** | Remembered; reflected | 뒤돌아보다 | (implied) "Thought about a year ago" | 회상 표현 |
| **Struggle with** | Had difficulty with | ~와 힘들다 | "You were struggling every day" | 어려움 표현 |
| **Takes the bus** | Uses bus transportation | 버스를 타다 | "I'm not taking the bus anymore" | 교통수단 표현 |

### 4.2 Phrasal Verb Comparison and Nuance

**"Ran into" vs "Met"**
- "Ran into my teacher" = Unexpected encounter (우연한 만남)
- "Met my teacher" = Planned, intentional meeting (의도적 만남)

**"Got back to" vs "Returned to"**
- "Got back to my room" = Casual, personal journey (편한 표현, 개인적)
- "Returned to my room" = Formal, official (격식적)

### 4.3 Extended Phrasal Verb Practice

```
Pattern: Got back to [location] and started [action]

Example from text: Got back to my room and started writing this entry
한국어: 방으로 돌아가서 이 항목을 쓰기 시작했다

Alternative sentences:
- Got back to the office and started working on the project
- Got back to the kitchen and started preparing dinner
- Got back to the gym and started my workout routine
```

## PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

### 5.1 Question and Answer Patterns

#### Pattern 1: Clarification Question

**Structure:**
```
A: [Statement with implied information]
B: "What [specific element]?"
A: [Provides clarification]
```

**Examples from text:**
```
"It's been a year"
"What has?" (= What's been a year?)
"Since I started running. From last March."
```

**Korean equivalent:** "뭐가?" "언제부터?"

**Alternative clarification questions:**
- "What do you mean?" (무슨 뜻이야?)
- "How so?" (어떻게?)
- "Since when?" (언제부터?)
- "Really?" (정말?)

#### Pattern 2: Reflective Comparison

**Structure:**
```
A: [Past state] → [present different state]
B: Acknowledgment of change
A: Affirmation and explanation
```

**Example from text:**
```
A: "I had really bad motion sickness on the bus, remember?"
B: "You don't have that problem now."
A: "Yeah. Because I'm not taking the bus anymore."
```

#### Pattern 3: Acknowledgment and Support

**Structure:**
```
A: [Expresses personal growth or change]
B: [Shows recognition/support]
A: [Confirms the change]
```

**Example:**
```
A: "I've changed a lot."
B: "Yeah. You've grown a lot."
(Implicit agreement and affirmation)
```

### 5.2 Dialogue Structure Analysis

#### Lunch Conversation Structure
- **Opening:** Nostalgic question ("Remember a year ago?")
- **Setup:** Presenting past difficulty (motion sickness issue)
- **Development:** Recognition of present improvement
- **Conclusion:** Positive affirmation ("That was a good choice")

#### Teacher Conversation Structure
- **Opening:** Time reference ("It's been a year")
- **Clarification:** Specifying what's been a year (running, since March)
- **Acknowledgment:** Teacher recognizes the time passage
- **Growth recognition:** Teacher affirms personal development

### 5.3 Conversational Patterns for Practice

```
PATTERN A: Reflecting on Past
Structure: "I remember [past difficulty]" + "Now I [positive change]"

Examples:
- I remember struggling with anxiety. Now I feel confident.
- I remember being stressed all the time. Now I manage it well.
- I remember hating mornings. Now I wake up energized.

PATTERN B: Time Passage Recognition
Structure: "It's been [time period]" + "I've [change/achievement]"

Examples:
- It's been a year since I started therapy. I've gotten so much better.
- It's been six months since I left that job. I've found peace.
- It's been two years. I've accomplished more than I thought I could.

PATTERN C: Peer Validation
Structure: "You [current positive state]" → Response: "Yeah, I [explanation]"

Examples:
- A: You look so healthy now!
- B: Thanks! I've been exercising regularly.

- A: You seem less stressed.
- B: Yeah, I've changed my daily routine.
```

## PART 7: TEEN/CASUAL SPEECH FEATURES (구어체 특징)

### 7.1 Incomplete Sentences and Fragments

| Feature | Example from Text | Korean | Function |
|---|---|---|---|
| **Single word/phrase responses** | "Yeah" / "What?" | 응 / 뭐? | Quick acknowledgment, seeking clarification |
| **Ellipsis (omitted subjects)** | "Running instead of the bus" | 버스 대신 뛰기 | Common in casual speech |
| **Short declarative statements** | "Morning. / 10km run. / School." | 아침. / 10km 달리기. / 학교. | Diary-style brevity |
| **Implied continuation** | "Now. / 7km every day on foot." | 지금. / 매일 7km를 걸어서. | Reader fills in context |

### 7.2 Casual Connectors and Fillers

| Feature | Example | Korean | Usage |
|---|---|---|---|
| **Implicit logical connectors** | States separated by dashes | 대시로 분리된 문장들 | Suggests cause-effect without explicit transition |
| **Direct address** | "Teacher" / "Hey" | "선생님" / "야" | Friendly, less formal register |
| **Repetition for emphasis** | "So much has changed" (appears twice) | "정말 많이 달라졌어" (두 번 반복) | Emphasizes key realization |

### 7.3 Diary Style Features

```
Characteristic: Fragmented, minimalist style

Pattern observed in entry:
1. Single-word time markers: "Morning" / "School" / "After school"
2. Short sentence fragments: "No bus. / No motion sickness."
3. Explicit transitions (dashes) between thoughts
4. Dialogue as conversation record
5. Final reflection drawing conclusions

This style creates:
- Immediacy and raw emotion
- Teenage authenticity
- Easy reading flow
- Personal, intimate tone
```

## PART 10: EXTENDED PRACTICE (연습)

### 10.1 Dialogue Practice Scenarios

#### Scenario 1: Parent-Child Conversation About Change

**Context:** Parent noticing improvements in child

```
Parent: You know, it's been a year since you started that running routine.
Child: Yeah. I remember being so worried about the bus back then.
Parent: You were. You complained every day about motion sickness.
Child: I don't even think about that anymore. Now I just run.
Parent: And you look so much healthier now. We're proud of you.
Child: Thanks. I've really changed a lot. Even preparing for a marathon now.
Parent: That's amazing. You're growing up so well.
Child: It's all because I made that one good choice.
```
