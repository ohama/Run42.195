# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Running & Performance Conversations

### "You've been running a lot lately?"

**Original Korean:** "요즘 많이 달리고 있어?"

**Grammar Point:** Present Perfect Continuous for recent habits

**Structure:** Have/Has + been + verb-ing + time expression

**Analysis:**
- 최근에 시작한 습관이 지속되고 있음
- 상대방의 활동을 관찰한 후 던지는 질문

**Similar Expressions:**

| Expression | Nuance | Example |
|------------|--------|---------|
| "You've been running a lot lately?" | 최근 변화 관찰 | 선생님이 학생에게 |
| "Have you been training hard?" | 훈련 강조 | 코치가 선수에게 |
| "I've noticed you've been running." | 더 직접적 | 친구가 친구에게 |
| "Are you running more these days?" | 더 캐주얼 | 일반적 질문 |

**Practice Dialogue:**
```
Teacher: You've been running a lot lately?
Student: Yeah, I run every day during the break.
Teacher: That's impressive. How much are you running?
Student: 10km.
Teacher: Wow, that's serious training.
```

| "How far are you going?" | 일반적 거리 | 단거리/장거리 |

**Related Running Expressions:**

| English | Korean | Usage |
|---------|--------|-------|
| "I'm running 10km." | 10km 달려 | 명확한 거리 |

### "That's impressive."

**Original Korean:** "멋있다", "인상적이다"

**Nuance Spectrum:**

| Expression | Intensity | Formality | Context |
|------------|-----------|-----------|---------|
| "Nice." | ★☆☆ | Very casual | 가벼운 칭찬 |
| "Cool." | ★☆☆ | Casual | 또래 칭찬 |
| "That's good." | ★★☆ | Neutral | 일반적 칭찬 |
| "That's impressive." | ★★★ | Neutral | 진지한 칭찬 |
| "That's remarkable." | ★★★★ | Formal | 공식적 칭찬 |
| "That's exceptional." | ★★★★★ | Very formal | 최고 칭찬 |

**Alternative Impressiveness Expressions:**

| Expression | Meaning | When to Use |
|------------|---------|-------------|
| "I'm impressed." | 인상받았어 | 진심 있게 |
| "That's quite an achievement." | 대단한 성과야 | 격식 있게 |
| "Wow, that's serious." | 와, 진심이네 | 놀라움과 칭찬 |
| "Not bad." | 나쁘지 않은데 | 캐주얼하고 다소 비꼬인 |
| "I'm blown away." | 깜놀했어 | 매우 놀라고 감동 |

**Responding to Praise:**
```
A: That's impressive.
B: Thank you. (감사)
B: Thanks, I'm trying my best. (겸손)
B: Thanks, I really enjoy it. (이유 추가)
```

## 1.2 Decision-Making Conversations

### "You interested in the 10km race?"

**Original Korean:** "10km 레이스 할 관심 있어?"

**Grammar Point:** Interested in + gerund

**Structure:** Are you + interested in + noun/gerund?

**Similar Interest Questions:**

| Question | Topic | Formality |
|----------|-------|-----------|
| "You interested in the race?" | 관심 | Casual |
| "Are you interested in racing?" | 활동 | Standard |
| "Would you be interested in joining?" | 제안 | More formal |
| "Do you have any interest in competing?" | 경쟁 | Formal |
| "Fancy giving it a try?" | 시도 | Very casual (British) |

**"Interested in" with Different Topics:**

| Expression | Topic | Context |
|------------|-------|---------|
| "You interested in the race?" | 레이스 | 운동 |
| "You interested in joining us?" | 함께하기 | 그룹 활동 |
| "You interested in that job?" | 일자리 | 취업 |
| "You interested in learning?" | 배우기 | 교육 |

**Natural Responses:**

```
A: You interested in the race?
B: Me? Race? (놀라움)
B: I don't know, maybe. (불확실)
B: Definitely! (적극적)
B: I'm thinking about it. (검토 중)
```

### "I'm thinking about it."

**Original Korean:** "생각해볼게", "고민 중이야"

**"Think about" Pattern:**

| Expression | Meaning | Commitment Level |
|------------|---------|------------------|
| "I'm thinking about it." | 검토 중 | 비커밋 |
| "I'll think about it." | 생각해볼게 | 미결정 |
| "Let me think about it." | 생각 시간 요청 | 보류 |
| "I've been thinking about it." | 계속 생각중 | 진지한 검토 |
| "I can't stop thinking about it." | 계속 떠올려짐 | 집착 |

**Decision-Making Stages:**

| Stage | Expression | Time Frame |
|-------|------------|-----------|
| 초기 거부 | "I don't know..." | 즉시 반응 |
| 검토 | "I'm thinking about it." | 며칠 |
| 호기심 | "Let me think..." | 하루 |
| 결정 직전 | "I've almost decided." | 몇 시간 |
| 확정 | "I'm going to do it." | 확정 |

**Encouraging Someone to Decide:**

```
A: I'm thinking about it.
B: Come on, just decide!
B: You should do it. It'll be fun.
B: Don't overthink it. Give it a try.
```

```

# PART 3: DESCRIBING RACING & COMPETITION (레이싱과 경쟁 묘사)

## 3.1 Race Concepts & Terminology

### "Running with other people. Official times. Rankings."

**Original Korean:** "다른 사람들과 달리기. 공식 기록. 순위."

**Racing Components:**

| Term | English | Korean | Meaning |
|------|---------|--------|---------|
| Race | 레이스 | 경주 | 공식 경쟁 |
| Official times | 공식 기록 | 기록 | 정확히 측정된 시간 |
| Rankings | 순위 | 등수 | 상대적 위치 |
| Finish line | 피니시 라인 | 결승선 | 끝나는 지점 |
| Personal record | 개인 최고기록 | 자신의 최고기록 | 자신의 가장 좋은 시간 |
| Placement | 입상 | 등위 | 일정 순위 내 들기 |

**Racing vs. Running Alone:**

| Solo Running | Racing |
|--------------|--------|
| Self-paced | Pace set by competition |
| No official record | Official times |
| No comparison | Rankings/standings |
| Personal satisfaction | External validation |
| Flexible | Structured |
| Solitude | Community |

**Expressions Contrasting the Two:**

```
"Running alone is different from racing."
"In a race, there's pressure I don't feel when running solo."
"Official races have timing, rankings, other runners."
"When you're racing, everyone's watching."
```

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "There's a [thing]."

**Structure:** There's + singular noun / There are + plural noun

**Uses:** 새로운 정보 소개, 기회 제시

| Example | Information Type |
|---------|------------------|
| "There's a marathon." | 이벤트 |
| "There's a problem." | 문제 |
| "There's a solution." | 해결책 |
| "There's time." | 시간 여유 |
| "There's a reason." | 이유 |

**Conversational Context:**

```
A: "There's a marathon in October."
B: "Really? What kind?"

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Teacher Suggesting a Race

```
Teacher: You've been running a lot lately?
Student: Yeah, I run every day during the break.
Teacher: Oh nice. How much are you running?
Student: 10km.
Teacher: Wow, that's impressive. You know what? There's a regional marathon in October.
Student: A marathon?
Teacher: Yeah. The Jeju Marathon. They have a 10km category too.
Student: Oh, interesting.
Teacher: You interested in the 10km race?
Student: Me? Race? [pause] I don't know...
Teacher: With your level, you could definitely finish it.
Student: But what if there are too many people?
Teacher: That's part of the experience. Racing's a great experience. More motivating than running solo.
Student: I'll think about it.
Teacher: Good. Registration closes mid-September.
```

## 10.4 Discussion Questions

1. **Have you ever decided to try something that scared you?** What helped you make the decision? What was the result?

2. **How do you react when someone suggests you try something new?** Are you usually enthusiastic, cautious, or somewhere in between?

3. **What makes a good motivator for trying challenging things?** Is it encouragement from others, personal pride, fear of missing out, or something else?

4. **How do emotions like nervousness and excitement differ physically?** Can they happen at the same time?

5. **Would you participate in a local race?** Why or why not? What category would you choose and why?

7. **What's your strategy for making important decisions?** Do you think immediately, or do you need time?
