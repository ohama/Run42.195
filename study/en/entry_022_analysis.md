# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Running & Athletic Routine

### "I tried running the whole way for the first time."

**Original Korean:** "처음으로 전 거리를 뛰어가 봤다"

**Grammar Point:** Try + gerund for attempting new actions

**Analysis:**
- 처음 시도하는 행동을 표현하는 방법
- "try" + "-ing" = ~해 보다 (처음)
- "for the first time" = 처음으로

**Variations by Context:**

| Expression | Meaning | Context |
|------------|---------|---------|
| "I tried running..." | 처음 시도해 봤다 | 새로운 활동 |
| "I attempted to run..." | 시도했다 | 더 격식적 |
| "I gave it a try." | 해 봤다 | 가벼움 |
| "I took a shot at running." | 도전해 봤다 | 캐주얼 |
| "I finally ran the whole way." | 드디어 전체를 뛰었다 | 성공 강조 |

**Similar First-Time Expressions:**

| English | Korean | Example |
|---------|--------|---------|
| "I attempted it." | 시도했다 | "I attempted the 5k run." |
| "I gave it a go." | 해 봤다 | "I gave running a go." |
| "I went for it." | 도전했다 | "I went for the marathon." |
| "I challenged myself." | 나를 도전했다 | "I challenged myself to run 7km." |
| "I did it for the first time." | 처음 했다 | "I ran 7km for the first time." |

**Practice Dialogue:**
```
A: Did you finish the race?
B: I tried running the whole way for the first time.
A: Really? How did you do?
B: I made it! Barely, but I did it.
```

### "7km. From home to school. Every step."

**Original Korean:** "7km. 집에서 학교까지. 한 발 한 발."

**Grammar Point:** Fragment listing for emphasis

**Analysis:**
- 완전한 문장 없이 핵심 정보를 나열
- 독자에게 같은 감정을 전달
- 리듬감 있는 표현 (rhythm)

**Why Fragments Work:**
- 강조 효과 (emphasis)
- 신속한 정보 전달 (quick info)
- 감정 전달 (emotion)
- 산문 vs 시 같은 효과 (poetic)

**Similar Fragment Patterns:**

| Fragments | Effect | Korean |
|-----------|--------|--------|
| "7km. Non-stop. Achievement." | 업적 강조 | 한 번에, 멈추지 않고, 성공 |
| "Early morning. Dark sky. Nervous." | 상황 강조 | 이른 아침, 어두운 하늘, 긴장 |
| "One kilometer. Two kilometers. Three." | 진행 과정 | 1km, 2km, 3km |
| "My legs. My breathing. My will." | 신체 부분 강조 | 다리, 호흡, 의지 |

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Telling Your Friend About Your Achievement

```
A: How did it go this morning?
B: I ran the whole 7km to school.
A: No way! The whole way?
B: Yeah. Without stopping.
A: That's insane. How did it feel?
B: It was tough. Really tough. There was this moment at 4km where I almost stopped.
A: But you didn't?
B: No. I told myself "just a little more," and somehow I made it.
A: How long did it take?
B: 45 minutes. Usually it takes an hour and ten minutes on the bus.
A: That's amazing. You're faster than the bus!
B: I know, right? And then I got this runner's high...
A: Runner's high?
B: Yeah, you know that feeling when you're struggling and suddenly you feel fine?
A: That sounds awesome.
B: It was. My whole body was tingling.
```

### Scenario 2: Dealing with Doubt

```
A: I'm thinking of running 5km tomorrow.
B: 5km? That's a long way!
A: I know. I'm scared I can't do it.
B: Maybe you should take it easier the first time.
A: Yeah, maybe you're right. But...
B: But what?
A: I've been running more, and... I don't know. Maybe I could do it?
B: What made you think you could?
A: I ran 7km last week. And I thought, if I can do that, why not 5km?
A: Plus, there's this moment around the halfway point where things get easier.
B: Really?
A: Yeah, it's called a second wind. Everything suddenly feels lighter.
B: Okay, maybe you could do it. Just don't push yourself too hard.
A: Don't worry. If it gets too hard, I'll walk part of it.
B: Smart thinking.
```

### Scenario 3: Marathon Dreams

```
A: Minseo asked me if I was gonna run a marathon.
B: A marathon? That's 42km!
A: I know! It sounds crazy.
B: Do you want to run a marathon?
A: I mean... not today. But a few months ago, even 1km was impossible.
B: And now?
A: Now I ran 7km without stopping.
B: So you're saying...?
A: I'm saying I never could have imagined this before. So maybe... someday?
B: That's an amazing goal!
A: I don't know. It's scary to even think about it. But it doesn't seem completely impossible.
B: What will you do next?
A: Just keep practicing. Keep running 7km until I'm used to it. Then try farther.
B: Slowly?
A: Yeah, slowly. Little by little. That's how I got here.
```
