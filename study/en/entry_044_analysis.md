# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Expressing Progress & Achievement

### "7km got easy."

**Original Korean:** "7km이 쉬워졌다"

**Analysis:**
- 간단하지만 강력한 표현
- 주어 + 목적어 + got + 형용사 = 상태 변화 표현
- 성취감을 효과적으로 전달

**Structure:** Subject + got + adjective (상태 변화)

| Examples | Korean |
|----------|--------|
| "7km got easy." | 7km이 쉬워졌다 |
| "Math got easier." | 수학이 더 쉬워졌다 |
| "The routine got easier." | 루틴이 쉬워졌다 |
| "It got better." | 더 좋아졌다 |
| "Running got fun." | 달리기가 재미있어졌다 |

**Progress Expressions Spectrum:**

| Stage | Expression | Korean |
|-------|------------|--------|
| 아직 어려움 | "Still hard." | 아직 어렵다 |
| 조금 쉬워짐 | "Getting a little easier." | 조금 쉬워지고 있다 |
| 눈에 띄는 진전 | "It got easier." | 쉬워졌다 |
| 완전히 쉬워짐 | "7km got easy." | 이제 쉽다 |
| 너무 쉬움 | "7km is nothing." | 7km은 아무것도 아니다 |

**Conversational Variations:**
- "7km isn't hard anymore." (더 이상 어렵지 않다)
- "I can do 7km without struggling." (힘들지 않고 할 수 있다)
- "7km feels manageable now." (더 관리할 수 있게 느껴진다)

**Practice Dialogue:**
```
A: How's your running going?
B: It's getting better. 7km got easy!
A: Really? You didn't think you could do that a few months ago.
B: I know, right? I used to die after 5km.
A: That's amazing progress.
B: Thanks. I'm thinking about trying 10km next.
```

**Grammar Point:** Present Perfect for recent improvement

**Structure:** "My stamina really has improved."

|------|--------|---------|
| stamina | 체력/지구력 | 오래 지속할 수 있는 능력 |
| endurance | 인내력 | 힘든 상황을 견디는 능력 |
| fitness level | 체력 수준 | 전체적인 신체 상태 |
| aerobic capacity | 유산소 능력 | 산소 사용 능력 |
| muscle strength | 근력 | 근육의 힘 |

**Similar Achievement Sentences:**

| Sentence | Emphasis |
|----------|----------|
| "My stamina has improved." | 객관적 서술 |
| "My stamina really has improved." | 강조 (정말로) |

**Alternative Phrases:**
- "I'm in better shape." (더 좋은 상태다)

## 1.2 Describing Physical Changes

### "It wasn't hard."

**Original Korean:** "힘들지 않았다"

**Simplicity & Impact:**
- 3 단어로 복잡한 감정 표현
- 과거와 현재 비교의 함축
- 강한 확신 전달

**Difficulty Spectrum:**

| Expression | Level | Meaning |
|------------|-------|---------|
| "It was impossible." | ★★★★★ | 불가능했다 |
| "It was very hard." | ★★★★ | 매우 어려웠다 |
| "It was hard." | ★★★ | 어려웠다 |
| "It was kind of hard." | ★★ | 약간 어려웠다 |
| "It wasn't hard." | ★ | 어렵지 않았다 |
| "It was easy." | ☆ | 쉬웠다 |

**Conversational Use:**
```
A: Did you have trouble with today's run?
B: No, it wasn't hard.
A: Really? Yesterday you were struggling.
B: I know. I'm getting better.
```

### "No particularly hard stretch."

**Original Korean:** "특별히 힘든 구간이 없었다"

**Nuance:**
- "stretch" = 구간/거리
- 어떤 특정 시점도 특별히 힘들지 않았다는 의미
- 전체적으로 균등하게 쉬워졌다는 뜻

**"Stretch" in Running Context:**

| Term | Korean | Example |
|------|--------|---------|
| first stretch | 처음 구간 | "The first 2km was easy." |
| the hard stretch | 힘든 구간 | "3km to 5km was the hardest." |
| the easy stretch | 쉬운 구간 | "The last 2km was great." |
| second wind | 제2의 힘 | "I got my second wind at 4km." |

**Related Expressions:**
- "There was no difficult part." (어려운 부분이 없었다)
- "It was consistent." (일관되게 쉬웠다)
- "Every part was manageable." (모든 부분이 감당할 수 있었다)

### "Got a little winded, but nothing I couldn't handle."

**Original Korean:** "조금 숨차긴 했지만, 견딜 수 없을 정도는 아니었다"

**"Winded" Definition:** 숨이 차다 (호흡이 가빠지다)

**Breathing Terms in Running:**

| Expression | Intensity | Meaning |
|------------|-----------|---------|
| "breathing hard" | ★★ | 숨을 쉬기 힘들다 |
| "winded" | ★★★ | 숨이 차다 |
| "out of breath" | ★★★★ | 숨을 못 쉬다 |
| "gasping for air" | ★★★★★ | 헐떡거리다 |

**"Nothing I couldn't handle" Pattern:**

| Expression | Meaning |
|------------|---------|
| "Nothing I couldn't handle." | 견딜 수 있었다 |
| "I could manage." | 처리할 수 있었다 |
| "It was manageable." | 감당할 수 있었다 |
| "I got through it." | 견뎌냈다 |
| "It was okay." | 괜찮았다 |

**Practice Dialogue:**
```
A: How was today's 7km?
B: Pretty good. Got a little winded, but nothing I couldn't handle.
A: Nice. You're really improving.
B: Yeah, my breathing is so much better now.
```

## 1.3 Reflecting on Past vs. Present

### "In May, I was gasping for air after 7km. Now it's just 'okay, done with today's run.'"

**Original Korean:** "5월에는 7km 후에 숨이 차서 힘들었는데, 지금은 그냥 '오늘 운동 끝' 정도."

**Contrast Structure:** Past tense + Present tense

**The Power of Comparison:**
- 과거 어려움 vs 현재 쉬움
- 진전을 구체적으로 보여줌
- 격려적 톤

**Time Transition Phrases:**

| Expression | Usage |
|------------|-------|
| "Back in May" | 5월에는 |
| "In May" | 5월에 |
| "At first" | 처음에는 |
| "Six months ago" | 6개월 전 |
| "Three months later" | 3개월 후 |
| "Now" | 지금은 |
| "These days" | 요즘은 |

**Casual Dismissal Expression:**
- "It's just..." = 그냥... (쉽게 생각함)
- "No big deal" = 별 것 아니다
- "Easy peasy" = 너무 쉽다 (매우 캐주얼)

## 1.4 Telling Someone About Achievement

### "Mom, 7km got easy."

**Original Korean:** "엄마, 7km이 쉬워졌어"

**Why This Matters:**
- 부모와의 정서적 소통
- 단순하지만 직접적인 표현
- 성취의 순간을 공유

**Similar Achievement Announcements:**

| Situation | Sentence |
|-----------|----------|
| 일 완료 | "I finished the project!" |
| 기술 습득 | "I can swim now!" |
| 시험 합격 | "I passed the test!" |
| 목표 달성 | "I made my 5km goal!" |


| Response | Tone |

# PART 3: DESCRIBING PROGRESS & IMPROVEMENT (진전 묘사)

## 3.1 Comparing Past and Present

**Original Korean:** "5월에 처음 풀 거리를 뛸 때는 45분 걸렸고 죽을 것 같았다"

**Extreme Exaggeration in Casual Speech:**
- "felt like I was dying" = 매우 힘들었다 (실제로는 아님)
- 과장으로 강조 효과 낸다

**Time Indicators:**

| Expression | Meaning |
|------------|---------|
| "Back in May" | 5월에는 |
| "when I first ran" | 처음 뛸 때 |
| "the whole distance" | 전체 거리 |
| "it took 45 minutes" | 45분 걸렸다 |

**Exaggeration for Emphasis:**

| Casual | Literal |
|--------|---------|
| "felt like I was dying" | "was very exhausted" |
| "nearly passed out" | "felt dizzy" |

**Practice Dialogue:**

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: Sharing Recent Progress with Parent

**Structure:** "I [action], [positive observation]."

```
Doyun: "Mom, 7km got easy."
Mom: "Easy?"
Doyun: "Yeah. It used to be hard, but now I can just run it."
Mom: "That's amazing. Our Doyun."
Doyun: "I'm going to work up to 10km."
Mom: "Okay, take it slow."
```

**Key Elements:**
- Direct announcement
- Parent's surprise/confirmation question
- Explanation of change
- Parent's pride/warning
- Child's ambition

---

## Pattern 2: Mutual Recognition of Progress

**Structure:** "Hey, [compliment]? Yeah, [agree + explain]."

```
Minseo: "Hey, am I getting better too?"
Doyun: "Yeah. You definitely look better."
Minseo: "Really?"
Doyun: "You're running faster and breathing easier."
```

**Why This Works:**
- Reciprocal encouragement
- Specific observations
- Builds confidence
- Motivates continued effort

---

## Pattern 3: Understanding Universal Process

**Structure:** "I think that's how it is for [group]. [Universal truth]."

```
"I think that's how it is for everyone.
It's hard in the beginning.
But if you keep at it, you get better."
```

**Generalizing from Personal Experience:**
- Personal observation → universal principle
- Wisdom-sharing
- Mentoring tone (even as teen)

**Similar Patterns:**
- "That's how it works." (그렇게 작동한다)
- "That's always the case." (항상 그렇다)
- "It's the same for everyone." (누구나 그렇다)
- "That's just how things are." (그게 현실이다)

# PART 7: TEEN/CASUAL SPEECH FEATURES (구어체 특징)

## 7.1 Fragment Sentences for Impact

**Why Fragments Work:**
- 강조 (주의 집중)
- 진정한 생각의 흐름
- 다이나믹한 페이싱

| Fragment | Full Sentence | Effect |
|----------|---------------|--------|

| "Something was different." | "I noticed something was different." | 미스터리 |
| "It wasn't hard." | "I found that it wasn't hard." | 확실성 |
| "Forty-five minutes." | "It took forty-five minutes." | 강조 |
| "Seven minutes faster." | "That's seven minutes faster." | 인상 |

## 7.2 Casual Connectors

**Common Connectors in Teen Speech:**

| Connector | Usage | Example |
|-----------|-------|---------|
| "but" | contrast | "Got a little winded, but nothing I couldn't handle." |
| "and" | addition | "My stamina improved. For sure." |
| "so" | result | "7km got easy, so it's time to try for 10km." |

**Running Sentences Together:**
```
"I just ran. No particularly hard stretch. Got a little winded,
but nothing I couldn't handle."
```

## 7.3 Hedging & Confidence

```

```
