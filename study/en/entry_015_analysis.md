## 1.3 Variations and Alternatives

### "Something amazing happened"
- Something incredible happened (더 강한 놀라움)
- The craziest thing happened (구어체, 젊은 층)
- You won't believe what happened (상대방 관심 유도)
- Guess what happened (대화 시작용)

### "I hate X. No, hated."
- I used to hate X (과거 습관 강조)
- I hated X, but not anymore (변화 명시)
- X was never my thing... until now (극적 변화)

### "You killed it"
- You nailed it (정확하게 해냈다)
- You crushed it (압도적으로 잘했다)
- You smashed it (영국식)
- You aced it (시험/과제에 주로 사용)

## 1.4 Practice Dialogue 1: Sharing Good News

```
A: Hey, something amazing happened at work today!
B: Really? What happened?
A: I ended up giving the presentation instead of my boss.
B: Wow! How did it go?
A: Everyone said I killed it.
B: That's awesome! Keep it up!
```

**한국어 번역:**
```
A: 야, 오늘 회사에서 놀라운 일이 있었어!
B: 진짜? 무슨 일인데?
A: 결국 상사 대신 내가 발표를 하게 됐어.

A: 다들 내가 완전 잘했대.
B: 대박이다! 계속 그렇게 해!
```

# PART 4: PHRASAL VERBS (구동사)

## 4.1 Phrasal Verbs from the Text

| Phrasal Verb | Korean | Example from Text |
|--------------|--------|-------------------|
| keep running | 계속 달리다 | "I kept running" |
| end up ~ing | 결국 ~하게 되다 | "I ended up running six laps" |
| go up | 올라가다 | "I just... go up" (climbing stairs) |
| keep up | 계속하다, 유지하다 | "Keep it up" |
| think about | ~에 대해 생각하다 | "I thought about it" |
| turn into | ~로 변하다 | "That turned into stamina" |

## 4.2 Detailed Analysis: "end up"

**Meaning:** 예상치 못하게 결국 ~하게 되다

| Form | Example | Korean |
|------|---------|--------|
| end up + ~ing | I ended up running six laps | 결국 여섯 바퀴를 뛰게 됐다 |
| end up + 장소 | We ended up at a coffee shop | 우리는 결국 카페에 가게 됐다 |
| end up + 형용사 | He ended up tired | 그는 결국 지치게 됐다 |
| end up + with | She ended up with a cold | 그녀는 결국 감기에 걸렸다 |

**Similar expressions:**
- wound up (더 구어체): I wound up staying late
- turned out (결과 강조): It turned out to be fun
- eventually (시간 경과): I eventually finished

## 4.3 Detailed Analysis: "keep + ~ing"

**Meaning:** 계속 ~하다

| Expression | Korean | Nuance |
|------------|--------|--------|
| keep running | 계속 달리다 | 중단 없이 지속 |
| keep trying | 계속 노력하다 | 포기하지 않고 |
| keep going | 계속 가다 | 멈추지 않고 |
| keep it up | 계속 해 | 격려의 표현 |

**Contrast with "keep on":**
- keep running = 계속 달리다 (일반적)
- keep on running = 계속 달리다 (강조, 약간 고집스러운 느낌)

## 4.4 Detailed Analysis: "turn into"

**Meaning:** ~로 변하다, ~이 되다

| Example | Korean | Context |
|---------|--------|---------|
| That turned into stamina | 그것이 체력이 됐다 | Physical change |
| The rain turned into snow | 비가 눈으로 바뀌었다 | Weather |
| The argument turned into a fight | 말다툼이 싸움으로 번졌다 | Escalation |
| He turned into a different person | 그는 다른 사람이 됐다 | Character change |

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## 5.1 Pattern: "Huh?" as Realization

**Structure:** Huh? (+ realization statement)

**Usage:** 무언가를 깨달았을 때 사용하는 감탄사

| Example | Korean | Context |
|---------|--------|---------|
| Huh? This is actually good. | 어? 이거 진짜 맛있네. | Unexpected discovery |
| Huh? I can do this. | 어? 나 이거 할 수 있네. | Surprising ability |
| Huh? It's already 5 o'clock. | 어? 벌써 5시야. | Time realization |

## 5.2 Pattern: "I thought X, but Y"

**Structure:** I thought + 절, but + 다른 결과

| Example | Korean |
|---------|--------|
| I thought I hated exercise. But I like walking. | 운동을 싫어하는 줄 알았다. 근데 걷기는 좋다. |
| I thought it would be hard, but it was easy. | 어려울 줄 알았는데 쉬웠다. |
| I thought he was angry, but he was just tired. | 화난 줄 알았는데 그냥 피곤했던 거야. |

## 5.3 Pattern: "When I got home, I told X"

**Structure:** When I got home, I + 과거동사

| Example | Korean |
|---------|--------|
| When I got home, I told Mom. | 집에 가서 엄마한테 말했다. |
| When I got home, I took a shower. | 집에 가서 샤워했다. |
| When I got home, I collapsed on the bed. | 집에 가자마자 침대에 쓰러졌다. |

## 5.4 Pattern: "Now that I think about it"

**Structure:** Now that I think about it, + 깨달은 내용

| Example | Korean |
|---------|--------|
| Now that I think about it, climbing stairs feels easier. | 생각해보니까 계단 오르기가 더 쉬워졌다. |
| Now that I think about it, she was right. | 생각해보니까 그녀 말이 맞았다. |
| Now that I think about it, I should have called. | 생각해보니까 전화했어야 했다. |

## 5.5 Pattern: "Nothing X about that"

**Structure:** Nothing + 형용사 + about that

| Example | Korean |
|---------|--------|
| Nothing bad about that. | 그것에 대해 나쁠 건 없다. |
| Nothing wrong about that. | 그게 뭐가 잘못됐어. |
| Nothing strange about that. | 그게 뭐 이상해. |
| Nothing special about that. | 뭐 특별할 것도 없어. |

## 5.6 Pattern: Self-Correction with "No, X"

**Structure:** Statement. No, + 수정된 statement.

| Example | Korean |
|---------|--------|
| I hate endurance runs. No, hated. | 오래달리기 싫어. 아니, 싫었어. |
| I can't do this. No, I can. | 나 못해. 아니, 할 수 있어. |
| She's 30. No, 31. | 그녀는 30살이야. 아니, 31살. |

## 5.7 Extended Practice Dialogue

```
A: I thought you hated running.
B: I did. No, I used to. Something changed.
A: Huh? What do you mean?
B: Now that I think about it, all that walking built up my stamina.
A: That makes sense. Walking is exercise too.
B: Right? Nothing bad about that.
A: Keep it up! You might end up loving it.
```

**한국어 번역:**
```
A: 너 달리기 싫어하는 줄 알았는데.
B: 그랬어. 아니, 그랬었어. 뭔가 달라졌어.
A: 어? 무슨 말이야?
B: 생각해보니까 그 걷기가 체력을 길러줬어.
A: 그게 말이 되네. 걷기도 운동이잖아.
B: 그치? 나쁠 거 없지.
A: 계속 해! 좋아하게 될지도 몰라.
```

## Bonus: Grammar Deep Dive (문법 심화)

### 1. "Not as... as before" Structure

**Formula:** Subject + be + not as + adjective + as + before/usual/expected

| Tense | Example |
|-------|---------|
| Present | I'm not as tired as before |
| Past | I wasn't as nervous as before |
| Present Perfect | I haven't been as sick as before |
| Future | I won't be as slow as before |

**Variations with different comparisons:**
- not as... as I used to be
- not as... as I expected
- not as... as last time
- not as... as everyone else

### 2. "End up + ~ing" vs "Wind up + ~ing"

Both mean "결국 ~하게 되다" but have subtle differences:

| Expression | Nuance | Example |
|------------|--------|---------|
| end up ~ing | Neutral, common | I ended up staying late |
| wind up ~ing | More unexpected | I wound up in the hospital |

**Note:** "wound up" is the past tense of "wind up" (pronounced /waʊnd/)

### 3. Using "Just" for Emphasis

"Just" can minimize or emphasize depending on context:

| Usage | Example | Meaning |
|-------|---------|---------|
| Minimizing | "I just nodded" | Only nodded, nothing more |
| Emphasizing ease | "I just... go up" | It's become effortless |
| Time | "I just got home" | Very recently |
| Exactly | "Just right" | Perfectly |
