# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Coach-Student Interaction

### "Come here for a second."

**Original Korean:** "여기 잠깐 와봐"

**Analysis:**
- 친근하면서도 권위 있는 요청 (코치의 학생에 대한 부름)
- "for a second" = 짧은 시간을 의미하지만 실제로는 더 오래 대화할 수 있음 (예의 표현)

**Variations by Formality:**

| Casual | Standard | Formal |
|--------|----------|--------|
| "Come here a sec." | "Come here for a moment." | "Could you come over here, please?" |
| "Hey, come over." | "Come here for a second." | "I need to speak with you." |
| "Yo, get over here." | "Come this way." | "I would appreciate your presence." |

**Coach's Request Spectrum:**

| Expression | Tone | Context |
|------------|------|---------|
| "Come here." | Neutral | 일반적 부름 |
| "Come here for a second." | Friendly | 짧지만 중요한 이야기 |
| "I need to tell you something." | Serious | 중요한 내용 |
| "Sit down, we need to talk." | Very serious | 심각한 상황 |

**Practice Dialogue:**
```
Coach: "Come here for a second."
Doyun: "Yes, Coach?"
Coach: "I want to teach you something important today."
Doyun: "What is it?"
Coach: "It's about race strategy."
```

## 1.2 Learning & Understanding

### "You know about pacing?"

**Original Korean:** "페이싱이 뭔지 알아?"

**Teaching Question Pattern:**
- Coach는 학생이 이미 알 수도 있다는 가정 하에 질문
- 상대방의 기초 지식을 확인하는 방식

**Knowledge Check Patterns:**

| Pattern | Meaning | Example |
|---------|---------|---------|
| "Do you know about...?" | 기초 지식 확인 | "You know about pacing?" |
| "Are you familiar with...?" | 경험 확인 | "Are you familiar with tempo runs?" |
| "Have you heard of...?" | 정보 확인 | "Have you heard of split times?" |
| "What do you know about...?" | 깊이 있는 확인 | "What do you know about heart rate zones?" |

**Dialogue:**
```
Coach: "You know about pacing?"
Doyun: "Yeah. Like controlling your speed."
Coach: "Right. That's exactly it."
```

---

### "Like controlling your speed."

**Original Korean:** "속도를 조절하는 것 같은"

**Simple Definition with "like":**
- "like" = ~와 같은, ~처럼
- 정확하지 않은 정의를 할 때 사용 (I think, I guess와 유사)

**"Like" as Definition Strategy:**

| Usage | Example | Meaning |
|-------|---------|---------|
| Simple comparison | "Like controlling your speed." | 속도 조절 같은 것 |
| Rough equivalent | "It's like a warm-up jog." | 워밍업 같은 조깅 |
| Approximation | "Like five minutes per km." | 대략 5분 |
| Understanding check | "So it's like... managing energy?" | 에너지 관리? |

---

### "Right. And that's the most important thing in a race."

**Original Korean:** "맞아. 그리고 그게 경기에서 제일 중요한 거야"

**Confirmation + Emphasis:**
- "Right" = 맞다는 동의
- "And that's" = 추가적 중요성 강조

**Confirmation Patterns:**

| Pattern | Meaning | Effect |
|---------|---------|--------|
| "Right." | 맞다 | Simple agreement |
| "Right. And that's the most important thing." | 강조하며 동의 | 신뢰성 높임 |
| "Exactly. That's why..." | 정확히 맞다 | 이유도 제시 |
| "Yes. In fact..." | 동의 + 추가정보 | 더 깊이 있는 설명 |

## 1.3 Warnings & Explanations

### "If you run too fast at the beginning, you'll burn out toward the end."

**Original Korean:** "처음에 너무 빨리 뛰면, 나중에는 지칠 거야"

**"Burn out" Analysis:**
- burn = 연소하다, 불타다
- out = 다 소진되다
- **Meaning:** 에너지를 다 사용해서 피곤해지다

**Conditional Warning Pattern:**

| Structure | Meaning | Example |
|-----------|---------|---------|
| "If you... you'll..." | 결과 경고 | "If you run too fast, you'll burn out." |
| "If you... you might..." | 가능성 경고 | "If you sprint early, you might hit the wall." |
| "If you... you won't..." | 부정적 결과 | "If you go too slow, you won't get a good time." |

**Related Physical Expressions:**

```
"You'll burn out."
"You'll hit a wall."
"You'll run out of energy."
"You'll suffer later."
"You won't be able to finish strong."
```

---

### "Can't finish strong."

**Original Korean:** "강하게 마무리할 수 없어"

**"Finish Strong" Concept:**
- Racing에서 중요한 개념
- 마지막 부분에서 빠르고 강하게 마무리하기

**Race Performance Language:**

| Expression | Stage | Meaning |
|------------|-------|---------|
| "Start strong." | Beginning | 처음부터 힘 있게 |
| "Keep steady." | Middle | 일정한 속도 유지 |
| "Finish strong." | End | 끝을 강하게 |
| "Sprint the finish." | Final | 마지막 전력질주 |

**Energy Management Dialogue:**
```
A: How should I run the race?
B: Start easy, keep steady, finish strong.
A: That's the strategy?
B: That's exactly it.
```

---

### "On the flip side, if you go too slow at first, you won't get a good time."

**Original Korean:** "반대로, 처음에 너무 천천히 가면, 좋은 기록을 못 낼 거야"

**"On the flip side" = 한편으로는, 반대로는**
- flip = 뒤집기
- side = 한쪽
- **Usage:** 대조적인 상황을 설명할 때

**Contrast Expressions:**

| Expression | Meaning | Context |
|------------|---------|---------|
| "On the flip side" | 반대 관점 | 상반된 문제 제시 |
| "On the other hand" | 한편으로는 | 다른 각도 |
| "But on the other side" | 그런데 반대쪽 | 균형잡힌 설명 |
| "However" | 하지만 | 대조 |
| "In contrast" | 대조적으로 | 비교 |

**Practice Dialogue:**
```
Coach: "Too fast at the start, you'll burn out."
Doyun: "So I should go slow?"
Coach: "On the flip side, if you go too slow, you won't get a good time."
Doyun: "So there's a balance?"
Coach: "Exactly. That's called pacing."
```

# PART 2: PHYSICAL SENSATIONS & BODY AWARENESS (신체 감각 표현)

## 2.1 Body Fatigue Language

### "Burn out"

**Original Korean:** "지쳐서 못 나감"

**Physical Breakdown Stages:**

| Stage | Expression | How You Feel |
|-------|------------|--------------|
| Normal | "I feel good." | 기분이 좋음 |
| Tired | "I'm getting tired." | 피곤해짐 |
| Very tired | "I'm wearing out." | 체력이 떨어짐 |
| Exhausted | "I'm burning out." | 에너지 완전 소진 |
| Critical | "I'm hitting the wall." | 더 이상 못함 |

**Energy Depletion Expressions:**

```
"I burned out halfway."
"I ran out of gas."
"I hit the wall at 8km."
"My legs gave out."
"I completely bonked."
"I crashed near the end."
```

---

### "Your body's still warming up."

**Original Korean:** "너의 몸이 아직 워밍업 중이야"

**Warm-up Concept:**
- 운동 시작 시 신체가 준비 단계에 있음
- 처음 몇 분은 최고 성능을 낼 수 없음

**Warm-up Stages:**

| Stage | Time | Description |
|-------|------|-------------|
| Cold start | 0-2 min | 신체 준비 중 |
| Warming up | 2-5 min | 점차 활발해짐 |
| Warmed up | 5-10 min | 본 활동 준비 완료 |
| Peak performance | 10+ min | 최고 능력 발휘 |

**Body Temperature Expressions:**

```
"Your body's still warming up."
"You need a few minutes to warm up."
"Don't push too hard while you're warming up."
"Once you're warmed up, you can increase pace."
"Your muscles need time to warm up."
```

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: Confirmation Question

**Structure:** "You [know/understand/have]...?"

**Meaning:** 기초 개념 확인

**Examples:**
```
"You know about pacing?"
"You have a running watch?"
"You understand the strategy?"
"You know what I mean?"
```

**Response Patterns:**
```
"Yeah, I do."
"Yeah. Like..."
"Not really, can you explain?"
"Kind of, but I'm not sure."
```

---

## Pattern 2: Conditional Warning

**Structure:** "If you [do X], you'll [experience Y]."

**Meaning:** 행동의 결과 경고

**Examples from Text:**
```
"If you run too fast at the beginning, you'll burn out toward the end."
"If you get excited and run too fast, you'll suffer later."
"If you go too slow at first, you won't get a good time."
```

**Real-world Usage:**
```
"If you don't warm up, you'll get injured."
"If you don't drink water, you'll get dehydrated."
"If you push too hard too soon, you'll hit the wall."
```

**Dialogue Practice:**
```
A: Should I run really fast in the beginning?
B: No. If you do that, you'll burn out.
A: How much slower should I go?
B: About 10 seconds per kilometer slower.
```

---

## Pattern 3: Multi-Part Instructions

**Structure:** Divide race into sections, give separate instructions for each

**Formula:**
```
"[Section 1]: [Instruction]"
"[Section 2]: [Instruction]"
"[Section 3]: [Instruction]"
```

**Example from Text:**
```
"For the first 1-2km, start a little slower—about 5 minutes 10 seconds."
"Kilometers 3 through 7, hold steady. Keep that 5-minute pace."
"For the last 8-10km, use everything you've got left. You can push down to 4 minutes 50 seconds."
```

**Other Multi-Step Instructions:**
```
1. Eat a light breakfast
2. Arrive 30 minutes early
3. Do a warm-up jog
4. Check your gear
5. Get to the starting line
```

---

## Pattern 4: Explanation → Reason

**Structure:** "Why?" → "[Reason]"

**Examples:**
```
Q: "Why start slower?"
A: "Your body's still warming up. If you get excited and run too fast at the start, you'll suffer later."

Q: "Why 3-7km at steady pace?"
A: "That's your aerobic zone. You want to run comfortably hard here."

Q: "Why finish strong?"
A: "You have energy left, and it shows you trained properly."
```

---

## Pattern 5: Giving & Receiving Help

**Structure:** Offer → Accept → Gratitude

**Example from Text:**
```
Coach: "Wait. [pulls something out] A running watch. You can borrow it."
Doyun: "Really?"
Coach: "Give it back after the race."
Doyun: "Thank you so much!"
```

**Dialogue Variations:**
```
A: "You need a watch?"
B: "Yeah, I don't have a good one."
A: "Here, use mine."
B: "Seriously? Thanks!"
A: "No problem. Good luck."

A: "Want some help?"
B: "That would be great."
A: "Let me teach you about pacing."
B: "I appreciate that."
```
