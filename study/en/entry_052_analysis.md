# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Conversational Opening & Question Patterns

### "Coach, I have a question."

**Original Korean:** "코치님, 질문이 있는데요."

**Analysis:**
- Formal yet direct way to ask for advice
- Common in sports/mentoring contexts
- Sets up a question that follows

**Variations by Formality:**

| Casual | Standard | Formal |
|--------|----------|--------|
| "Hey, quick question." | "I have a question." | "If I may ask a question..." |
| "Can I ask something?" | "Coach, I have a question." | "Coach, might I inquire..." |
| "Question for you." | "Can I ask you something?" | "If it's not too much trouble..." |

**Practice Dialogue:**
```
A: Coach, I have a question.
B: Yeah, what is it?
A: How should I train for the race?
B: Good question. Sit down and let's talk.
```

## 1.2 Instructional Language

### "Sit down."

**Original Korean:** "앉아."

**Grammar Point:** Imperative (명령문)

**Usage in Coaching Context:**

Korean coaches often use direct commands as a sign of care/engagement. In English, this can sound blunt, so context matters:

| Context | Softening Techniques |
|---------|---------------------|
| Coach to student | "Sit down" is fine |
| Parent to child | Add "please" → "Sit down, please" |
| Peer to peer | "Wanna sit down?" or "Let's sit down" |

**Alternative Forms:**

```
Direct:     "Sit down."
Softer:     "Have a seat."
Inviting:   "Let's sit and talk."
Polite:     "Would you mind sitting down?"
```

---

### "Can't do every day. You need rest days."

**Original Korean:** "매일은 안 돼. 휴식일이 필요해."

**Grammar Point:** Sentence fragments in instruction/advice

**Why Fragments Work Here:**
- Direct and emphatic
- Coach-like authority
- Emphasizes importance

**Full vs. Fragment Versions:**

| Fragment | Full Version |
|----------|--------------|
| "Can't do every day." | "You can't run every day." |
| "You need rest days." | "You're going to need rest days." |
| "Injury is the scariest thing." | "Injury is the scariest thing that could happen." |

**Advice Giving Pattern:**

```
[What not to do]. [Why]. [What to do instead].

"Can't overdo it. Injury is the scariest thing. Rest days are essential."
```

---

### "How should I organize this?"

**Original Korean:** "이걸 어떻게 짜야 되나?"

**"How should I" Pattern for Requesting Advice:**

| Question | Context |
|----------|---------|
| "How should I train?" | General advice |
| "How should I organize this?" | Structure/planning |
| "How should I approach this?" | Strategy |
| "How should I prepare?" | Readiness |
| "How should I manage this?" | Time/resources |

**Receptive Learning Signals:**

These phrases show the speaker is ready to learn:
- "How should I...?"
- "What should I do?"
- "Can you explain...?"
- "Should I...?"

# PART 3: DESCRIBING TRAINING STRUCTURES & SCHEDULES (훈련 구조 및 일정 설명)

## 3.1 Types of Running Training

### "Long runs. Running slow over a long distance. Good for building endurance."

**Original Korean:** "장거리 달리기. 느리게 오래 뛰는 거. 지구력 기르는데 좋아."

**Three Training Types Explained:**

| Type | Description | Purpose | Example |
|------|-------------|---------|---------|
| Long run | 느리게 오래 | 지구력 구축 | "Run 10-12km at a slow pace" |
| Interval training | 빠르고 느리고 반복 | 스피드 향상 | "Fast 2min, slow 2min, repeat 5x" |
| Tempo run | 레이스 페이스로 꾸준히 | 실전 연습 | "Run at race pace for 30 minutes" |

**Educational Language Pattern:**

Coach uses this structure:
1. **Name the concept**
2. **Explain it simply**
3. **State the benefit**

```
"Second, intervals. Running fast, then slow, then repeating. Good for building speed."
```

---

### "Intervals. Running fast, then slow, then repeating."

**Original Korean:** "인터벌. 빨리 뛰고 천천히 뛰고 반복."

**Interval Training Vocabulary:**

| Term | English | Korean |
|------|---------|--------|
| interval | 인터벌 | 반복 구간 |
| rep | 반복 | 한 번의 반복 |
| set | 세트 | 여러 반복의 그룹 |
| recovery period | 회복 시간 | 천천히 달리는 구간 |
| fast segment | 빠른 구간 | 고강도 달리기 |

**Example Interval Workout:**

```
"Do 5 reps of 2 minutes fast, 2 minutes slow."
= "Run fast for 2 min, then slow for 2 min. Repeat this 5 times."
```

---

**Original Korean:** "템포 런. 레이스 페이스로 꾸준히 뛰는 거. 실전 연습."

## 3.2 Weekly Schedule Planning

### "Monday long run, Wednesday intervals, Friday tempo run."

**Original Korean:** "월요일은 장거리 달리기, 수요일은 인터벌, 금요일은 템포 런."

**Days of the Week + Activity Pattern:**

```
Monday:    Long run (10-12km, slow)
Tuesday:   Rest or easy jog
Wednesday: Intervals (fast-slow repetition)
Thursday:  Morning run to school 7km
Friday:    Tempo run (race pace)
Saturday:  Run with Minseow
Sunday:    Rest
```

**Planning Conversation Pattern:**

| Question | Answer Structure |
|----------|-----------------|
| "How should I organize this?" | [Day] + [Activity] + [Details] |
| "What should I do on Monday?" | "Monday long run (10-12km, slow)" |
| "When's my hard day?" | "Wednesday intervals, Friday tempo run" |

**"The rest, light jogging or rest":**

This means the remaining days for easier recovery activities.

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "But you can't [action]. [Reason]."

**Structure:** But + can't + verb + reason

**From Text:** "But you can't overdo it. Injury is the scariest thing."

**This Pattern Conveys:**
- Caution/warning
- Coach's experience
- Importance of the rule

**More Examples:**

| Warning | Reason |
|---------|--------|
| "But you can't overdo it." | "Injury is the scariest thing." |
| "But you can't skip rest days." | "Your muscles need to recover." |
| "But you can't run the same pace every day." | "You'll get bored and lose motivation." |
| "But you can't compare yourself to others." | "Everyone has their own pace." |

---

## Pattern 2: "How [many/often/should I]...?"

**Structure:** How + question word + verb?

**From Text:**
- "How should I prepare for the 10km race?"
- "How should I organize this?"
- "How many days a week are you running?"

**This Pattern Shows:**
- Seeking specific advice
- Willingness to learn
- Respect for coach's expertise

**Practice Variations:**

```
"How often should I train?"
"How many kilometers should I run?"
"How should I structure my week?"
"How can I improve my speed?"
"How much rest do I need?"
```

---

## Pattern 3: "There are [number] kinds. First, [X]. Second, [Y]. Third, [Z]."

**Structure:** Numbered list with "First," "Second," "Third"

**From Text:**
```
"There are three kinds. First, long runs. Running slow over a long distance. Good for building endurance.
Second, intervals. Running fast, then slow, then repeating. Good for building speed.
Third, tempo runs. Running at race pace steadily. Real-world practice."
```

**Why This Pattern Works:**
- Clear organization
- Easy to follow
- Good for teaching
- Memorable structure

**This is Educational/Instructional Language:**

| Context | Pattern | Example |
|---------|---------|---------|
| Teaching | "There are 3 types..." | Coach explaining training |
| Explaining | "First... Second... Third..." | Breaking down concepts |
| Instructing | Each item + explanation | Benefits of each type |

**Practice Your Own:**

```
"There are four things to consider. First, [rest].
Second, [nutrition]. Third, [pacing]. Fourth, [mindset]."
```

---

## Pattern 4: "[Name]. [Action]. [Benefit]."

**Structure:** Title + Description + Purpose

**From Coach's Teaching Method:**

| Title | Action | Benefit |
|-------|--------|---------|
| "Long runs" | "Running slow over a long distance" | "Good for building endurance" |
| "Intervals" | "Running fast, then slow, then repeating" | "Good for building speed" |
| "Tempo runs" | "Running at race pace steadily" | "Real-world practice" |

**This is Effective Teaching:**
- Concrete (what to do)
- Clear (why to do it)
- Connected (how it helps)

---

## Pattern 5: "Yeah, but... [Counterpoint]."

**Structure:** Agreement + but + clarification/caution

**From Text:**

| Doyun Says | Coach Responds |
|-----------|----------------|
| "Almost every day." | "Can't do every day. You need rest days." |

**Coach acknowledges the effort, then corrects:**

```
"Yeah, but that's too much."
"Yeah, but you need to be smarter about it."
"Yeah, but quality over quantity."
```

**This Pattern in Mentoring:**
- Validates effort
- Provides better guidance
- Shows expertise
- Protects the student

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Asking Coach for Training Advice

```
A: Coach, I have a question.
B: Yeah, what is it?
A: How should I prepare for the 10km race?
B: Good question. What's your current time?
A: About 55 minutes.
B: That's decent. With a month, you could get under 50 minutes.
   But you can't overdo it. Injury is the scariest thing.
A: How often should I train?
B: 4-5 days a week. The rest, just rest or light jogging.
A: What kinds of training?
B: Three kinds. First, long runs for endurance. Second, intervals
   for speed. Third, tempo runs for race practice.
A: Got it. So how should I organize this?
B: Monday long run, Wednesday intervals, Friday tempo run.
   Let me show you.
```

### Scenario 2: Discussing Training Progress

```
A: I've been running almost every day.
B: Can't do every day. You need rest days.
A: Why? The more I run, the better, right?
B: Not really. Your muscles need time to recover. You'll get injured.
A: Oh, I didn't think of that.
B: Most people don't. Quality over quantity. That's the key.
A: So how many days?
B: 4-5 is fine. Mix it up: long runs, intervals, tempo runs.
A: That makes sense. I'm ready to train properly now.
B: I like your attitude. You're taking this seriously.
```

### Scenario 3: Talking About Training Types

```
A: What's the difference between these training types?
B: Good question. Long runs build endurance. You run slow, long distance.
A: For how long?
B: 10-12km at a slow pace. Then we do intervals for speed.
A: What are intervals?
B: Running fast, then slow, then repeating. Good for building speed.
A: And the third one?
B: Tempo runs. You run at race pace steadily. Real-world practice.
A: So I need to do all three?
B: Yes. Each one helps different aspects of your fitness.
A: Got it. I'll organize my training like you said.
```
