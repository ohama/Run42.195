## PART 1: EVERYDAY EXPRESSIONS (일상 표현)

### Core Expressions from the Text

| Expression | Korean | Context | Formality | Alternatives |
|---|---|---|---|---|
| "Hold on!" | 잠깐, 기다려! | Someone calls you to wait | Casual | "Wait up!" "Hold it!" "Just a second!" |
| "Getting serious, aren't you?" | 진지하게 준비하는군요. | Noticing someone is committed | Casual-Informal | "Really dedicating yourself?" "Going all in?" |
| "That's what I liked about it" | 그게 좋았어요. | Expressing fondness | Casual | "I enjoyed that" "That's what appealed to me" |
| "You're doing well" | 잘하고 있어요. | Expressing encouragement | Neutral | "You're doing great" "You're coming along well" |
| "You won't be able to later" | 나중엔 못 해요. | Warning about future limitations | Casual | "You won't be able to do it later" "Won't have the chance later" |
| "Keep at it" | 계속 해봐요. | Encouragement to persist | Casual | "Keep going" "Don't give up" "Stay with it" |

### Formality Spectrum

```
Formal:     "You appear to be making good progress"
Standard:   "You're doing well"
Casual:     "You're doing great"
Very casual: "You're crushing it"
```

### Practice Dialogue 1: Noticing Progress

**Scenario: A friend notices you've started a new hobby**

```
Person A: "Hey, I see you running by the house every morning now."
Person B: "Yeah, I started a few weeks ago. It's become a regular thing."
Person A: "Getting serious, aren't you? Did you buy new shoes?"
Person B: "Yeah, got proper running shoes this weekend. Makes a huge difference."
Person A: "That's awesome. You're doing well with it."
Person B: "Thanks. It's hard sometimes, but it feels good."
```

### Practice Dialogue 2: Workplace Recognition

**Scenario: Manager acknowledging employee's effort**

```
Manager: "I've noticed you coming in early. Starting a morning routine?"
Employee: "Yes, I run before work now. Helps me focus."
Manager: "That's great. You're doing well. Keep it up."
Employee: "Thank you. It's challenging but rewarding."
```

### Practice Dialogue 3: Intergenerational Advice

**Scenario: Grandparent sharing life wisdom**

```
Teen: "Grandpa, you told me to do this while I'm young. Why can't I do it later?"
Grandpa: "Your body changes. Things you can do easily now become harder."
Teen: "That's depressing."
Grandpa: "Not really. The memories stay with you. That's the good part."
```

### Common Mistakes

| ❌ Incorrect | ✓ Correct | Explanation |
|---|---|---|
| "You do well" | "You're doing well" | Use present continuous for ongoing action |
| "I saw you running by all time" | "I saw you running by all the time" | Use "all the time" (not "all time") |
| "Getting more serious?" | "Getting serious, aren't you?" | Use tag question for British English naturalness |
| "Run while you're young. Won't be able after." | "Run while you're young. You won't be able to later." | Complete the second clause for clarity |

## PART 4: PHRASAL VERBS (구동사)

### Phrasal Verbs in Entry 025

| Phrasal Verb | Meaning (한국어) | Literal vs Figurative | Example | Alternatives |
|---|---|---|---|---|
| **hold on** | 잠깐 기다려 | Literal = stop | "Doyun! Hold on!" | wait (up), wait for me, just a second |
| **keep going** | 계속하다 | Literal = continue moving | "Keep going" (implied in context) | keep at it, keep it up, persist, continue |
| **pass by** | ~를 지나가다 | Literal = move past | "I passed Grandfather...at the orchard" | go past, pass, walk past, run past |
| **run by** | ~를 지나 뛰다 / 설명하다 | Could be: move past while running OR explain | "I see you running by all the time" | speed past, dash past, jog past |
| **call out** | 소리쳐서 말하다 | Literal = shout | "He called out" | shout, yell, holler, cry out |
| **catch breath** | 숨을 고르다 | Literal = regain breathing | "catching my breath" | get wind back, recover breath, breathe normally |
| **start out** | 시작하다 | Literal = begin journey | "Started walking, then ended up running" | begin, commence, start off |
| **end up** | ~가 되다 | Figurative = result in | "ended up running" | become, turn out to be, wound up |
| **keep at it** | 계속하다 / 포기하지 않다 | Figurative = persist | "Keep at it. Just keep going." | persist, keep going, don't give up |

### Phrasal Verb Comparison Table

| Phrasal Verb | Meaning | Formality | Example |
|---|---|---|---|
| hold on | wait, stop | Casual | "Hold on a second" |
| wait up | wait for me | Casual-Informal | "Wait up! I'm coming!" |
| stop by | visit briefly | Casual | "I'll stop by later" |
| hold up | delay, sustain | Formal-Casual | "What's holding you up?" |

### Phrasal Verbs with GET

In "Getting serious, aren't you?" - "get" can combine with many adjectives:

```
get + adjective = to become
- Get serious = become serious
- Get tired = become tired
- Get better = improve
- Get comfortable = become comfortable
- Get excited = become excited
```

## PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

### Pattern 1: Recognizing Progress

**Formula:** [Observation] + [Duration] + [Evaluation]

```
"I see you running by all the time these days. You're doing well."
"I've noticed you [action] every [time period]. You're [positive evaluation]."
```

**Examples:**
- "I see you studying every evening. You're doing well."
- "I've noticed you helping out a lot lately. That's great."
- "You've been going to the gym regularly. It's paying off."
- "I see you practicing every day. You're improving."
- "You've been working hard on this. You're doing great."

### Pattern 2: Sharing Past Experience

**Formula:** [Past action statement] + [Context] + [Qualification] + [Positive assessment]

```
"I used to run when I was young. Ran a lot in the army. Running was training.
But running felt good. Hard, sure. But good. That's what I liked about it."
```

**Examples:**
- "I used to play tennis in college. It was intense, but I loved it. That's what I enjoyed."
- "I used to work in sales. Long hours, tough job. But rewarding. That's what made it worthwhile."
- "I used to garden with my father. Hard work, dirty. But peaceful. That's what appealed to me."

### Pattern 3: Offering Wisdom About Age/Timing

**Formula:** [Imperative: action now] + [Contrast: future limitation] + [Reason/Consequence]

```
"Run a lot while you're young. You won't be able to later. These legs, they ran when I was young.
Now even walking's hard."
```

**Examples:**
- "Travel while you're young. You won't have the energy later. Trust me."
- "Learn languages while your brain is fresh. It gets harder as you age."
- "Enjoy your strength now. One day these bones will ache."
- "Make memories while you can. Time moves faster than you think."
- "Build your foundation now. It pays off later."

### Pattern 4: Awkward But Happy Response

**Formula:** [Emotional contradiction] + [Reason for both emotions]

```
"It was embarrassing but also made me happy."
"[Situation] was embarrassing, but it made me happy / felt good."
```

**Examples:**
- "Getting praised in front of everyone was embarrassing, but it felt good."
- "The compliment was mortifying, but I appreciated it."
- "Being the center of attention was awkward, but I loved it."
- "The recognition was uncomfortable, but I was grateful."

### Pattern 5: Internal Monologue / Realization

**Formula:** [Italicized thought] + [Character action] + [Reinforcement]

```
"*So this is what running shoes are like.*
I felt stupid for running in regular sneakers before."

"*Run a lot while you're young. You won't be able to later.*
He's right. Run while I can."
```

**Examples:**
- "*This is what determination feels like.* I finally understand what everyone talks about."
- "*So this is what a real victory tastes like.* I never want to forget this feeling."
- "*This is what trust means.* He actually believed in me."

### Pattern 6: Peer Validation / Identity Shift

**Formula:** [Peer statement defining] + [Self-questioning response] + [Acceptance/Integration]

```
Minseo: "You're totally a runner now."
Doyun: "A runner? Am I really a runner?"
[Later] "Runner... It all started when I missed the bus in March. And now I've become a runner."
```

**Examples:**
- Friend: "You're an athlete now."  You: "Am I really?"
- Colleague: "You're a leader."  You: "Do they actually see me that way?"
- Teacher: "You're a scholar."  You: "Could I really be?"

## PART 7: TEEN/CASUAL SPEECH FEATURES (구어체 특징)

### Incomplete Sentences (단편적 표현)

**Pattern 1: Subject omitted in sequence**

```
"I bought running shoes this weekend.
Mom and I went to a sporting goods store. The running shoe section had so many options.
I had no idea which ones to pick."

(Subject "I" implied in all sentences after first)
```

**Pattern 2: Object dropped when understood**

```
"'First pair?' 'Yes.'"
(Object of "first pair" understood)

"'I'll take these.' 55,000 won."
(Understood: "It costs" or "They are")
```

**Pattern 3: Trailing thoughts**

```
"My feet didn't hurt. The impact felt absorbed. My toes didn't rub against anything.
The shoes fit perfectly.
*So this is what running shoes are like.*"

(Fragments show quick realization)
```

### Casual Connectors (구어 연결어)

| Connector | Korean | Usage | Example |
|---|---|---|---|
| And/then | 그리고/그러면 | Sequential actions | "I bought shoes and wore them today" |
| But | 그런데 | Contrast | "It's hard, but good" |
| So | 그래서 | Cause-effect | "I felt good, so I kept going" |
| Like | 예를 들어 / ~처럼 | Comparison | "Like my feet were floating on clouds" |
| Really/Just | 정말 / 그냥 | Emphasis/filler | "I just kept running" |

### Dialogue Patterns: Teen Speech

**Pattern 1: Short response confirmation**

```
"First pair?"
"Yes."

"Getting serious, aren't you?"
(implied: "Yeah, I suppose I am")
```

**Pattern 2: Casual explanation**

```
"Being a soldier. Running was training."
(Fragment + explanation)
```

**Pattern 3: Repetition for emphasis**

```
"Hard, sure. But good."
(Verb implied; emotion emphasized)
```

### Filler Words & Hedging (완화 표현)

| Expression | Korean | Usage | Intensity |
|---|---|---|---|
| "sure" | 물론, ~하지만 | Concession | Medium |
| "I guess" | 아마 / ~인 것 같아 | Uncertainty | Weak |

| "like" | ~처럼, 대충 | Approximation | Weak |

```

### Practice: Rewriting Formal to Casual

**Exercise: Convert formal to teen speech**

```

### Common Mistakes in Teen Speech

| ❌ Incorrect | ✓ Correct | Explanation |
