# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Decision & Commitment Expressions

### "I've decided."

**Original Korean:** "나는 결심했다"

**Analysis:**
- 완전한 결정을 표현하는 과거완료형
- 상황 설명 없이 짧게 선언 (강조 효과)

**Variations by Formality:**

| Casual | Standard | Formal |
|--------|----------|--------|
| "I've decided." | "I've made a decision." | "I have come to a resolution." |
| "I'm doing it." | "I'm going to do it." | "I have determined to proceed." |
| "I've made up my mind." | "My mind is made up." | "The decision has been finalized." |

**Commitment Level Spectrum:**

| Expression | Certainty | Meaning |
|------------|-----------|---------|
| "Maybe I will." | ★☆☆ | 할 수도 있어 |
| "I think I'll..." | ★★☆ | 아마 할 것 같아 |
| "I've decided." | ★★★★ | 결심했어 |
| "I'm definitely doing it." | ★★★★★ | 절대 할 거야 |

**Practice Dialogue:**
```
A: You're really running the race?
B: Yeah, I've decided. I'm doing it.
A: When did you decide?
B: A few days ago. The more I thought about it, the more I knew I had to do it.
A: I respect that.
```

---

### "I'm going to do it."

**Original Korean:** "나는 그걸 할 거다"

**Grammar Point:** Future commitment (미래에 대한 확실한 의지)

**"Going to" vs. "Will":**

| Going to | Will | Context |
|----------|------|---------|
| 이미 계획/결정함 | 즉흥적 결정 | "I'm going to run." vs. "I'll help." |
| 증거/징후 있음 | 단순 미래 | "It's going to rain." vs. "It will rain." |
| 개인적 계획 | 약속 | "I'm going to apply." vs. "I promise I will." |

**Examples with Different Intensities:**

| Sentence | Intensity |
|----------|-----------|
| "I might do it." | ★☆☆ |
| "I could do it." | ★★☆ |
| "I think I'll do it." | ★★☆ |
| "I'm going to do it." | ★★★★ |
| "I'm doing it, no matter what." | ★★★★★ |

## 1.2 Hesitation & Inner Dialogue

### "Should I? Shouldn't I?"

**Original Korean:** "해야 하나? 아니 말아야 하나?"

**Structure:** Fragment questions showing internal debate

**Meaning:** 불안한 마음으로 같은 질문을 반복

**Mental Debate Patterns:**

| Pattern | Meaning | Example |
|---------|---------|---------|
| "Should I?" | 해야 할까? | Should I sign up? |
| "Shouldn't I?" | 아니 말아야 할까? | Shouldn't I just quit? |
| "What if...?" | 만약 ~하면? | What if I fail? |
| "But what if...?" | 그런데 만약? | But what if it rains? |

**Related Hesitation Expressions:**

```
"To go or not to go."
"Do it or don't do it."
"Take the risk or play it safe."
"I can't decide."
"Part of me wants to, but..."
```

**Dialogue Practice:**
```
A: Are you signing up for the race?
B: I don't know. Should I? Shouldn't I?
A: What's making you hesitate?
B: I'm scared I won't be able to finish.
A: But you've been training for months.
B: Yeah... I guess you're right.
```

### "Can I really do this?"

**Original Korean:** "나 정말 할 수 있을까?"

**Grammar Point:** Can + reality check

**Self-Doubt Expressions:**

| Expression | Doubt Level |
|------------|-------------|

| "Can I really do this?" | ★★★ |
| "There's no way I can do this." | ★★★★ |
| "I don't think I can do this." | ★★★★ |

**Building Confidence Response:**


# PART 2: TEXT MESSAGE ENGLISH (문자 표현)

## 2.1 Informal/Casual Text Language

### "yo i signed up"

**Original Context:** Text message to friend Minseo

**Text Message Features:**
- No capital letters (casual)
- No punctuation
- Shortened word forms
- Emoji likely (not shown here)

**Text vs. Formal Writing:**

| Text Version | Formal Version |
|--------------|----------------|
| "yo i signed up" | "I've signed up for the race." |
| "fr? for real?" | "Seriously? You're joking?" |
| "yeah 10km" | "Yes, the 10km category." |
| "no way!!!" | "That's incredible!" |
| "lol im so nervous" | "I must admit I'm quite nervous." |

**Common Text Message Abbreviations:**

| Text | Full | Korean |
|------|------|--------|
| yo | hey | 야 (친구 호칭) |
| fr | for real | 진짜? |
| lol | laugh out loud | ㅋㅋ |
| omg | oh my god | 헐/어? |
| idk | I don't know | 몰라 |
| imo | in my opinion | 내 생각엔 |
| tbh | to be honest | 솔직히 |
| thx | thanks | 고마워 |
| k | okay | 응 |

### "no way!!! should i?"

**Original Korean:** "대박!!! 나도 할까?"

**Surprise & Question in Text:**

| Feature | Meaning | Example |
|---------|---------|---------|
| Multiple !! | 강한 흥분 | "no way!!!" |
| Abbreviations | 친밀감 | "fr? for real?" |
| All lowercase | casual | "sick thats awesome" |
| Fragment | 빠른 말투 | "yeah 10km" |

**Enthusiasm Progression in Text:**

```
Mild: "that's cool"
Moderate: "that's awesome"
High: "sick! that's awesome"
Very High: "no way!!! should i?"
```

---

### "do it do it we should do this together"

**Original Korean:** "해, 해 우리 함께 해"

**Text Persuasion Techniques:**

| Technique | Example | Effect |
|-----------|---------|--------|
| Repetition | "do it do it" | 강한 권유 |
| "we/us" | "we should do this" | 함께의 느낌 |
| Multiple reasons | 나열 | 힘 더하기 |
| Exclamation | "lol" | 가볍게 만들기 |

**Peer Pressure (Positive):**

```
Soft: "You should try it."
Moderate: "Come on, do it."
Strong: "do it do it we should do this together"
Very Strong: "I won't do it unless you do it too."
```

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "You've decided?"

**Structure:** Fragment question confirming decision

**Meaning:** 정말로 결심했니?

**Decision Confirmation Pattern:**

| Pattern | Meaning | Example |
|---------|---------|---------|
| "You've decided?" | 확인 질문 | "You've decided?" |
| "You're sure?" | 확신 물음 | "You're sure?" |
| "Really?" | 놀라움 | "Really?" |
| "You're going to do it?" | 명확한 질문 | "You're going to do it?" |

**Dialogue with Pattern:**
```
A: I want to sign up for the marathon race.
B: You've decided?
A: Yeah. I want to try it.
B: Good. That's a good idea.
```

---

## Pattern 2: "If I don't do it, I'll regret it."

**Structure:** Conditional + regret

**Meaning:** 안 하면 후회할 거야 (후회 회피 동기)

**Regret Prevention Patterns:**

| Expression | Meaning |
|------------|---------|
| "If I don't do it, I'll regret it." | 안 하면 후회 |
| "I'd regret it if I didn't try." | 시도하지 않으면 후회 |
| "I can't live with that regret." | 그 후회를 못 견디겠어 |
| "I have to at least try." | 최소한 시도해야 해 |

**Fear of Regret:**

```
"What if I don't try and spend years thinking about it?"
"I'd always wonder 'what if?'"
"I can't pass this up."
"I'd kick myself if I didn't go for it."
"I have to do this or I'll never forgive myself."
```

---

## Pattern 3: "I run 10km in 58 minutes so that's plenty of time."

**Structure:** Fact + logical conclusion

**Meaning:** 능력이 있으므로 충분해 (자신감 표현)

**Confidence Building Patterns:**

| Pattern | Effect |
|---------|--------|
| "I can [action] so [conclusion]." | 능력 기반 확신 |
| "I've [experience] so [confidence]." | 경험 기반 확신 |
| "I've been [time] so I'm ready." | 준비 기반 확신 |

**Examples:**

```
"I've trained for 6 months so I'm ready."
"I run three times a week so I'm in shape."
"I've done a 5km before so a 10km is doable."
"I have a good base so I can do this."
```

---

## Pattern 4: "But a race will be different though."

**Structure:** Concession + contrast

**Meaning:** 경기는 다를 것이다 (현실감 인식)

**Realistic Concern Pattern:**

| Stage | Expression |
|-------|------------|
| 낙관 | "I run 58 minutes." |
| 현실화 | "But a race will be different." |
| 인정 | "Lots of people. Nervous energy." |
| 준비 | "I'll find out when I do it." |

**Hedging Expressions:**

```
"In training, yes. But in a race? Different."
"Sure, I can do 10km. But under pressure? We'll see."
"That's my normal pace. But race day will be different."
"My training times don't guarantee race results."
```
