# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Expressing Nervousness & Worry

### "The nerves are kicking in."

**Original Korean:** "긴장이 몰려오다"

**Analysis:**
- Vivid, active expression using "kick in"
- Implies growing, building anxiety
- More dynamic than "I'm nervous"

**Variations by Intensity:**

| Expression | Intensity | Korean |
|-----------|-----------|--------|
| "I'm a little nervous." | ★☆☆ | 조금 긴장돼 |
| "The nerves are kicking in." | ★★★ | 긴장이 몰려오다 |
| "I'm freaking out." | ★★★★ | 완전히 패닉 중이야 |
| "I'm terrified." | ★★★★★ | 정말 무서워 |

**Similar Expressions:**
- "Nervousness is setting in."
- "I'm starting to feel nervous."
- "The anxiety is building."
- "I'm getting the jitters." (불안감)

**Practice Dialogue:**
```
A: How are you feeling about the race?
B: The nerves are kicking in, to be honest.
A: That's completely normal.
B: Is it? I feel like I'm going crazy.
```

### "Can I actually run 21km?"

**Original Korean:** "내가 정말 21km를 뛸 수 있을까?"

**Grammar Point:** Question expressing self-doubt

**Structure:** Can I + [verb] + [challenge]? (implies doubt)

**Self-Doubt Spectrum:**

| Question | Doubt Level | Tone |
|----------|-----------|------|
| "Can I do this?" | ★★☆ | Uncertain |
| "Can I actually run 21km?" | ★★★ | Significant doubt |
| "There's no way I can do this." | ★★★★ | Strong doubt |
| "I'll never be able to do this." | ★★★★★ | Despair |

**Variations:**
- "Is there any way I can run 21km?"
- "Do you think I can actually do 21km?"
- "Will I be able to finish 21km?"

**Note:** The question mark at the end conveys internal struggle, not seeking information.

---

### "But 21km? That's a first."

**Original Korean:** "하지만 21km는 처음이야."

**Grammar Point:** Fragment + emphasis for impact

**Structure:** [Noun/phrase]? That's a first.

**Meaning:** "I've never done this before / This is unprecedented for me"

**Examples:**
| Situation | Expression |
|-----------|------------|
| new experience | "Public speaking? That's a first for me." |
| challenging distance | "Running 21km? That's a first." |
| unusual event | "A movie date with my crush? That's a first!" |

**Variations:**
- "21km is new territory for me."
- "I've never run that far before."
- "That's uncharted territory."

### "You nervous?"

**Original Korean:** "긴장됐어?"

**Grammar Point:** Sentence fragment for directness

**Uses:** Quick check-in with friends/coaches

**Response Options:**

| "Kind of." | 좀 | Casual |
| "Not really." | 별로 | Downplaying |

**Follow-up Patterns:**
```

```

### "It's different this time."

**Original Korean:** "이번엔 달라."

**Grammar Point:** Simple statement emphasizing change

**Meaning:** "This situation is unique/more challenging"

**Escalation Pattern:**

| Comparison | Context | Expression |
|-----------|---------|------------|
| similar to before | past experience was same | "It's similar to last time." |
| slightly different | some changes | "It's a bit different." |
| significantly different | major changes | "It's different this time." |
| completely different | unprecedented | "It's nothing like before." |

**Examples:**
```
"The 10km race? I did that. But 21km? It's different this time."
"I've given presentations before, but this one's different. It's live-streamed."
"Previous jobs were fine, but this role's different. Way more responsibility."
```

### "Because the guy who was dreading the bus back in March is running 17km now."

**Original Korean:** "3월에 버스 타기를 미워하던 아이가 지금 17km를 뛰고 있잖아."

**Grammar Point:** Relative clause showing transformation

**Structure:** [Person description] + is + [current action]

**Uses:** Pointing out growth/progress as proof

**Key Word: "dreading" (거절하다, 두려워하다)**

**Alternatives:**
- "hated"
- "was afraid of"
- "couldn't stand"

**Similar Proof Statements:**
| Proof Type | Example |
|-----------|---------|
| Past behavior | "The you from last month wouldn't recognize yourself now." |
| Achievement | "You already completed 17km. This is just 4km more." |
| Consistency | "You've shown up every single day for training." |
| Progress | "Look how far you've come." |

### "You've kept going this far. You can do it this time too."

**Original Korean:** "여기까지 와봤잖아. 이번에도 할 수 있어."

**Grammar Point:** Two-part encouragement (past proof + future confidence)

**Structure:**
1. [Past achievement] (past perfect)
2. You can + [future action]

**Encouragement Pattern Analysis:**

**Part 1 - Evidence Building:**
- "You've kept going this far."
- "You made it through before."
- "You've trained hard."

**Part 2 - Future Confidence:**
- "You can do it this time too."
- "You can finish this."
- "You'll make it."

**Why This Works:**
- Acknowledges past effort
- Removes novelty fear (you've done hard things)
- Projects confidence forward

### "My legs weren't cooperating."

**Original Korean:** "다리가 말을 안 들었어."

**Idiom Alert:** "weren't cooperating" = 말을 안 듣다 (literally "not listening")

**Personification in English:**
- English uses humor/metaphor to describe body parts acting independently
- "My legs were refusing"
- "My body wasn't cooperating"
- "My mind said yes but my body said no"

**Physical Struggle Expressions:**

| Expression | Meaning | Tone |
|-----------|---------|------|

| "My legs were like jelly." | 다리가 후들거림 | Dramatic |

| "I felt like I was moving in slow motion." | 느린 것 같음 | Tired |

### "But you kept going."

**Structure:** But you + kept + [verb+ing]

**Usage Examples:**


### "There's your answer."

**Original Korean:** "그게 답이야."

**Grammar Point:** Simple demonstrative statement

**Uses:** Pointing to proof/evidence

**When to Use:**
- After person answers their own concern
- When evidence is clear
- To redirect focus to accomplishment

**Context:** Coach using runner's own experience to answer their doubt

### "The last 4km is going to be the toughest."

**Original Korean:** "마지막 4km이 가장 힘들 거야."

**Grammar Point:** Prediction using "going to"

**Coach Wisdom:** Being honest about challenge instead of denying it

**Honesty in Coaching:**

| Approach | Effect |
|----------|--------|

### "But as long as you don't give up then, it's fine."

**Original Korean:** "그때 포기하지 않으면 괜찮아."

**Grammar Point:** Conditional with emphasis on one factor

**Structure:** As long as + [negative condition], [result]

**Conditional Patterns in Coaching:**

| Condition | Implication |
|-----------|-----------|

**Similar Expressions:**
- "Just don't give up."

### "You can keep moving forward, even slowly."

**Original Korean:** "느리더라도 계속 나아갈 수 있어."

**Grammar Point:** Acceptance of alternative success

**Structure:** You can + [verb], even + [negative condition]

**Reframing Failure:**

# PART 2: PHYSICAL SENSATIONS & EMOTIONAL STATES (신체 감각 & 감정 표현)

## 2.1 Nervousness Manifestations

### "Everyone says I can do it. But I'm the one freaking out."

**Original Korean:** "다들 할 수 있대. 근데 난 패닉 중이야."

**Emotional Contrast Pattern:** What others say vs. What I feel

**Structure:** Everyone says [positive] + But + I'm [negative]

**"Freaking out" Definition:**
- 공포/불안으로 패닉 상태
- Losing emotional control
- Spiral of worry

**Freaking Out Spectrum:**

| Expression | Intensity | Korean |
|-----------|-----------|--------|
| "I'm worried." | ★★☆ | 걱정돼 |
| "I'm anxious." | ★★★ | 불안해 |
| "I'm freaking out." | ★★★★ | 패닉 중이야 |
| "I'm losing my mind." | ★★★★★ | 미칠 것 같아 |

**The Isolation in Anxiety:**
- Others' reassurance can't reach the anxious person
- Gap between rational knowledge ("I can do it") and emotional state ("I'm panicking")

---

### "21km scares me."

**Original Korean:** "21km가 무서워."

**Directness:** Simple, vulnerable statement

**Fear Spectrum:**

| Expression | Meaning | Tone |
|-----------|---------|------|
| "I'm unsure about 21km." | 불안해 | Measured |
| "I'm worried about 21km." | 걱정돼 | Concerned |
| "21km intimidates me." | 위협적이야 | Intimidated |
| "21km scares me." | 두려워 | Vulnerable |
| "I'm terrified of 21km." | 정말 무서워 | Terrified |

**Why "Scares" Works:**
- Admits genuine fear
- Not "I'm scared I'll fail"—the distance itself is the fear
- Childlike honesty

---

### "Back in March, even walking 7km felt impossible."

**Original Korean:** "3월엔 7km를 걸으면서도 불가능해 보였어."

**Grammar Point:** Past tense for historical contrast

**Structure:** [Time in past] + [activity] + felt impossible

**Key Word: "felt impossible" (불가능해 보였다)**

**Progress Timeline:**

| Time | Distance | Activity | Emotion |
|------|----------|----------|---------|
| March | 7km | walking | impossible |
| Now | 17km | running | accomplished |
| Future (10 days) | 21km | half-marathon | challenging |

**Felt vs. Was:**
- "It was impossible" = absolute statement
- "It felt impossible" = emotional reality (even if objectively possible)
- More vulnerable/honest phrasing

**Similar Reflections:**
```
"Back in March, I couldn't run at all. Now I'm running 21km."
"Six months ago, I hated exercise. Now it's part of my life."
"A year ago, I would never have thought I could do this."
```

---

### "Now I'm running 17km."

**Original Korean:** "지금 난 17km를 달리고 있어."

**Present Progressive:** "am running"

**Implication:** This is my current reality (not hypothetical)

**Transformation Statement:**
- Marks concrete change
- Not "I ran 17km once"—ongoing capability
- Evidence of lasting change

---

### "I was scared then too, but I did it."

**Original Korean:** "그때도 무서웠는데 해냈어."

**Grammar Point:** "scared...but...did it" pattern

**Parallelism Structure:**
- Emotion acknowledged: "I was scared"
- Action completed: "but I did it"
- Implies emotions ≠ ability to act

**Key Insight:** Fear doesn't prevent accomplishment

**Similar Patterns:**

| Pattern | Meaning |
|---------|---------|
| "I was scared, but I did it." | Fear didn't stop me |
| "I doubted myself, but I proved I could." | Doubt was wrong |
| "I thought it was impossible, but I made it." | Reality exceeded expectations |

---

### "I can do this too. Actually, I'm going to do it."

**Original Korean:** "난 할 수 있어. 아니, 해낼 거야."

**Progression of Statements:**

| Statement | Certainty | Korean |
|-----------|-----------|--------|
| "I can do this." | Possible | 할 수 있어 |
| "I'm going to do it." | Committed | 해낼 거야 |

**Shift from Possibility to Certainty:**
- First statement: intellectual possibility
- Second statement: emotional commitment
- "Going to" = decided, committed future

**Determination Progression:**

| Level | Expression |
|-------|-----------|
| Hopeful | "I can do this." |
| Determined | "I will do this." |
| Committed | "I'm going to do it." |
| Absolute | "I'll finish this race." |

# PART 3: DESCRIBING DOUBT & CONFIDENCE (의심 & 확신 묘사)

## 3.1 Thought Patterns During Anxiety

### "What my coach said. What Minseo said."

**Original Korean:** "코치가 한 말. 민서가 한 말."

- Weighing different voices

**Original Korean:** "다들 할 수 있대."

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: Expressing Doubt + Seeking Reassurance

**Structure:** [Doubt question] → [Validation response] → [Deeper doubt] → [Deeper validation]

**From Entry:**
```
Doyun: "Can I really finish 21km?"
Coach: "You ran 17km, didn't you?"
Doyun: "Yeah."
Coach: "There's your answer. You pushed through even when it was hard."
```

**Pattern Analysis:**
1. Question doubt = admission
2. Question answered with evidence
3. Doubt acknowledged but overcome with proof

**Using This Pattern:**

```
Person A: Do you think I can do this job?
Person B: You managed the last project, right?
Person A: Yeah, but this one's bigger.
Person B: And you handled the bigger challenges. You can handle this too.
```

---

## Pattern 2: "I was [emotion], but I [action]"

**Structure:** I was + [past emotion] + but + I + [accomplished action]

**From Entry:**
- "I was scared then too, but I did it."

**Meaning:** Emotions don't determine outcomes

**Practice Examples:**

| Emotion | Action | Full Sentence |
|---------|--------|---------------|
| scared | did it | "I was scared, but I did it." |
| nervous | succeeded | "I was nervous, but I succeeded." |
| unsure | finished | "I was unsure, but I finished." |
| terrified | tried | "I was terrified, but I tried." |

**Broader Life Application:**
```
"I was scared to try running. But I did it."
"I was nervous moving to a new country. But I did it."
"I was unsure about changing careers. But I did it."
```

## Pattern 3: Past Evidence for Future Confidence

**Structure:** [Past accomplishment] = [Future capability]

**From Entry:**
- "Back in March, even walking 7km felt impossible. Now I'm running 17km...I can do this too."

**Logic:**
- If I could do 7km→17km, I can do 17km→21km
- If I could overcome that, I can overcome this
- Pattern of growth suggests continued growth

**Using This Pattern:**

```
"I couldn't code three months ago. Now I'm building apps. I can learn this new framework."
"I was terrified of public speaking. I've given five presentations. I can do this speech."
"I couldn't run one mile. I ran a 10K. I can finish this half-marathon."
```

## Pattern 4: Reframing Fear as Normal

**Structure:** "Of course you [emotion]. It's your [first/important thing]."

**From Entry:**
- Coach: "Of course you are. It's your first half marathon."

**Effect:**
- Normalizes emotion
- Removes shame
- Validates concern as appropriate

**Using This Pattern:**

| Emotion | Event | Response |
|---------|-------|----------|
| nervous | first speech | "Of course. It's your first public presentation." |
| anxious | first date | "Of course. First dates are nerve-wracking." |
| scared | first day | "Of course. First days are always scary." |
| excited | promotion | "Of course. It's a big step." |

## Pattern 5: Coach Wisdom - Honesty About Challenge + Confidence in Ability

**Structure:** [Honest difficulty] + But + [confidence statement]

**From Entry:**
- "The last 4km is going to be the toughest. But as long as you don't give up then, it's fine."

**Effect:** Respect through honesty

**Why This Works:**
- Doesn't minimize challenge (dismissive)
- Doesn't exaggerate difficulty (demoralizing)
- Focuses on what matters: willpower, not pain

**Using This Pattern:**

```
"This project is going to be the hardest. But your team is capable."
"The next year will be challenging. But you've overcome obstacles before."
"That exam is going to be brutal. But if you keep studying, you'll pass."
```

## Pattern 6: Turning Worry Into Determination

**Freaking Out Definition:** Losing emotional control from anxiety

**Intensity Levels:**

## 6.3 Emotional Isolation in Anxiety

### "Everyone says I can do it. But I'm the one freaking out."

**The Paradox:**
- Rational support ≠ emotional comfort (when anxious)
- Others' confidence can't immediately penetrate personal doubt
- Knowing you can do it ≠ feeling ready

**Common Anxiety Paradox:**

| Support Received | Internal Reality |
|-----------------|-----------------|
| "You'll be fine." | "What if I'm not?" |
| "You've done it before." | "But what if this time is different?" |
| "Everyone believes in you." | "I don't believe in myself." |
| "You're ready." | "I don't feel ready." |

**This is NOT irrational:**
- Multiple emotions valid
- Support is real AND anxiety is real
- Both can coexist

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Pre-Race Conversation with Coach

```
Athlete: Coach, I'm nervous. Really nervous.
Coach: Of course you are. It's your first half marathon.
Athlete: But what if I can't finish? 21km is so far.
Coach: You ran 17km, didn't you?
Athlete: Yeah, but it was hard. Especially the last part.
Coach: Did you finish?
Athlete: Yeah, I did.
Coach: Then you know you can push through when it's hard.
Athlete: You make it sound simple.
Coach: It is simple. The only thing that matters is finishing.
    Don't give up in the last 4km, and you'll be fine.
Athlete: Okay. I can do this.
Coach: You've done the training. You're ready.
```
