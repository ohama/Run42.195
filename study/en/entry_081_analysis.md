# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Time-Related Expressions

### "Clear. Cold."

**Original Korean:** "날씨 맑음. 춥다."

**Analysis:**
- Fragmentary sentence structure for impact and pace
- Weather description using minimal words
- Typical diary entry opening style

**Weather Description Patterns:**

| Full Sentence | Fragment | Effect |
|---|---|---|
| "It was clear and cold." | "Clear. Cold." | More vivid, immediate |
| "The weather was clear." | "Clear." | Stripped down, factual |
| "It was a cold, clear day." | "Clear. Cold." | Poetic, atmospheric |

**Common Weather Fragments:**

| Fragment | Meaning | Korean |
|---|---|---|
| "Clear." | 맑다 | Perfect running weather |
| "Cold." | 춥다 | Need extra layers |
| "Rainy." | 비 온다 | Challenging conditions |
| "Humid." | 습하다 | Difficult running |
| "Windy." | 바람 분다 | Extra resistance |

**Practice Dialogue:**
```
A: What's the weather like for your run?
B: Clear. Cold.
A: Any wind?
B: Not too bad. Perfect running weather.
```

## 1.3 Advice & Caution Expressions

### "Don't push yourself too hard."

**Original Korean:** "무리하지 마."

**Grammar Point:** Negative imperative with emphasis

**Structure:** Don't + push/drive/exhaust + yourself + too + adverb

**Caution Expressions:**

| Expression | Intensity | Korean |
|---|---|---|
| "Don't push yourself." | ★★☆ | 너무 무리하지 마 |
| "Don't push too hard." | ★★★ | 너무 열심히 하지 마 |
| "Don't overdo it." | ★★★ | 무리하지 마 |
| "Take it easy." | ★☆☆ | 느긋하게 해 |
| "Pace yourself." | ★★☆ | 페이스 조절해 |
| "Don't injure yourself." | ★★★★ | 다치지 마 |
| "Rest when you need to." | ★☆☆ | 필요하면 쉬어 |

**Context-Specific Warnings:**

| Situation | Expression | Korean |
|---|---|---|
| Hard training | "Don't push too hard." | 무리하지 마 |
| Holiday week | "Take it easy." | 쉬어 |
| After injury | "Don't overdo it." | 무리하지 마 |
| Long run | "Pace yourself." | 페이스 조절해 |

**Dialogue Practice:**
```
Coach: Next week is 17km.
Runner: Should I go all out?
Coach: No, it's Christmas week. Don't push yourself too hard.
Runner: Understood. I'll take it easy.
```

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: Question-Response About Daily Activity

**Structure:**
```
A: "How many kilometers today?"
B: "[Number]km."
A: "In the morning?"
B: "Yeah."
```

**Variations for Different Sports:**

| Question | Sport | Answer |
|---|---|---|
| "How many kilometers?" | Running | "15km" |
| "How many miles?" | Cycling | "10 miles" |
| "How many sets?" | Gym | "5 sets" |
| "How long?" | Swimming | "30 minutes" |
| "How many laps?" | Track | "50 laps" |

**Response Patterns:**

| Question Type | Answer Type | Example |
|---|---|---|
| Amount | Number | "15km" |
| Time | Yes/No | "In the morning?" → "Yeah" |
| Difficulty | Opinion | "Was it hard?" → "It was fine" |
| Comparison | Comparison | "More than usual?" → "About the same" |

---

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Friend Asking About Morning Run

```
A: Hey, how did your run go this morning?
B: Pretty good. Did 15km.
A: 15? In this cold?
B: Yeah, I started before sunrise. By the time the sun came up, I was 7km in.
A: Weren't your hands freezing?
B: They did at first. I had to put on gloves after 2km.
A: How much longer until your marathon?
B: Still training. 42km is a long way off.
A: But you're getting better though, right?
B: Yeah, every week gets a little easier.
```

### Scenario 2: Coach Feedback After Run

```
Runner: Coach, I did 15km today.
Coach: Good job. Was it hard?
Runner: It was fine. Easier than the half-marathon.
Coach: Your fitness level is improving.
Runner: Thank you. When do I increase distance again?
Coach: Next week is 17km.
Runner: That's only 2km more. I can do that.
Coach: It's Christmas week, so don't push yourself too hard.
Runner: Understood. I'll be careful.
Coach: Good. Rest well between runs.
```

### Scenario 3: Diary Reflection Conversation

```
Friend: What are you doing?
You: Writing about this morning's run.
Friend: 15km again? You're crazy.
You: Not really. It felt easier this time.
Friend: How's your training going overall?
You: Good. My body has adapted to the distance.
Friend: So you'll actually complete a marathon?
You: That's the goal. But it's still a long way to 42km.
Friend: You'll make it. You're getting better every week.
You: Thanks. I hope so.
```
