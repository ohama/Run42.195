# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Time and Schedule Expressions

### "Winter break has started."

**Original Korean:** "겨울방학이 시작되었다"

**Analysis:**
- 단순히 사건을 진술하는 표현
- 더 자연스러운 변형: "Winter break is here." / "Break's finally here."

**Variations by Context:**

| Formal | Neutral | Casual |
|--------|---------|--------|
| "Winter break has begun." | "Winter break has started." | "Break's here!" |
| "The break commences." | "Break is starting." | "Finally on break!" |
| "Break is underway." | "Break has started." | "Breaktime, baby!" |

**Dialogue Practice:**
```
A: What are you doing this week?
B: Winter break has started, so I'm free!
A: Lucky! What are your plans?
B: Mostly training. Trying to build mileage.
```

---

### "Right on schedule."

**Original Korean:** "계획대로"

**Meaning:** 예정대로, 일정 대로

**"On schedule" variations:**

| Expression | Nuance |
|------------|--------|
| "Right on schedule." | 정확히 계획대로 |
| "On track." | 진행 중 |
| "According to plan." | 계획 따라 |
| "Dead on." | 정확히 |
| "Bang on." | 정확히 (British) |
| "Right on time." | 정시에 |

**Running/Training Context:**
```
"I did my morning run—right on schedule."
"The 17km long run came in at 1:47—right on schedule."
"Training plan is on track so far."
```

**Common Mistakes:**
- ❌ "I'm on the schedule." (틀림)
- ✅ "I'm on schedule." / "Right on schedule." (맞음)

## 1.2 Family Dinner Conversation

### "What are you going to do during break?"

**Original Korean:** "방학 동안 뭐 할 거야?"

**Grammar:** "Going to" for near future plans

**Asking About Plans:**

| Question | Formality | Context |
|----------|-----------|---------|
| "What are you going to do?" | Standard | 일반적 질문 |
| "What will you do?" | Slightly formal | 격식체 |
| "What are your plans?" | Casual | 친근함 |
| "What's the plan?" | Very casual | 친구 사이 |
| "Any plans?" | Very casual | 간단히 |

**Different Answers:**
```
A: What are you going to do during break?
B: "Run." (한 단어 답변 - 자조적 톤)
B: "I'm going to focus on training." (구체적)
B: "Mostly run, honestly. Maybe see some friends." (균형잡힌)
```

### "Run." / "Of course."

**Original Korean:** "달린다" / "역시"

**One-Word Answers:**

**Why they're effective:**
- 간결함 (brevity)
- 확신 표현 (confidence)
- 강조 (emphasis)
- 약간의 자조적 톤 (self-deprecating humor)

**Dialogue Context:**
```
Dad: What are you going to do during break?
Doyun: Run.
Dad: [laughs] Of course.
```

**"Of course" Nuance:**

| Expression | Meaning | Tone |
|------------|---------|------|
| "Of course." | 당연하지 | 예상함, 약간 웃김 |
| "Naturally." | 당연히 | 격식체 |
| "Obviously." | 당연히 | 당연하다는 톤 |
| "What else?" | 그 외 뭐? | 자조적 |
| "Like I'd do anything else." | 다른 거 할 리 없지 | 자조적 |

### "Are you going to run on Christmas too?"

**Original Korean:** "크리스마스도 달릴 거야?"

**"Too" / "Also" Distinctions:**

| Word | Position | Example |
|------|----------|---------|
| "too" | 문장 끝 | "Are you going to run on Christmas too?" |
| "also" | 동사 앞 | "Are you also going to run on Christmas?" |
| "even" | 강조 | "Are you going to run even on Christmas?" |

**Concern in Parent's Tone:**
- Mom is gently worried about overtraining
- "Too" emphasizes inclusivity (even special days?)
- Implies: "You're running every single day?"

**Similar Parental Concern Expressions:**
```
"You're not going to rest at all?"
"Every day, even Christmas?"
"Isn't that too much?"
"Don't you need a break?"
```


---

## Pattern 2: "I can focus on [activity] like I did [time period]"

**Structure:** I can + verb + like I did + [specific time]

**Uses:** Comparing current ability to past performance

| Example | Meaning |
|---------|---------|
| "I can focus on training like I did during summer break." | 여름처럼 훈련에 집중할 수 있다 |

```

```
