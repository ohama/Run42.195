# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Weather and Temperature Expressions

### "Clear skies. Seriously hot."

**Original Korean:** "맑음, 진짜 덥다."

**Analysis:**
- Fragment structure (불완전 문장) for immediacy
- "Seriously" = 진짜로, 정말 (intensifier)
- Alternative: "It was clear. And seriously hot."

**Variations by Formality:**
| Casual | Standard | Formal |
|--------|----------|--------|
| "Clear and hot." | "The skies were clear and it was hot." | "Clear skies with elevated temperature." |
| "Seriously hot." | "Really hot." | "Exceptionally warm conditions." |
| "Super hot." | "Very hot." | "Notably warm." |

**Similar Weather Openings:**
| Expression | Meaning |
|------------|---------|
| "Clear skies." | 맑은 하늘 |
| "Sunny." | 햇빛이 좋다 |
| "Partly cloudy." | 흐린 부분이 있다 |
| "Overcast." | 흐린 날씨 |
| "Drizzling." | 보슬비 내리다 |

**Practice Dialogue:**
```
A: How's the weather where you are?
B: Clear skies. Seriously hot.
A: Any chance of rain?
B: Not in the forecast. Nothing but sun for days.
```

### "The rainy season's taking a break."

**Original Korean:** "장마가 잠깐 쉬어가는 중이다"

**Grammar Point:** Present progressive for seasonal patterns

**"Taking a break" Nuance:**
- 일시적으로 쉬어가다 (temporary pause)
- 곧 다시 올 것 같은 뉘앙스

**Similar Seasonal Expressions:**
| Expression | Meaning |
|------------|---------|
| "The rainy season's coming." | 장마가 온다 |
| "The rainy season's ending." | 장마가 끝난다 |
| "The rainy season's taking a break." | 장마가 잠깐 쉰다 |
| "Summer's approaching." | 여름이 다가온다 |
| "It's getting warmer." | 따뜻해지고 있다 |

**Additional Weather Phrases:**
```
"The monsoon is about to hit."
"We're in that hot, humid stretch."
"The heat breaks in autumn."
"Spring turned to summer overnight."
```

---

### "Already 26 degrees."

**Original Korean:** "벌써 26도였다"

**"Already" Emphasis:**
- 이른 시간 대비 높은 온도 표현
- 놀라움/우려 담음

**Temperature Conversation Patterns:**
| Temperature | Expression | Feeling |
|------------|------------|---------|
| 10-15°C | "Chilly." | Cool |
| 15-20°C | "Pleasant." | Comfortable |
| 20-25°C | "Warm." | Nice |
| 25-30°C | "Hot." | Starting to sweat |
| 30+°C | "Really hot." / "Extremely hot." | Unbearable |

**How to Talk About Temperature (Metric):**
```
"It's 26 degrees (Celsius)."
"The temperature reached 32C."
"It dropped to 8 degrees overnight."
"We're looking at 28 or 29 today."
```

**Note:** The entry uses Celsius (metric), which is standard in Korea. English speakers in metric countries (UK, Europe, Australia) also use Celsius.

## 1.2 Physical Activity Expressions

### "The moment I started running, I was sweating."

**Original Korean:** "뛰기 시작하니까 바로 땀이 났다"

**"The moment" Pattern:**
- 즉시, 바로 (instantaneous action)
- 원인과 결과 표현

**Structure:** The moment + 주어 + 동사, 결과

**Similar Instant Action Expressions:**
| Expression | Meaning |
|------------|---------|
| "The moment I woke up..." | 일어나자마자 |
| "The moment I saw you..." | 보자마자 |
| "The moment I started..." | 시작하자마자 |
| "As soon as I..." | 하는 순간 |
| "The second I..." | ~하는 순간 (더 강한 표현) |

**Practice Examples:**
```
"The moment I stepped outside, the heat hit me."
"The moment I started exercising, my phone died."
"The moment the race began, my legs hurt."
```

- 적은 거리/시간에도 벌써 (놀라움)
- 의외로 빠른 진행

**Variations:**
| Structure | Meaning |
|-----------|---------|
| "Not even 1km in..." | 1km도 못 갔는데 |
| "Not even 5 minutes..." | 5분도 안 됐는데 |
| "Not even halfway..." | 반도 못 갔는데 |
| "Just 1km in..." | 겨우 1km 했는데 |

**Soaked/Wet Vocabulary:**
| Expression | Intensity |
|------------|-----------|
| "damp" | 약간 축축하다 |
| "wet" | 젖었다 |
| "soaked" | 흠뻑 젖었다 |
| "drenched" | 완전히 젖었다 |

**Dialogue Practice:**

```

### "Must be the heat."

**Original Korean:** "더위 때문인 것 같다"

**Speculative Language:**
- "Must be" = ~인 것 같다, 분명히 ~일 거야
- 확신은 없지만 합리적인 추측

**Speculation Expressions:**
| Expression | Certainty |
|------------|-----------|
| "It must be the heat." | 분명히 (높음) |
| "It's probably the heat." | 아마 |
| "It could be the heat." | 일 수도 있고 |
| "I think it's the heat." | 내 생각엔 |
| "Maybe it's the heat." | 혹시 모르지만 |

**Practice:**
```
"My stomach hurts."
"Must be something you ate."

"I'm so tired."
"Must be the long run yesterday."

"Why's the sky so red?"
"Must be the sunset."
```

**Original Korean:** "원래 7km까지 뛰려고 했는데, 너무 더워서 못 버티겠다"

| Pattern | Meaning |
|---------|---------|
| "I was planning... but..." | 계획했지만 |
| "I wanted... but..." | 원했지만 |
| "I was going to... but..." | 하려고 했는데 |

**"Unbearable" Intensity:**
| Expression | Intensity |
|------------|-----------|
| "uncomfortable" | ★☆☆ |
| "unpleasant" | ★★☆ |

```

# PART 2: PHYSICAL SENSATIONS (신체 감각 표현)

## 2.1 Heat and Exhaustion Effects

### "What am I going to do in summer?"

**Original Korean:** "여름엔 어떡하지"

**Rhetorical Question for Worry:**
- 실제 답을 기대하지 않음
- 내적 불안과 궁금증 표현

**Similar Worried Questions:**
| Expression | Meaning |
|------------|---------|
| "What am I going to do?" | 어떡하지? |
| "How am I going to manage?" | 어떻게 해야 해? |
| "What if I can't..." | 못하면 어떡해? |
| "How will I survive...?" | 어떻게 버티지? |

**Dialogue:**
```
A: The heat's getting worse.
B: Tell me about it. What am I going to do in summer?
A: Run early in the morning?
B: Maybe. I don't know.
```

---

### "July and August will be even hotter."

**Original Korean:** "7월, 8월은 더 덥겠지"

**Future Assumption:**
- "will be even hotter" = 더 더울 거야
- Certainty about seasonal patterns

**Comparative Heat Expressions:**
| Comparison | Expression |
|------------|------------|
| A와 B 비교 | "July is hotter than June." |
| 점진적 상승 | "It'll get hotter." |
| 최고조 | "August is the hottest month." |
| 예상 초과 | "It's even hotter than I expected." |

---

### "Can I still run then?"

**Original Korean:** "그때도 달릴 수 있을까"

**"Can I still...?" for Doubt:**
- 부정적 가능성 암시
- ~할 수 있을까? (가능성 의문)

**Similar Doubtful Questions:**
```
"Will I even be able to run?"
"Am I strong enough?"
"Is it even possible?"
"Can my body handle it?"
```

## 2.2 Elderly Advice and Dialogue

### "Run early in the morning. Before 6. That's when it's coolest."

**Original Korean:** "아침 일찍 뛰어. 6시 전에. 그때가 제일 시원해."

**Imperative Instructions:**
- Direct advice from elder
- Simple, sequential
- Fragment style reinforces action-oriented tone

**Breakdown:**
1. "Run early in the morning." - 일반 조언
2. "Before 6." - 구체적 시간
3. "That's when it's coolest." - 이유 설명

**Giving Advice Patterns:**
| Pattern | Formality |
|---------|-----------|
| "Run early." | Very casual (older to younger) |
| "You should run early." | Standard |
| "You ought to run early." | Slightly formal |
| "I'd recommend running early." | More formal |

---

### "Before sunrise is best."

**Original Korean:** "해 뜨기 전이 제일 낫지"

**Superlatives in Advice:**
- "Best" 사용으로 최고의 방법임을 강조
- Conversational ending with "-지" (한국식 문법) → "right?" (영어식)

**Sunrise/Sunset Vocabulary:**
| Expression | Meaning |
|------------|---------|
| "before sunrise" | 해 뜨기 전 |
| "at dawn" | 새벽에 |
| "early morning" | 이른 아침 |
| "golden hour" | 황금시간 (일출/일몰 후 1시간) |
| "daybreak" | 명시경 |
| "before the sun comes up" | 해 뜨기 전 |

**Dialogue:**
```
Grandpa: Run before 6. Before sunrise is best.
Doyun: How much earlier?

```
