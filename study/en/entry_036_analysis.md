# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Weather Forecast & Natural Phenomena

### "A typhoon is coming."

**Original Korean:** "태풍이 온다"

**Analysis:**
- Simple present for immediate future (앞으로 곧 일어날 일)
- 한국어의 단순한 표현과 동일하게 영어에서도 현재형 사용 가능
- 더 격식적인 표현: "A typhoon is heading our way." / "A typhoon is approaching."

**Variations by Formality:**
| Casual | Standard | Formal |
|--------|----------|--------|
| "A typhoon's coming." | "A typhoon is coming." | "A typhoon is expected to arrive." |
| "The typhoon's here." | "The typhoon has arrived." | "The typhoon has made landfall." |
| "Typhoon warning" | "There's a typhoon warning." | "A typhoon alert has been issued." |

**Practice Dialogue:**
```
A: Hey, did you hear? A typhoon is coming.
B: When?
A: Wednesday to Thursday, they say.
B: Seriously? What about the game on Saturday?
A: Should be fine by then.
```

### "The wind had picked up."

**Original Korean:** "바람이 세어졌다"

**Grammar Point:** Past perfect for background events

**"Pick up" meanings in weather:**

| Context | Meaning | Example |
|---------|---------|---------|
| 바람이 세어지다 | become stronger | "The wind had picked up." |

| 듣기 시작하다 | begin to hear/receive | "I picked up the signal." |

**Weather Progression Vocabulary:**

| Stage | Expression | Description |
|-------|------------|-------------|
| 시작 | "The wind is starting." | 바람이 시작됨 |
| 점진 | "The wind is picking up." | 점점 세어짐 |
| 강함 | "The wind is strong." | 바람이 강함 |
| 매우 강함 | "The wind is howling." | 바람이 윤윤거림 |
| 극심 | "The wind is fierce." | 바람이 심함 |

### "Expected to impact Jeju from Wednesday to Thursday."

**Original Korean:** "수요일부터 목요일까지 제주에 영향을 미칠 것 같다"

**Grammar Point:** Passive voice + time expression

**Structure:** Subject + "expected to" + verb + time frame

**Similar News Expressions:**

| Expression | Meaning | Context |
|------------|---------|---------|
| "Expected to hit" | 타격을 주다 | 태풍/돌풍 |
| "Expected to arrive" | 도착하다 | 시간/손님 |
| "Expected to impact" | 영향을 미치다 | 날씨/정책 |
| "Expected to affect" | 영향을 주다 | 일반적 상황 |

**Time Frame Expressions:**
- "from...to..." (~~부터...까지...)
- "over the next few days" (앞으로 며칠)
- "by Thursday" (목요일까지)
- "through the weekend" (주말까지)

## 1.2 Running & Training Expressions

### "I won't be able to run."

**Original Korean:** "달릴 수 없을 것 같다"

**Grammar Point:** Will + not + be able to (미래 불가능)

**Variations:**

| Expression | Meaning | Tone |
|------------|---------|------|
| "I won't be able to run." | 못 할 것 같아 | 확실 |
| "I might not be able to run." | 못 할 수도 있어 | 불확실 |
| "I can't run." | 못 달려 | 현재 사실 |
| "I don't think I can run." | 못 달릴 것 같아 | 생각 |

**Running-Related Phrases:**
- "I won't be able to train." (훈련할 수 없다)
- "I'll have to skip my run." (달리기 쉬어야겠다)
- "No running days ahead." (달릴 날이 없다)
- "I'll have to take a break." (휴식을 가져야겠다)

### "Left at 5:15 AM, ran 6 km, walked 1 km."

**Original Korean:** "오전 5시 15분에 출발했고, 6km 달렸고, 1km 걸었다"

**Analysis:** Fragmentary list style (시간, 거리, 운동 형태를 순서대로 나열)

**Why this works:**
- 훈련 기록을 정확하게 전달
- 일기 형식에서 효율적
- 사실적, 중립적 톤

**Running Diary Vocabulary:**

| Type | Expression | Example |
|------|------------|---------|
| 시작 시간 | "Left at [time]" | "Left at 5:15 AM" |
| 달린 거리 | "ran [km]" | "ran 6 km" |
| 걸은 거리 | "walked [km]" | "walked 1 km" |
| 총거리 | "Total [km]" | "Total 7 km" |
| 소요 시간 | "[time] minutes" | "45 minutes" |

**Full Entry Example:**
```
Morning Run - Monday, July 8
Departed: 5:15 AM
Ran: 6 km
Walked: 1 km
Total: 7 km
Duration: 45 minutes
Conditions: Cloudy, windy
```

# PART 2: PHYSICAL SENSATIONS (신체 감각 표현)

## 2.1 Running-Related Sensations

### "she started huffing and puffing"

**Original Korean:** "숨을 헐떡거리기 시작했다"

**Vocabulary:**
- **Huff:** 헐떡거리다 (짧고 빠르게)
- **Puff:** 팬프팬프 하다 (숨을 몰아쉬다)
- **Huffing and puffing:** 헐떡거림 (둘을 함께 사용하면 과장된 효과)

**Breathing Intensity Spectrum:**

| Expression | Intensity | Description |
|------------|-----------|-------------|
| "breathing hard" | ★★☆ | 숨이 찬 정도 |
| "out of breath" | ★★★ | 숨이 많이 찼다 |
| "huffing and puffing" | ★★★★ | 거칠게 숨 쉼 |
| "gasping for air" | ★★★★★ | 숨을 겨우 쉼 |

**Related Expressions:**
- "She was struggling to catch her breath." (숨을 고르려고 애쓰고 있었다)
- "Her breathing was heavy." (호흡이 거쳤다)
- "I could hear her breathing from across the room." (방 건너편에서도 호흡이 들렸다)

**Dialogue Practice:**
```
A: How are you doing?
B: I'm huffing and puffing. I need a break.
A: Sit down for a second. Catch your breath.
B: How much further?
A: We're almost there.
```

**Original Korean:** "총 2km. 달린 거 1.5km, 걸은 거 0.5km"

- Running portion
- Walking portion (run/walk method)

| Method | Description | Example |
|--------|-------------|---------|

| "interval training" | 인터벌 훈련 | "Ran 400m, walked 200m, repeat" |

**Training Log Language:**
```
Total: 7 km
Running: 6 km
Walking: 1 km

Or more detailed:
Distance: 7 km
Time: 45 minutes
Pace: 6:25 per km
Running intervals: 6 km
Walking intervals: 1 km
```

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "I want to do it again"

**Structure:** I want to + verb again

**From Text:** "But I want to do it again" / "Let's do it again Saturday"

**Uses:** Expressing desire to repeat a positive experience

| Example | Original Experience |
|---------|---------------------|
| "I want to do it again." | 첫 달리기를 다시 하고 싶어 |
| "Let's go again next week." | 다음 주에 또 가자 |
| "Can we run again soon?" | 곧 다시 달릴 수 있어? |

**Variations:**
- "I'd like to do it again." (다시 하고 싶어 - 더 격식)
- "Let's do it again sometime." (언젠가 또 하자 - 느슨함)
- "Can we repeat that?" (그걸 다시 해? - 직설)

---

## Pattern 2: "Weirdly satisfying"

**Structure:** [Adverb] + [emotion adjective]

**From Text:** "But weirdly satisfying."

**Uses:** Describing unexpected positive emotions

**Similar Patterns:**

| Adverb | Emotion | Full Expression |
|--------|---------|-----------------|
| strangely | satisfying | "strangely satisfying" |
| surprisingly | good | "surprisingly good" |
| oddly | relaxing | "oddly relaxing" |
| unexpectedly | fun | "unexpectedly fun" |
| kind of | nice | "kind of nice" |

**Examples:**
```
"The pain is weirdly satisfying."
"Running is unexpectedly fun."
"It's surprisingly easy once you start."
"It's oddly relaxing to just run without thinking."
```

## Pattern 3: "lol sore legs are normal"

**Structure:** Casual text/chat register

**From Text:** Text messages in the diary

**Features of Text Messaging English:**
- Lowercase (less formal)
- "lol" = laugh out loud (웃음)
- Period omission (종결 없음)

# PART 10: EXTENDED PRACTICE

## 10.1 Dialogue Practice

### Scenario 1: Running as a Friend

```
A: Hey, your legs must still be killing you from Saturday.
B: Yeah, they're so sore. But I want to do it again.
A: Really? I thought you hated running.
B: I did at first. But it was weirdly satisfying, you know?
A: Okay, so when? There's a typhoon coming Wednesday.
B: Right, but Saturday should be fine. Want to go again?
A: Absolutely. Maybe bring proper running shoes this time?
B: [laughs] Yeah, yeah. I learned that lesson.
```

### Scenario 2: Dealing with Weather Delays

```
A: So there's a typhoon coming, huh?
B: Yeah, the forecast says Wednesday to Thursday.
A: What about our run on Friday?
B: Should be fine by then. The worst should pass by Thursday.
A: Thank God. I was looking forward to it.
B: Yeah, me too. I've been thinking about it all week.
A: The fact that you're excited about running still amazes me.
B: I know, right? Who would've thought?
```

### Scenario 3: Encouraging a Beginner

```
A: I don't think I can do this. It's too hard.
B: Come on, you're doing great. Just a little more.
A: My legs are killing me.
B: I know. Grind your teeth and push through. We're almost there.
A: How much further?
B: About 500 meters. You can do it.
A: [gasping] That's it? Just 500 meters?
B: Yeah! And then we can walk. You're almost there!
```
