# PART 1: EVERYDAY EXPRESSIONS (일상 표현)

## 1.1 Weather & Time Expressions

### "Clear skies, cool breeze."

**Original Korean:** "맑은 날씨, 시원한 바람"

**Analysis:**
- 날씨를 간결하게 표현하는 방식
- 단편적인 명사 나열로 묘사 강화
- 계절 전환의 편안함 표현

**Variations by Formality:**

| Casual | Standard | Formal |
|--------|----------|--------|
| "Clear skies, cool breeze." | "The skies are clear and there's a cool breeze." | "The weather is pleasantly clear with a cool breeze." |
| "Beautiful day." | "It's a beautiful day." | "What a lovely day it is." |
| "Perfect weather." | "The weather is perfect." | "The weather conditions are ideal." |

**Seasonal Transition Expressions:**

| Season | Expression | Feeling |
|--------|------------|---------|
| Spring | "It's getting warmer." | 따뜻해지고 있어 |
| Summer | "It's getting hot." | 더워지고 있어 |
| Fall | "It's getting cooler." | 시원해지고 있어 |
| Winter | "It's getting cold." | 추워지고 있어 |

**Practice Dialogue:**
```
A: How's the weather today?
B: Clear skies, cool breeze. Perfect running weather.
A: Great day to be outside.
B: Absolutely. That's why I ran 7km this morning.
```

### "It was definitely cooler than August because it was September now."

**Original Korean:** "9월이라서 그런지 8월보다 확실히 시원했다"

**Grammar Point:** Causal connection with "because"

**Structure:** It was [adjective] because it was [time period]

**Examples:**

| Situation | Cause | Full Sentence |
|-----------|-------|---------------|
| 시원함 | 9월 | "It was cooler because it was September." |
| 바쁨 | 월요일 | "It was busy because it was Monday." |
| 조용함 | 일요일 | "It was quiet because it was Sunday." |
| 춥음 | 겨울 | "It was cold because it was winter." |

**Alternative Structures:**
- "Since it was September, it was definitely cooler than August."
- "Being September, the temperature had dropped noticeably compared to August."
- "The cooler weather reflected the seasonal change to September."

**Common Mistakes:**
❌ "It was cool because of it was September." (중복)
✅ "It was cool because it was September."

## 1.2 Daily Routine Expressions

### "Up at 5, left at 5:20."

**Original Korean:** "5시에 일어나, 5시 20분에 출발"

**Analysis:**
- 간결한 시간 표현
- 동사 생략으로 텍스트 속도 빠르게 함
- 아침 루틴의 효율성 강조

**Morning Time Expressions:**

| Time | Expression | Activity |
|------|------------|----------|
| 5:00 AM | "Up at 5" | 일어나다 |
| 5:20 AM | "Left at 5:20" | 출발하다 |
| 6:20 AM | "Arrived at 6:20" | 도착하다 |
| 7:00 AM | "Around 7" | ~쯤 |

**Dialogue Practice:**
```
A: When do you wake up for running?
B: Up at 5, left at 5:20.
A: That's early! Don't you sleep in on school days?
B: Not when I'm training. The morning is the best time.
```

## 3.2 Performance Improvement

**Original Korean:** "과거의 나였으면 힘들었을 거다"

**Grammar Point:** Conditional with "would have been"

**Structure:** If it were the old + [noun], it would have been + [adjective]

**Hypothetical Comparisons:**

| Old Self | New Self | Sentence |
|----------|----------|----------|
| couldn't run 2km | can run 10km | "If it were the old me, this 2km would've been torture. Now it's nothing." |

**Alternative Structures:**
- "The old me would've struggled. Now I'm fine."
- "A few months ago, this would've been impossible. Now it's easy."
- "I couldn't have done this before. Now I can."

# PART 5: CONVERSATIONAL PATTERNS (회화 패턴)

## Pattern 1: "I'll be [verb]ing there during [time]."

**Structure:** I'll be + gerund + there during + [time period]

**From Text:** "I'll be running there during PE class."

**Future Activity Expressions:**

| Expression | Example | Time Frame |
|------------|---------|------------|
| "I'll be running there" | 거기서 뛸 거야 | 가까운 미래 |
| "I'll be practicing there" | 거기서 연습할 거야 | 예정된 활동 |
| "I'll be waiting there" | 거기서 기다릴 거야 | 약속 |

**Practice Sentences:**
```
"I'll be running there during PE class."
"I'll be training there all semester."
"I'll be competing there next month."
"I'll be working there this summer."
```

---

## Pattern 2: "[Activity] used to be [negative]. It's different now."

**Structure:** [Activity] used to be + [adjective], + positive statement

**From Text:** "Long-distance running was torture... It's different now."

**Change-Over-Time Expressions:**

| Used to Be | Now | Example |
|------------|-----|---------|
| torture | enjoyable | "Running used to be torture. Now I enjoy it." |
| boring | exciting | "School was boring. Now I find it interesting." |
| scary | easy | "Exams were scary. Now they're manageable." |
| painful | natural | "Exercise was painful. Now it feels natural." |

**Alternative Structures:**
- "Running used to be torture, but now it's different."
- "I used to dread long-distance running. Not anymore."
- "Long-distance running: once torture, now just another workout."

---

## Pattern 3: "[Observation] + because [reason]."

**Structure:** [Fact/Observation] + because + [explanation]

**From Text:** "They didn't exercise much over break [because they're lazy]."

**Inference Expressions:**

| Expression | Example |
|------------|---------|
| "[Fact], I guess" | "Some kids were struggling, I guess they didn't train." |
| "[Fact] because [reason]" | "They were struggling because they didn't train." |
| "[Fact], probably because [reason]" | "They looked tired, probably because of the break." |

---
