# 스타일 가이드

## 시점과 시제
- **시점:** 1인칭 (나)
- **시제:** 과거형 회고체 ("~했다", "~였다")
- **화자:** 김도윤

## 일기 형식

### 헤더 형식
```
# 2024년 3월 11일 월요일

맑음, 아침 바람이 차가웠다.
```

### 본문 구조
1. **도입:** 오늘의 날씨나 상황
2. **전개:** 주요 사건이나 달리기 경험
3. **대화:** 누군가와 나눈 대화 (자주 포함)
4. **내면:** 느낀 감정이나 생각
5. **마무리:** 짧은 성찰이나 다음 다짐

### 일기 길이
- 평균: 800-1200자
- 중요한 날: 1500-2000자
- 짧은 날: 500-700자

## 톤과 문체

### 기본 톤
- 솔직하고 꾸밈없는 14세 소년의 목소리
- 관찰력 있고 감수성 풍부함
- 과장 없이 담담하게, 그러나 감정은 섬세하게

### 좋은 예시
```
버스에서 내리자마자 속이 울렁거렸다.
아, 또 이러네. 창문을 열 수도 없는 그 갑갑함.
땀 냄새와 향수 냄새가 뒤섞인 공기.
나는 학교 정문 앞 벤치에 주저앉아 한참을 쉬어야 했다.
```

### 피해야 할 예시
```
버스는 정말 너무너무 싫었다!!!
완전 최악이었다. 죽을 것 같았다.
```

## 대화 스타일

### 형식
```
"야, 도윤아. 너 오늘 또 걸어온 거야?"
민서가 교실에 들어서자마자 물었다.
"어. 날씨가 좋아서."
"미쳤다, 진짜. 한 시간 넘게 걸리잖아."
나는 그냥 웃었다. 설명하기 귀찮았다.
```

### 말투 특징
- **도윤:** 짧고 담백한 대답, 내면은 길게 생각
- **민서:** 활발하고 직설적, 감탄사 많음
- **엄마:** 걱정 섞인 잔소리, 따뜻함
- **할아버지:** 제주 사투리 살짝, 말수 적음
- **선생님:** 격식체, 하지만 친근함

### 제주 사투리 (할아버지)
- "혼저 옵서" (어서 오세요)
- "경 헤" (그렇게 해)
- "조심헹 강" (조심해서 가)
- *과하지 않게, 분위기 정도로만*

## 자연 묘사 가이드

### 오감 활용
- **시각:** 풍경, 색깔, 빛
- **청각:** 새소리, 바람, 발소리
- **후각:** 꽃향기, 흙냄새, 바다 냄새
- **촉각:** 바람, 온도, 땀
- **미각:** (가끔) 물맛, 감귤 맛

### 계절별 키워드

#### 봄
- 유채꽃 노란 물결, 벚꽃 흩날림
- 차갑지만 부드러운 바람
- 흙냄새, 풀 냄새
- 새벽 공기의 상쾌함

#### 여름
- 짙은 녹음, 눈부신 햇살
- 습한 공기, 땀, 매미 소리
- 소나기 후 흙냄새
- 바다 바람

#### 가을
- 억새, 코스모스, 단풍
- 청명한 하늘, 선선한 바람
- 감귤 익는 향기
- 아침 안개

#### 겨울
- 동백꽃, 귤밭의 주황빛
- 차가운 공기, 하얀 입김
- 한라산 눈
- 고요함

## 달리기 묘사 가이드

### 신체 감각
- 발이 땅을 차는 느낌
- 심장 박동, 호흡
- 근육의 움직임
- 땀의 흐름

### 예시
```
발바닥이 아스팔트를 찰 때마다 작은 충격이 무릎까지 올라왔다.
처음엔 불편했던 그 느낌이 이젠 리듬이 됐다.
심장이 쿵쿵 뛰고, 숨이 거칠어지고,
그런데 이상하게 머릿속은 점점 맑아졌다.
```

### 성장 표현
- 초반: 힘듦, 숨참, 다리 아픔
- 중반: 리듬 찾기, 편안해짐
- 후반: 자유로움, 기쁨

## 감정 묘사 가이드

### 직접 말하지 않고 보여주기
```
(피하기)
나는 정말 기뻤다.

(좋은 예)
가슴이 터질 것 같았다.
결승선을 지나는 순간, 눈물이 났다.
왜 우는지도 모르겠는데, 그냥 눈물이.
```

### 감정과 신체 연결
- 긴장: 손에 땀, 심장 두근거림
- 기쁨: 가슴 벅참, 웃음
- 지침: 다리 무거움, 눈 감김
- 뿌듯함: 어깨 펴짐, 깊은 숨

## 금지 사항
- 어른스러운 표현이나 어휘
- 과도한 비유나 화려한 수사
- 설명적 서술 ("왜냐하면 ~이기 때문이다")
- 감정의 직접적 명명 ("기뻤다", "슬펐다"만으로 끝내기)
- 인터넷 신조어 과다 사용
