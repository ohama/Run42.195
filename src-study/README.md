# English Study Guide

Comprehensive English conversation study guides based on "A Jeju Boy's Marathon Journey."

- [한국어 원작](https://ohama.github.io/Run42.195/ko/) - 104 entries complete
- [English Translation](https://ohama.github.io/Run42.195/en/) - 104/104 entries
- [English Study Guide](https://ohama.github.io/Run42.195/study/) - Conversation practice guides

## About This Study Guide

Each diary entry comes with a detailed study guide designed for Korean adult English learners (40s demographic). Learn natural English expressions through the story of a 14-year-old boy's marathon journey.

## What You'll Learn

- **Everyday Expressions** - Natural phrases for daily conversation
- **Physical Sensations** - How to describe how your body feels
- **Describing Spaces/Situations** - Setting the scene in English
- **Phrasal Verbs** - Essential verb + preposition combinations
- **Conversational Patterns** - Reusable sentence structures
- **Expressing Emotions** - From frustration to joy
- **Teen/Casual Speech** - Informal English features
- **Sentence Starters** - Beginning sentences naturally
- **Vocabulary by Category** - Organized word lists
- **Practice Exercises** - Dialogues, fill-in-the-blank, discussion questions

## How to Use

1. Read the English translation first
2. Study the companion guide for that entry
3. Practice the dialogues out loud
4. Try the writing prompts
5. Use the Quick Reference Cards for review
