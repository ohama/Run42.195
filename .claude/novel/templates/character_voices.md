### 말버릇과 특징

```yaml
verbal_tics:
  - 자주 쓰는 감탄사: "아..." / "음..."
  - 말끝 습관: "~인 것 같아" / "~거든"
  - 특정 단어 반복: "진짜" / "그니까"
  - 회피 표현: "뭐랄까..." / "그게..."

speech_rhythm:
  - 평소: 차분하고 천천히
  - 흥분할 때: 빨라지고 문장이 짧아짐
  - 긴장할 때: 더듬거림, 말끝 흐림
```

### 대화 샘플 (상황별)

**인사할 때:**
```
"안녕... 오늘 날씨 좋다."
"어, 왔어?"
```

**갈등 상황:**
```
"그건... 아닌 것 같은데."
"왜 그래야 하는 건데?"
```

**감정 표현할 때:**
```
"좀... 무서웠어. 진짜로."
"기분이 이상해. 뭔가 두근거린다고 해야 하나."
```

### 성장에 따른 음성 변화

| 시점 | 변화 |
|------|------|
| **초기** | 소극적, 짧은 문장, 자신감 없음 |
| **중반** | 조금씩 적극적, 의견 표현 시작 |
| **후반** | 확신 있는 어조, 긴 문장도 사용 |

```yaml
voice_evolution:
  - stage: "초기 (1-3개월)"
    traits:
      - 짧은 문장 위주
      - "~것 같아" 자주 사용
      - 눈치 보는 말투
    sample: "나... 걸어가 볼까? 안 되려나..."

  - stage: "중기 (4-8개월)"
    traits:
      - 문장이 길어짐
      - 의견 표현 증가
      - 유머 섞기 시작
    sample: "오늘은 좀 뛰어볼까. 다리가 근질근질해."

  - stage: "후기 (9-12개월)"
    traits:
      - 확신 있는 어조
      - 다른 사람 격려
      - 자기 철학 표현
    sample: "한 발 한 발이 쌓이면 마라톤도 되는 거야."
```
