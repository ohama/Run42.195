# 제주 소년의 마라톤 도전기 - 전체 아웃라인

## 개요
제주에 사는 중학교 1학년 김도윤이 만원 버스를 피해 걷기 시작하고, 1년 만에 마라톤 풀코스를 완주하기까지의 성장 일기

## 구조
- **형식:** 일기체
- **기간:** 2024년 3월 ~ 2025년 3월 (1년)
- **빈도:** 주 2회 (월/목)
- **총 회차:** 약 104회

---

## 제1부: 시작 (3월-5월) - 약 26회

### 테마
- 불편함에서 벗어나려는 작은 용기
- 걷기의 발견과 기쁨
- 느린 시간 속에서 찾는 자신만의 리듬

### 주요 사건
1. **새 학기, 버스의 고통** (3월 초)
   - 중학교 첫 등교, 만원 버스 경험
   - 멀미, 답답함, 불쾌한 냄새

2. **우연한 도보 통학** (3월 중순)
   - 버스 놓쳐서 걸어감
   - 1시간 30분 걸림, 하지만 상쾌함 발견

3. **걷기의 습관화** (4월)
   - 주 1-2회 걷기 시작
   - 통학길 풍경 발견 (유채꽃, 감귤밭)
   - 할아버지와의 첫 만남

4. **뛰기의 시작** (5월)
   - 늦잠 자서 뛰어봄
   - 의외로 기분 좋음
   - 민서의 관심 ("너 달리기 잘하네?")

### 감정 곡선
```
불쾌함 → 호기심 → 작은 기쁨 → 자신감 싹틈
```

---

## 제2부: 성장 (6월-8월) - 약 26회

### 테마
- 본격적인 달리기와 체력 성장
- 시련과 극복 (더위, 태풍, 슬럼프)
- 동료의 등장

### 주요 사건
1. **러닝의 시작** (6월)
   - 첫 러닝화 구매
   - 5km 논스톱 달성
   - 비 오는 날 달리기의 매력

2. **여름의 시련** (7월)
   - 폭염과의 싸움
   - 태풍으로 못 달린 며칠의 답답함
   - 체육 선생님의 관심

3. **여름방학 훈련** (8월)
   - 매일 아침 달리기
   - 10km 첫 도전과 성공
   - 민서와 함께 달리기 시작
   - "마라톤 대회"라는 것을 알게 됨

### 감정 곡선
```
열정 → 좌절 → 회복 → 성취감 → 새로운 목표
```

---

## 제3부: 도전 (9월-11월) - 약 26회

### 테마
- 구체적 목표의 설정
- 첫 대회 도전
- 한계와 돌파

### 주요 사건
1. **목표 설정** (9월)
   - 2학기 시작, 체력 변화 실감
   - 10km 대회 존재 알게 됨
   - 대회 참가 결심

2. **첫 대회** (10월)
   - 10km 대회 참가
   - 긴장, 떨림, 그리고 완주
   - 결승선의 감동
   - "풀코스도 할 수 있을까?"

3. **하프마라톤** (11월)
   - 21km 도전
   - 중간에 찾아온 슬럼프 ("왜 이렇게까지?")
   - 포기하고 싶은 순간
   - 하프 완주와 자신감

### 감정 곡선
```
설렘 → 긴장 → 감동 → 야망 → 의심 → 극복
```

---

## 제4부: 완성 (12월-3월) - 약 26회

### 테마
- 최종 목표를 향한 여정
- 1년의 회고와 성장 확인
- 완주의 기쁨과 새로운 시작

### 주요 사건
1. **겨울 훈련** (12월)
   - 추위 속 달리기의 또 다른 매력
   - 30km 롱런 성공
   - 1년 전 자신과의 비교

2. **마지막 준비** (1월)
   - 풀코스 대회 신청
   - 최종 조정 훈련
   - 가족의 응원 확인

3. **풀코스 완주** (2월 말-3월 초)
   - 42.195km 도전
   - 35km 지점의 벽
   - 완주의 순간
   - 눈물과 감동

4. **에필로그** (3월)
   - 1년 전 만원 버스의 나
   - 지금의 나
   - 새로운 목표?

### 감정 곡선
```
결의 → 불안 → 고통 → 한계 → 돌파 → 벅참 → 감사
```

## 서브플롯

### 1. 할아버지와의 관계
- 매일 지나가는 감귤밭의 할아버지
- 처음엔 인사만 → 가끔 대화 → 묵묵한 응원
- 풀코스 완주 후 찾아가서 인사

### 2. 민서와의 우정
- 같은 버스 노선 → "왜 버스 안 타?" → 함께 달리기
- 라이벌이 아닌 동료
- 서로 다른 속도이지만 같은 방향

### 3. 가족의 변화
- 엄마: 걱정 → 이해 → 응원 → 자랑
- 아빠: 무관심 → 관심 → 함께 새벽 달리기 (1회)
- 여동생: 놀림 → 부러움

### 4. 자연과의 교감
- 계절 변화를 온몸으로 느낌
- 제주의 사계절이 달리기와 함께
- 자연이 주는 위로와 에너지
