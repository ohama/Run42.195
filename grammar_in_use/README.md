# Grammar in Use - Novel Examples

Based on "English Grammar in Use" (Intermediate) by Raymond Murphy.
Examples drawn from "A Jeju Boy's Marathon Journey" (제주 소년의 마라톤 도전기).

## Structure

This directory contains grammar units matching the structure of "English Grammar in Use."
Each unit uses examples exclusively from our novel to teach grammar points.

## How to Use

Each unit file contains:
1. **Grammar explanation** (simplified from the original)
2. **Examples from the novel** (with Korean-English pairs)
3. **Practice exercises** using novel vocabulary
4. **Key sentences** for memorization

## Source

Based on: *English Grammar in Use* (5th Edition) by Raymond Murphy, Cambridge University Press.
All example sentences are original content from "A Jeju Boy's Marathon Journey."
