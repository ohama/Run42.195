# Unit 10: Present Perfect Continuous and Simple

## A. 핵심 차이 (Key Difference)

### English
Both **Present Perfect Continuous** and **Present Perfect Simple** connect past to present, but they have different focuses:
- **Continuous** emphasizes the **activity/process** (how long, still happening)
- **Simple** emphasizes the **result/completion** (how much, how many times)

### Korean
**현재완료진행형**과 **현재완료**는 모두 과거와 현재를 연결하지만 초점이 다릅니다:
- **진행형**: **활동/과정** 강조 (얼마나 오래, 계속 진행 중)
- **단순형**: **결과/완료** 강조 (얼마나 많이, 몇 번)

---

## B. 비교표 (Comparison Table)

| Aspect | Present Perfect Continuous | Present Perfect Simple |
|--------|---------------------------|----------------------|
| Focus | 활동/과정 | 결과/완료 |
| Question | How long...? | How many/much...? |
| Duration | 기간 강조 | 횟수/양 강조 |
| State | 아직 진행 중일 수 있음 | 완료된 느낌 |

### From the Original Book:
> "I'**ve been reading** your book." (activity in progress)
> 책을 읽고 있었어. (아직 안 끝남)

> "I'**ve read** your book." (finished)
> 책을 읽었어. (다 읽음)

---

## C. 소설 속 예문 비교 (Novel Examples)

### Example 1: 활동 vs 결과

**Continuous (활동 강조):**
> **"I've been walking to school."**
> 걸어서 학교에 다녀왔다. (활동 강조)

**Simple (결과/횟수 강조):**
> **"I've walked to school ten times."**
> 열 번 걸어서 학교에 갔다. (횟수)

### Example 2: 기간 vs 횟수

**Continuous (How long?):**
> **"I've been running since March."**
> 3월부터 달려왔다. (기간)

**Simple (How many times?):**
> **"I've run 7km twice this week."**
> 이번 주에 7km를 두 번 뛰었다. (횟수)

### Example 3: 과정 vs 완료

**Continuous (진행 중인 과정):**
> **"I've been preparing for the race."**
> 대회를 준비해 왔다. (과정)

**Simple (완료):**
> **"I've finished my preparation."**
> 준비를 마쳤다. (완료)

### Example 4: 변화 강조

**Continuous (계속되는 변화):**
> **"My stamina has been improving."**
> 체력이 계속 좋아지고 있다.

**Simple (변화된 상태):**
> **"My stamina has improved."**
> 체력이 좋아졌다.

### Example 5: 현재 증거

**Continuous (최근 활동의 증거):**
> **"I'm sweating. I've been running."**
> 땀이 나. 달려왔어. (최근 활동)

**Simple (결과):**
> **"I've run 42km. I finished the marathon."**
> 42km를 뛰었어. 마라톤을 완주했어. (결과)

## D. 사용 가이드 (Usage Guide)

### Continuous를 사용할 때:

| Situation | Example |
|-----------|---------|
| 얼마나 오래? (How long?) | I've been running for 6 months. |
| 계속 진행 중인 활동 | I've been training hard. |
| 최근 활동의 현재 증거 | I'm tired. I've been running. |
| 과정 강조 | I've been preparing for this. |

### Simple을 사용할 때:

| Situation | Example |
|-----------|---------|
| 몇 번? 얼마나 많이? | I've run 3 times this week. |
| 완료된 행동 | I've finished the race. |
| 결과 강조 | I've lost 5kg. |
| 경험 | I've been to Japan. |

---

## E. 동사별 사용 구분 (Verbs and Their Usage)

### 일부 동사는 Continuous가 더 자연스러움:

| Verb | Continuous (자연) | Simple (가능) |
|------|------------------|--------------|
| live | I've been living here for 10 years. | I've lived here for 10 years. |
| work | I've been working all day. | I've worked here for 5 years. |
| wait | I've been waiting for an hour. | I've waited long enough. |
| rain | It's been raining all day. | It's rained a lot this month. |
| learn | I've been learning English. | I've learned a lot. |
| study | I've been studying hard. | I've studied this book. |

### 일부 동사는 Simple만 사용:

| Verb | Simple Only | Wrong |
|------|-------------|-------|
| know | I've known him for years. | ~~I've been knowing~~ |
| have (소유) | I've had this watch for 5 years. | ~~I've been having~~ |
| believe | I've always believed in you. | ~~I've been believing~~ |
| like | I've always liked running. | ~~I've been liking~~ |
| love | I've loved her forever. | ~~I've been loving~~ |

---

## F. 같은 의미로 쓰일 때 (Interchangeable Cases)

가끔 두 형태가 비슷한 의미로 쓰일 수 있습니다:

> **"I've lived here since 2020."**
> **"I've been living here since 2020."**
> (둘 다 가능, 약간의 뉘앙스 차이)

**하지만 횟수/양을 말할 때는 반드시 Simple:**

> **"I've run 500km this year."** (O)
> ~~"I've been running 500km this year."~~ (X)

---

## G. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I've been running 10km.~~ | I've **run** 10km. | 양/거리 = Simple |
| ~~I've been finishing.~~ | I've **finished**. | 완료 = Simple |
| ~~I've been knowing him for years.~~ | I've **known** him. | know = 상태동사 |
| ~~How many times have you been running?~~ | How many times **have you run**? | 횟수 = Simple |
| ~~I've run since March.~~ | I've **been running** since March. | 기간 강조 = Continuous |

## Exercises

### A. 올바른 형태를 고르세요.

1. I (have run / have been running) for two hours. I need a break.
2. She (has walked / has been walking) to school five times this week.
3. How long (have you trained / have you been training)?
4. They (have finished / have been finishing) the race!
5. I (have known / have been knowing) him since elementary school.
6. I (have read / have been reading) three books this month.
7. He (has worked / has been working) all day. He's exhausted.
8. How many times (have you visited / have you been visiting) Jeju?

**Answers:**
1. have been running (과정)
2. has walked (횟수 - five times)
3. have you been training (기간)
4. have finished (완료)
5. have known (know = 상태동사)
6. have read (횟수 - three books)
7. has been working (과정 + 결과)
8. have you visited (횟수)

### B. Continuous vs Simple 선택

Choose the better option:

1. "I'm tired." → I (have run / have been running) all morning.
2. "How many km?" → I (have run / have been running) 10km.
3. "Since when?" → I (have walked / have been walking) since March.
4. "How often?" → I (have walked / have been walking) three times this week.
5. "Look at my progress!" → I (have lost / have been losing) 5kg.
6. "I'm still reading it." → I (have read / have been reading) this book.

**Answers:**
1. have been running (피곤함의 이유 = 과정)
2. have run (양)
3. have been walking (기간)
4. have walked (횟수)
5. have lost (결과)
6. have been reading (진행 중)

### C. 빈칸 채우기

1. I _______ (train) for six months for this race.
2. She _______ (complete) three marathons in her life.
3. How long _______ you _______ (run)?
4. We _______ (wait) here for two hours already.
5. He _______ (run) 7km twice this week.
6. I _______ (study) English for ten years.
7. They _______ (visit) five countries so far.
8. She _______ (work) hard lately.

**Answers:**
1. have been training
2. has completed
3. have ... been running
4. have been waiting
5. has run
6. have been studying (또는 have studied)
7. have visited
8. has been working

### D. 문장 완성

Complete with the appropriate form:

1. (Activity + tired) I'm exhausted because I _______.
   → have been running

2. (Result) I _______ the race. I'm a finisher now.
   → have finished

3. (How long) He _______ since 5 AM.
   → has been running

4. (How many) She _______ this route ten times.
   → has walked

5. (In progress) I _______ about it all day.
   → have been thinking

6. (Completion) They _______ all their homework.
   → have done

## Key Sentences (암기 문장)

| Continuous (과정) | Simple (결과) |
|------------------|--------------|
| I've been running since March. | I've run 500km this year. |
| (3월부터 달려왔다) | (올해 500km를 뛰었다) |
| I've been preparing for the race. | I've finished my preparation. |
| (대회를 준비해 왔다) | (준비를 마쳤다) |
| How long have you been training? | How many times have you trained? |
| (얼마나 오래 훈련했어?) | (몇 번 훈련했어?) |
| I've been reading this book. | I've read three books. |
| (이 책을 읽고 있었어) | (세 권을 읽었어) |
