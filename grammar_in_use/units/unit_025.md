# Unit 25: When I do / When I've done / When and if

## A. 문법 설명 (Grammar Explanation)

### 시간/조건 부사절에서의 시제

**When, if, after, before, until, as soon as** 등의 접속사 뒤에는 **미래 시제(will)를 쓰지 않습니다.**
대신 현재시제나 현재완료를 사용합니다.

| 주절 | 부사절 |
|------|--------|
| 미래 (will) | 현재 (do) 또는 현재완료 (have done) |

### When vs If

- **When**: 확실히 일어날 일 (~ 할 때)
- **If**: 불확실한 일 (만약 ~라면)

---

## B. 소설 속 예문 (Examples from the Novel)

### Example 1: When + Present Simple

> **"When I get to school, I'll rest on the bench."**
> 학교에 도착하면 벤치에서 쉴 거야.

- When + 현재 (get), 주절은 will

### Example 2: When + Present Perfect

> **"When I've finished the race, I'll celebrate."**
> 대회를 끝내면 축하할 거야.

- When + 현재완료 (have finished) = 완료 후에

### Example 3: If + Present Simple

> **"If it rains, I'll run anyway."**
> 비가 오면 그래도 달릴 거야.

- If + 현재 (rains), 불확실한 조건

### Example 4: As soon as + Present Simple

> **"As soon as I cross the finish line, I'll call my mom."**
> 결승선을 넘자마자 엄마한테 전화할 거야.

- As soon as + 현재시제

### Example 5: Until + Present Simple

> **"I won't stop until I finish."**
> 끝낼 때까지 멈추지 않을 거야.

- Until + 현재 (finish)

---

## C. 추가 설명 (Additional Points)

### 현재 vs 현재완료 (when 절에서)

| When + Present | When + Present Perfect |
|----------------|------------------------|
| ~할 때 (동시) | ~하고 나서 (완료 후) |
| When I **see** him, I'll say hi. | When I**'ve finished**, I'll rest. |

### 자주 쓰이는 접속사

| 접속사 | 의미 | 예문 |
|--------|------|------|
| when | ~할 때 | When I arrive, I'll call you. |
| if | 만약 ~라면 | If I'm tired, I'll walk. |
| before | ~하기 전에 | Before I run, I'll stretch. |
| after | ~한 후에 | After I finish, I'll rest. |
| until | ~할 때까지 | I'll run until I finish. |
| as soon as | ~하자마자 | As soon as I see him, I'll wave. |

### 주의: 부사절 vs 명사절

**명사절**에서는 will 사용 가능:
> "I don't know **when he will arrive**." (언제 올지 모르겠어)

**부사절**에서는 will 사용 불가:
> "**When he arrives**, I'll tell him." (그가 도착하면 말할게)

---

## D. 비교 (Comparison)

### When vs If

| When (확실) | If (불확실) |
|-------------|-------------|
| When I finish (끝내면) | If I finish (만약 끝내면) |
| When it rains (비 오면 - 확실히 올 것) | If it rains (비가 오면 - 올지 모름) |

### 소설에서의 예

> **"When Monday comes, I'll ride the bus again."**
> 월요일이 오면 또 버스를 탈 거야. (월요일은 확실히 옴)

> **"If my legs hurt, I'll slow down."**
> 다리가 아프면 속도를 줄일 거야. (아플지 모름)

## Exercises

### A. 올바른 형태 고르기

1. When I (arrive / will arrive), I'll call you.
2. If it (rains / will rain) tomorrow, I'll stay home.
3. I'll rest when I (finish / will finish) the race.
4. As soon as I (get / will get) home, I'll take a shower.
5. I won't stop until I (reach / will reach) the finish line.

**Answers:**
1. arrive
2. rains
3. finish
4. get
5. reach

### B. 문장 완성하기

1. When I _______ (see) Minseo, I'll tell him.
2. If you _______ (be) tired, take a rest.
3. After I _______ (run) 10km, I'll walk home.
4. I'll celebrate when I _______ (complete) the marathon.
5. Before the race _______ (start), I'll stretch.

**Answers:**
1. see
2. are
3. have run / run
4. have completed / complete
5. starts

### C. When vs If 선택

1. _______ Monday comes, I'll go back to school. (확실히 옴)
2. _______ it rains, we'll cancel the run. (불확실)
3. _______ I finish the race, I'll be so happy. (확실히 끝낼 것)
4. _______ I'm too tired, I'll stop. (불확실)

**Answers:**
1. When
2. If
3. When
4. If
