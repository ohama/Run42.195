# Unit 39: If I knew... / I wish I knew...

## A. 현재 사실과 반대되는 가정 (Present Unreal Conditions)

**If + 과거시제** = 현재 사실과 **반대**되는 상황을 가정

### 핵심 원리:

| 현재 사실 | 가정 (If...) | 결과 |
|-----------|--------------|------|
| I **don't** have a car. | If I **had** a car... | ...I would drive. |
| I **can't** run fast. | If I **could** run fast... | ...I would win. |
| I **am** weak. | If I **weren't** weak... | ...I could run more. |
| I **don't** know the way. | If I **knew** the way... | ...I wouldn't get lost. |

### 구조:
```
If + 과거시제, ... would/could/might + 동사원형
```

**중요:** 과거시제를 사용하지만 **현재**에 대한 이야기입니다!

---

## B. If I knew... (현재 사실의 반대)

### From the Original Book:

> "**If I knew** his number, **I would phone** him."
> 그의 번호를 알면 전화할 텐데. (지금 모른다)

> "**If we had** a bigger house, **we could** invite friends."
> 집이 더 크면 친구들을 초대할 수 있을 텐데. (집이 작다)

> "**If I didn't** have to work, **I would travel** the world."
> 일하지 않아도 되면 세계 여행을 할 텐데. (일해야 한다)

### Examples from the Novel:

> **"If I knew how to get there faster, I wouldn't take the bus."**
> 더 빨리 가는 방법을 알면 버스를 안 탈 텐데. (모른다)

> **"If I had more stamina, I could run further."**
> 체력이 더 있으면 더 멀리 뛸 수 있을 텐데. (체력이 부족하다)

> **"If the bus weren't so crowded, I wouldn't feel sick."**
> 버스가 이렇게 붐비지 않으면 속이 안 좋지 않을 텐데. (붐빈다)

> **"If I could ride at this time every day, it would be great."**
> 매일 이 시간에 탈 수 있으면 좋을 텐데. (못 탄다)

### 가정의 결과절 표현:

| 조동사 | 의미 | 예문 |
|--------|------|------|
| would | ~할 텐데 | I **would** go if I had time. |
| could | ~할 수 있을 텐데 | I **could** help if I knew how. |
| might | ~할지도 모를 텐데 | She **might** agree if you asked. |

## C. I wish I knew... (현재에 대한 소망)

**I wish + 과거시제** = 현재 상황에 대한 아쉬움/소망

### 핵심 개념:
"I wish..."는 **현재 사실과 반대**되는 상황을 바랄 때 사용합니다.

### From the Original Book:

> "**I wish I knew** what to do."
> 어떻게 해야 할지 알면 좋겠다. (모른다)

> "**I wish I didn't have** to work tomorrow."
> 내일 일하지 않아도 되면 좋겠다. (일해야 한다)

> "**I wish I could** speak French."
> 프랑스어를 할 수 있으면 좋겠다. (못한다)

### 현재 사실과 I wish 비교:

| 현재 사실 | I wish... |
|-----------|-----------|
| I'm not strong. | I **wish** I **were** strong. |
| I can't run fast. | I **wish** I **could** run fast. |
| I don't have time. | I **wish** I **had** time. |
| The bus is crowded. | I **wish** the bus **weren't** crowded. |
| I have to work. | I **wish** I **didn't have** to work. |

### Examples from the Novel:

> **"I wish I could ride at this time every day."**
> 매일 이 시간에 탈 수 있으면 좋겠다. (못 탄다)

> **"I wish I had more stamina."**
> 체력이 더 있으면 좋겠다. (없다)

> **"I wish the bus weren't so crowded."**
> 버스가 이렇게 붐비지 않으면 좋겠다. (붐빈다)

> **"I wish I knew what to do."**
> 어떻게 해야 할지 알면 좋겠다. (모른다)

---

## D. I wish I were... (가정법에서의 be동사)

I wish 뒤에서 be동사는 모든 주어에 **were**를 사용합니다 (격식체).

### From the Original Book:

> "**I wish I were** taller."
> 키가 더 컸으면 좋겠다.

> "**I wish it weren't** so cold."
> 이렇게 춥지 않으면 좋겠다.

### Examples from the Novel:

> **"I wish I were stronger."**
> 내가 더 강하면 좋겠다.

> **"I wish I weren't so weak."**
> 이렇게 약하지 않으면 좋겠다.

> **"I wish it weren't raining."**
> 비가 안 오면 좋겠다.

### 격식 vs 비격식:

| Standard (격식) | Alternative (구어체) |
|-----------------|---------------------|
| I wish I **were** | I wish I **was** |
| I wish it **weren't** | I wish it **wasn't** |
| I wish he **were** | I wish he **was** |

*격식체에서는 were, 일상 대화에서는 was도 사용*

## E. If only... (I wish보다 강한 표현)

**If only** = I wish (더 강한 감정/간절함)

### From the Original Book:

> "**If only I knew** his number!"
> 그의 번호를 알면 좋으련만!

> "**If only I could** help!"
> 도울 수 있으면 좋으련만!

### Examples from the Novel:

> **"If only I had more time!"**
> 시간이 더 있으면 좋으련만!

> **"If only I could run faster!"**
> 더 빨리 달릴 수 있으면 좋으련만!

> **"If only the bus weren't so crowded!"**
> 버스가 이렇게 붐비지 않으면 좋으련만!

### I wish vs If only 비교:

| I wish | If only |
|--------|---------|
| 아쉬움/바람 | 더 강한 아쉬움/간절함 |
| 평범한 어조 | 감정적인 어조 |
| I wish I knew. | If only I knew! |

## F. I wish vs I hope 비교

| 구분 | I wish | I hope |
|------|--------|--------|
| 시제 | + 과거시제 | + 현재시제/will |
| 가능성 | 불가능/어려운 일 | 가능한 일 |
| 의미 | 현재 사실과 반대 | 미래 가능성 |
| 감정 | 아쉬움, 후회 | 기대, 희망 |

### 비교 예문:

| I wish (불가능/반대) | I hope (가능/기대) |
|----------------------|-------------------|
| I **wish** it **weren't** raining. | I **hope** it **doesn't** rain tomorrow. |
| (지금 비가 온다 - 안 오면 좋겠다) | (내일 비가 안 오길 바란다 - 가능) |
| I **wish** I **could** run a marathon. | I **hope** I **can** finish the marathon. |
| (지금은 못 뛴다 - 뛸 수 있으면) | (완주할 수 있기를 바란다 - 가능) |
| I **wish** I **had** a car. | I **hope** I **get** a car someday. |
| (지금 없다 - 있으면 좋겠다) | (언젠가 갖기를 바란다 - 가능) |

### From the Novel:

> **"I hope I can run 10km."** (가능한 목표)
> 10km 완주할 수 있기를 바란다.

> **"I wish I could run a full marathon right now."** (지금은 불가능)
> 지금 당장 풀 마라톤을 뛸 수 있으면 좋겠다.

## G. 흔한 실수 (Common Mistakes)

| 틀린 문장 (X) | 올바른 문장 (O) | 설명 |
|---------------|-----------------|------|
| I wish I **can** run fast. | I wish I **could** run fast. | wish + 과거시제 |
| I wish I **am** taller. | I wish I **were** taller. | wish + were |
| I wish I **will have** time. | I wish I **had** time. | wish + 과거시제 |
| If I **know**, I would tell. | If I **knew**, I would tell. | if + 과거시제 |
| I wish it **isn't** raining. | I wish it **weren't** raining. | wish + 과거시제 |
| I hope I **were** rich. | I hope I **will be** rich. | hope + 현재/미래 |

### 가장 흔한 오류:

```
(X) I wish I can help you.
(O) I wish I could help you.

(X) I wish the weather is better.
(O) I wish the weather were better.

(X) If I have money, I would buy it.
(O) If I had money, I would buy it.
```

## Exercises

### A. 현재 사실을 보고 If... 문장을 만드세요.

1. I don't have a car. → If I _______ a car, I would drive to school.
2. The bus is crowded. → If the bus _______ crowded, I wouldn't feel sick.
3. I can't run fast. → If I _______ run fast, I would join the track team.
4. I don't know the way. → If I _______ the way, I wouldn't get lost.
5. It's raining. → If it _______ raining, I would run outside.
6. I have to wake up early. → If I _______ to wake up early, I would sleep more.
7. He doesn't speak English. → If he _______ English, he could travel easily.
8. We don't live near the school. → If we _______ near the school, we would walk.

**Answers:**
1. had
2. weren't
3. could
4. knew
5. weren't
6. didn't have
7. spoke
8. lived

**Answers:**

### D. 틀린 부분을 찾아 고치세요.

1. I wish I can help you.
   → _______________________

2. If I know the answer, I would tell you.
   → _______________________

3. I wish I am not so tired.
   → _______________________

4. If only I have more time!
   → _______________________

5. I hope I were stronger.
   → _______________________

**Answers:**
1. I wish I **could** help you.
2. If I **knew** the answer, I would tell you.
3. I wish I **weren't** so tired.
4. If only I **had** more time!
5. I **wish** I were stronger. (또는 I hope I **will be** stronger.)

## Summary: 현재 가정법 정리

### 구조 비교표:

| 구조 | 형태 | 의미 | 예문 |
|------|------|------|------|
| If + 과거 | If + 과거시제, would + 동사 | 조건 + 결과 | If I had time, I would go. |
| I wish | I wish + 과거시제 | 바람/아쉬움 | I wish I had time. |
| If only | If only + 과거시제 | 강한 바람 | If only I had time! |

### 공식:
```
현재 사실의 반대를 가정할 때:

If + PAST, ... would/could/might + verb
I wish + PAST
If only + PAST

예: 현재 사실: I don't have time.
→ If I had time, I would run.
→ I wish I had time.
→ If only I had time!
```

### be동사 규칙:
- 모든 주어에 **were** 사용 (격식체)
- 구어체에서는 was도 허용
