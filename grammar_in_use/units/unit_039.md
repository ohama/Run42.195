# Unit 39: If I knew... / I wish I knew...

## A. 현재 사실과 반대되는 가정

**If + 과거시제** = 현재 사실과 **반대**되는 상황을 가정

| 현재 사실 | 가정 (If...) |
|-----------|--------------|
| I **don't** have a car. | If I **had** a car... |
| I **can't** run fast. | If I **could** run fast... |
| I **am** weak. | If I **weren't** weak... |
| I **don't** know the way. | If I **knew** the way... |

---

## B. If I knew... (현재 사실의 반대)

과거시제를 사용하지만 **현재**에 대한 이야기입니다.

### Examples from the Novel:

> **"If I knew how to get there faster, I wouldn't take the bus."**
> 더 빨리 가는 방법을 알면 버스를 안 탈 텐데. (모른다)

> **"If I had more stamina, I could run further."**
> 체력이 더 있으면 더 멀리 뛸 수 있을 텐데. (체력이 부족하다)

> **"If the bus weren't so crowded, I wouldn't feel sick."**
> 버스가 이렇게 붐비지 않으면 속이 안 좋지 않을 텐데. (붐빈다)

> **"If I could ride at this time every day, it would be great."**
> 매일 이 시간에 탈 수 있으면 좋을 텐데. (못 탄다)

---

## C. I wish I knew... (현재에 대한 소망)

**I wish + 과거시제** = 현재 상황에 대한 아쉬움/소망

| 현재 사실 | I wish... |
|-----------|-----------|
| I'm not strong. | I **wish** I **were** strong. |
| I can't run fast. | I **wish** I **could** run fast. |
| I don't have time. | I **wish** I **had** time. |
| The bus is crowded. | I **wish** the bus **weren't** crowded. |

### Examples from the Novel:

> **"I wish I could ride at this time every day."**
> 매일 이 시간에 탈 수 있으면 좋겠다. (못 탄다)

> **"I wish I had more stamina."**
> 체력이 더 있으면 좋겠다. (없다)

> **"I wish the bus weren't so crowded."**
> 버스가 이렇게 붐비지 않으면 좋겠다. (붐빈다)

> **"I wish I knew what to do."**
> 어떻게 해야 할지 알면 좋겠다. (모른다)

---

## D. I wish I were... (가정법에서의 be동사)

I wish 뒤에서 be동사는 모든 주어에 **were**를 사용합니다.

> **"I wish I were stronger."**
> 내가 더 강하면 좋겠다.

> **"I wish I weren't so weak."**
> 이렇게 약하지 않으면 좋겠다.

> **"I wish it weren't raining."**
> 비가 안 오면 좋겠다.

| Standard | Alternative (구어체) |
|----------|---------------------|
| I wish I **were** | I wish I **was** |
| I wish it **weren't** | I wish it **wasn't** |

*격식체에서는 were, 일상 대화에서는 was도 사용*

---

## E. If only... (I wish보다 강한 표현)

**If only** = I wish (더 강한 감정)

> **"If only I had more time!"**
> 시간이 더 있으면 좋으련만!

> **"If only I could run faster!"**
> 더 빨리 달릴 수 있으면 좋으련만!

> **"If only the bus weren't so crowded!"**
> 버스가 이렇게 붐비지 않으면 좋으련만!

---

## F. I wish vs I hope 비교

| I wish | I hope |
|--------|--------|
| 현재 사실과 반대 | 미래 가능성 |
| 불가능/어려운 일 | 가능한 일 |
| + 과거시제 | + 현재시제 |

### 비교:
> **"I wish it weren't raining."** (지금 비가 온다 - 안 오면 좋겠다)
> **"I hope it doesn't rain tomorrow."** (내일 비가 안 오길 바란다 - 가능)

> **"I wish I could run a marathon."** (지금은 못 뛴다 - 뛸 수 있으면)
> **"I hope I can finish the marathon."** (완주할 수 있기를 바란다 - 가능)

## Exercises

### A. 현재 사실을 보고 If... 문장을 만드세요.

1. I don't have a car. → If I _______ a car, I would drive to school.
2. The bus is crowded. → If the bus _______ crowded, I wouldn't feel sick.
3. I can't run fast. → If I _______ run fast, I would join the track team.
4. I don't know the way. → If I _______ the way, I wouldn't get lost.
5. It's raining. → If it _______ raining, I would run outside.

**Answers:**
1. had
2. weren't
3. could
4. knew
5. weren't

### B. I wish... 문장으로 바꾸세요.

1. I don't have more time. → I wish _______________________.
2. I can't run a marathon. → I wish _______________________.
3. The bus is always crowded. → I wish _______________________.
4. I'm not stronger. → I wish _______________________.
5. I don't know how to train. → I wish _______________________.

**Answers:**
1. I wish I had more time.
2. I wish I could run a marathon.
3. I wish the bus weren't always crowded.
4. I wish I were stronger.
5. I wish I knew how to train.

### C. If... 또는 I wish... 중 알맞은 것을 고르세요.

1. _______ I could run faster. (소망)
2. _______ I ran faster, I would win the race. (조건)
3. _______ the weather were better! (강한 소망)
4. _______ I had known earlier. (아쉬움)

**Answers:**
1. I wish
2. If
3. If only
4. I wish

## Key Sentences (암기 문장)

| English | Korean |
|---------|--------|
| If I knew the answer, I would tell you. | 답을 알면 말해줄 텐데. |
| I wish I could run faster. | 더 빨리 달릴 수 있으면 좋겠다. |
| If I had more time, I would train harder. | 시간이 더 있으면 더 열심히 훈련할 텐데. |
| I wish the bus weren't so crowded. | 버스가 이렇게 붐비지 않으면 좋겠다. |
| If only I were stronger! | 더 강하면 좋으련만! |

## Summary: 가정법 현재

```
현재 사실의 반대를 가정할 때:

If + PAST, ... would/could/might + verb
I wish + PAST
If only + PAST

예: 현재 사실: I don't have time.
→ If I had time, I would run.
→ I wish I had time.
→ If only I had time!
```
