# Unit 38: If I do... and If I did...

## A. 두 가지 조건문 유형 비교

**Type 1 (현실적 조건):** If + 현재시제, will + 동사원형
**Type 2 (비현실적/가정 조건):** If + 과거시제, would + 동사원형

| Type | If clause | Main clause | 의미 |
|------|-----------|-------------|------|
| Type 1 | If I **miss** the bus | I **will** be late | 실제 가능성 |
| Type 2 | If I **missed** the bus | I **would** walk | 가정/상상 |

---

## B. Type 1: If + 현재시제 (Real/Possible conditions)

실제로 일어날 **가능성이 있는** 상황을 표현합니다.

### Examples from the Novel:

> **"If I walk, I'd arrive around 8:50."**
> 걸으면 8시 50분쯤 도착할 거야.

> **"If someone asked how my day was, honestly, all I can think about is the bus."**
> 누가 오늘 어땠냐고 물으면, 솔직히 버스 생각밖에 안 나.

> **"If your body hurts, rest."**
> 몸이 아프면 쉬어라.

> **"If you get injured, you'll have to skip the race."**
> 부상당하면 대회에 못 나가게 돼.

---

## C. Type 2: If + 과거시제 (Unreal/Imaginary conditions)

**지금 사실이 아닌** 상황을 가정하거나 상상합니다.

### Examples from the Novel:

> **"I wish I could ride at this time every day."**
> 매일 이 시간에 탈 수 있으면 좋을 텐데. (하지만 못 탄다)

- "could ride" = 과거형이지만 현재의 가정을 표현
- "would be" = 가정의 결과

> **"If I stopped, it would be hard to run again."**
> 멈추면 다시 뛰기 어려울 거야.

> **"If I walked for an hour every day, I would be too tired."**
> 매일 한 시간씩 걸으면 너무 피곤할 거야. (실제로는 안 걸음)

---

## D. Type 1 vs Type 2 비교

| Situation | Type 1 (Real) | Type 2 (Unreal) |
|-----------|---------------|-----------------|
| 버스 놓침 | If I **miss** the bus, I **will** be late. | If I **missed** the bus, I **would** walk. |
| 비 옴 | If it **rains**, I **won't** run. | If it **rained** every day, I **would** get depressed. |
| 운동 | If I **run** today, I **will** feel good. | If I **ran** 42km tomorrow, I **would** die. |

### Type 1 (실제 가능성)
> "**If I miss** the bus, **I'll** walk."
> 버스 놓치면 걸을 거야. (실제로 놓칠 가능성)

### Type 2 (가정/상상)
> "**If I missed** the bus every day, **I'd** have to wake up earlier."
> 매일 버스를 놓친다면 더 일찍 일어나야 할 텐데. (가정)

---

## E. If I were... (가정법에서 was 대신 were)

Type 2에서 be동사는 모든 주어에 **were**를 사용합니다 (격식체).

### Examples:

> **"If I were stronger, I wouldn't get motion sick."**
> 내가 더 튼튼했다면 멀미를 안 했을 텐데.

> **"If I were Minseo, I would just take the bus."**
> 내가 민서였다면 그냥 버스를 탔을 거야.

| Formal | Informal |
|--------|----------|
| If I **were** you | If I **was** you |
| If she **were** here | If she **was** here |

*회화에서는 was도 많이 사용하지만, 격식체에서는 were가 맞습니다.*

---

## F. Unless = If not

**Unless** = "~하지 않으면" (= if ... not)

### Examples:

> **"Unless I run, I won't make it on time."**
> = If I **don't** run, I won't make it on time.
> 뛰지 않으면 제시간에 못 갈 거야.

> **"Unless you practice, you won't improve."**
> = If you **don't** practice, you won't improve.
> 연습하지 않으면 늘지 않을 거야.

## Exercises

### A. Type 1 또는 Type 2를 사용해 빈칸을 채우세요.

1. If I _______ (miss) the bus tomorrow, I will be late.
2. If I _______ (be) taller, I would play basketball.
3. If it _______ (rain) tomorrow, we won't run outside.
4. If I _______ (have) more time, I would train every day.
5. If you _______ (run) with me, we will finish together.

**Answers:**
1. miss (Type 1 - 실제 가능성)
2. were/was (Type 2 - 가정)
3. rains (Type 1 - 실제 가능성)
4. had (Type 2 - 가정)
5. run (Type 1 - 실제 가능성)

### B. 문장을 완성하세요.

1. If I had a car, _______________________.
2. If I wake up early tomorrow, _______________________.
3. If I were you, _______________________.
4. If the weather is good, _______________________.

**Sample Answers:**
1. I would drive to school. (Type 2)
2. I will go for a run. (Type 1)
3. I would take the bus. (Type 2)
4. I will run outside. (Type 1)

### C. 다음 상황에 맞는 조건문을 쓰세요.

1. 버스를 놓칠 가능성이 있다 (Type 1)
   → If I miss the bus, I will walk to school.

2. 매일 달리기를 한다고 상상 (Type 2)
   → If I ran every day, I would get stronger.

3. 비가 올 수도 있다 (Type 1)
   → If it rains, I won't run outside.

## Summary Chart

```
Type 1 (Real Possible):
If + PRESENT, ... will/can/may + verb
→ 실제로 일어날 수 있는 상황

Type 2 (Unreal/Imaginary):
If + PAST, ... would/could/might + verb
→ 가정, 상상, 현재 사실과 반대
```
