# Unit 38: If I do... and If I did...

## A. 조건문의 두 가지 유형 (Two Types of Conditionals)

영어에는 조건을 표현하는 두 가지 주요 구조가 있습니다.

| Type | 구조 | 의미 | 사용 상황 |
|------|------|------|-----------|
| **Type 1** | If + 현재시제, will + 동사원형 | 실제 가능한 조건 | 미래에 일어날 수 있는 일 |
| **Type 2** | If + 과거시제, would + 동사원형 | 가정/상상 조건 | 현재 사실과 반대이거나 상상 |

### 형태 비교:

| Type | If clause | Main clause | 예문 |
|------|-----------|-------------|------|
| Type 1 | If I **miss** the bus | I **will** be late | 버스 놓치면 늦을 거야 |
| Type 2 | If I **missed** the bus | I **would** walk | 버스를 놓친다면 걸을 텐데 |

---

## B. Type 1: If + 현재시제 (Real/Possible Conditions)

### 핵심 개념:
**실제로 일어날 가능성이 있는** 상황에 대한 조건과 결과를 표현합니다.

### From the Original Book:
> "**If I'm late** again, my boss **will be** angry."
> 또 늦으면 상사가 화낼 거야.

> "**If it rains** tomorrow, we **won't go** to the park."
> 내일 비가 오면 공원에 안 갈 거야.

> "**If you don't hurry**, you'**ll miss** the train."
> 서두르지 않으면 기차를 놓칠 거야.

### Examples from the Novel:

> **"If I walk, I'd arrive around 8:50."**
> 걸으면 8시 50분쯤 도착할 거야.

> **"If someone asked how my day was, honestly, all I can think about is the bus."**
> 누가 오늘 어땠냐고 물으면, 솔직히 버스 생각밖에 안 나.

> **"If your body hurts, rest."**
> 몸이 아프면 쉬어라.

> **"If you get injured, you'll have to skip the race."**
> 부상당하면 대회에 못 나가게 돼.

### Type 1에서 사용 가능한 결과절:

| 결과절 형태 | 예문 | 의미 |
|-------------|------|------|
| will + 동사 | If it rains, I **will stay** home. | 확실한 결과 |
| can + 동사 | If you finish early, you **can leave**. | 가능성/허락 |
| may/might + 동사 | If he calls, I **might answer**. | 불확실한 결과 |
| 명령문 | If you see him, **tell** him I called. | 지시 |

---

## C. Type 2: If + 과거시제 (Unreal/Imaginary Conditions)

### 핵심 개념:
**지금 사실이 아닌 상황**을 가정하거나 상상할 때 사용합니다. 과거시제를 사용하지만 **현재 또는 미래**에 대한 이야기입니다.

### From the Original Book:
> "**If I knew** his number, I **would phone** him."
> 그의 번호를 알면 전화할 텐데. (지금 모른다)

> "**If I had** a car, I **would drive** to work."
> 차가 있으면 운전해서 출근할 텐데. (차가 없다)

> "**If I were** you, I **wouldn't do** that."
> 내가 너라면 그렇게 안 할 텐데.

### Examples from the Novel:

> **"I wish I could ride at this time every day."**
> 매일 이 시간에 탈 수 있으면 좋을 텐데. (하지만 못 탄다)

> **"If I stopped, it would be hard to run again."**
> 멈추면 다시 뛰기 어려울 거야.

> **"If I walked for an hour every day, I would be too tired."**
> 매일 한 시간씩 걸으면 너무 피곤할 거야. (실제로는 안 걸음)

### Type 2에서 사용 가능한 결과절:

| 결과절 형태 | 예문 | 의미 |
|-------------|------|------|
| would + 동사 | If I had time, I **would help**. | 가정의 결과 |
| could + 동사 | If I had money, I **could buy** it. | 능력/가능성 |
| might + 동사 | If I asked, she **might agree**. | 불확실한 결과 |

## D. Type 1 vs Type 2 상세 비교

### 같은 상황, 다른 의미:

| 상황 | Type 1 (Real) | Type 2 (Unreal) |
|------|---------------|-----------------|
| 버스 놓침 | If I **miss** the bus, I **will** be late. | If I **missed** the bus, I **would** walk. |
| | 버스 놓치면 늦을 거야. (가능성 있음) | 버스를 놓친다면 걸을 텐데. (상상) |
| 비 옴 | If it **rains**, I **won't** run. | If it **rained** every day, I **would** get depressed. |
| | 비가 오면 달리지 않을 거야. (가능) | 매일 비가 온다면 우울할 텐데. (가정) |
| 운동 | If I **run** today, I **will** feel good. | If I **ran** 42km tomorrow, I **would** die. |
| | 오늘 달리면 기분이 좋을 거야. (실제 계획) | 내일 42km 뛴다면 죽을 거야. (과장된 상상) |

### 화자의 의도 차이:

```
Type 1: 실제로 일어날 가능성이 있다고 생각
"If I miss the bus, I'll walk."
→ 버스를 놓칠 수도 있다고 생각함

Type 2: 상상이거나 현재 사실과 반대
"If I missed the bus every day, I'd have to wake up earlier."
→ 매일 놓치지 않으니까 가정하는 것
```

## E. If I were... (가정법에서의 be동사)

Type 2에서 be동사는 모든 주어에 **were**를 사용합니다 (격식체).

### From the Original Book:
> "**If I were** you, I'd be careful."
> 내가 너라면 조심할 거야.

> "**If I were** rich, I wouldn't have to work."
> 내가 부자라면 일할 필요가 없을 텐데.

### Examples from the Novel:

> **"If I were stronger, I wouldn't get motion sick."**
> 내가 더 튼튼했다면 멀미를 안 했을 텐데.

> **"If I were Minseo, I would just take the bus."**
> 내가 민서였다면 그냥 버스를 탔을 거야.

### 격식 vs 비격식:

| Formal (격식) | Informal (비격식) |
|---------------|-------------------|
| If I **were** you | If I **was** you |
| If she **were** here | If she **was** here |
| If it **were** possible | If it **was** possible |

*회화에서는 was도 많이 사용하지만, 문법 시험이나 격식체에서는 were가 올바릅니다.*

## F. Unless = If not

**Unless** = "~하지 않으면" (= if ... not)

### From the Original Book:
> "**Unless** you hurry, you'll be late."
> = **If** you **don't** hurry, you'll be late.
> 서두르지 않으면 늦을 거야.

> "I won't go **unless** you come with me."
> = I won't go **if** you **don't** come with me.
> 네가 같이 안 가면 나도 안 갈 거야.

### Examples from the Novel:

> **"Unless I run, I won't make it on time."**
> = If I **don't** run, I won't make it on time.
> 뛰지 않으면 제시간에 못 갈 거야.

> **"Unless you practice, you won't improve."**
> = If you **don't** practice, you won't improve.
> 연습하지 않으면 늘지 않을 거야.

### Unless vs If not 사용법:

| Unless | If not |
|--------|--------|
| 조건이 부정적일 때 | 모든 상황 |
| 더 간결함 | 더 유연함 |

**주의:** Unless는 이미 부정의 의미를 포함하므로 not을 추가하지 않습니다.
- (X) Unless you don't hurry...
- (O) Unless you hurry...

## G. 흔한 실수 (Common Mistakes)

| 틀린 문장 (X) | 올바른 문장 (O) | 설명 |
|---------------|-----------------|------|
| If I **will miss** the bus... | If I **miss** the bus... | If절에 will 사용 안 함 |
| If I **would know**... | If I **knew**... | If절에 would 사용 안 함 |
| If I **was** you... (격식) | If I **were** you... | 가정법에서 were 사용 |
| If I have time, I **would** go. | If I **had** time, I **would** go. | Type 혼합 금지 |
| Unless you **don't** hurry... | Unless you hurry... | Unless에 not 불필요 |
| **If** you call, tell him. | **If** you call, **tell** him. (O) | 명령문 결과절 가능 |

### Type 혼합 오류 (가장 흔한 실수):

```
(X) If I have money, I would buy it.
    → Type 1 조건절 + Type 2 결과절 (혼합 불가)

(O) If I have money, I will buy it. (Type 1)
(O) If I had money, I would buy it. (Type 2)
```

## Exercises

### A. Type 1 또는 Type 2를 사용해 빈칸을 채우세요.

1. If I _______ (miss) the bus tomorrow, I will be late.
2. If I _______ (be) taller, I would play basketball.
3. If it _______ (rain) tomorrow, we won't run outside.
4. If I _______ (have) more time, I would train every day.
5. If you _______ (run) with me, we will finish together.
6. If she _______ (know) the answer, she would tell you.
7. If the weather _______ (be) good, we'll go for a run.
8. If I _______ (can) fly, I would visit Jeju every week.

**Answers:**
1. miss (Type 1 - 실제 가능성)
2. were/was (Type 2 - 가정)
3. rains (Type 1 - 실제 가능성)
4. had (Type 2 - 가정)
5. run (Type 1 - 실제 가능성)
6. knew (Type 2 - 가정)
7. is (Type 1 - 실제 가능성)
8. could (Type 2 - 가정)

### B. 문장을 완성하세요.

1. If I had a car, _______________________.
2. If I wake up early tomorrow, _______________________.
3. If I were you, _______________________.
4. If the weather is good, _______________________.
5. If I could run faster, _______________________.
6. Unless you hurry, _______________________.

**Sample Answers:**
1. I would drive to school. (Type 2)
2. I will go for a run. (Type 1)
3. I would take the bus. (Type 2)
4. I will run outside. (Type 1)
5. I would join the track team. (Type 2)
6. you will miss the bus. (= If you don't hurry)

### C. 틀린 부분을 고치세요.

1. If I will see him, I'll tell him.
   → _______________________
2. If I would have time, I would help you.
   → _______________________
3. Unless you don't study, you'll fail.
   → _______________________
4. If I was you, I would rest. (격식체로)
   → _______________________
5. If I have money, I would buy it.
   → _______________________

**Answers:**
1. If I **see** him, I'll tell him.
2. If I **had** time, I would help you.
3. Unless you study, you'll fail. (또는 If you don't study)
4. If I **were** you, I would rest.
5. If I **had** money, I would buy it. (또는 If I have money, I **will** buy it.)

### D. 상황에 맞게 Type 1 또는 Type 2로 문장을 쓰세요.

1. 버스를 놓칠 가능성이 있다 (Type 1)
   → _______________________

2. 매일 달리기를 한다고 상상 (Type 2)
   → _______________________

3. 비가 올 수도 있다 (Type 1)
   → _______________________

4. 새처럼 날 수 있다면 (Type 2)
   → _______________________

**Sample Answers:**
1. If I miss the bus, I will walk to school.
2. If I ran every day, I would get stronger.
3. If it rains, I won't run outside.
4. If I could fly like a bird, I would travel the world.

## Summary: Type 1 vs Type 2 비교표

| 구분 | Type 1 (Real) | Type 2 (Unreal) |
|------|---------------|-----------------|
| If절 시제 | 현재시제 | 과거시제 |
| 결과절 | will/can/may + 동사 | would/could/might + 동사 |
| 의미 | 실제 가능한 상황 | 가정, 상상, 현재 반대 |
| be동사 | am/is/are | were (모든 주어) |
| 예문 | If it rains, I won't go. | If it rained, I wouldn't go. |
| 해석 | 비가 오면 안 갈 거야. | 비가 온다면 안 갈 텐데. |

```
Type 1 (Real Possible):
If + PRESENT, ... will/can/may + verb
→ 실제로 일어날 수 있는 상황

Type 2 (Unreal/Imaginary):
If + PAST, ... would/could/might + verb
→ 가정, 상상, 현재 사실과 반대
```
