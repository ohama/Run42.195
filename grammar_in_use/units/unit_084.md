# Unit 84: There... and it...

## A. 형태와 기본 개념 (Form and Basic Concept)

### There is/are: 존재 표현

**There is/are**는 무언가가 **존재한다**는 것을 표현합니다.

| 시제 | 단수 | 복수 |
|------|------|------|
| 현재 | There **is** a bus. | There **are** buses. |
| 과거 | There **was** a bench. | There **were** people. |
| 미래 | There **will be** a race. | There **will be** races. |
| 현재완료 | There **has been** a problem. | There **have been** problems. |

### It: 가주어/특정 대상

| It 용법 | Example |
|---------|---------|
| 날씨 | It's cold. |
| 시간 | It's 7:30. |
| 거리 | It's 7 km to school. |
| 소요 시간 | It takes 40 minutes. |
| 특정 대상 | It (= the bus) was crowded. |

---

## B. There is/are 상세 (Existence)

### From the Original Book:

> **"There's a supermarket near here."**
> 근처에 슈퍼마켓이 있다.

> **"There are some books on the table."**
> 테이블 위에 책이 몇 권 있다.

### Examples from the Novel:

> **"There was no space to move."**
> 움직일 공간이 없었다.
- There was + 단수 (no space)

> **"There were too many people on the bus."**
> 버스에 사람이 너무 많았다.
- There were + 복수 (people)

> **"There's a problem."**
> 문제가 있다.
- There's = There is

> **"There are still 7 kilometers left."**
> 아직 7킬로미터 남았다.

> **"There was a bench near the school."**
> 학교 근처에 벤치가 있었다.

---

## C. It: 특정 대상 or 가주어 (It for Reference or Dummy Subject)

**It**은 두 가지 용법이 있습니다:

### 1. 특정 대상 (Referring to something):

> **"The bus was packed. It was hard to breathe."**
> 버스가 꽉 찼다. 숨 쉬기 힘들었다.
- It = the situation (또는 문맥상 대상)

> **"I saw the school. It was beautiful."**
> 학교를 봤다. 예뻤다.
- It = the school (특정 대상)

### 2. 가주어 (Dummy subject):

| Category | Example | Meaning |
|----------|---------|---------|
| 날씨 | **It** was cold this morning. | 오늘 아침은 추웠다. |
| 시간 | **It** was 7:23. | 7시 23분이었다. |
| 거리 | **It**'s 7 km to school. | 학교까지 7km이다. |
| 소요 시간 | **It** takes 40 minutes. | 40분 걸린다. |
| 상황 | **It** was hard. | 힘들었다. |

---

## D. 소설 속 It 예문 (It from the Novel)

### Examples:

> **"It was 7:23 when I opened my eyes."**
> 눈을 떴을 때 7시 23분이었다.
- It = 시간

> **"It felt like winter."**
> 겨울 같았다.
- It = 날씨/상황

> **"It took an hour and ten minutes."**
> 한 시간 십 분 걸렸다.
- It = 소요 시간

> **"It was hard."**
> 힘들었다.
- It = 상황

> **"It's 7 kilometers from home to school."**
> 집에서 학교까지 7킬로미터다.
- It = 거리

## E. There vs It 비교 (Comparison)

| There | It |
|-------|-----|
| 존재 (있다) | 특정 대상/가주어 |
| There **is** a problem. | **It** is a problem. |
| 문제가 있다. (존재) | 그것이 문제다. (정체) |

### From the Novel:

> **"There was a bench."** (벤치가 있었다 - 존재)
> **"It was comfortable."** (그게 편했다 - It = the bench)

> **"There were flowers."** (꽃이 있었다 - 존재)
> **"They were beautiful."** (예뻤다 - They = the flowers)

### 핵심 차이:

| There | It |
|-------|-----|
| 새로운 정보 도입 | 이미 언급된 것 참조 |
| "~이 있다" | "그것은 ~이다" |
| 존재 확인 | 묘사/설명 |

---

## F. It + to부정사 / that절 (It as Dummy Subject)

### It + to부정사:

| Structure | Example | Meaning |
|-----------|---------|---------|
| It's + 형용사 + to... | **It's hard to run** in the morning. | 아침에 달리기는 힘들다. |
| It takes + 시간 + to... | **It takes time to improve**. | 발전하는 데 시간이 걸린다. |
| It's + 명사 + to... | **It's a pleasure to meet** you. | 만나서 반갑습니다. |

> **"It's hard to run in the morning."**
> 아침에 달리기는 힘들다.
- It = to run in the morning (진주어)

> **"It takes time to improve."**
> 발전하는 데 시간이 걸린다.

### It + that절:

| Structure | Example | Meaning |
|-----------|---------|---------|
| It's + 형용사 + that... | **It's true that I've changed**. | 내가 변한 건 사실이다. |
| It's + 명사 + that... | **It's a fact that running is hard**. | 달리기가 힘들다는 건 사실이다. |

> **"It's true that I've changed."**
> 내가 변한 건 사실이다.
- It = that I've changed (진주어)

## G. There + be vs have (Existence vs Possession)

| There is/are | have |
|--------------|------|
| 일반적 존재 | 소유 |
| **There is** a bench. | **I have** a bag. |
| 벤치가 있다. (일반 위치에) | 가방이 있다. (내 것) |

### From the Novel:

> **"There's a bus at 7:30."** (버스가 있다 - 일반 존재)
> **"I have a friend named Minseo."** (친구가 있다 - 소유)

> **"There were many people."** (사람이 많았다 - 존재)
> **"I have many friends."** (친구가 많다 - 소유)

## H. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | 설명 |
|-----------|-------------|------|
| **It is** many people here. | **There are** many people here. | 존재는 There |
| **There takes** 40 minutes. | **It takes** 40 minutes. | 시간은 It |
| **There** was cold. | **It** was cold. | 날씨는 It |
| **It has** a problem. (존재) | **There is** a problem. | 존재는 There |
| **There are** raining. | **It is** raining. | 날씨는 It |

### 추가 실수 예시:

| Wrong (X) | Correct (O) |
|-----------|-------------|
| **It** is a bus stop near here. | **There** is a bus stop near here. |
| **There** is 7 o'clock. | **It** is 7 o'clock. |
| **There** seems difficult. | **It** seems difficult. |
| **It were** many students. | **There were** many students. |

## Summary: There vs It

| There | It |
|-------|-----|
| 존재 표현 | 가주어 (날씨, 시간, 거리) |
| **There is** a bus. | **It is** cold. |
| **There were** people. | **It takes** time. |
| 뒤에 **명사** | 뒤에 **형용사/to부정사/that절** |

### There 구문:
- There is/are + 명사 (존재)
- There was/were + 명사 (과거 존재)
- There will be + 명사 (미래 존재)

### It 구문:
- It is + 형용사 (날씨, 상황)
- It is + 시간/거리
- It takes + 시간 (소요)
- It is + 형용사 + to... (가주어)
- It is + 형용사 + that... (가주어)

### 구분 요령:
- **"~이 있다"** → There
- **"~이다" (날씨/시간/상황)** → It
- **새로운 정보 도입** → There
- **이미 알려진 것 참조** → It (또는 대명사)
