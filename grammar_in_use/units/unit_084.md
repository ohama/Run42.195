# Unit 84: There... and it...

## A. There is/are: 존재 표현

**There is/are**는 무언가가 **존재한다**는 것을 표현합니다.

| 단수 | 복수 |
|------|------|
| There is a bus. | There are buses. |
| There was a bench. | There were people. |

---

## B. 소설 속 There is/are

### Examples from the Novel:

> **"There was no space to move."**
> 움직일 공간이 없었다.

- There was + 단수 (no space)

> **"There were too many people on the bus."**
> 버스에 사람이 너무 많았다.

- There were + 복수 (people)

> **"There's a problem."**
> 문제가 있다.

- There's = There is

> **"There are still 7 kilometers left."**
> 아직 7킬로미터 남았다.

---

## C. It: 특정 대상 or 가주어

**It**은 두 가지 용법이 있습니다:

### 1. 특정 대상:

> **"The bus was packed. It was hard to breathe."**
> 버스가 꽉 찼다. 숨 쉬기 힘들었다.

- It = the bus (특정 대상)

### 2. 가주어 (날씨, 시간, 거리, 상황):

> **"It was cold this morning."**
> 오늘 아침은 추웠다.

- It = 날씨 (가주어)

> **"It takes 40 minutes by bus."**
> 버스로 40분 걸린다.

- It = 거리/시간 (가주어)

---

## D. 소설 속 It

### Examples from the Novel:

> **"It was 7:23 when I opened my eyes."**
> 눈을 떴을 때 7시 23분이었다.

- It = 시간

> **"It felt like winter."**
> 겨울 같았다.

- It = 날씨/상황

> **"It took an hour and ten minutes."**
> 한 시간 십 분 걸렸다.

- It = 소요 시간

> **"It was hard."**
> 힘들었다.

- It = 상황

---

## E. There vs It 비교

| There | It |
|-------|-----|
| 존재 (있다) | 특정 대상/가주어 |
| There is a problem. | It is a problem. |
| 문제가 있다. | 그것이 문제다. |

### From the Novel:

> **"There was a bench."** (벤치가 있었다 - 존재)
> **"It was comfortable."** (그게 편했다 - It = the bench)

> **"There were flowers."** (꽃이 있었다 - 존재)
> **"They were beautiful."** (예뻤다 - They = the flowers)

---

## F. It + to부정사 / that절

### It + to부정사:

> **"It's hard to run in the morning."**
> 아침에 달리기는 힘들다.

- It = to run in the morning (진주어)

> **"It takes time to improve."**
> 발전하는 데 시간이 걸린다.

### It + that절:

> **"It's true that I've changed."**
> 내가 변한 건 사실이다.

- It = that I've changed (진주어)

---

## G. 비교: There + be vs have

| There is | have |
|----------|------|
| 일반적 존재 | 소유 |
| There is a bench. | I have a bag. |
| 벤치가 있다. | 가방이 있다. |

### From the Novel:

> **"There's a bus at 7:30."** (버스가 있다 - 존재)
> **"I have a friend named Minseo."** (친구가 있다 - 소유)

## Exercises

### A. There 또는 It을 넣으세요.

1. _______ was cold this morning.
2. _______ were many people on the bus.
3. _______ takes 40 minutes by bus.
4. _______ is a problem.
5. _______ was 7:23 when I woke up.

**Answers:**
1. It
2. There
3. It
4. There
5. It

### B. 올바른 형태를 고르세요.

1. (There is / It is) a bench near the school.
2. (There was / It was) hard to breathe.
3. (There takes / It takes) an hour to walk.
4. (There are / They are) many flowers along the road.

**Answers:**
1. There is
2. It was
3. It takes
4. There are

### C. 빈칸을 채우세요.

1. _______ was no space to move.
2. _______ was raining this morning.
3. _______ are 7 kilometers to school.
4. _______ felt like summer.
5. _______ is hard to run every day.

**Answers:**
1. There
2. It
3. There
4. It
5. It

## Summary

| There | It |
|-------|-----|
| 존재 표현 | 가주어 (날씨, 시간, 거리) |
| There is a bus. | It is cold. |
| There were people. | It takes time. |
| 뒤에 명사 | 뒤에 형용사/to부정사/that절 |
