# Unit 12: For and Since / When...? and How long...?

## A. 문법 설명 (Grammar Explanation)

### English
- **For** + period of time (how long)
- **Since** + point in time (when it started)
- **When...?** asks about a specific point in time
- **How long...?** asks about duration

### Korean
- **For** + 기간 (얼마나 오래)
- **Since** + 시점 (언제부터)
- **When...?** 특정 시점을 묻는다
- **How long...?** 기간을 묻는다

---

## B. For vs Since 상세 비교

### FOR + 기간 (Period of time)

| Expression | Korean |
|------------|--------|
| for two days | 이틀 동안 |
| for a week | 일주일 동안 |
| for three months | 3개월 동안 |
| for a year | 1년 동안 |
| for a long time | 오랫동안 |
| for ages | 아주 오랫동안 |

### SINCE + 시점 (Point in time)

| Expression | Korean |
|------------|--------|
| since yesterday | 어제부터 |
| since Monday | 월요일부터 |
| since March | 3월부터 |
| since 2024 | 2024년부터 |
| since 8 o'clock | 8시부터 |
| since I was young | 어렸을 때부터 |
| since I started school | 학교 다닌 이후로 |

## C. 소설 속 예문 (Examples from the Novel)

### Example 1: For + 기간

> **"Forty minutes by bus from home to school."**
> 집에서 학교까지 버스로 40분.

- **for forty minutes** = 40분 동안

> **"I've been walking for over an hour."**
> 한 시간 넘게 걸어왔다.

### Example 2: Since + 시점

> **"I started walking in March."**
> 3월에 걷기 시작했다.

- **since March** = 3월부터

> **"I've been running since May."**
> 5월부터 달려왔다.

### Example 3: 둘 다 사용 가능

> **"I've been preparing for a whole year."**
> 1년 동안 준비해 왔다. (for)

> **"I've been preparing since last March."**
> 작년 3월부터 준비해 왔다. (since)

### Example 4: Since + 절 (clause)

> **"Since I missed the bus, I've been walking."**
> 버스를 놓친 이후로 걸어다녔다.

> **"Since I started running, my stamina has improved."**
> 달리기 시작한 이후로 체력이 좋아졌다.

## D. When...? vs How long...?

### When...? (언제?)

> **"When did you start walking to school?"**
> 언제 걸어서 등교하기 시작했어?

**Answer:** "In March." / "On March 11th."

### How long...? (얼마나 오래?)

> **"How long have you been walking to school?"**
> 얼마나 오래 걸어서 등교했어?

**Answer:** "For three months." / "Since March."

---

## E. 시제 사용 주의

### Present Perfect + for/since:

> **"I've lived in Jeju for 14 years."** (O)
> 14년 동안 제주에 살았다.

> **"I've lived in Jeju since I was born."** (O)
> 태어날 때부터 제주에 살았다.

### Past Simple + for (끝난 기간):

> **"I lived in Seoul for two years."** (지금은 아님)
> 서울에서 2년 살았다.

### 주의: Since + Present Perfect

> **"Since I started, I have improved a lot."** (O)
> 시작한 이후로 많이 좋아졌다.

> ~~"Since I started, I improved a lot."~~ (X)
> (since와 현재완료 함께 사용)

## F. 대화 패턴 (Dialogue Patterns)

### Pattern 1: When did you...?

```
A: When did you start running?
B: In March.
A: And how long have you been running now?
B: For about six months.
```

### Pattern 2: How long have you...?

```
Coach: How long have you been training?
Doyun: Since March. So about three months.
Coach: That explains your improvement.
```

### Pattern 3: 소설 속 대화

```
Teacher: Have you been running a lot lately?
Doyun: Yes, sir.
Teacher: Since when?
Doyun: I started walking in March, then started running in May.
Teacher: So it's been three months.
```

## Exercises

### A. For or Since?

1. I've been walking _______ March.
2. I've lived here _______ 14 years.
3. She's been training _______ last summer.
4. He's waited _______ two hours.
5. I've known him _______ we were kids.

**Answers:**
1. since (시점)
2. for (기간)
3. since (시점)
4. for (기간)
5. since (시점/절)

### B. When or How long?

1. _______ did you start walking to school?
2. _______ have you been running?
3. _______ was the first time you ran 7km?
4. _______ has she lived in Jeju?

**Answers:**
1. When (시점 질문)
2. How long (기간 질문)
3. When (시점 질문)
4. How long (기간 질문)

### C. 질문에 답하기

1. Q: How long have you been running?
   A: _______ six months. (기간)

2. Q: When did you start?
   A: _______ March. (시점)

3. Q: How long did it take?
   A: _______ an hour and ten minutes. (기간)

4. Q: Since when have you been training?
   A: _______ I decided to run the marathon. (시점/절)

**Answers:**
1. For
2. In
3. For
4. Since

### D. 문장 만들기

Complete with for, since, or appropriate time expressions:

1. I've been walking to school _______ (3월).
2. I've been preparing _______ (1년).
3. How long have you lived in Jeju? _______ (태어났을 때부터).
4. When did you finish? _______ (2시간 전에).

**Answers:**
1. since March
2. for a year
3. Since I was born
4. Two hours ago

### E. 오류 수정

Find and correct the errors:

1. I've been running since three months.
2. I live here for ten years.
3. Since when you have been walking?
4. I've known him for I was young.

**Answers:**
1. I've been running **for** three months. (or: since March)
2. I **have lived** here for ten years.
3. Since when **have you** been walking?
4. I've known him **since** I was young.

## Key Sentences (암기 문장)

| For (기간) | Since (시점) |
|-----------|-------------|
| I've been running for six months. | I've been running since March. |
| I walked for an hour. | I've walked since the morning. |
| I've lived here for 14 years. | I've lived here since I was born. |
| I've known him for a year. | I've known him since middle school. |

| When...? | How long...? |
|----------|--------------|
| When did you start? - In March. | How long have you been running? - For six months. |
| When was the race? - Last October. | How long did the race take? - About 4 hours. |

## 정리표 (Summary)

| 표현 | 사용 | 예문 |
|------|------|------|
| for | + 기간 | for three months, for a year |
| since | + 시점 | since March, since 2024 |
| since | + 절 | since I started running |
| When? | 시점 질문 | When did you start? |
| How long? | 기간 질문 | How long have you been running? |
