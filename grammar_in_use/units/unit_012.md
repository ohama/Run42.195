# Unit 12: For and Since / When...? and How long...?

## A. For vs Since 기본 개념

### English
- **For** + period of time (how long)
- **Since** + point in time (when it started)
- **When...?** asks about a specific point in time
- **How long...?** asks about duration

### Korean
- **For** + 기간 (얼마나 오래)
- **Since** + 시점 (언제부터)
- **When...?** 특정 시점을 묻는다
- **How long...?** 기간을 묻는다

---

## B. For + 기간 상세 (For + Period of Time)

### From the Original Book:
> "I've been waiting **for two hours**."
> 두 시간 동안 기다렸다.

### Common Expressions with For:

| Expression | Korean | Example |
|------------|--------|---------|
| for two days | 이틀 동안 | I've been sick for two days. |
| for a week | 일주일 동안 | I've been training for a week. |
| for three months | 3개월 동안 | I've been running for three months. |
| for a year | 1년 동안 | I've lived here for a year. |
| for a long time | 오랫동안 | I've known him for a long time. |
| for ages | 아주 오랫동안 | I haven't seen her for ages. |
| for hours | 몇 시간 동안 | I've been walking for hours. |
| for 40 minutes | 40분 동안 | The ride takes for 40 minutes. |

---

## C. Since + 시점 상세 (Since + Point in Time)

### From the Original Book:
> "I've been waiting **since 8 o'clock**."
> 8시부터 기다렸다.

### Common Expressions with Since:

| Expression | Korean | Example |
|------------|--------|---------|
| since yesterday | 어제부터 | I've been tired since yesterday. |
| since Monday | 월요일부터 | I've been training since Monday. |
| since March | 3월부터 | I've been running since March. |
| since 2024 | 2024년부터 | I've lived here since 2024. |
| since 8 o'clock | 8시부터 | I've been waiting since 8 o'clock. |
| since I was young | 어렸을 때부터 | I've loved running since I was young. |
| since I started school | 학교 다닌 이후로 | I've walked since I started school. |
| since the beginning | 처음부터 | I've been here since the beginning. |

---

## D. 소설 속 예문 (Examples from the Novel)

### Example 1: For + 기간

> **"Forty minutes by bus from home to school."**
> 집에서 학교까지 버스로 40분.

- **for forty minutes** = 40분 동안

> **"I've been walking for over an hour."**
> 한 시간 넘게 걸어왔다.

### Example 2: Since + 시점

> **"I started walking in March."**
> 3월에 걷기 시작했다.

- **since March** = 3월부터

> **"I've been running since May."**
> 5월부터 달려왔다.

### Example 3: 둘 다 사용 가능

> **"I've been preparing for a whole year."**
> 1년 동안 준비해 왔다. (for)

> **"I've been preparing since last March."**
> 작년 3월부터 준비해 왔다. (since)

### Example 4: Since + 절 (clause)

> **"Since I missed the bus, I've been walking."**
> 버스를 놓친 이후로 걸어다녔다.

> **"Since I started running, my stamina has improved."**
> 달리기 시작한 이후로 체력이 좋아졌다.

## E. When...? vs How long...?

### When...? (언제?)

Asks about a **specific point** in time.

| Question | Answer |
|----------|--------|
| When did you start running? | In March. / On March 11th. |
| When did you arrive? | At 8 o'clock. / Yesterday. |
| When were you born? | In 2010. |

### How long...? (얼마나 오래?)

Asks about **duration**.

| Question | Answer |
|----------|--------|
| How long have you been running? | For six months. / Since March. |
| How long did it take? | About an hour. |
| How long have you known him? | Since middle school. |

### From the Novel:

**When...?**
> **"When did you start walking to school?"**
> 언제 걸어서 등교하기 시작했어?
> **Answer:** "In March." / "On March 11th."

**How long...?**
> **"How long have you been walking to school?"**
> 얼마나 오래 걸어서 등교했어?
> **Answer:** "For three months." / "Since March."

---

## F. 시제 사용 주의 (Tense with For/Since)

### Present Perfect + for/since (계속):

> **"I've lived in Jeju for 14 years."** (O)
> 14년 동안 제주에 살았다. (지금도 살고 있음)

> **"I've lived in Jeju since I was born."** (O)
> 태어날 때부터 제주에 살았다.

### Past Simple + for (끝난 기간):

> **"I lived in Seoul for two years."** (지금은 아님)
> 서울에서 2년 살았다. (이제 서울에 안 삶)

### 주의: Since는 항상 현재완료와 함께

> **"Since I started, I have improved a lot."** (O)
> 시작한 이후로 많이 좋아졌다.

> ~~"Since I started, I improved a lot."~~ (X)
> (since와 현재완료 함께 사용해야 함)

## G. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I've been running since three months.~~ | I've been running **for** three months. | 기간 = for |
| ~~I've been running for March.~~ | I've been running **since** March. | 시점 = since |
| ~~Since 2020, I live here.~~ | Since 2020, I **have lived** here. | since = 현재완료 |
| ~~I know him since we were kids.~~ | I **have known** him since we were kids. | since = 현재완료 |
| ~~How long are you waiting?~~ | **How long have you been waiting?** | 현재완료 필요 |
| ~~When you started running?~~ | **When did you start** running? | 의문문 어순 |

## H. 대화 패턴 (Dialogue Patterns)

### Pattern 1: When did you...?

```
A: When did you start running?
B: In March.
A: And how long have you been running now?
B: For about six months.
```

### Pattern 2: How long have you...?

```
Coach: How long have you been training?
Doyun: Since March. So about three months.
Coach: That explains your improvement.
```

### Pattern 3: Since when...?

```
Teacher: Have you been running a lot lately?
Doyun: Yes, sir.
Teacher: Since when?
Doyun: I started walking in March, then started running in May.
Teacher: So it's been three months.
```

## Exercises

### A. For or Since?

1. I've been walking _______ March.
2. I've lived here _______ 14 years.
3. She's been training _______ last summer.
4. He's waited _______ two hours.
5. I've known him _______ we were kids.
6. I've been tired _______ yesterday.
7. She's worked here _______ three years.
8. I've had this phone _______ 2023.

**Answers:**
1. since (시점)
2. for (기간)
3. since (시점)
4. for (기간)
5. since (시점/절)
6. since (시점)
7. for (기간)
8. since (시점)

### B. When or How long?

1. _______ did you start walking to school?
2. _______ have you been running?
3. _______ was the first time you ran 7km?
4. _______ has she lived in Jeju?
5. _______ did the race start?
6. _______ did it take to finish?
7. _______ have you known each other?
8. _______ did you decide to enter the marathon?

**Answers:**
1. When (시점 질문)
2. How long (기간 질문)
3. When (시점 질문)
4. How long (기간 질문)
5. When (시점 질문)
6. How long (기간 질문)
7. How long (기간 질문)
8. When (시점 질문)

### C. 질문에 답하기

1. Q: How long have you been running?
   A: _______ six months. (기간)

2. Q: When did you start?
   A: _______ March. (시점)

3. Q: How long did it take?
   A: _______ an hour and ten minutes. (기간)

4. Q: Since when have you been training?
   A: _______ I decided to run the marathon. (시점/절)

5. Q: How long have you lived here?
   A: _______ I was born. (시점/절)

**Answers:**
1. For
2. In
3. (About) / For
4. Since
5. Since

### D. 오류 수정

Find and correct the errors:

1. I've been running since three months.
   → I've been running **for** three months.

2. I live here for ten years.
   → I **have lived** here for ten years.

3. Since when you have been walking?
   → **Since when have you been walking?**

4. I've known him for I was young.
   → I've known him **since** I was young.

5. Since I started, I improved a lot.
   → Since I started, I **have improved** a lot.

## Key Sentences (암기 문장)

| For (기간) | Since (시점) |
|-----------|-------------|
| I've been running for six months. | I've been running since March. |
| I walked for an hour. | I've walked since the morning. |
| I've lived here for 14 years. | I've lived here since I was born. |
| I've known him for a year. | I've known him since middle school. |
| I've been waiting for two hours. | I've been waiting since 8 o'clock. |

| When...? | How long...? |
|----------|--------------|
| When did you start? - In March. | How long have you been running? - For six months. |
| When was the race? - Last October. | How long did the race take? - About 4 hours. |
| When did you arrive? - At 3 PM. | How long have you been here? - For 30 minutes. |
