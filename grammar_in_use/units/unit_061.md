# Unit 61: Be/get used to... (I'm used to...)

## A. 세 가지 표현의 형태 (Form)

**주의:** "used to + 동사원형" (예전에 ~했다)과 "be used to + -ing" (익숙하다)는 완전히 다릅니다!

| Expression | Structure | Meaning | Example |
|------------|-----------|---------|---------|
| be used to | -ing / 명사 | ~에 익숙하다 | I'm used to running. |
| get used to | -ing / 명사 | ~에 익숙해지다 | I'm getting used to it. |
| used to | **동사원형** | 예전에 ~했다 | I used to run. |

### From the Original Book:

> "I'm **used to driving** on the left." (현재 익숙함)
> 왼쪽으로 운전하는 데 익숙해.

> "I **used to drive** to work, but now I take the bus." (과거 습관)
> 예전에는 차로 출근했는데, 지금은 버스 타.

---

## B. be used to + -ing (Primary Usage - 현재 익숙함)

**"be used to + -ing"** = ~에 익숙하다 (현재 상태)

### From the Original Book:

> "I find it hard to get up early. I'm not **used to** it."
> 일찍 일어나기 힘들어. 익숙하지 않아.

### Examples from the Novel:

> **"Now I'm used to walking long distances."**
> 이제 먼 거리를 걷는 데 익숙하다.

- **be used to + -ing**: 현재 익숙한 상태

> **"I'm used to waking up early."**
> 일찍 일어나는 데 익숙하다.

> **"He's used to running in the heat."**
> 그는 더위 속에서 달리는 데 익숙하다.

### 상태 설명:

```
be used to = 현재 상태
═══════════════════════════════►
[익숙한 상태가 지속됨]
```

---

## C. get used to + -ing (Process - 익숙해지다)

**"get used to + -ing"** = ~에 익숙해지다 (과정)

### From the Original Book:

> "I'm slowly **getting used to** my new job."
> 새 직장에 서서히 익숙해지고 있어.

### Examples from the Novel:

> **"I'm getting used to running every day."**
> 매일 달리는 데 익숙해지고 있다.

- **get used to + -ing**: 익숙해지는 과정

> **"It took time, but I got used to the early mornings."**
> 시간이 걸렸지만, 이른 아침에 익숙해졌다.

> **"You'll get used to it."**
> 익숙해질 거야.

### 과정 설명:

```
get used to = 변화 과정
낯섦 ──────────────────► 익숙함
     [점점 익숙해지는 중]
```

---

## D. used to + 동사원형 (Past Habit - 과거 습관)

**"used to + 동사원형"** = 예전에는 ~했다 (지금은 아님)

### From the Original Book:

> "I **used to play** tennis, but now I prefer golf."
> 예전에는 테니스를 쳤는데, 지금은 골프를 더 좋아해.

### Examples from the Novel:

> **"I used to hate running."**
> 예전에는 달리기가 싫었다.

- **used to + 동사원형**: 과거에는 그랬지만 지금은 아님

> **"I used to get carsick on the bus."**
> 예전에는 버스에서 멀미를 했다.

> **"I used to take the bus every day."**
> 예전에는 매일 버스를 탔다.

### 시간적 구조:

```
used to = 과거 상태/습관 (지금은 아님)
───────────────┐
과거            │현재
[그랬다]        │[아님]
```

## E. 세 표현 상세 비교

| Expression | Time | Meaning | Example |
|------------|------|---------|---------|
| be used to + -ing | 현재 | 익숙하다 | I'm used to running. |
| get used to + -ing | 과정 | 익숙해지고 있다 | I'm getting used to it. |
| used to + 동사원형 | 과거 | 예전에 ~했다 | I used to run. |

### From the Novel - 도윤이의 변화:

> **"I used to hate taking the bus."** (과거: 싫었다)
> 예전에는 버스 타는 게 싫었다.

> **"Now I'm used to walking."** (현재: 익숙하다)
> 이제 걷는 데 익숙하다.

> **"I got used to running every morning."** (과정 완료)
> 매일 아침 달리는 데 익숙해졌다.

### Timeline Visualization:

```
과거                    현재                    미래
──────────────────────────────────────────────────►

[used to hate]          [am used to]
예전에 싫었다            이제 익숙하다

         [got used to / getting used to]
         익숙해지는 과정
```

## F. 부정문과 의문문 (Negative and Questions)

### 부정문:

> **"I'm not used to running in the rain."**
> 비 오는 날 달리는 게 익숙하지 않다.

> **"I wasn't used to waking up at 5 AM."**
> 새벽 5시에 일어나는 게 익숙하지 않았다.

> **"I didn't use to like running."**
> 예전에는 달리기를 좋아하지 않았다.

### 의문문:

> **"Are you used to running long distances?"**
> 장거리 달리는 데 익숙해?

> **"Is it hard to get used to?"**
> 익숙해지기 어려워?

> **"Did you use to play sports?"**
> 예전에 운동했어?

### 부정/의문 형태 정리:

| Expression | Negative | Question |
|------------|----------|----------|
| be used to | I'm **not** used to... | **Are** you used to...? |
| get used to | I can't get used to... | Will you get used to...? |
| used to | I **didn't use to**... | **Did** you **use to**...? |

## G. be used to + 명사 (Noun Complement)

**be used to** 뒤에는 명사도 올 수 있습니다.

### Examples:

> **"I'm used to the early mornings now."**
> 이제 이른 아침에 익숙하다.

> **"I'm not used to the heat."**
> 더위에 익숙하지 않다.

> **"He got used to the routine."**
> 그는 그 루틴에 익숙해졌다.

### -ing vs 명사:

| + -ing | + 명사 |
|--------|--------|
| I'm used to **running** early. | I'm used to **early mornings**. |
| I'm used to **walking** long distances. | I'm used to **the distance**. |
| I'm getting used to **waking** up early. | I'm getting used to **the alarm**. |

---

## H. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I'm used to **wake** up early.~~ | I'm used to **waking** up early. | be used to + -ing |
| ~~I used to **waking** up early.~~ | I **used to wake** up early. | used to + 동사원형 |
| ~~I'm use to running.~~ | I'm **used** to running. | used, not use |
| ~~I am used to **run**.~~ | I am used to **running**. | be used to + -ing |
| ~~I was used to **take** the bus.~~ | I was used to **taking** the bus. | be used to + -ing |
| ~~I'm getting used to **run**.~~ | I'm getting used to **running**. | get used to + -ing |

## Exercises

### A. 올바른 형태를 고르세요.

1. I'm used to (wake / waking) up early.
2. I used to (take / taking) the bus.
3. She's getting used to (run / running) long distances.
4. He used to (hate / hating) exercise.
5. Are you used to (walk / walking) to school?
6. I didn't use to (like / liking) running.
7. It took time to get used to (the heat / heating).
8. I'm not used to (be / being) so tired.

**Answers:**
1. waking (be used to + -ing)
2. take (used to + 동사원형)
3. running (get used to + -ing)
4. hate (used to + 동사원형)
5. walking (be used to + -ing)
6. like (used to + 동사원형)
7. the heat (be used to + 명사)
8. being (be used to + -ing)

### B. 문장을 완성하세요.

1. I used _______ (take) the bus, but now I walk.
2. I'm used _______ (wake) up at 5 AM.
3. It took time, but I got used _______ (run) every day.
4. She isn't used _______ (exercise) in the morning.
5. He used _______ (hate) running, but now he loves it.
6. Are you getting used _______ (the new schedule)?
7. I didn't use _______ (enjoy) running.
8. They're used _______ (train) in cold weather.

**Answers:**
1. to take
2. to waking
3. to running
4. to exercising
5. to hate
6. to (명사이므로 to만)
7. to enjoy
8. to training

### C. be used to / get used to / used to 구분하기

1. 예전에는 버스를 탔다 (과거 습관)
   → I _______ the bus.

2. 이제 걷는 데 익숙하다 (현재 상태)
   → I _______ walking.

3. 아직 익숙해지는 중이다 (과정)
   → I _______ walking.

4. 예전에는 달리기를 싫어했다 (과거)
   → I _______ hate running.

5. 이른 아침에 익숙해졌다 (과정 완료)
   → I _______ the early mornings.

**Answers:**
1. used to take
2. 'm used to / am used to
3. 'm getting used to / am getting used to
4. used to
5. got used to

### D. 틀린 부분을 고치세요.

1. I'm used to wake up early.
   → _______

2. I use to play soccer.
   → _______

3. She's getting used to run every day.
   → _______

4. I was used to take the bus.
   → _______

**Answers:**
1. I'm used to **waking** up early.
2. I **used** to play soccer.
3. She's getting used to **running** every day.
4. I was used to **taking** the bus.

## Summary: 세 표현 비교표

| 표현 | 구조 | 의미 | 시제 |
|------|------|------|------|
| **be used to** | + -ing / 명사 | ~에 익숙하다 | 현재 상태 |
| **get used to** | + -ing / 명사 | ~에 익숙해지다 | 과정 |
| **used to** | + **동사원형** | 예전에 ~했다 | 과거 습관 |

### Quick Check:

| Question | Answer |
|----------|--------|
| 지금 익숙해? | → **be used to** + -ing |
| 익숙해지고 있어? | → **get used to** + -ing |
| 예전에 그랬어? | → **used to** + 동사원형 |

### Remember:
- **be/get used to** = **전치사 to** → + **-ing**
- **used to** = **to부정사** → + **동사원형**
