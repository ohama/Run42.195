# Unit 85: Some and any

## A. 형태와 기본 개념 (Form and Basic Concept)

| some | any |
|------|-----|
| 긍정문 | 부정문, 의문문 |
| I have **some** water. | I don't have **any** water. |
| 약간의, 조금 | (부정) 전혀, (의문) 조금이라도 |

### 기본 규칙:

| Sentence Type | Use |
|---------------|-----|
| 긍정문 (+) | **some** |
| 부정문 (-) | **any** |
| 의문문 (?) | **any** (일반 질문) |
| 의문문 (?) | **some** (권유/요청) |

---

## B. some: 긍정문 (Positive Statements)

### From the Original Book:

> **"I've got some money."**
> 돈이 좀 있어.

> **"I saw some beautiful flowers."**
> 예쁜 꽃을 좀 봤다.

### Examples from the Novel:

> **"I still have some of the candy Minseo gave me."**
> 민서가 준 사탕이 아직 좀 남아 있다.
- some = 약간의 (긍정문)

> **"I drank some water."**
> 물을 좀 마셨다.

> **"Some kids looked at me."**
> 몇몇 애들이 나를 쳐다봤다.

> **"I made some progress."**
> 좀 나아졌다.

> **"I saw some flowers along the road."**
> 길가에 꽃이 좀 보였다.

---

## C. any: 부정문, 의문문 (Negative and Questions)

### 부정문 (Negative):

> **"I didn't say anything."**
> 나는 아무 말도 안 했다.
- anything = any + thing (부정문)

> **"I don't have any experience."**
> 경험이 전혀 없다.

> **"There wasn't any space."**
> 공간이 전혀 없었다.

> **"I didn't see anyone."**
> 아무도 못 봤다.

### 의문문 (Questions):

> **"Do you have any questions?"**
> 질문 있어?

> **"Is there any problem?"**
> 문제 있어?

> **"Have you seen any flowers?"**
> 꽃 좀 봤어?

---

## D. 소설 속 예문 (Examples from the Novel)

### some (긍정문):

> **"I saw some flowers."**
> 꽃이 좀 보였다.

> **"Some students were walking."**
> 몇몇 학생들이 걷고 있었다.

### any (부정문):

> **"I couldn't hear anything."**
> 아무것도 안 들렸다.

> **"There wasn't any wind."**
> 바람이 전혀 없었다.

> **"I didn't have any energy left."**
> 남은 힘이 전혀 없었다.

### any (의문문):

> **"Do you have any advice?"**
> 조언 있어?

> **"Is there any water?"**
> 물 있어?

---

## E. some in questions: 권유, 요청 (Offers and Requests)

**제안/권유/요청** 의문문에는 **some**을 씁니다. (Yes를 기대하는 질문)

### From the Original Book:

> **"Would you like some coffee?"**
> 커피 좀 드실래요? (권유)

> **"Can I have some water?"**
> 물 좀 주실 수 있어요? (요청)

### Examples:

> **"Would you like some water?"**
> 물 좀 드실래요?
- some = 권유 (Yes를 기대)

> **"Can I have some help?"**
> 좀 도와줄 수 있어?
- some = 요청

### From the Novel:

> **"Do you want some candy?"**
> 사탕 좀 먹을래?
- some = 권유

> **"Can I get some water?"**
> 물 좀 줄래?
- some = 요청

---

## F. any: "어떤 것이든" (Any = It doesn't matter which)

**긍정문에서 any**가 쓰이면 **"어떤 것이든, 무엇이든"**의 의미입니다.

### From the Original Book:

> **"You can take any bus."**
> 아무 버스나 타면 돼.

> **"Come any day you like."**
> 아무 날이나 오세요.

### Examples:

> **"Any bus will take you there."**
> 아무 버스나 타면 돼.

> **"You can run at any time."**
> 언제든 달릴 수 있어.

### From the Novel:

> **"Anything's fine."**
> 아무거나 괜찮아.
- anything = 어떤 것이든

> **"Any day is fine for running."**
> 아무 날이나 달리기에 괜찮아.

> **"I'll take any advice you have."**
> 어떤 조언이든 받을게.

## G. some/any + 복합어 (Compounds)

| some- | any- (부정/의문) | any- (긍정: 어떤~이든) |
|-------|-----------------|---------------------|
| some**body/one** | any**body/one** | any**body/one** |
| some**thing** | any**thing** | any**thing** |
| some**where** | any**where** | any**where** |
| some**time** | any**time** | any**time** |

### Examples:

| Positive | Negative | Question | "Any..." |
|----------|----------|----------|----------|
| I saw **someone**. | I didn't see **anyone**. | Did you see **anyone**? | **Anyone** can do it. |
| I heard **something**. | I didn't hear **anything**. | Did you hear **anything**? | **Anything** is possible. |
| I went **somewhere**. | I didn't go **anywhere**. | Did you go **anywhere**? | Go **anywhere** you like. |

## H. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | 설명 |
|-----------|-------------|------|
| I don't have **some** money. | I don't have **any** money. | 부정문은 any |
| Do you have **some** questions? | Do you have **any** questions? | 일반 의문문은 any |
| I have **any** friends. | I have **some** friends. | 긍정문은 some |
| Would you like **any** coffee? | Would you like **some** coffee? | 권유는 some |
| **Some** bus will work. | **Any** bus will work. | "아무~나"는 any |

### 추가 실수 예시:

| Wrong (X) | Correct (O) |
|-----------|-------------|
| I didn't say **something**. | I didn't say **anything**. |
| Is there **some** problem? (일반 질문) | Is there **any** problem? |
| Can I have **any** water? (요청) | Can I have **some** water? |

## Exercises

### A. some 또는 any를 넣으세요.

1. I drank _______ water.
2. I don't have _______ experience.
3. Do you have _______ questions?
4. Would you like _______ candy?
5. _______ bus will work.
6. I saw _______ flowers along the road.
7. I didn't see _______ people.
8. Can I have _______ help?

**Answers:**
1. some
2. any
3. any
4. some
5. Any
6. some
7. any
8. some

### B. 올바른 형태를 고르세요.

1. I saw (some / any) flowers along the road.
2. There wasn't (some / any) space to move.
3. Can I have (some / any) water? (요청)
4. I didn't say (something / anything).
5. (Some / Any) day is good for practice.
6. Do you know (someone / anyone) here?
7. Would you like (some / any) coffee?
8. I can go (somewhere / anywhere) you want.

**Answers:**
1. some
2. any
3. some
4. anything
5. Any
6. anyone
7. some
8. anywhere

### C. 긍정문, 부정문, 의문문으로 바꾸세요.

1. I have some candy. (부정문) → _______
2. Do you have any money? (긍정문) → _______
3. There is some water. (부정문) → _______
4. I saw someone. (부정문) → _______
5. Did you go anywhere? (긍정문) → _______

**Answers:**
1. I don't have any candy.
2. I have some money.
3. There isn't any water.
4. I didn't see anyone.
5. I went somewhere.

### D. some과 any 중 알맞은 것을 고르고, 이유를 쓰세요.

1. I need _______ help. (긍정문)
2. Do you want _______ water? (권유)
3. Is there _______ problem? (일반 질문)
4. _______ time is fine for me. (어느 때든)
5. I didn't hear _______. (부정문)

**Answers:**
1. some (긍정문)
2. some (권유 - Yes 기대)
3. any (일반 질문)
4. Any (어느 때든 = 긍정문의 any)
5. anything (부정문)

## Summary: some vs any

| | some | any |
|-|------|-----|
| 긍정문 | O (기본) | O (어떤 것이든) |
| 부정문 | X | O (전혀) |
| 의문문 | O (권유/요청) | O (일반 질문) |

### 핵심 규칙:
1. **긍정문** → some (기본)
2. **부정문** → any
3. **의문문** → any (일반), some (권유/요청)
4. **"아무~나/어떤~이든"** → any (긍정문에서)

### 복합어:
- some/any + body/one = 사람
- some/any + thing = 것
- some/any + where = 장소
- some/any + time = 시간
