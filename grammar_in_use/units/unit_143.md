# Unit 143: Phrasal Verbs (7) - up (1)

## A. 개념 설명 (Grammar Explanation)

### English
This unit focuses on additional meanings of **up** in phrasal verbs. **Up** often indicates **completion**, **increase in intensity**, **approaching**, or **preparation**. Understanding these extended meanings helps you use phrasal verbs more naturally.

### Korean
이 유닛에서는 **up**의 추가적인 의미를 다룹니다. **up**은 종종 **완료**, **강도 증가**, **접근**, **준비**를 나타냅니다. 이러한 확장된 의미를 이해하면 구동사를 더 자연스럽게 사용할 수 있습니다.

---

## B. UP의 확장 의미 (Extended Meanings of UP)

### 1. 완료/끝까지 (Completion)

| Phrasal Verb | Meaning |
|--------------|---------|
| drink up | 다 마시다 |
| eat up | 다 먹다 |
| use up | 다 쓰다 |
| clean up | 청소하다 |
| finish up | 마치다 |

> **"I drank up all the water."**
> 물을 다 마셨다.

> **"Eat up your breakfast."**
> 아침을 다 먹어.

### 2. 나타남/도착 (Appearing/Arriving)

| Phrasal Verb | Meaning |
|--------------|---------|
| show up | 나타나다 |
| come up | 나타나다 (화면에) |
| turn up | 나타나다 |
| pop up | 갑자기 나타나다 |

> **"Minseo showed up in running shoes."**
> 민서가 러닝화를 신고 나타났다.

> **"A bunch of articles came up."**
> 많은 기사들이 나왔다.

### 3. 증가/강화 (Increasing)

| Phrasal Verb | Meaning |
|--------------|---------|
| pick up | 올리다/강해지다 |
| speed up | 속도 높이다 |
| build up | 쌓다/증가하다 |
| heat up | 뜨거워지다 |

> **"I picked up the pace."**
> 속도를 올렸다.

> **"The wind had picked up."**
> 바람이 세졌다.

### 4. 준비/설정 (Setting up)

| Phrasal Verb | Meaning |
|--------------|---------|
| set up | 설정하다 |
| warm up | 워밍업하다 |
| make up | 만들어내다 |
| fix up | 고치다/준비하다 |

> **"I set up my alarm."**
> 알람을 맞춰놨다.

> **"My body warmed up nicely."**
> 몸이 잘 풀렸다.

## C. 소설 속 예문 (Examples from the Novel)

### Example 1: warm up (워밍업하다)

> **"My body warmed up nicely."**
> 몸이 잘 풀렸다.

> **"You'll warm up once we start moving."**
> 움직이기 시작하면 몸이 풀릴 거야.

> **"By 5 km, my body was warmed up."**
> 5km 지점에 몸이 풀렸다.

### Example 2: pick up (올리다/나아지다)

> **"Then I gradually picked up the pace."**
> 그러다 점차 속도를 올렸다.

> **"The wind had picked up."**
> 바람이 세졌다.

> **"After a week of running, they'd picked up some dust."**
> 일주일 달리고 나니 먼지가 좀 묻었다.

**pick up의 다양한 의미:**
| Meaning | Example |
|---------|---------|
| 속도 올리다 | pick up the pace |
| 강해지다 | The wind picked up. |
| 줍다 | I picked up a coin. |
| 데리러 가다 | I'll pick you up at 5. |
| 배우다 | I picked up some Korean. |

### Example 3: show up (나타나다)

> **"Minseo showed up in running shoes. White ones."**
> 민서가 러닝화를 신고 나타났다. 흰색이었다.

> **"Minseo showed up in workout clothes."**
> 민서가 운동복 차림으로 나타났다.

### Example 4: come up (나타나다/떠오르다)

> **"A bunch of articles came up."**
> 많은 기사들이 나왔다.

> **"The sun came up."**
> 해가 떠올랐다.

### Example 5: speed up (속도를 높이다)

> **"But I held back. Fought the urge to speed up."**
> 하지만 참았다. 속도를 높이고 싶은 충동을 억눌렀다.

## D. UP 구동사 분류표 (UP Phrasal Verbs by Category)

| 의미 | 구동사 | 예문 |
|------|--------|------|
| 완료 | drink up, eat up, use up | Drink up your water. |
| 나타남 | show up, come up, turn up | She showed up late. |
| 증가 | pick up, speed up, build up | I picked up the pace. |
| 준비 | warm up, set up | My body warmed up. |
| 기상 | wake up, get up | I woke up at 5. |

## E. 워밍업 관련 표현 (Warming up expressions)

| Expression | Meaning | Example |
|------------|---------|---------|
| warm up | 몸을 풀다 | I need to warm up first. |
| be warmed up | 몸이 풀린 상태 | My body was warmed up. |
| warm-up (명사) | 워밍업 | I did a 10-minute warm-up. |
| warming up | 워밍업 중 | I'm just warming up. |

### From the Novel:

> **"My body's not warmed up yet."**
> 몸이 아직 안 풀렸다.

> **"By 5 km, my body was warmed up."**
> 5km 지점에 몸이 풀렸다.

## F. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~warm out~~ | **warm up** | 워밍업은 warm up |
| ~~I picked the pace up.~~ | ✓ / **I picked up the pace.** | 둘 다 가능 (분리 가능) |
| ~~speed down~~ | **slow down** | 느려지다 = slow down |
| ~~show on~~ | **show up** | 나타나다 = show up |
| ~~build on~~ (쌓다) | **build up** | 쌓다 = build up |

### pick up 대명사 위치:
- I **picked up** the pace. ✓
- I **picked** the pace **up**. ✓
- I **picked it up**. ✓ (대명사는 중간에)
- ~~I picked up it.~~ ✗

## UP의 다양한 의미 요약 (Summary)

| Meaning | Phrasal Verbs |
|---------|---------------|
| **완료** (completion) | drink up, eat up, use up, finish up |
| **나타남** (appearing) | show up, come up, turn up, pop up |
| **증가** (increase) | pick up, speed up, build up, heat up |
| **준비** (preparation) | warm up, set up, make up |
| **기상** (waking) | wake up, get up, stay up |
