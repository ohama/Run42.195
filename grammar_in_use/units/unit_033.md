# Unit 33: Should (1)

## A. 문법 설명 (Grammar Explanation)

### Should: 조언/의견 (~하는 게 좋다)

**Should**는 "~해야 한다/~하는 게 좋다"는 의미로 조언이나 의견을 표현합니다.

- Must보다 부드러운 표현
- Ought to와 같은 의미

### 형태:
- 긍정: should + 동사원형
- 부정: should not (shouldn't) + 동사원형
- 의문: Should + 주어 + 동사원형?

---

## B. 소설 속 예문 (Examples from the Novel)

### Example 1: 조언 (Advice)

> **"You should rest well this weekend."**
> 이번 주말에 푹 쉬어야 해.

- 부드러운 조언

### Example 2: 의견 (Opinion)

> **"I think you should try running earlier."**
> 더 일찍 달려보는 게 좋을 것 같아.

- I think와 함께 쓰는 의견

### Example 3: 부정 (Shouldn't)

> **"You shouldn't run on an empty stomach."**
> 공복에 달리면 안 돼.

- ~하지 않는 게 좋다

### Example 4: 의문문 (Question)

> **"Should I see a doctor?"**
> 병원 가봐야 하나?

- 조언 구하기

### Example 5: 당연한 일 (Expected)

> **"I should be able to finish in time."**
> 제시간에 끝낼 수 있을 거야.

- 예상/기대

## C. 추가 설명 (Additional Points)

### Should vs Must

| Should | Must |
|--------|------|
| 조언/권유 | 의무/필수 |
| ~하는 게 좋다 | ~해야 한다 |
| You **should** rest. | You **must** rest. |
| (쉬는 게 좋아) | (꼭 쉬어야 해) |

### Should vs Ought to

거의 같은 의미:
> "You **should** rest." = "You **ought to** rest."

Ought to는 더 격식체, should가 더 일반적

### I think ... should

의견을 부드럽게 표현:
> "**I think** you should try walking first."
> 먼저 걷기를 해보는 게 좋을 것 같아.

### Should for Expected Things

당연히 그럴 것이라는 기대:
> "He should be here by now."
> 지금쯤 여기 있어야 하는데.

## D. 비교 (Comparison)

| 표현 | 강도 | 예문 |
|------|------|------|
| must | 강함 (의무) | You **must** rest. |
| should | 중간 (조언) | You **should** rest. |
| could | 약함 (제안) | You **could** rest. |

### 일상 대화에서

| 상황 | 표현 |
|------|------|
| 강한 조언 | You should definitely rest. |
| 일반 조언 | You should rest. |
| 부드러운 제안 | Maybe you should rest. |
| 의견 | I think you should rest. |

## Exercises

### A. Should로 조언하기

1. 더 일찍 일어나는 게 좋겠어.
   → You _______ earlier.

2. 공복에 달리면 안 돼.
   → You _______ on an empty stomach.

3. 병원 가봐야 하나?
   → _______ I see a doctor?

**Answers:**
1. should wake up / should get up
2. shouldn't run
3. Should

### B. Must vs Should 선택

1. You _______ rest before the race. (강한 의무)
2. You _______ try running with music. (부드러운 제안)
3. The doctor said I _______ stop running for a week. (의무)
4. I think you _______ drink more water. (조언)

**Answers:**
1. must
2. should
3. must
4. should

### C. 문장 완성

1. 이번 주말에 푹 쉬어야 해. (조언)
   → You _______ this weekend.

2. 더 일찍 달려보는 게 좋을 것 같아. (의견)
   → I think you _______ earlier.

3. 지금쯤 여기 있어야 하는데. (기대)
   → He _______ here by now.

**Answers:**
1. should rest well
2. should try running
3. should be
