# Unit 33: Should (1)

## A. 형태 (Form)

### Should: 조언/의견

**Should**는 "~해야 한다/~하는 게 좋다"는 의미로 조언이나 의견을 표현합니다.

| Form | Example |
|------|---------|
| 긍정 | You should rest. |
| 부정 | You shouldn't run. |
| 의문 | Should I rest? |

### 구조:

| Type | Structure | Example |
|------|-----------|---------|
| Affirmative | should + 동사원형 | You should rest. |
| Negative | should not (shouldn't) + 동사원형 | You shouldn't worry. |
| Question | Should + 주어 + 동사원형? | Should I see a doctor? |

---

## B. 조언 (Advice)

**Should** = "~하는 게 좋다" (Must보다 부드러움)

### From the Original Book:

> "You look tired. You **should** go to bed."
> 피곤해 보여. 자는 게 좋겠어.

> "You've been working too hard. You **should** take a break."
> 너무 열심히 일했어. 쉬는 게 좋겠어.

> "You **shouldn't** believe everything you read in the newspaper."
> 신문에 나온 걸 다 믿으면 안 돼.

### Examples from the Novel:

> **"You should rest well this weekend."**
> 이번 주말에 푹 쉬어야 해.

- 부드러운 조언

> **"You shouldn't run on an empty stomach."**
> 공복에 달리면 안 돼.

- ~하지 않는 게 좋다

> **"You should drink more water."**
> 물 더 마시는 게 좋아.

---

## C. 의견 (Opinions)

### I think ... should

의견을 부드럽게 표현:

> **"I think you should try running earlier."**
> 더 일찍 달려보는 게 좋을 것 같아.

> **"I don't think you should run today."**
> 오늘 달리지 않는 게 좋을 것 같아.

### From the Original Book:

> "Do you think I **should** buy it?"
> 사는 게 좋을까?

> "**I think** the government **should** do more for education."
> 정부가 교육에 더 투자해야 한다고 생각해.

### Softening Expressions:

| Expression | Example |
|------------|---------|
| I think you should... | I think you should rest. |
| Maybe you should... | Maybe you should try this. |
| Perhaps you should... | Perhaps you should see a doctor. |
| You might want to... | You might want to reconsider. |

---

## D. 질문으로 조언 구하기 (Asking for Advice)

> **"Should I see a doctor?"**
> 병원 가봐야 하나?

> **"What should I do?"**
> 어떻게 해야 해?

> **"Do you think I should run tomorrow?"**
> 내일 달리는 게 좋을까?

### Common Questions:

| Question | Korean |
|----------|--------|
| Should I...? | ~해야 할까? |
| What should I do? | 뭘 해야 하지? |
| Where should I go? | 어디로 가야 하지? |
| Do you think I should...? | ~하는 게 좋을까? |

---

## E. 당연한 일/기대 (Expected Things)

**Should**는 "당연히 ~할 것"이라는 기대를 표현합니다.

> **"I should be able to finish in time."**
> 제시간에 끝낼 수 있을 거야.

> **"He should be here by now."**
> 지금쯤 여기 있어야 하는데.

> **"The package should arrive tomorrow."**
> 택배가 내일 도착할 거야.

### From the Original Book:

> "Tom has been studying hard, so he **should** pass the exam."
> 톰이 열심히 공부했으니까 시험에 붙을 거야.

> "It's 10 o'clock. She **should** be at work now."
> 10시야. 지금 회사에 있을 거야.

---

## F. Should vs Must vs Could

| Expression | Strength | Meaning | Example |
|------------|----------|---------|---------|
| must | 강함 | 의무/필수 | You **must** rest. |
| should | 중간 | 조언/권유 | You **should** rest. |
| could | 약함 | 가능한 제안 | You **could** rest. |

### Comparison:

| Must | Should |
|------|--------|
| 의무/필수 | 조언/권유 |
| ~해야 한다 (강함) | ~하는 게 좋다 (부드러움) |
| You **must** rest. (꼭 쉬어야 해) | You **should** rest. (쉬는 게 좋아) |

---

## G. Should vs Ought to

**Should**와 **ought to**는 같은 의미입니다.

| Should | Ought to |
|--------|----------|
| 더 일반적 | 더 격식체 |
| You should rest. | You ought to rest. |
| You shouldn't run. | You ought not to run. |

### Note:
일상 회화에서는 **should**가 더 자주 사용됩니다.

---

## H. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~You should to rest.~~ | You **should rest**. | to 없음 |
| ~~He shoulds rest.~~ | He **should rest**. | should는 변하지 않음 |
| ~~I don't should go.~~ | I **shouldn't go**. | 부정은 shouldn't |
| ~~Do I should go?~~ | **Should** I go? | Should가 맨 앞 |
| ~~You should must rest.~~ | You **should rest**. / You **must rest**. | 조동사 하나만 |
| ~~I think you should to try.~~ | I think you **should try**. | to 없음 |

### Should vs Would 구분:

| Should (조언) | Would (가정) |
|---------------|-------------|
| You should try. | I would try. |
| (해보는 게 좋아) | (나라면 해볼 거야) |

## Exercises

### A. Should로 조언하기 (Give advice using should)

1. 더 일찍 일어나는 게 좋겠어.
   → You _______ earlier.

2. 공복에 달리면 안 돼.
   → You _______ on an empty stomach.

3. 병원 가봐야 하나?
   → _______ I see a doctor?

4. 물 더 마시는 게 좋아.
   → You _______ more water.

5. 뭘 해야 하지?
   → What _______ I do?

**Answers:**
1. should wake up / should get up
2. shouldn't run
3. Should
4. should drink
5. should

### B. Must vs Should 선택 (Choose the correct form)

1. You _______ rest before the race. It's required. (강한 의무)
2. You _______ try running with music. It's nice. (부드러운 제안)
3. The doctor said I _______ stop running for a week. (의무)
4. I think you _______ drink more water. (조언)
5. You _______ wear a seatbelt. It's the law. (의무)
6. You _______ try this restaurant. The food is great. (권유)
7. Students _______ arrive on time. It's a rule. (의무)
8. Maybe you _______ rest today. (부드러운 조언)

**Answers:**
1. must
2. should
3. must
4. should
5. must
6. should
7. must
8. should

### C. 문장 완성 (Complete the sentences)

1. 이번 주말에 푹 쉬어야 해. (조언)
   → You _______ this weekend.

2. 더 일찍 달려보는 게 좋을 것 같아. (의견)
   → I think you _______ earlier.

3. 지금쯤 여기 있어야 하는데. (기대)
   → He _______ here by now.

4. 오늘 달리지 않는 게 좋을 것 같아.
   → I don't think you _______ today.

5. 어떻게 해야 해?
   → What _______ I do?

**Answers:**
1. should rest well
2. should try running
3. should be
4. should run
5. should

### D. 상황에 맞게 조언하기 (Give advice for each situation)

1. Your friend looks tired.
   → You **should get some rest**.

2. Someone wants to improve their running.
   → You **should train regularly**.

3. A friend is unsure about joining the track team.
   → Maybe you **should try it**.

4. Someone is running on an injury.
   → You **shouldn't run** with that injury.

## Summary Table (요약표)

| 형태 | 의미 | 예문 |
|------|------|------|
| should + 동사원형 | ~하는 게 좋다 | You **should** rest. |
| shouldn't + 동사원형 | ~하지 않는 게 좋다 | You **shouldn't** run. |
| Should + 주어...? | ~해야 할까? | **Should** I go? |

| should be | ~일 거야 (기대) | He **should be** here. |

## Advice Strength Scale (조언 강도)

```
강함                                          약함
|------------------------------------------|
must           should           could/might
(해야 해)       (하는 게 좋아)      (해볼 수도)
```
