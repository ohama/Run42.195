# Unit 23: I will and I'm going to

## A. 핵심 차이 (Key Difference)

**Will**과 **be going to** 모두 미래를 나타내지만, 용법이 다릅니다.

### 핵심 비교표:

| Situation | Will | Be going to |
|-----------|------|-------------|
| 즉석 결정 (decide now) | O | X |
| 이미 결정한 계획 (already decided) | X | O |
| 일반적인 예측 (general prediction) | O | O (둘 다 가능) |
| 증거 기반 예측 (evidence-based) | X | O |
| 약속/제안 (promise/offer) | O | X |

### 형태 비교:

| Form | Will | Be going to |
|------|------|-------------|
| 긍정 | I will run. | I'm going to run. |
| 부정 | I won't run. | I'm not going to run. |
| 의문 | Will you run? | Are you going to run? |

---

## B. 즉석 결정 vs 이미 결정 (Spontaneous vs Pre-planned)

### 즉석 결정 = Will

말하는 **순간** 결정할 때:

> **A:** "There's no milk."
> **B:** "Oh, I**'ll** go and get some." (지금 결정)

> **A:** "I'm cold."
> **B:** "I**'ll** close the window." (지금 결정)

### 이미 결정 = Be going to

말하기 **전에** 이미 결정한 것:

> **A:** "Why are you wearing running shoes?"
> **B:** "I**'m going to** run. I decided this morning." (이미 결정)

> **A:** "What are your plans for tomorrow?"
> **B:** "I**'m going to** run 10km." (이미 계획)

### From the Original Book:

> "**A:** Why are you turning on the TV?"
> "**B:** I**'m going to** watch the news."
> 왜 TV 켜? - 뉴스 볼 거야. (이미 결정)

> "**A:** I can't do this math problem."
> "**B:** Don't worry. I**'ll** help you."
> 이 수학 문제 못 풀겠어. - 걱정 마, 도와줄게. (지금 결정)

### Examples from the Novel:

> **"I'll take these."** (running shoes)
> 이걸로 할게요.

- 가게에서 신발을 보고 그 자리에서 결정
- **Will** = 즉석 결정

> **"I'm going to run 10km tomorrow."**
> 내일 10km 달릴 거야.

- 이미 마음먹고 있던 계획
- **Be going to** = 미리 결정한 의도

---

## C. 예측 (Predictions)

### 일반적 예측 = Will (or Be going to)

특별한 증거 없이 **생각/의견**으로:

> **"I think I'll feel better soon."**
> 곧 나아질 것 같아.

> **"I think the race will be hard."**
> 대회가 힘들 것 같아.

> **"I think it'll rain tomorrow."**
> 내일 비 올 것 같아.

### 증거 기반 예측 = Be going to

**지금 보이는 증거**를 바탕으로:

> **"Look at those clouds! It's going to rain."**
> 저 구름 봐! 비 올 것 같아.

> **"My legs are heavy. I'm going to slow down."**
> 다리가 무거워. 속도가 느려질 것 같아.

### Comparison Table:

| Type | Form | Example | Korean |
|------|------|---------|--------|
| 일반 예측 | Will | I think it**'ll** rain. | 비 올 것 같아. |
| 증거 기반 | Be going to | Look! It**'s going to** rain. | 봐! 비 올 것 같아. |

---

## D. 약속과 제안 (Promises and Offers)

약속과 제안에는 주로 **Will**을 사용합니다.

> **"I'll be waiting at the finish line."**
> 결승선에서 기다릴게. (약속)

> **"I'll help you with your training."**
> 훈련 도와줄게. (제안)

> **"I'll make dinner tonight."**
> 오늘 저녁 내가 만들게. (제안)

| Situation | Use | Example |
|-----------|-----|---------|
| 약속 | Will | I'll be there. |
| 제안 | Will | I'll help you. |
| 결심 | Will | I won't give up. |

---

## E. 대화에서의 자연스러운 흐름 (Conversation Flow)

### Pattern 1: 계획 말하기

**A:** "What's your plan for tomorrow?"
**B:** "I**'m going to** run 10km." (이미 결정된 계획)

### Pattern 2: 즉석으로 도와주기

**A:** "Oh, I'm thirsty."
**B:** "I**'ll** get you some water." (지금 결정)

### Pattern 3: 둘 다 사용하는 경우

**A:** "I**'m going to** run tomorrow." (계획 말하기)
**B:** "Great! I**'ll** join you!" (지금 결정)

## F. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I'm going to take these.~~ (가게에서 즉석 결정) | I**'ll** take these. | 즉석 결정 = will |
| ~~I'll run tomorrow.~~ (이미 계획함) | I**'m going to** run tomorrow. | 미리 결정 = be going to |
| ~~Look! It will rain.~~ (구름 보고) | Look! It**'s going to** rain. | 증거 기반 = be going to |
| ~~I'll help him tomorrow.~~ (이미 약속) | I**'m going to** help him tomorrow. | 미리 결정 = be going to |
| ~~I going to run.~~ | I**'m going to** run. | be 동사 필요 |
| ~~I'm go to run.~~ | I**'m going to** run. | going to |

### 상황별 올바른 선택:

| 상황 | 선택 | 이유 |
|------|------|------|
| 전화가 울린다 | I'll answer it. | 지금 결정 |
| 주말 계획 말할 때 | I'm going to visit grandpa. | 이미 계획 |
| 검은 구름 보고 | It's going to rain. | 증거 기반 |
| 누가 물을 흘렸다 | I'll get a towel. | 지금 결정 |
| 내일 대회에 대해 | The race is going to start at 8. | 확정된 일정 |

---

## G. 둘 다 사용 가능한 경우 (Both are possible)

미래에 대한 **일반적인 예측**에서는 둘 다 가능:

> "I think the race **will be** hard."
> "I think the race **is going to be** hard."

> "It **will** probably rain tomorrow."
> "It**'s probably going to** rain tomorrow."

## Exercises

### A. Will vs Be going to 선택 (Choose the correct form)

1. A: "There's no milk." B: "I know. I _______ (get) some later." (이미 알고 있었고 계획함)
2. A: "There's no milk." B: "Oh, I _______ (go) get some now." (지금 결정)
3. Look at those clouds! It _______ (rain).
4. I think the exam _______ (be) difficult.
5. I _______ (run) the marathon in March. I've been training for months.
6. "Who wants to help?" - "I _______ (do) it!"
7. We _______ (visit) grandma this weekend. We planned it last week.
8. Don't worry, I _______ (not tell) anyone.

**Answers:**
1. am going to get
2. will go / 'll go
3. is going to rain
4. will be (또는 is going to be)
5. am going to run
6. will do / 'll do
7. are going to visit
8. won't tell

### B. 상황에 맞게 문장 완성 (Complete based on context)

1. (미리 계획) 내일 학교까지 달릴 거야.
   → I _______ to school tomorrow.

2. (지금 결정) 물 가져다 줄게.
   → I _______ you some water.

3. (현재 증거) 다리가 아파. 곧 멈춰야 할 것 같아.
   → My legs hurt. I _______ stop soon.

4. (약속) 결승선에서 기다릴게.
   → I _______ at the finish line.

**Answers:**
1. am going to run
2. will get / 'll get
3. am going to have to
4. will be waiting / 'll be waiting

### C. 대화 완성 (Complete the dialogues)

1. A: "Why are you wearing running shoes?"
   B: "I _______ (run). I decided this morning."
   → **am going to run**

2. A: "Oh no, I forgot my water bottle!"
   B: "Don't worry. I _______ (share) mine."
   → **will share / 'll share**

3. A: "What are you doing this weekend?"
   B: "I _______ (train) for the marathon."
   → **am going to train**

4. A: "I can't carry all these bags."
   B: "Here, I _______ (help) you."
   → **will help / 'll help**

### D. 틀린 부분 고치기 (Error Correction)

1. Look! He will fall!
   → Look! He**'s going to** fall!

2. I'm going to answer the phone. (방금 울린 전화)
   → I**'ll** answer the phone.

3. I'll visit my friend tomorrow. (지난주에 약속함)
   → I**'m going to** visit my friend tomorrow.

4. It's getting dark. It will rain.
   → It's getting dark. It**'s going to** rain.

## Decision Tree (결정 트리)

```
미래를 말하려고 할 때:

1. 지금 결정했나요?
   → Yes: Will (I'll help you.)
   → No: 계속...

2. 이미 계획했나요?
   → Yes: Be going to (I'm going to run.)
   → No: 계속...

3. 지금 보이는 증거가 있나요?
   → Yes: Be going to (It's going to rain.)
   → No: 계속...

4. 일반적인 예측인가요?
   → Yes: Will or Be going to (둘 다 가능)

5. 약속이나 제안인가요?
   → Yes: Will (I'll be there.)
```
