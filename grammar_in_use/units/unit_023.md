# Unit 23: I will and I'm going to

## A. 문법 설명 (Grammar Explanation)

**Will**과 **be going to** 모두 미래를 나타내지만, 용법이 다릅니다.

### 핵심 차이:

| Will | Be going to |
|------|-------------|
| 말하는 순간 결정 (즉석 결정) | 이미 결정한 계획 |
| 일반적인 예측 | 현재 증거 기반 예측 |

### 예측에서의 차이:
- **Will**: 일반적인 의견, 생각
- **Be going to**: 지금 보이는 증거를 바탕으로

---

## B. 소설 속 예문 (Examples from the Novel)

### Example 1: 즉석 결정 (Will)

> **"I'll take these."** (running shoes)
> 이걸로 할게요.

- 가게에서 신발을 보고 그 자리에서 결정
- **Will** = 즉석 결정

### Example 2: 이미 결정한 계획 (Be going to)

> **"I'm going to run 10km tomorrow."**
> 내일 10km 달릴 거야.

- 이미 마음먹고 있던 계획
- **Be going to** = 미리 결정한 의도

### Example 3: 일반적 예측 (Will)

> **"I think I'll feel better soon."**
> 곧 나아질 것 같아.

- 특별한 증거 없이 일반적인 생각
- **Will** = 의견/예측

### Example 4: 증거 기반 예측 (Be going to)

> **"My legs are heavy. I'm going to slow down."**
> 다리가 무거워. 속도가 느려질 것 같아.

- 지금 다리가 무거운 것(증거)을 보고 예측
- **Be going to** = 현재 상황 기반 예측

### Example 5: 약속/제안 (Will)

> **"I'll be waiting at the finish line."**
> 결승선에서 기다릴게.

- 약속
- **Will** = 약속/제안

---

## C. 추가 설명 (Additional Points)

### 둘 다 사용 가능한 경우

미래에 대한 일반적인 예측에서는 둘 다 가능:
> "I think the race **will be** hard."
> "I think the race **is going to be** hard."

### 대화에서의 자연스러운 흐름

A: "What's your plan for tomorrow?"
B: "I**'m going to** run 10km." (이미 결정된 계획)

A: "Oh, I'm thirsty."
B: "I**'ll** get you some water." (지금 결정)

---

## D. 비교 (Comparison)

| 상황 | Will | Be going to |
|------|------|-------------|
| 즉석 결정 | I'll help you. | X |
| 미리 결정한 계획 | X | I'm going to run tomorrow. |
| 약속/제안 | I'll be there. | X |
| 일반 예측 | I think it'll rain. | O (둘 다 가능) |
| 증거 기반 예측 | X | Look! It's going to rain. |

### 틀린 예 vs 맞는 예

| 틀린 예 | 맞는 예 | 이유 |
|---------|---------|------|
| "I'm going to take these." (가게에서 즉석 결정) | "I'll take these." | 즉석 결정은 will |
| "I'll run tomorrow." (이미 계획함) | "I'm going to run tomorrow." | 미리 결정한 계획은 be going to |

## Exercises

### A. Will vs Be going to 선택

1. A: "There's no milk." B: "I know. I _______ (get) some later." (이미 알고 있었고 계획함)
2. A: "There's no milk." B: "Oh, I _______ (go) get some now." (지금 결정)
3. Look at those clouds! It _______ (rain).
4. I think the exam _______ (be) difficult.
5. I _______ (run) the marathon in March. I've been training for months.

**Answers:**
1. am going to get
2. will go / 'll go
3. is going to rain
4. will be (또는 is going to be)
5. am going to run

### B. 상황에 맞게 문장 완성

1. (미리 계획) 내일 학교까지 달릴 거야.
   → I _______ to school tomorrow.

2. (지금 결정) 물 가져다 줄게.
   → I _______ you some water.

3. (현재 증거) 다리가 아파. 곧 멈춰야 할 것 같아.
   → My legs hurt. I _______ stop soon.

**Answers:**
1. am going to run
2. will get / 'll get
3. am going to have to

### C. 대화 완성

A: "Why are you wearing running shoes?"
B: "I _______ (run). I decided this morning." → **am going to run**

A: "Oh no, I forgot my water bottle!"
B: "Don't worry. I _______ (share) mine." → **will share / 'll share**
