# Unit 35: Had better / It's time...

## A. 문법 설명 (Grammar Explanation)

### Had better: 강한 조언/경고

"~하는 게 좋겠다" - should보다 강한 표현, 안 하면 문제가 생길 수 있음을 암시

> "You**'d better** rest." = "You **had better** rest."
> 쉬는 게 좋겠어. (안 쉬면 안 좋을 수 있음)

### It's time: ~할 시간이다

1. **It's time + to + 동사원형**: 일반적인 표현
2. **It's time + 주어 + 과거동사**: 늦었다는 뉘앙스 (가정법)

> "It's time **to go**." (갈 시간이야)
> "It's time **we went**." (이제 가야 해 - 늦었다는 느낌)

---

## B. 소설 속 예문 (Examples from the Novel)

### Example 1: Had better (강한 조언)

> **"You'd better go to bed early."**
> 일찍 자는 게 좋겠어.

- 안 자면 내일 힘들 것이라는 암시

### Example 2: Had better not (경고)

> **"You'd better not skip breakfast before the race."**
> 대회 전에 아침 거르지 않는 게 좋겠어.

- 거르면 문제가 생길 수 있음

### Example 3: It's time to + 동사원형

> **"It's time to start running."**
> 달리기 시작할 시간이야.

- 일반적인 시간 표현

### Example 4: It's time + 과거동사 (가정법)

> **"It's time I started training seriously."**
> 이제 진지하게 훈련을 시작해야 해.

- 이미 시작했어야 했다는 뉘앙스

### Example 5: It's about time

> **"It's about time you stopped making excuses."**
> 이제 변명 좀 그만해야지.

- 더 강한 표현, 이미 늦었다는 느낌

---

## C. 추가 설명 (Additional Points)

### Had better의 특징

1. **과거형이 아님**: "had"가 있지만 현재/미래를 말함
2. **축약형**: I'd better, You'd better (대화에서 자주 사용)
3. **부정형**: had better not (hadn't better는 잘 안 씀)

### Had better vs Should

| Had better | Should |
|------------|--------|
| 강한 조언/경고 | 일반적인 조언 |
| 안 하면 문제 | 권유 |
| You'd better rest. | You should rest. |
| (안 쉬면 안 좋아) | (쉬는 게 좋아) |

### It's time 구조

| 구조 | 예문 | 뉘앙스 |
|------|------|--------|
| It's time to + 동사원형 | It's time to go. | 중립적 |
| It's time + 과거동사 | It's time we went. | 늦었음 |
| It's about time + 과거동사 | It's about time we went. | 더 늦었음 |
| It's high time + 과거동사 | It's high time we went. | 매우 늦었음 |

---

## D. 비교 (Comparison)

| 표현 | 강도 | 예문 |
|------|------|------|
| should | 약함 | You should go to bed. |
| had better | 강함 | You'd better go to bed. |
| must | 매우 강함 | You must go to bed. |

### It's time 비교

| 표현 | 강도 |
|------|------|
| It's time to go. | 중립 |
| It's time we went. | 약간 늦음 |
| It's about time we went. | 꽤 늦음 |
| It's high time we went. | 매우 늦음 |

## Exercises

### A. Had better / Had better not 완성

1. 일찍 자는 게 좋겠어.
   → You _______ to bed early.

2. 대회 전에 무리하지 않는 게 좋겠어.
   → You _______ too hard before the race.

3. 물 많이 마시는 게 좋겠어.
   → You _______ plenty of water.

**Answers:**
1. had better go / 'd better go
2. had better not push / 'd better not push
3. had better drink / 'd better drink

### B. It's time 문장 완성

1. 달리기 시작할 시간이야. (일반)
   → It's time _______ running.

2. 이제 진지하게 훈련해야 해. (늦었음)
   → It's time I _______ seriously.

3. 이제 좀 쉬어야지. (늦었음)
   → It's about time you _______.

**Answers:**
1. to start
2. started training / trained
3. rested / took a rest

### C. Should vs Had better 선택

1. You _______ try running earlier. (일반 조언)
2. You _______ leave now or you'll miss the bus. (경고)
3. I think you _______ drink more water. (의견)
4. You _______ not run on that injured leg. (강한 경고)

**Answers:**
1. should
2. had better / 'd better
3. should
4. had better / 'd better
