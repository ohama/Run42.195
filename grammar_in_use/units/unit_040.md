# Unit 40: If I had known... / I wish I had known...

## A. 과거 사실과 반대되는 가정 (Third Conditional)

**If + had + p.p.** = 과거 사실과 **반대**되는 상황을 가정

### 핵심 원리:

| 과거 사실 | 가정 (If I had...) | 결과 |
|-----------|-------------------|------|
| I **didn't** wake up early. | If I **had woken** up early... | ...I wouldn't have been late. |
| I **missed** the bus. | If I **hadn't missed** the bus... | ...I would have arrived on time. |
| I **didn't** know about running. | If I **had known** about running... | ...I would have started earlier. |
| I **got** carsick. | If I **hadn't gotten** carsick... | ...I would have felt better. |

### 구조:
```
If + had + p.p., ... would/could/might + have + p.p.
```

---

## B. If I had known... (과거 사실의 반대)

### From the Original Book:

> "**If I had known** you were coming, **I would have met** you at the airport."
> 네가 온다는 걸 알았으면 공항에서 마중했을 텐데. (몰랐다)

> "**If I had seen** him, **I would have said** hello."
> 그를 봤으면 인사했을 텐데. (못 봤다)

> "**If we hadn't taken** a taxi, **we would have missed** the plane."
> 택시를 안 탔으면 비행기를 놓쳤을 거야. (택시를 탔다)

### Examples from the Novel:

> **"If I had woken up earlier, I wouldn't have missed the bus."**
> 더 일찍 일어났으면 버스를 안 놓쳤을 텐데. (늦게 일어났다)

- 과거 사실: I woke up late and missed the bus.
- 가정: If I had woken up earlier...

> **"If I hadn't missed the bus, I wouldn't have started walking."**
> 버스를 안 놓쳤으면 걷기 시작하지 않았을 텐데. (놓쳤다)

> **"If I had known about the shortcut, I would have arrived earlier."**
> 지름길을 알았으면 더 일찍 도착했을 텐데. (몰랐다)

> **"If I hadn't stayed up late watching YouTube, I would have heard my alarm."**
> 유튜브 보느라 늦게까지 안 있었으면 알람 소리를 들었을 텐데.

### 가정의 결과절 표현:

| 조동사 | 의미 | 예문 |
|--------|------|------|
| would have p.p. | ~했을 텐데 | I **would have gone** if I had known. |
| could have p.p. | ~할 수 있었을 텐데 | I **could have helped** if you had asked. |
| might have p.p. | ~했을지도 모를 텐데 | She **might have agreed** if I had tried. |

## C. I wish I had known... (과거에 대한 후회)

**I wish + had + p.p.** = 과거 일에 대한 아쉬움/후회

### From the Original Book:

> "**I wish I hadn't said** that."
> 그 말을 안 했으면 좋았을 텐데. (말했다)

> "**I wish I had studied** harder."
> 더 열심히 공부했으면 좋았을 텐데. (안 했다)

> "**I wish I had known** about the party."
> 파티에 대해 알았으면 좋았을 텐데. (몰랐다)

### 과거 사실과 I wish 비교:

| 과거 사실 | I wish... |
|-----------|-----------|
| I didn't wake up early. | I **wish** I **had woken** up early. |
| I missed the bus. | I **wish** I **hadn't missed** the bus. |
| I stayed up late. | I **wish** I **hadn't stayed** up late. |
| I didn't know. | I **wish** I **had known**. |
| I said something stupid. | I **wish** I **hadn't said** that. |

### Examples from the Novel:

> **"I wish I had set my alarm earlier."**
> 알람을 더 일찍 맞춰놨으면 좋았을 텐데.

> **"I wish I hadn't stayed up late watching YouTube."**
> 유튜브 보느라 늦게까지 안 있었으면 좋았을 텐데.

> **"I wish I had started running sooner."**
> 달리기를 더 일찍 시작했으면 좋았을 텐데.

> **"I wish I hadn't been so weak before."**
> 전에 그렇게 약하지 않았으면 좋았을 텐데.

## D. 가정법 시제 비교 (현재 vs 과거)

| Type | If절 시제 | 결과절 | 의미 | 시점 |
|------|-----------|--------|------|------|
| **현재 가정** | If + 과거 | would + 동사원형 | 현재 반대 | 지금 |
| **과거 가정** | If + had p.p. | would have + p.p. | 과거 반대 | 그때 |

### 비교 예문:

**현재 가정 (지금 모른다):**
> "If I **knew** how to run, I **would** run a marathon."
> 달리는 법을 알면 마라톤을 뛸 텐데. (지금 모른다)

**과거 가정 (그때 몰랐다):**
> "If I **had known** how to run, I **would have run** a marathon."
> 달리는 법을 알았으면 마라톤을 뛰었을 텐데. (그때 몰랐다)

### 더 많은 비교:

| 현재 가정 | 과거 가정 |
|-----------|-----------|
| If I **had** time, I **would** go. | If I **had had** time, I **would have gone**. |
| (지금 시간이 있으면 갈 텐데) | (그때 시간이 있었으면 갔을 텐데) |
| If she **were** here, I **would** ask. | If she **had been** here, I **would have asked**. |
| (지금 여기 있으면 물어볼 텐데) | (그때 여기 있었으면 물어봤을 텐데) |

## E. If only I had... (후회의 강한 표현)

**If only + had p.p.** = I wish had (더 강한 후회)

### From the Original Book:

> "**If only I had listened** to her advice!"
> 그녀의 조언을 들었으면 좋았을 텐데!

> "**If only I hadn't lost** my phone!"
> 휴대폰을 안 잃어버렸으면 좋았을 텐데!

### Examples from the Novel:

> **"If only I had woken up earlier!"**
> 더 일찍 일어났으면 좋았을 텐데!

> **"If only I hadn't missed the bus!"**
> 버스를 안 놓쳤으면 좋았을 텐데!

> **"If only I had started running sooner!"**
> 달리기를 더 일찍 시작했으면 좋았을 텐데!

### I wish vs If only (과거):

| I wish | If only |
|--------|---------|
| I wish I had known. | If only I had known! |
| 후회/아쉬움 | 더 강한 후회/탄식 |

---

## F. Should have done vs If I had done

둘 다 과거에 대한 후회를 표현하지만 뉘앙스가 다릅니다.

| 표현 | 의미 | 초점 |
|------|------|------|
| **should have done** | ~했어야 했다 | 의무/당위 |
| **If I had done** | ~했으면 ...했을 텐데 | 가정 + 결과 |

### 비교 예문:

> **"I should have woken up earlier."**
> 더 일찍 일어났어야 했다. (의무/당위)

> **"If I had woken up earlier, I wouldn't have missed the bus."**
> 더 일찍 일어났으면 버스를 안 놓쳤을 텐데. (가정 + 결과)

### From the Novel:

> **"I should have trained harder."**
> 더 열심히 훈련했어야 했다.

> **"If I had trained harder, I would have run faster."**
> 더 열심히 훈련했으면 더 빨리 달렸을 텐데.

## G. 흔한 실수 (Common Mistakes)

| 틀린 문장 (X) | 올바른 문장 (O) | 설명 |
|---------------|-----------------|------|
| If I **would have known**... | If I **had known**... | if절에 would 사용 안 함 |
| If I **knew**, I would have told. | If I **had known**, I would have told. | 과거 가정은 had + p.p. |
| I wish I **know** earlier. | I wish I **had known** earlier. | 과거 후회는 had + p.p. |
| If I **had went**... | If I **had gone**... | 과거분사 형태 확인 |
| I wish I **didn't say** that. | I wish I **hadn't said** that. | 과거 후회는 had + p.p. |
| If I **have known**... | If I **had known**... | have가 아닌 had |

### 시제 혼동 (가장 흔한 오류):

```
(X) If I knew about it yesterday, I would have come.
(O) If I had known about it yesterday, I would have come.

과거 사실에 대한 가정 = If + had + p.p.
```

### 현재 가정과 과거 가정 구분:

```
현재: If I had money NOW, I would buy it.
      (지금 돈이 있으면 살 텐데)

과거: If I had had money THEN, I would have bought it.
      (그때 돈이 있었으면 샀을 텐데)
```

## Exercises

### A. 과거 사실을 보고 If I had... 문장을 만드세요.

1. I didn't hear my alarm.
   → If I _______ (hear) my alarm, I wouldn't have been late.

2. I stayed up late.
   → If I _______ (not stay) up late, I would have woken up on time.

3. I missed the bus.
   → If I _______ (not miss) the bus, I wouldn't have walked.

4. I didn't know about the shortcut.
   → If I _______ (know) about the shortcut, I would have arrived earlier.

5. It rained that day.
   → If it _______ (not rain), I would have run outside.

6. She didn't tell me.
   → If she _______ (tell) me, I would have known.

7. We took a taxi.
   → If we _______ (not take) a taxi, we would have been late.

8. He didn't study.
   → If he _______ (study), he would have passed.

**Answers:**
1. had heard
2. hadn't stayed
3. hadn't missed
4. had known
5. hadn't rained
6. had told
7. hadn't taken
8. had studied

### B. I wish... 문장으로 바꾸세요.

1. I didn't start running earlier.
   → I wish _______________________.

2. I missed the bus that morning.
   → I wish _______________________.

3. I stayed up late watching YouTube.
   → I wish _______________________.

4. I didn't listen to my coach.
   → I wish _______________________.

5. I gave up too easily before.
   → I wish _______________________.

6. He didn't come to my race.
   → I wish _______________________.

7. I said something stupid.
   → I wish _______________________.

8. We didn't meet earlier.
   → I wish _______________________.

**Answers:**
1. I wish I had started running earlier.
2. I wish I hadn't missed the bus that morning.
3. I wish I hadn't stayed up late watching YouTube.
4. I wish I had listened to my coach.
5. I wish I hadn't given up too easily before.
6. I wish he had come to my race.
7. I wish I hadn't said something stupid.
8. I wish we had met earlier.

### C. 현재 가정과 과거 가정을 구분하세요.

1. If I _______ (know) his number, I would call him. (현재)
2. If I _______ (know) his number, I would have called him. (과거)
3. I wish I _______ (be) stronger. (현재)
4. I wish I _______ (be) stronger back then. (과거)
5. If she _______ (have) time, she would help. (현재)
6. If she _______ (have) time, she would have helped. (과거)

**Answers:**
1. knew (현재 가정)
2. had known (과거 가정)
3. were (현재 소망)
4. had been (과거 후회)
5. had (현재 가정)
6. had had (과거 가정)

### D. 올바른 형태를 고르세요.

1. If I (had known / knew) about the party, I (would go / would have gone).
2. I wish I (didn't say / hadn't said) that yesterday.
3. If she (had been / was) there, I (would ask / would have asked) her.
4. I wish I (studied / had studied) harder when I was young.
5. If only I (listened / had listened) to my parents!

**Answers:**
1. had known / would have gone
2. hadn't said
3. had been / would have asked
4. had studied
5. had listened

## Summary: 가정법 과거완료 정리

### 구조 비교표:

| 구조 | 형태 | 의미 |
|------|------|------|
| If + had p.p. | If + had + p.p., would have + p.p. | 과거 조건 + 결과 |
| I wish | I wish + had + p.p. | 과거 후회/아쉬움 |
| If only | If only + had + p.p. | 강한 과거 후회 |

### 공식:
```
과거 사실의 반대를 가정할 때:

If + HAD + P.P., ... would/could/might + HAVE + P.P.
I wish + HAD + P.P.
If only + HAD + P.P.

예: 과거 사실: I missed the bus.
→ If I hadn't missed the bus, I wouldn't have walked.
→ I wish I hadn't missed the bus.
→ If only I hadn't missed the bus!
```

### 시제 비교 요약:

| 시점 | If절 | 결과절 | 예문 |
|------|------|--------|------|
| 현재 | If + 과거 | would + 동사 | If I knew, I would tell you. |
| 과거 | If + had p.p. | would have + p.p. | If I had known, I would have told you. |
