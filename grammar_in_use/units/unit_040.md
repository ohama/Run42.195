# Unit 40: If I had known... / I wish I had known...

## A. 과거 사실과 반대되는 가정 (Third Conditional)

**If + had + p.p.** = 과거 사실과 **반대**되는 상황을 가정

| 과거 사실 | 가정 (If I had...) |
|-----------|-------------------|
| I **didn't** wake up early. | If I **had woken** up early... |
| I **missed** the bus. | If I **hadn't missed** the bus... |
| I **didn't** know about running. | If I **had known** about running... |
| I **got** carsick. | If I **hadn't gotten** carsick... |

---

## B. If I had known... (과거 사실의 반대)

과거에 일어난 일의 **반대**를 가정합니다.

### Examples from the Novel:

> **"If I had woken up earlier, I wouldn't have missed the bus."**
> 더 일찍 일어났으면 버스를 안 놓쳤을 텐데. (늦게 일어났다)

- 과거 사실: I woke up late and missed the bus.
- 가정: If I had woken up earlier...

> **"If I hadn't missed the bus, I wouldn't have started walking."**
> 버스를 안 놓쳤으면 걷기 시작하지 않았을 텐데. (놓쳤다)

> **"If I had known about the shortcut, I would have arrived earlier."**
> 지름길을 알았으면 더 일찍 도착했을 텐데. (몰랐다)

> **"If I hadn't stayed up late watching YouTube, I would have heard my alarm."**
> 유튜브 보느라 늦게까지 안 있었으면 알람 소리를 들었을 텐데.

---

## C. I wish I had known... (과거에 대한 후회)

**I wish + had + p.p.** = 과거 일에 대한 아쉬움/후회

| 과거 사실 | I wish... |
|-----------|-----------|
| I didn't wake up early. | I **wish** I **had woken** up early. |
| I missed the bus. | I **wish** I **hadn't missed** the bus. |
| I stayed up late. | I **wish** I **hadn't stayed** up late. |
| I didn't know. | I **wish** I **had known**. |

### Examples from the Novel:

> **"I wish I had set my alarm earlier."**
> 알람을 더 일찍 맞춰놨으면 좋았을 텐데.

> **"I wish I hadn't stayed up late watching YouTube."**
> 유튜브 보느라 늦게까지 안 있었으면 좋았을 텐데.

> **"I wish I had started running sooner."**
> 달리기를 더 일찍 시작했으면 좋았을 텐데.

> **"I wish I hadn't been so weak before."**
> 전에 그렇게 약하지 않았으면 좋았을 텐데.

---

## D. 가정법 시제 비교

| Type | 시제 | 의미 | 예문 |
|------|------|------|------|
| 현재 가정 | If + 과거 | 현재 반대 | If I **knew**, I would tell you. |
| 과거 가정 | If + had p.p. | 과거 반대 | If I **had known**, I would have told you. |

### 비교 예문:

**현재 가정 (지금 모른다):**
> "If I **knew** how to run, I **would** run a marathon."
> 달리는 법을 알면 마라톤을 뛸 텐데. (지금 모른다)

**과거 가정 (그때 몰랐다):**
> "If I **had known** how to run, I **would have run** a marathon."
> 달리는 법을 알았으면 마라톤을 뛰었을 텐데. (그때 몰랐다)

---

## E. If only I had... (후회의 강한 표현)

**If only + had p.p.** = I wish had (더 강한 후회)

> **"If only I had woken up earlier!"**
> 더 일찍 일어났으면 좋았을 텐데!

> **"If only I hadn't missed the bus!"**
> 버스를 안 놓쳤으면 좋았을 텐데!

> **"If only I had started running sooner!"**
> 달리기를 더 일찍 시작했으면 좋았을 텐데!

---

## F. Should have done vs If I had done

둘 다 과거에 대한 후회를 표현하지만 뉘앙스가 다릅니다.

| Should have done | If I had done |
|------------------|---------------|
| ~했어야 했다 (의무/당위) | ~했으면 ...했을 텐데 (가정) |

> **"I should have woken up earlier."**
> 더 일찍 일어났어야 했다. (의무)

> **"If I had woken up earlier, I wouldn't have missed the bus."**
> 더 일찍 일어났으면 버스를 안 놓쳤을 텐데. (가정+결과)

## Exercises

### A. 과거 사실을 보고 If I had... 문장을 만드세요.

1. I didn't hear my alarm.
   → If I _______ (hear) my alarm, I wouldn't have been late.

2. I stayed up late.
   → If I _______ (not stay) up late, I would have woken up on time.

3. I missed the bus.
   → If I _______ (not miss) the bus, I wouldn't have walked.

4. I didn't know about the shortcut.
   → If I _______ (know) about the shortcut, I would have arrived earlier.

5. It rained that day.
   → If it _______ (not rain), I would have run outside.

**Answers:**
1. had heard
2. hadn't stayed
3. hadn't missed
4. had known
5. hadn't rained

### B. I wish... 문장으로 바꾸세요.

1. I didn't start running earlier.
   → I wish _______________________.

2. I missed the bus that morning.
   → I wish _______________________.

3. I stayed up late watching YouTube.
   → I wish _______________________.

4. I didn't listen to my coach.
   → I wish _______________________.

5. I gave up too easily before.
   → I wish _______________________.

**Answers:**
1. I wish I had started running earlier.
2. I wish I hadn't missed the bus that morning.
3. I wish I hadn't stayed up late watching YouTube.
4. I wish I had listened to my coach.
5. I wish I hadn't given up too easily before.

### C. 현재 가정과 과거 가정을 구분하세요.

1. If I _______ (know) his number, I would call him. (현재)
2. If I _______ (know) his number, I would have called him. (과거)
3. I wish I _______ (be) stronger. (현재)
4. I wish I _______ (be) stronger back then. (과거)

**Answers:**
1. knew (현재 가정)
2. had known (과거 가정)
3. were (현재 소망)
4. had been (과거 후회)

## Summary: 가정법 과거완료

```
과거 사실의 반대를 가정할 때:

If + HAD + P.P., ... would/could/might + HAVE + P.P.
I wish + HAD + P.P.
If only + HAD + P.P.

예: 과거 사실: I missed the bus.
→ If I hadn't missed the bus, I wouldn't have walked.
→ I wish I hadn't missed the bus.
→ If only I hadn't missed the bus!
```
