# Unit 102: So and Such

## A. so와 such의 차이 (The Difference)

**so**와 **such**는 모두 "매우, 정말"의 의미로 강조할 때 사용하지만, 뒤에 오는 구조가 다릅니다.

### Core Difference:

| 단어 | 구조 | 예시 |
|------|------|------|
| **so** | so + 형용사/부사 | so tired, so fast |
| **such** | such + (a/an) + 명사구 | such a long race |

### From the Original Book:

> "The weather was **so** nice."
> = "It was **such** nice weather."

> "She's **so** patient."
> = "She's **such** a patient person."

---

## B. so + 형용사/부사 (so + adjective/adverb)

### Structure:

```
so + 형용사: I was so tired.
so + 부사: He ran so fast.
```

### From the Original Book:

> "Everything happened **so** quickly."
> 모든 것이 너무 빨리 일어났다.

> "It was **so** hot that I couldn't sleep."
> 너무 더워서 잠을 잘 수가 없었다.

### Examples from the Novel:

> **"It was so hard."**
> 정말 힘들었다.

> **"I was so tired after the race."**
> 대회 후에 너무 피곤했다.

> **"The scenery was so beautiful."**
> 경치가 너무 아름다웠다.

> **"I ran so fast that I got there early."**
> 너무 빨리 달려서 일찍 도착했다.

> **"The bus was so crowded."**
> 버스가 너무 붐볐다.

> **"I was so happy when I finished."**
> 완주했을 때 너무 행복했다.

---

## C. such + (a/an) + 명사구 (such + noun phrase)

### Structure:

```
such + a/an + 형용사 + 단수명사: such a long race
such + 형용사 + 불가산명사: such hard work
such + 형용사 + 복수명사: such nice people
```

### From the Original Book:

> "It was **such** a good book."
> 정말 좋은 책이었다.

> "I've had **such** a busy day."
> 정말 바쁜 하루였다.

> "Why did you buy **such** an expensive coat?"
> 왜 그렇게 비싼 코트를 샀어?

### Examples from the Novel:

> **"It was such a long race."**
> 정말 긴 대회였다.

- such + a + 형용사 + 명사

> **"It was such a beautiful morning."**
> 정말 아름다운 아침이었다.

> **"He is such a kind grandfather."**
> 그는 정말 친절한 할아버지다.

> **"It was such hard work."**
> 정말 힘든 일이었다.

- such + 형용사 + 불가산명사 (a 없음)

> **"There were such friendly people at the race."**
> 대회에 정말 친절한 사람들이 있었다.

- such + 형용사 + 복수명사 (a 없음)

---

## D. so vs such 비교 (Comparison)

### Conversion Table:

| so + 형용사 | such + 명사구 |
|-------------|---------------|
| so **tired** | such **a tiring race** |
| so **beautiful** | such **a beautiful view** |
| so **hard** | such **hard work** |
| so **nice** | such **a nice person** |
| so **expensive** | such **an expensive coat** |

### With many/much vs a lot of:

| so | such |
|----|------|
| so **many** people | such **a lot of** people |
| so **much** effort | such **a lot of** effort |
| so **few** friends | such **a small number of** friends |
| so **little** time | such **a small amount of** time |

### From the Original Book:

> "There were **so many** people in the room."
> = "There were **such a lot of** people in the room."

---

## E. so...that / such...that (결과절)

**that**절을 사용하여 결과를 나타낼 수 있습니다.

### Pattern:

```
so + 형용사 + that + 결과
I was so tired that I couldn't move.

such + (a) + 명사구 + that + 결과
It was such a hard race that I almost gave up.
```

### From the Original Book:

> "It was **so** cold **that** we didn't go out."
> 너무 추워서 밖에 나가지 않았다.

> "It was **such** nice weather **that** we went to the beach."
> 날씨가 너무 좋아서 해변에 갔다.

### Examples from the Novel:

> **"I was so exhausted that I collapsed at the finish line."**
> 너무 지쳐서 결승선에서 쓰러졌다.

> **"It was such a hard race that I thought about quitting."**
> 너무 힘든 대회여서 포기할까 생각했다.

> **"The view was so beautiful that I stopped to look."**
> 경치가 너무 아름다워서 멈춰서 봤다.

> **"It was such a proud moment that I cried."**
> 너무 자랑스러운 순간이어서 울었다.

> **"There were so many runners that I couldn't see the start line."**
> 주자가 너무 많아서 출발선이 보이지 않았다.

---

## F. so의 다른 용법 (Other uses of 'so')

### 1. so = 그래서 (therefore)

> **"I was tired, so I rested."**
> 피곤해서 쉬었다.

> **"The bus was late, so I walked."**
> 버스가 늦어서 걸었다.

### 2. I think so / I hope so / I don't think so

| Expression | Meaning | Korean |
|------------|---------|--------|
| I think so. | 그런 것 같아. | 그렇게 생각해. |
| I hope so. | 그랬으면 좋겠어. | 그러길 바라. |
| I don't think so. | 그렇게 생각하지 않아. | 그렇지 않을 것 같아. |
| I'm afraid so. | 유감이지만 그래. | 안타깝지만 그래. |

### From the Original Book:

> "Will you be ready by tomorrow?" "I **think so**."
> "내일까지 준비될 거야?" "그런 것 같아."

### Examples from the Novel:

> **"Will you finish the marathon?" "I hope so."**
> "마라톤 완주할 거야?" "그랬으면 좋겠어."

> **"Is it going to rain?" "I don't think so."**
> "비 올까?" "그렇지 않을 것 같아."

## G. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | 설명 |
|-----------|-------------|------|
| ~~such tired~~ | **so** tired | 형용사 앞 → so |
| ~~so a long race~~ | **such** a long race | a + 명사 앞 → such |
| ~~such hard~~ | **so** hard | 형용사만 있으면 → so |
| ~~so a nice day~~ | **such** a nice day | a + 형용사 + 명사 → such |
| ~~such beautiful~~ | **so** beautiful | 형용사만 → so |
| ~~It was so nice weather~~ | It was **such** nice weather | 명사 포함 → such |

### Decision Guide:

```
형용사/부사만? → so (so tired, so fast)
명사 포함? → such (such a race, such hard work)
```

### C. so 또는 such를 사용하여 두 가지 방법으로 바꾸세요 (Rewrite in two ways)

1. The race was very hard.
   → The race was **so hard**.
   → It was **such a hard race**.

2. The morning was very beautiful.
   → The morning was **so beautiful**.
   → It was **such a beautiful morning**.

3. There were very many runners.
   → There were **so many** runners.
   → (such a lot of runners)

4. The medal was very heavy.
   → The medal was **so heavy**.
   → It was **such a heavy medal**.

5. The grandfather is very kind.
   → The grandfather is **so kind**.
   → He is **such a kind grandfather**.

---

### D. 틀린 부분을 고치세요 (Error correction)

1. It was so nice day that I decided to walk.
   → It was **such a** nice day that I decided to walk.

2. I was such happy that I cried.
   → I was **so** happy that I cried.

3. There was such much traffic.
   → There was **so** much traffic.

4. He gave me so good advice.
   → He gave me **such** good advice.

5. It was such difficult race.
   → It was **such a** difficult race.

6. I feel so a tired person.
   → I feel **so** tired. / I'm **such a** tired person.

## Summary

### so vs such Comparison:

| | so | such |
|---|-----|------|
| 구조 | so + 형용사/부사 | such + (a/an) + 명사 |
| 예시 | so tired | such a long race |
| many/much | so many/much | such a lot of |
| that절 | so...that | such...that |

### Quick Decision:

| 뒤에 오는 것 | 사용 | 예시 |
|--------------|------|------|
| 형용사만 | so | so tired |
| a/an + 명사 | such | such a nice day |
| 불가산명사 | such | such nice weather |
| 복수명사 | such | such nice people |
| many/much | so | so many people |

### Pattern Summary:

```
so + 형용사: so tired
so + 부사: so quickly
so + many/much: so many people

such + a + 형용사 + 단수명사: such a nice day
such + 형용사 + 불가산명사: such nice weather
such + 형용사 + 복수명사: such nice people
```
