# Unit 102: So and Such

## A. so와 such의 차이

**so**와 **such**는 모두 "매우, 정말"의 의미로 강조할 때 사용하지만, 뒤에 오는 구조가 다릅니다.

| 단어 | 구조 | 예시 |
|------|------|------|
| **so** | so + 형용사/부사 | so tired, so fast |
| **such** | such + (a/an) + 명사구 | such a long race |

---

## B. so + 형용사/부사

### Examples from the Novel:

> **"It was so hard."**
> 정말 힘들었다.

> **"I was so tired after the race."**
> 대회 후에 너무 피곤했다.

> **"The scenery was so beautiful."**
> 경치가 너무 아름다웠다.

> **"I ran so fast that I got there early."**
> 너무 빨리 달려서 일찍 도착했다.

> **"The bus was so crowded."**
> 버스가 너무 붐볐다.

### Pattern:
```
so + 형용사 + that + 결과
I was so tired that I couldn't move.
```

---

## C. such + (a/an) + 명사구

### Examples from the Novel:

> **"It was such a long race."**
> 정말 긴 대회였다.

- such + a + 형용사 + 명사

> **"It was such a beautiful morning."**
> 정말 아름다운 아침이었다.

> **"He is such a kind grandfather."**
> 그는 정말 친절한 할아버지다.

> **"It was such hard work."**
> 정말 힘든 일이었다.

- such + 형용사 + 불가산명사 (a 없음)

> **"There were such friendly people at the race."**
> 대회에 정말 친절한 사람들이 있었다.

- such + 형용사 + 복수명사 (a 없음)

### Pattern:
```
such + a/an + 형용사 + 단수명사 + that + 결과
It was such a hard race that I almost gave up.
```

---

## D. so vs such 비교

| so | such |
|----|------|
| so **tired** | such **a tiring race** |
| so **beautiful** | such **a beautiful view** |
| so **hard** | such **hard work** |
| so **many** people | such **friendly** people |
| so **much** pain | such **terrible** pain |

### 주의: so many/much vs such a lot of

| so | such |
|----|------|
| so **many** runners | such **a lot of** runners |
| so **much** effort | such **a lot of** effort |

---

## E. so...that / such...that (결과절)

**that**절을 사용하여 결과를 나타낼 수 있습니다.

### Examples from the Novel:

> **"I was so exhausted that I collapsed at the finish line."**
> 너무 지쳐서 결승선에서 쓰러졌다.

> **"It was such a hard race that I thought about quitting."**
> 너무 힘든 대회여서 포기할까 생각했다.

> **"The view was so beautiful that I stopped to look."**
> 경치가 너무 아름다워서 멈춰서 봤다.

> **"It was such a proud moment that I cried."**
> 너무 자랑스러운 순간이어서 울었다.

---

## F. so의 다른 용법

### 1. so = 그래서 (therefore)

> **"I was tired, so I rested."**
> 피곤해서 쉬었다.

### 2. I think so / I hope so

> **"Will you finish? - I think so."**
> 완주할 거야? - 그럴 것 같아.

### Examples from the Novel:

> **"I hope so."**
> 그랬으면 좋겠다.

> **"I don't think so."**
> 그렇게 생각하지 않아.

## Exercises

### A. so와 such 중 알맞은 것을 고르세요.

1. It was (so / such) a beautiful morning.
   → **such**

2. I was (so / such) tired after the race.
   → **so**

3. It was (so / such) hard work.
   → **such**

4. There were (so / such) many people.
   → **so**

5. He is (so / such) a kind person.
   → **such**

### B. 문장을 완성하세요.

1. The race was _______ hard that I almost gave up.
   → **so**

2. It was _______ a long distance that I got tired.
   → **such**

3. I ran _______ fast that I arrived early.
   → **so**

4. It was _______ a proud moment that I cried.
   → **such**

5. The bus was _______ crowded that I couldn't breathe.
   → **so**

### C. so 또는 such를 사용하여 문장을 바꾸세요.

1. The race was very hard.
   → The race was **so hard**.
   → It was **such a hard race**.

2. The morning was very beautiful.
   → The morning was **so beautiful**.
   → It was **such a beautiful morning**.

3. There were very many runners.
   → There were **so many** runners.

4. The medal was very heavy.
   → The medal was **so heavy**.
   → It was **such a heavy medal**.

### D. 빈칸을 채우세요.

1. It was _______ a nice day that I decided to walk.
   → **such**

2. I was _______ happy that I cried.
   → **so**

3. There was _______ much traffic.
   → **so**

4. He gave me _______ good advice.
   → **such**
