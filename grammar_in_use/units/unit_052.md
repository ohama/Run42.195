# Unit 52: Question Tags (do you? isn't it?, etc.)

## A. 부가 의문문이란? (What are Question Tags?)

문장 끝에 붙여서 **확인, 동의, 강조**를 구하는 짧은 질문

### From the Original Book:

> "It's a beautiful day, **isn't it**?"
> 날씨 좋지, 그렇지?

> "You speak English, **don't you**?"
> 영어 하지, 그렇지?

> "They haven't arrived, **have they**?"
> 그들 안 왔지, 그렇지?

### 기본 규칙:

| 문장 유형 | 태그 |
|-----------|------|
| 긍정문 | + 부정 태그 |
| 부정문 | + 긍정 태그 |

---

## B. 부가 의문문의 기본 규칙 (Basic Rules)

### 4가지 핵심 규칙:

1. **긍정문 → 부정 태그**
2. **부정문 → 긍정 태그**
3. **같은 조동사/be동사 사용**
4. **주어는 대명사로**

### 긍정문 + 부정 태그:

| 긍정문 | 부정 태그 | 완성 문장 |
|--------|-----------|-----------|
| You are tired, | aren't you? | You are tired, **aren't you**? |
| He can run fast, | can't he? | He can run fast, **can't he**? |
| She finished, | didn't she? | She finished, **didn't she**? |

### 부정문 + 긍정 태그:

| 부정문 | 긍정 태그 | 완성 문장 |
|--------|-----------|-----------|
| You aren't tired, | are you? | You aren't tired, **are you**? |
| He can't run fast, | can he? | He can't run fast, **can he**? |
| You didn't finish, | did you? | You didn't finish, **did you**? |

### Examples from the Novel:

> **"You're tired, aren't you?"**
> 피곤하지, 그렇지?

> **"He can run fast, can't he?"**
> 그는 빨리 달릴 수 있지, 그렇지?

> **"You finished the race, didn't you?"**
> 대회 끝냈지, 그렇지?

> **"You aren't tired, are you?"**
> 안 피곤하지, 그렇지?

> **"He can't run fast, can he?"**
> 그는 빨리 못 달리지, 그렇지?

> **"You didn't finish, did you?"**
> 못 끝냈지, 그렇지?

## C. 다양한 시제의 부가 의문문 (Tags with Different Tenses)

### From the Original Book:

| 시제 | 긍정문 + 부정 태그 |
|------|-------------------|
| 현재 be | You **are** running, **aren't** you? |
| 현재 일반 | You **run** every day, **don't** you? |
| 과거 be | It **was** hard, **wasn't** it? |
| 과거 일반 | You **finished**, **didn't** you? |
| 현재완료 | You **have** trained, **haven't** you? |
| 미래 | You **will** come, **won't** you? |
| 조동사 | You **can** run, **can't** you? |

### Examples from the Novel:

> **"The bus comes at 7:30, doesn't it?"**
> 버스 7시 30분에 오지, 그렇지?

> **"You ran this morning, didn't you?"**
> 오늘 아침에 달렸지, 그렇지?

> **"The race was hard, wasn't it?"**
> 대회 힘들었지, 그렇지?

> **"You've been training, haven't you?"**
> 훈련하고 있었지, 그렇지?

> **"You'll run tomorrow, won't you?"**
> 내일 달릴 거지, 그렇지?

## D. 특수한 경우 (Special Cases)

### 1. I am → aren't I?

> **"I'm running well, aren't I?"**
> 나 잘 달리고 있지, 그렇지?

*주의: I amn't (X) → aren't I (O)*

### 2. Let's → shall we?

> **"Let's run together, shall we?"**
> 같이 달리자, 그래?

### 3. 명령문 → will you? / won't you?

> **"Run faster, will you?"**
> 더 빨리 달려, 알았지?

> **"Don't stop, will you?"**
> 멈추지 마, 알았지?

> **"Sit down, won't you?"**
> 앉아, 그래?

### 4. There is/are → is/are there?

> **"There's a finish line ahead, isn't there?"**
> 앞에 결승선이 있지, 그렇지?

> **"There are many runners, aren't there?"**
> 주자가 많지, 그렇지?

### 5. Nobody/Nothing/No one → 긍정 태그

부정어가 있으면 이미 부정이므로 긍정 태그 사용

> **"Nobody came, did they?"**
> 아무도 안 왔지, 그렇지?

> **"Nothing happened, did it?"**
> 아무 일도 없었지, 그렇지?

> **"No one called, did they?"**
> 아무도 안 전화했지, 그렇지?

### 특수 경우 요약표:

| 주어/문장 | 태그 |
|-----------|------|
| I am | aren't I? |
| Let's | shall we? |
| 명령문 | will you? / won't you? |
| There is/are | isn't/aren't there? |
| Nobody/Nothing | did they? / did it? |

## E. 억양 (Intonation)

### From the Original Book:

| 억양 | 의미 | 예문 |
|------|------|------|
| 내림↓ | 확인 (답을 안다) | You're tired, aren't you?↓ |
| 올림↑ | 진짜 질문 (모른다) | You're coming, aren't you?↑ |

### 확인 (내림 억양):
> "You finished the race, didn't you?↓"
> (완주한 거 아는데 확인)

### 진짜 질문 (올림 억양):
> "You finished the race, didn't you?↑"
> (정말 완주했는지 모름)

### Examples from the Novel:

> **"Total hell, isn't it?"** ↓ (확인 - 둘 다 안다)
> 완전 지옥이지, 그렇지?

> **"You're running every day, aren't you?"** ↓ (확인)
> 매일 달리고 있지, 그렇지?

## F. 부정어가 포함된 문장 (Negative Words)

never, hardly, rarely, seldom, few, little 등은 부정적 의미이므로 **긍정 태그** 사용

### From the Original Book:

> "She **hardly** ever works, **does** she?"
> 그녀는 거의 일하지 않지, 그렇지?

> "He **never** comes on time, **does** he?"
> 그는 절대 제시간에 안 오지, 그렇지?

### 부정어 + 긍정 태그:

| 부정어 | 예문 |
|--------|------|
| never | He **never** runs, **does** he? |
| hardly | She **hardly** eats, **does** she? |
| rarely | They **rarely** come, **do** they? |
| seldom | He **seldom** talks, **does** he? |
| few | **Few** people came, **did** they? |
| little | **Little** was said, **was** it? |

## G. 흔한 실수 (Common Mistakes)

| 틀린 문장 (X) | 올바른 문장 (O) | 설명 |
|---------------|-----------------|------|
| You run, **aren't** you? | You run, **don't** you? | 같은 조동사 사용 |
| She finished, **doesn't** she? | She finished, **didn't** she? | 시제 일치 |
| I am right, **amn't** I? | I am right, **aren't** I? | amn't 없음 |
| Let's go, **will** we? | Let's go, **shall** we? | Let's = shall we |
| Nobody came, **didn't** they? | Nobody came, **did** they? | 부정어 = 긍정 태그 |
| There is a bus, **isn't it**? | There is a bus, **isn't there**? | There = there |

### 조동사 불일치:

```
(X) You can swim, don't you?
(O) You can swim, can't you?

같은 조동사/be동사 사용!
```

### 시제 불일치:

```
(X) He ran yesterday, doesn't he?
(O) He ran yesterday, didn't he?

시제가 일치해야 함!
```

## Summary: 부가 의문문 완전 정리

### 기본 규칙:
```
1. 긍정문 → 부정 태그
   You are running, aren't you?

2. 부정문 → 긍정 태그
   You aren't running, are you?

3. 같은 조동사/be동사 사용
4. 주어는 대명사로
```

### 특수 경우:
```
I am → aren't I?
Let's → shall we?
명령문 → will you?
There is → isn't there?
Nobody/Nothing → 긍정 태그 (did they? / did it?)
부정어 (never, hardly) → 긍정 태그
```

### 억양:
```
내림↓ = 확인 (답을 안다)
올림↑ = 진짜 질문 (모른다)
```

### 시제별 태그:

| 시제 | 긍정 태그 | 부정 태그 |
|------|-----------|-----------|
| 현재 be | is/are | isn't/aren't |
| 과거 be | was/were | wasn't/weren't |
| 현재 일반 | do/does | don't/doesn't |
| 과거 일반 | did | didn't |
| 현재완료 | have/has | haven't/hasn't |
| 미래 | will | won't |
| 조동사 | can/should/etc. | can't/shouldn't/etc. |
