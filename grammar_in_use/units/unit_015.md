# Unit 15: Past Perfect (I had done)

## A. 문법 설명 (Grammar Explanation)

### English
**Past Perfect** (had + past participle) describes an action that happened **before another past action**.

It's the "past of the past" - looking back from a past moment.

### Korean
**과거완료** (had + 과거분사)는 **과거의 어떤 시점보다 더 이전에** 일어난 일을 표현합니다.

"과거의 과거" - 과거 시점에서 더 이전을 돌아볼 때 사용합니다.

---

## B. 과거완료의 형태 (Form)

| Subject | Had | Past Participle |
|---------|-----|-----------------|
| I/You/He/She/It/We/They | had | done, walked, run |

**축약형:** I had → I'd, He had → He'd
**부정문:** had not → hadn't
**의문문:** Had + 주어 + p.p.?

---

## C. 시간 순서 이해하기 (Timeline)

```
더 이전 ─────────── 과거 ─────────── 현재
   |                  |                |
 (먼저 일어난 일)    (나중 일어난 일)
   Past Perfect       Past Simple
   had done           did
```

---

## D. 소설 속 예문 (Examples from the Novel)

### Example 1: 과거보다 더 이전의 사건

> **"When I got to school, I realized I had forgotten my lunch."**
> 학교에 도착했을 때, 도시락을 잊어버렸다는 걸 깨달았다.

- **got** (Past Simple) = 도착한 과거 시점
- **had forgotten** (Past Perfect) = 그보다 더 이전에 잊어버림

### Example 2: 과거의 경험 언급

> **"I had never walked this long before."**
> 이렇게 오래 걸어본 적이 없었다.

- 걷기 시작하기 **전까지** 그런 경험이 없었음
- **before** = 그 과거 시점 이전에

### Example 3: 연속된 과거 사건

> **"Mom had already left for work when I woke up."**
> 내가 일어났을 때, 엄마는 이미 출근하셨다.

- **woke up** = 일어난 시점 (과거)
- **had left** = 그보다 먼저 출근함 (과거완료)

### Example 4: 과거 상황의 배경

> **"The race started at 8 AM. I had trained for a whole year for this moment."**
> 대회가 8시에 시작했다. 이 순간을 위해 1년 동안 훈련해 왔었다.

- **started** = 대회 시작 (과거)
- **had trained** = 대회 전 1년간의 훈련 (과거완료)

### Example 5: 상태 변화

> **"By the time I finished the race, I had run 42 kilometers."**
> 레이스를 끝냈을 때, 나는 42km를 달렸다.

- **finished** = 끝낸 시점 (과거)
- **had run** = 그 시점까지 달린 거리 (과거완료)

## E. 과거완료 vs 과거시제 비교

### Past Perfect + Past Simple:

> **"When I arrived, the bus had already left."**
> 도착했을 때, 버스는 이미 떠났다.

- 버스가 먼저 떠남 (had left) → 내가 도착 (arrived)

### 두 과거시제 (시간 순서대로):

> **"The bus left, and then I arrived."**
> 버스가 떠났고, 그 다음 내가 도착했다.

- 두 사건을 시간 순서대로 나열

---

## F. 과거완료를 자주 쓰는 상황

### 1. before/after와 함께:

> **"I had never felt this good before."**
> 전에 이렇게 기분 좋은 적 없었다.

### 2. when/by the time과 함께:

> **"When I got there, everyone had already started."**
> 도착했을 때, 모두 이미 시작했었다.

### 3. already/just/never와 함께:

> **"I had just finished eating when Mom called."**
> 엄마가 전화했을 때 방금 다 먹었다.

### 4. 보고/깨달음과 함께:

> **"I realized I had made a mistake."**
> 실수했다는 걸 깨달았다.

## Exercises

### A. 올바른 형태를 고르세요.

1. When I arrived, the race (started / had started) already.
2. I realized I (forgot / had forgotten) my water bottle.
3. By the time I finished, I (ran / had run) 10km.
4. She (trained / had trained) for months before the race.
5. The bus (left / had left) before I got to the stop.

**Answers:**
1. had started (도착 전에 시작)
2. had forgotten (깨닫기 전에 잊어버림)
3. had run (끝낼 때까지 달린 양)
4. had trained (대회 전에 훈련)
5. had left (도착 전에 떠남)

### B. 문장 연결하기

Connect using Past Perfect:

1. I ate breakfast. Then I left for school.
   → I had eaten breakfast before I left for school.

2. The bus left. Then I arrived at the stop.
   → The bus _______ before I arrived.

3. She finished training. Then the race started.
   → She _______ before the race started.

4. I ran 5km. Then I took a break.
   → I _______ before I took a break.

**Answers:**
2. had left
3. had finished training
4. had run 5km

### C. 빈칸 채우기

1. When I woke up, Mom _______ (already / leave) for work.
2. I _______ (never / run) a marathon before that day.
3. By the time I arrived, the race _______ (start).
4. I realized I _______ (forget) to stretch.
5. She _______ (just / finish) eating when I called.

**Answers:**
1. had already left
2. had never run
3. had started
4. had forgotten
5. had just finished

### D. 문장 만들기

Make sentences using Past Perfect:

1. I / not run / so far / before
   → I had never run so far before.

2. The race / already / begin / when I arrived
   → _______

3. I / train / for a year / by the time the race started
   → _______

4. Mom / already / go to bed / when I got home
   → _______

**Answers:**
2. The race had already begun when I arrived.
3. I had trained for a year by the time the race started.
4. Mom had already gone to bed when I got home.

## 정리: 과거완료 사용 공식

| 상황 | 공식 | 예문 |
|------|------|------|
| ~했을 때, 이미 | When + Past, had + p.p. | When I arrived, it had started. |
| ~하기 전에 | had + p.p. + before + Past | I had finished before he came. |
| ~할 때까지 | By the time + Past, had + p.p. | By the time I finished, I had run 42km. |
| ~한 적 없었다 | had never + p.p. | I had never felt this good. |
| 깨달았다 ~했다는 것을 | realized + had + p.p. | I realized I had forgotten. |
