# Unit 20: (I'm) going to (do)

## A. 문법 설명 (Grammar Explanation)

**be going to + 동사원형**은 미래를 표현하는 중요한 구조입니다.

### 두 가지 주요 용법:

1. **이미 결정한 의도/계획 (Intentions)**: 말하기 전에 이미 결정한 일
2. **현재 증거에 기반한 예측 (Predictions based on present evidence)**: 지금 보이는 상황으로 미래를 예측

### 형태:
- I **am going to** run
- He/She/It **is going to** run
- We/You/They **are going to** run

---

## B. 소설 속 예문 (Examples from the Novel)

### Example 1: 이미 결정한 의도

> **"I'm going to run to school tomorrow."**
> 내일 학교까지 달릴 거야.

- 이미 결정한 계획 - 말하기 전에 마음먹은 일

### Example 2: 계획/의도

> **"I'm going to run 10km."**
> 10km를 달릴 거야.

- 미리 세운 목표/계획

### Example 3: 현재 증거 기반 예측

> **"My legs are heavy. I'm going to slow down."**
> 다리가 무거워. 속도가 느려질 것 같아.

- 현재 상태(다리가 무거움)를 보고 예측

### Example 4: 계획된 미래

> **"I'm going to finish the race."**
> 나는 대회를 완주할 거야.

- 굳은 결심/의도

### Example 5: 현재 상황 기반 예측

> **"It's getting dark. It's going to rain."**
> 어두워지고 있어. 비가 올 것 같아.

- 지금 보이는 증거로 예측

---

## C. 추가 설명 (Additional Points)

### Going to의 발음

구어체에서는 "gonna" /ˈɡɒnə/로 발음하기도 합니다:
- I'm **gonna** run = I'm going to run
- (글에서는 "going to"로 씀)

### be going to vs will

| be going to | will |
|-------------|------|
| 이미 결정한 계획 | 즉석에서 결정 |
| 현재 증거 기반 예측 | 일반적인 예측 |

### 소설에서의 변화

도윤이의 계획이 점점 구체화되는 모습:
- March: "Maybe I should walk." (불확실)
- August: "I'm going to run 10km." (확실한 계획)
- October: "I'm going to finish the race." (굳은 결심)

---

## D. 비교 (Comparison)

| be going to | 의미 | Example |
|-------------|------|---------|
| 의도/계획 | 이미 결정함 | I**'m going to** run tomorrow. |
| 예측 | 증거 기반 | Look at those clouds. It**'s going to** rain. |

### Present Continuous vs Be going to

| Present Continuous | Be going to |
|--------------------|-------------|
| 확정된 약속 | 의도/계획 |
| I**'m meeting** him at 3. | I**'m going to meet** him someday. |
| (시간/장소 확정) | (아직 정확히 안 정함) |

## Exercises

### A. 빈칸 채우기

1. I _______ (run) 10km tomorrow. (계획)
2. Look at those dark clouds! It _______ (rain).
3. He _______ (finish) the marathon. He's been training hard.
4. We _______ (meet) at the park this weekend.
5. My legs hurt. I _______ (slow down).

**Answers:**
1. am going to run
2. is going to rain
3. is going to finish
4. are going to meet
5. am going to slow down

### B. 문장 변환

1. I plan to run tomorrow.
   → I **am going to run** tomorrow.

2. He decided to join the track team.
   → He **is going to join** the track team.

3. The sky is getting dark. (rain - 예측)
   → It **is going to rain**.

### C. 올바른 형태 고르기

1. "I (will / am going to) run a marathon." - 이미 결정한 계획
   → **am going to**

2. "Look! He (will / is going to) fall!" - 현재 상황 기반 예측
   → **is going to**
