# Unit 58: Verb + -ing or to... (3) (like, would like, etc.)

## A. like / love / hate + -ing or to (Form)

**like, love, hate, prefer**는 -ing와 to부정사 둘 다 쓸 수 있습니다.
의미 차이가 거의 없지만, 뉘앙스가 약간 다릅니다.

| + -ing | + to |
|--------|------|
| 일반적인 취향/즐거움 | 특정 상황에서의 선택/습관 |
| 영국식에서 더 선호 | 미국식에서 더 선호 |

### From the Original Book:

> "I **like getting** up early." (일반적 취향)
> "I **like to get** up early on weekends." (특정 습관)

### Examples from the Novel:

> **"I don't really like running."**
> 나는 달리기를 별로 좋아하지 않는다.

- **like + -ing**: 일반적인 취향

> **"I like to run in the morning."**
> 나는 아침에 달리는 것을 좋아한다.

- **like + to**: 습관적 선택을 강조

---

## B. like vs would like (Primary Usage)

**중요한 차이!** would like는 to만 가능합니다.

| Expression | Meaning | Structure | Tense |
|------------|---------|-----------|-------|
| like | 좋아하다 (일반) | -ing 또는 to | 현재 습관 |
| would like | ~하고 싶다 (지금) | **to만 가능** | 현재의 바람 |

### From the Original Book:

> "I **like** swimming." (수영을 좋아해)
> "I **would like** to swim." (수영하고 싶어 - 지금)

### Examples from the Novel:

> **"I like walking to school."**
> 나는 걸어서 학교 가는 걸 좋아한다. (일반적 취향)

> **"I would like to run a marathon."**
> 마라톤을 뛰고 싶다. (현재의 바람)

> **"What would you like to eat?"**
> 뭐 먹고 싶어?

- **would like + to**: ~하고 싶다 (want보다 공손)

### would like vs want:

| want | would like |
|------|------------|
| 직접적 | 더 공손함 |
| I want to go. | I would like to go. |
| 친구에게 | 격식 상황에서 |

---

## C. love / hate + -ing or to (Additional Uses)

### Examples from the Novel:

> **"I hate endurance runs. No, hated."**
> 오래달리기가 싫다. 아니, 싫**었다**.

- **hate + -ing**: 일반적으로 싫어하는 것

> **"I used to hate taking the bus."**
> 버스 타는 게 싫었다.

> **"I love running in the morning now."**
> 이제 아침에 달리는 게 좋다.

- **love + -ing**: 좋아하는 것

### 감정 강도:

```
love > like > don't mind > dislike > hate
좋아함                                 싫어함
```

| Expression | Example |
|------------|---------|
| I love running. | 달리기가 정말 좋아. |
| I like running. | 달리기가 좋아. |
| I don't mind running. | 달리기 괜찮아. |
| I dislike running. | 달리기가 별로야. |
| I hate running. | 달리기가 싫어. |

---

## D. begin / start / continue + -ing or to (Special Cases)

이 동사들은 -ing와 to 둘 다 쓸 수 있고, 의미 차이가 거의 없습니다.

### From the Original Book:

> "It **started raining**." = "It **started to rain**."
> 비가 오기 시작했어.

### Examples from the Novel:

> **"I started walking."** = I started to walk.
> 걷기 시작했다.

> **"My stomach started churning."** = started to churn.
> 속이 울렁거리기 시작했다.

> **"The rapeseed flowers are starting to bloom."**
> 유채꽃이 피기 시작하고 있다.

> **"I continued running even though I was tired."**
> 피곤했지만 계속 달렸다.

### 주의할 점:

**-ing가 이미 진행 중일 때는 to를 선호합니다:**

| Avoid | Prefer |
|-------|--------|
| ~~It's starting raining.~~ | It's **starting to rain**. |
| ~~I'm beginning liking it.~~ | I'm **beginning to like** it. |

## F. like doing vs like to do (Detailed Comparison)

### From the Original Book:

> "I **like cooking**." (요리를 즐긴다)
> "I **like to cook** the meals myself." (직접 요리하는 것을 선호한다)

| like doing | like to do |
|------------|------------|
| 즐거움 강조 | 선택/습관 강조 |
| 일반적 취향 | 특정 상황의 선택 |
| I like swimming. | I like to swim every day. |
| (수영이 좋아) | (매일 수영하길 좋아해) |

### From the Novel:

> **"I like walking."** (걷는 게 좋아 - 일반적)
> **"I like to walk in the morning."** (아침에 걷는 게 좋아 - 습관)

## G. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I would like running.~~ | I **would like to run**. | would like + to |
| ~~Would you like coming?~~ | **Would you like to come?** | would like + to |
| ~~I'm starting raining.~~ | It's **starting to rain**. | 진행형 + to |
| ~~She would love swimming.~~ | She **would love to swim**. | would love + to |
| ~~Do you would like coffee?~~ | **Would you like** coffee? | would like는 would가 필요 |
| ~~I like very much running.~~ | I **like running very much**. | 어순 오류 |

## Exercises

### A. 올바른 형태를 고르세요.

1. I would like (run / to run) a marathon.
2. She loves (run / running) in the morning.
3. We started (walk / walking) to school.
4. He hates (get / getting) up early.
5. Would you like (come / to come) with me?
6. It's beginning (rain / to rain).
7. I'd love (join / to join) you.
8. They continued (train / training) hard.

**Answers:**
1. to run (would like + to)
2. running (또는 to run)
3. walking (또는 to walk)
4. getting (또는 to get)
5. to come (would like + to)
6. to rain (진행형이므로 to 선호)
7. to join (would love + to)
8. training (또는 to train)

### B. 문장을 완성하세요.

1. I like _______ (listen) to birds while walking.
2. Would you like _______ (join) me for a run?
3. She began _______ (train) for the marathon.
4. I hate _______ (take) the crowded bus.
5. I would love _______ (finish) under 4 hours.
6. He started _______ (run) two years ago.
7. Do you like _______ (exercise) in the morning?
8. It's starting _______ (get) cold.

**Answers:**
1. listening / to listen
2. to join
3. training / to train
4. taking / to take
5. to finish
6. running / to run
7. exercising / to exercise
8. to get

### C. like vs would like 구분하세요.

1. I _______ running. (일반적으로 좋아함)
   a) like  b) would like

2. I _______ to run a marathon someday. (하고 싶음)
   a) like  b) would like

3. _______ you _______ some water? (물 좀 드시겠어요?)
   a) Do...like  b) Would...like

4. I _______ walking to school. (걷는 게 좋아)
   a) like  b) would like

5. I _______ to go home now. (집에 가고 싶어)
   a) like  b) would like

**Answers:**
1. a) like
2. b) would like
3. b) Would...like
4. a) like
5. b) would like

### D. 틀린 부분을 고치세요.

1. I would like running a marathon.
   → _______

2. Would you like going to the race?
   → _______

3. I'm beginning liking running.
   → _______

4. She would love coming with us.
   → _______

**Answers:**
1. I would like to run a marathon.
2. Would you like to go to the race?
3. I'm beginning to like running.
4. She would love to come with us.

## Summary: -ing vs to 선택 가이드

### 둘 다 가능한 경우:
| Verb | Example |
|------|---------|
| like | I like running / to run. |
| love | I love walking / to walk. |
| hate | I hate waiting / to wait. |
| prefer | I prefer walking / to walk. |
| start | I started running / to run. |
| begin | I began training / to train. |
| continue | I continued running / to run. |

### to만 가능한 경우:
| Expression | Example |
|------------|---------|
| would like | I'd like **to go**. |
| would love | I'd love **to come**. |
| would hate | I'd hate **to miss** it. |
| would prefer | I'd prefer **to stay**. |

### Quick Rule:
- **would + 감정동사** = 반드시 **to**!
