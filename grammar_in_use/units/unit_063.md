# Unit 63: Expressions + -ing

## A. -ing와 함께 쓰이는 표현들 (Form)

특정 표현들은 **-ing 형태**와 함께 쓰입니다.

| Expression | Meaning | Structure |
|------------|---------|-----------|
| It's no use | ~해봤자 소용없다 | It's no use + -ing |
| It's no good | ~해봤자 좋을 게 없다 | It's no good + -ing |
| There's no point (in) | ~해도 의미없다 | There's no point (in) + -ing |
| It's (not) worth | ~할 가치가 있다/없다 | It's (not) worth + -ing |
| have trouble/difficulty | ~하는 데 어려움이 있다 | have trouble/difficulty + -ing |
| spend/waste time | ~하는 데 시간을 쓰다/낭비하다 | spend/waste time + -ing |
| be busy | ~하느라 바쁘다 | be busy + -ing |
| go + activity | ~하러 가다 | go + -ing |

---

## B. It's no use / It's no good + -ing (Primary Usage)

### From the Original Book:

> "**It's no use crying** over spilled milk."
> 엎질러진 물에 대해 울어봤자 소용없어.

> "**It's no good trying** to persuade him."
> 그를 설득하려고 해봤자 좋을 게 없어.

### Examples from the Novel:

> **"It's no use complaining about the bus."**
> 버스에 대해 불평해봤자 소용없다.

- **It's no use + -ing**: ~해도 소용없다

> **"It's no good worrying about the race."**
> 레이스에 대해 걱정해봤자 안 좋다.

- **It's no good + -ing**: ~해도 좋을 게 없다

### 강도 비교:

```
It's no use    = 소용없다 (practical)
It's no good   = 좋을 게 없다 (no benefit)
```

---

## C. There's no point (in) + -ing (Meaninglessness)

### From the Original Book:

> "**There's no point in waiting** any longer."
> 더 이상 기다려봤자 의미가 없어.

### Examples from the Novel:

> **"There's no point in taking the bus if I feel sick."**
> 아프면 버스 타봤자 의미가 없다.

- **There's no point (in) + -ing**: ~해도 의미가 없다

> **"There's no point in running if you can't enjoy it."**
> 즐기지 못하면 달려봤자 의미가 없다.

### in의 생략:

| Full form | Short form |
|-----------|------------|
| There's no point **in** waiting. | There's no point waiting. |
| There's no point **in** trying. | There's no point trying. |

---

## D. It's (not) worth + -ing (Value Assessment)

### From the Original Book:

> "The movie **isn't worth seeing**."
> 그 영화는 볼 가치가 없어.

> "**It's worth considering** his offer."
> 그의 제안을 고려해볼 가치가 있어.

### Examples from the Novel:

> **"It's worth getting up early to run."**
> 달리기 위해 일찍 일어나는 건 가치가 있다.

- **It's worth + -ing**: ~할 가치가 있다

> **"The pain is worth enduring."**
> 그 고통은 견딜 가치가 있다.

> **"Walking 7km? It's not worth it."**
> 7km 걷기? 그럴 가치가 없어.

### worth의 다양한 형태:

| Structure | Example |
|-----------|---------|
| It's worth + -ing | It's worth trying. |
| It's not worth + -ing | It's not worth waiting. |
| It's worth it | The effort is worth it. |
| be worth + 명사 | The book is worth $20. |

## E. have trouble / difficulty + -ing (Difficulty)

### From the Original Book:

> "I **had trouble finding** your house."
> 너희 집 찾는 데 어려움이 있었어.

> "She **has difficulty remembering** names."
> 그녀는 이름 기억하는 데 어려움이 있어.

### Examples from the Novel:

> **"I had trouble breathing on the bus."**
> 버스에서 숨쉬기가 힘들었다.

- **have trouble + -ing**: ~하는 데 어려움이 있다

> **"I had difficulty sleeping before the race."**
> 레이스 전날 밤 잠들기가 어려웠다.

- **have difficulty + -ing**: ~하는 데 어려움이 있다

> **"I didn't have any trouble finishing 10km."**
> 10km 완주하는 데 아무 문제 없었다.

### 관련 표현:

| Expression | Example |
|------------|---------|
| have trouble | I have trouble sleeping. |
| have difficulty | I have difficulty breathing. |
| have a hard time | I had a hard time finishing. |
| have problems | I had problems concentrating. |

## F. spend / waste time + -ing (Time Usage)

### From the Original Book:

> "I **spent two hours cleaning** the house."
> 집 청소하는 데 두 시간을 썼어.

> "Don't **waste time worrying** about it."
> 그것에 대해 걱정하느라 시간 낭비하지 마.

### Examples from the Novel:

> **"I spent an hour walking to school."**
> 학교까지 걸어가는 데 한 시간을 썼다.

- **spend time + -ing**: ~하는 데 시간을 쓰다

> **"Don't waste time worrying."**
> 걱정하느라 시간 낭비하지 마.

- **waste time + -ing**: ~하느라 시간을 낭비하다

> **"I spent the morning running 10km."**
> 아침에 10km를 달리는 데 시간을 썼다.

### spend/waste + 시간/돈 + -ing:

| Pattern | Example |
|---------|---------|
| spend + 시간 + -ing | I spent 2 hours running. |
| spend + 돈 + -ing | I spent $100 buying shoes. |
| waste + 시간 + -ing | Don't waste time waiting. |
| waste + 돈 + -ing | I wasted money buying that. |

## G. be busy + -ing (Activity)

### From the Original Book:

> "She's **busy preparing** for the party."
> 그녀는 파티 준비하느라 바빠.

### Examples from the Novel:

> **"I was busy preparing for the race."**
> 레이스 준비하느라 바빴다.

- **be busy + -ing**: ~하느라 바쁘다

> **"Mom was busy packing my things."**
> 엄마가 내 짐 싸느라 바빴다.

### busy의 다양한 형태:

| Structure | Example |
|-----------|---------|
| be busy + -ing | I'm busy studying. |
| be busy with + 명사 | I'm busy with homework. |
| keep busy + -ing | I keep busy running. |

## H. go + -ing (Activities/Sports)

**go** 뒤에 운동이나 활동을 나타내는 -ing가 옵니다.

### From the Original Book:

> "Let's **go swimming** this afternoon."
> 오늘 오후에 수영하러 가자.

### Examples from the Novel:

> **"I went running this morning."**
> 오늘 아침에 달리러 갔다.

- **go running**: 달리러 가다

> **"Let's go walking together."**
> 같이 걸으러 가자.

### go + -ing 활동들:

| Activity | Expression |
|----------|------------|
| running | go running |
| walking | go walking |
| jogging | go jogging |
| swimming | go swimming |
| hiking | go hiking |
| shopping | go shopping |
| fishing | go fishing |
| skiing | go skiing |
| cycling | go cycling |
| camping | go camping |

## I. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~It's no use **to complain**.~~ | It's no use **complaining**. | It's no use + -ing |
| ~~There's no point **to wait**.~~ | There's no point **waiting**. | no point + -ing |
| ~~It's worth **to try**.~~ | It's worth **trying**. | worth + -ing |
| ~~I have trouble **to sleep**.~~ | I have trouble **sleeping**. | have trouble + -ing |
| ~~I spent time **to study**.~~ | I spent time **studying**. | spend time + -ing |
| ~~I'm busy **to work**.~~ | I'm busy **working**. | be busy + -ing |
| ~~Let's go **to run**.~~ | Let's **go running**. | go + -ing |

**Answers:**

## Summary: Expression + -ing 정리

| Expression | Meaning | Example |
|------------|---------|---------|
| It's no use + -ing | ~해봤자 소용없다 | It's no use complaining. |
| It's no good + -ing | ~해봤자 좋을 게 없다 | It's no good worrying. |
| There's no point (in) + -ing | ~해도 의미없다 | There's no point waiting. |
| It's worth + -ing | ~할 가치가 있다 | It's worth trying. |
| have trouble + -ing | ~하는 데 어려움 | I have trouble sleeping. |
| have difficulty + -ing | ~하는 데 어려움 | I have difficulty breathing. |
| spend time + -ing | ~하는 데 시간을 쓰다 | I spent time running. |
| waste time + -ing | ~하느라 낭비하다 | Don't waste time worrying. |
| be busy + -ing | ~하느라 바쁘다 | I'm busy training. |
| go + -ing | ~하러 가다 | Let's go running. |

### Quick Rule:
이 모든 표현들은 **to부정사가 아닌 -ing**를 사용합니다!
