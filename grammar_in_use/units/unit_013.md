# Unit 13: Present Perfect and Past (1)

## A. 문법 설명 (Grammar Explanation)

### English
**Present Perfect** and **Past Simple** both talk about the past, but:
- **Present Perfect**: past connected to present (no specific time)
- **Past Simple**: completed past (often with specific time)

### Korean
**현재완료**와 **과거시제** 모두 과거를 말하지만:
- **현재완료**: 과거와 현재의 연결 (특정 시간 없이)
- **과거시제**: 완료된 과거 (특정 시간과 함께)

---

## B. 핵심 차이 (Key Difference)

| Present Perfect | Past Simple |
|-----------------|-------------|
| 시간 특정 안 함 | 시간 특정 |
| 현재와 연결 | 과거만 언급 |
| 지금까지의 경험 | 과거의 사실 |
| I've walked to school. | I walked to school yesterday. |

---

## C. 시간 표현과 시제 (Time Expressions)

### Present Perfect (시간 특정 X):
- ever, never, before
- just, already, yet
- recently, lately
- today, this week (현재 포함 기간)
- How long...?

### Past Simple (시간 특정 O):
- yesterday
- last week/month/year
- ago (two days ago)
- in 2024, in March
- on Monday
- when I was young
- When...?

---

## D. 소설 속 예문 (Examples from the Novel)

### Example 1: 경험 vs 과거 사실

**Present Perfect (경험):**
> **"I've walked to school many times."**
> 학교에 걸어간 적이 많다.

**Past Simple (과거 특정 사건):**
> **"I walked to school yesterday."**
> 어제 학교에 걸어갔다.

### Example 2: 시간 표현의 차이

**Present Perfect (시간 불특정):**
> **"I've decided to run the marathon."**
> 마라톤을 뛰기로 결심했다. (최근에, 언제인지 중요하지 않음)

**Past Simple (시간 특정):**
> **"I decided to run the marathon in August."**
> 8월에 마라톤을 뛰기로 결심했다.

### Example 3: Just vs Ago

**Present Perfect (just):**
> **"I've just finished running."**
> 방금 달리기를 끝냈다.

**Past Simple (ago):**
> **"I finished running two hours ago."**
> 두 시간 전에 달리기를 끝냈다.

### Example 4: This week vs Last week

**Present Perfect (현재 포함 기간):**
> **"I've run 7km twice this week."**
> 이번 주에 7km를 두 번 뛰었다. (아직 이번 주)

**Past Simple (지난 기간):**
> **"I ran 7km twice last week."**
> 지난주에 7km를 두 번 뛰었다. (지난주 끝남)

### Example 5: Ever vs When I was young

**Present Perfect (ever):**
> **"Have you ever run a marathon?"**
> 마라톤 뛰어본 적 있어?

**Past Simple (과거 특정 시기):**
> **"I ran a lot when I was in elementary school."**
> 초등학교 때 많이 뛰었어.

## E. 대화 패턴 (Conversation Patterns)

### Pattern 1: Present Perfect → Past Simple

첫 문장은 현재완료, 추가 정보는 과거시제:

```
A: Have you ever run a marathon? (현재완료)
B: Yes, I have. I ran one last year. (과거시제)
```

```
A: I've decided to walk to school. (현재완료)
B: When did you decide? (과거시제)
A: I decided yesterday. (과거시제)
```

### Pattern 2: 소설 속 대화

```
Teacher: Have you been running a lot lately? (현재완료)
Doyun: Yes, I started walking in March. (과거시제)
Teacher: So it's been three months. (현재완료)
```

---

## F. 흔한 실수 (Common Mistakes)

| Wrong | Correct |
|-------|---------|
| ~~I have walked yesterday.~~ | I walked yesterday. |
| ~~Did you ever run a marathon?~~ | Have you ever run a marathon? |
| ~~I went there last week.~~ (X when asking about experience) | Have you been there? (경험) |
| ~~When have you decided?~~ | When did you decide? |

## Exercises

### A. Present Perfect or Past Simple?

1. I (have walked / walked) to school yesterday.
2. She (has finished / finished) the race last month.
3. (Have you ever run / Did you ever run) a marathon?
4. I (have decided / decided) to start training two weeks ago.
5. He (has just arrived / just arrived) at school.

**Answers:**
1. walked (yesterday)
2. finished (last month)
3. Have you ever run (경험)
4. decided (two weeks ago)
5. has just arrived (just + 현재완료)

### B. 빈칸 채우기

1. I _______ (see) that movie. It was great!
2. When _______ you _______ (see) it?
3. I _______ (see) it last weekend.
4. _______ you ever _______ (be) to Seoul?
5. Yes, I _______ (go) there in 2023.

**Answers:**
1. have seen
2. did ... see
3. saw
4. Have ... been
5. went

### C. 문장 완성

Complete with the correct tense:

1. I _______ (walk) to school many times. (경험)
2. I _______ (walk) to school yesterday. (어제)
3. She _______ (run) 10km this week. (이번 주)
4. She _______ (run) 10km last week. (지난주)
5. _______ you _______ (meet) him? (경험)

**Answers:**
1. have walked
2. walked
3. has run
4. ran
5. Have ... met

### D. 대화 완성

Complete the dialogue:

A: _______ you ever _______ (run) a marathon?
B: Yes, I _______. I _______ (run) my first one last October.
A: How _______ it?
B: It was amazing. I _______ (train) for six months before that.

**Answers:**
1. Have ... run
2. have
3. ran
4. was
5. had trained (or: trained)

## Key Sentences (암기 문장)

| Present Perfect | Past Simple |
|-----------------|-------------|
| I've finished the race. | I finished the race yesterday. |
| Have you ever run? | Did you run yesterday? |
| I've just arrived. | I arrived two hours ago. |
| I've walked many times. | I walked last Monday. |
| It's been great. | It was great yesterday. |

## 정리: 어떤 시제를 쓸까?

| 상황 | 시제 | 예문 |
|------|------|------|
| yesterday, last week, ago | Past Simple | I ran yesterday. |
| just, already, yet | Present Perfect | I've just finished. |
| ever, never | Present Perfect | Have you ever run? |
| in + 연도, on + 요일 | Past Simple | I started in March. |
| this week (아직 진행 중) | Present Perfect | I've run twice this week. |
| When...? | Past Simple | When did you start? |
| How long...? | Present Perfect | How long have you been running? |
