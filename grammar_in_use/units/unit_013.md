# Unit 13: Present Perfect and Past (1)

## A. 핵심 차이 (Key Difference)

### English
**Present Perfect** and **Past Simple** both talk about the past, but:
- **Present Perfect**: past connected to present (no specific time, result matters now)
- **Past Simple**: completed past (often with specific time, finished)

### Korean
**현재완료**와 **과거시제** 모두 과거를 말하지만:
- **현재완료**: 과거와 현재의 연결 (특정 시간 없이, 현재 결과 중요)
- **과거시제**: 완료된 과거 (특정 시간과 함께, 끝난 일)

---

## B. 시간 표현과 시제 (Time Expressions and Tense)

### From the Original Book:
> "Tom **has lost** his key. He can't get into the house."
> 톰이 열쇠를 잃어버렸다. 집에 못 들어간다. (현재 영향)

> "Tom **lost** his key yesterday. He couldn't get into the house."
> 톰이 어제 열쇠를 잃어버렸다. 집에 못 들어갔다. (과거 완료된 일)

### Time Expressions Chart:

| Present Perfect (시간 특정 X) | Past Simple (시간 특정 O) |
|------------------------------|--------------------------|
| ever, never, before | yesterday |
| just, already, yet | last week/month/year |
| recently, lately | ago (two days ago) |
| today, this week (현재 포함) | in 2024, in March |
| How long...? | on Monday |
| for, since | when I was young |
| - | When...? |

## C. 핵심 비교표 (Comparison Table)

| Present Perfect | Past Simple |
|-----------------|-------------|
| 시간 특정 안 함 | 시간 특정 |
| 현재와 연결 | 과거만 언급 |
| 지금까지의 경험 | 과거의 사실 |
| 현재 결과 중요 | 과거 사건 중요 |
| I've walked to school. | I walked to school yesterday. |

## D. 소설 속 예문 비교 (Novel Examples)

### Example 1: 경험 vs 과거 사실

**Present Perfect (경험):**
> **"I've walked to school many times."**
> 학교에 걸어간 적이 많다. (지금까지의 경험)

**Past Simple (과거 특정 사건):**
> **"I walked to school yesterday."**
> 어제 학교에 걸어갔다. (어제의 일)

### Example 2: 시간 표현의 차이

**Present Perfect (시간 불특정):**
> **"I've decided to run the marathon."**
> 마라톤을 뛰기로 결심했다. (최근에, 언제인지 중요하지 않음)

**Past Simple (시간 특정):**
> **"I decided to run the marathon in August."**
> 8월에 마라톤을 뛰기로 결심했다. (8월이라는 특정 시점)

### Example 3: Just vs Ago

**Present Perfect (just):**
> **"I've just finished running."**
> 방금 달리기를 끝냈다. (현재와 연결)

**Past Simple (ago):**
> **"I finished running two hours ago."**
> 두 시간 전에 달리기를 끝냈다. (특정 과거 시점)

### Example 4: This week vs Last week

**Present Perfect (현재 포함 기간):**
> **"I've run 7km twice this week."**
> 이번 주에 7km를 두 번 뛰었다. (아직 이번 주)

**Past Simple (지난 기간):**
> **"I ran 7km twice last week."**
> 지난주에 7km를 두 번 뛰었다. (지난주 끝남)

### Example 5: Ever vs When I was young

**Present Perfect (ever):**
> **"Have you ever run a marathon?"**
> 마라톤 뛰어본 적 있어? (지금까지 경험)

**Past Simple (과거 특정 시기):**
> **"I ran a lot when I was in elementary school."**
> 초등학교 때 많이 뛰었어. (그 시기의 일)

## E. 대화 패턴 (Conversation Patterns)

### Pattern 1: Present Perfect → Past Simple

첫 문장은 현재완료, 추가 정보(시간 특정)는 과거시제:

```
A: Have you ever run a marathon? (현재완료)
B: Yes, I have. I ran one last year. (과거시제)
```

```
A: I've decided to walk to school. (현재완료)
B: When did you decide? (과거시제)
A: I decided yesterday. (과거시제)
```

### Pattern 2: 소설 속 대화

```
Teacher: Have you been running a lot lately? (현재완료)
Doyun: Yes, I started walking in March. (과거시제)
Teacher: So it's been three months. (현재완료)
```

## F. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I have walked yesterday.~~ | I **walked** yesterday. | yesterday = 과거시제 |
| ~~Did you ever run a marathon?~~ | **Have** you **ever** run a marathon? | ever = 현재완료 |
| ~~I have seen him last week.~~ | I **saw** him last week. | last week = 과거시제 |
| ~~When have you decided?~~ | **When did you decide?** | When = 과거시제 |
| ~~I've finished two hours ago.~~ | I **finished** two hours ago. | ago = 과거시제 |
| ~~I lived here since 2020.~~ | I **have lived** here since 2020. | since = 현재완료 |

## G. 시제 선택 가이드 (Tense Selection Guide)

### Use Present Perfect when:

| Situation | Example |
|-----------|---------|
| 경험 (ever/never) | Have you ever run? |
| 방금 (just) | I've just arrived. |
| 이미/아직 (already/yet) | I've already finished. |
| 현재 결과 | I've lost my key. (can't find it now) |
| 현재 포함 기간 | I've run twice today. |
| 시간 불특정 | I've seen that movie. |

### Use Past Simple when:

| Situation | Example |
|-----------|---------|
| yesterday | I walked yesterday. |
| last week/month/year | I ran last week. |
| ago | I finished two hours ago. |
| in + 연도 | I started in March. |
| on + 요일 | I saw him on Monday. |
| When...? | When did you start? |
| 특정 과거 시점 | I met her at the race. |

## Exercises

### A. Present Perfect or Past Simple?

1. I (have walked / walked) to school yesterday.
2. She (has finished / finished) the race last month.
3. (Have you ever run / Did you ever run) a marathon?
4. I (have decided / decided) to start training two weeks ago.
5. He (has just arrived / just arrived) at school.
6. I (have seen / saw) that movie three times.
7. They (have gone / went) to Japan last summer.
8. (Have you eaten / Did you eat) yet?

**Answers:**
1. walked (yesterday)
2. finished (last month)
3. Have you ever run (경험)
4. decided (two weeks ago)
5. has just arrived (just + 현재완료)
6. have seen (경험)
7. went (last summer)
8. Have you eaten (yet)

### B. 빈칸 채우기

1. I _______ (see) that movie. It was great!
2. When _______ you _______ (see) it?
3. I _______ (see) it last weekend.
4. _______ you ever _______ (be) to Seoul?
5. Yes, I _______ (go) there in 2023.
6. I _______ (not finish) my homework yet.
7. She _______ (start) running in March.
8. I _______ (already / eat) breakfast.

**Answers:**
1. have seen
2. did ... see
3. saw
4. Have ... been
5. went
6. haven't finished
7. started
8. have already eaten

### C. 문장 완성

Complete with the correct tense:

1. I _______ (walk) to school many times. (경험)
2. I _______ (walk) to school yesterday. (어제)
3. She _______ (run) 10km this week. (이번 주)
4. She _______ (run) 10km last week. (지난주)
5. _______ you _______ (meet) him? (경험)
6. When _______ you _______ (meet) him? (언제)
7. I _______ (just / finish) my homework.
8. I _______ (finish) my homework an hour ago.

**Answers:**
1. have walked
2. walked
3. has run
4. ran
5. Have ... met
6. did ... meet
7. have just finished
8. finished

### D. 대화 완성

Complete the dialogue:

```
A: _______ you ever _______ (run) a marathon?
B: Yes, I _______. I _______ (run) my first one last October.
A: How _______ it?
B: It was amazing. I _______ (train) for six months before that.
A: _______ you _______ (decide) to run another one?
B: Yes, I _______ already _______ (sign up) for next year's race.
```

**Answers:**
1. Have ... run
2. have
3. ran
4. was
5. trained (또는 had trained)
6. Have ... decided
7. have ... signed up

## Key Sentences (암기 문장)

| Present Perfect | Past Simple |
|-----------------|-------------|
| I've finished the race. | I finished the race yesterday. |
| Have you ever run? | Did you run yesterday? |
| I've just arrived. | I arrived two hours ago. |
| I've walked many times. | I walked last Monday. |
| It's been great. | It was great yesterday. |
| I've seen that movie. | I saw that movie in 2023. |
| She's already left. | She left at 3 o'clock. |
| I haven't eaten yet. | I didn't eat this morning. |

## Summary: 시제 선택 가이드

| Signal Word | Tense | Example |
|-------------|-------|---------|
| yesterday, last week, ago | Past Simple | I ran yesterday. |
| just, already, yet | Present Perfect | I've just finished. |
| ever, never | Present Perfect | Have you ever run? |
| in + 연도, on + 요일 | Past Simple | I started in March. |
| this week (현재 포함) | Present Perfect | I've run twice this week. |
| When...? | Past Simple | When did you start? |
| How long...? (since/for) | Present Perfect | How long have you been running? |
