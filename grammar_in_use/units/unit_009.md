# Unit 9: Present Perfect Continuous (I have been doing)

## A. 문법 설명 (Grammar Explanation)

### English
**Present Perfect Continuous** (have/has + been + -ing) describes an activity that:
1. **Started in the past and is still continuing**
2. **Recently stopped but has present results** (you can see/feel the effects now)

### Korean
**현재완료진행형** (have/has + been + -ing)은 다음을 표현합니다:
1. **과거에 시작해서 아직 계속되는 활동**
2. **최근에 끝났지만 현재 결과가 보이는 활동** (지금 그 영향이 느껴짐)

---

## B. 현재완료진행형의 형태 (Form)

| Subject | Have/Has | Been | -ing |
|---------|----------|------|------|
| I/You/We/They | have | been | running |
| He/She/It | has | been | walking |

**부정문:** have/has + not + been + -ing
**의문문:** Have/Has + 주어 + been + -ing?

---

## C. 소설 속 예문 (Examples from the Novel)

### Example 1: 과거부터 지금까지 계속 (Still continuing)

> **"I've been preparing for a whole year."**
> 1년 동안 준비해 왔다.

- **I've been preparing** = 1년 전에 시작해서 지금도 계속
- Activity started in the past and continues to the present

### Example 2: 얼마나 오래 (How long)

> **"I've been walking to school since March."**
> 3월부터 걸어서 학교에 다녔다.

- **I've been walking** = 3월에 시작, 지금도 계속
- Emphasizes the duration of the activity

### Example 3: 최근 멈췄지만 결과가 보임

> **"I'm sweating. I've been running."**
> 땀이 나. 달려서 왔어.

- **I've been running** = 방금 전까지 달렸음
- The sweating is evidence of the recent activity

### Example 4: 진행 중인 활동 강조

> **"I've been thinking about joining the track team."**
> 육상부에 들어갈까 생각 중이었다.

- **I've been thinking** = 한동안 계속 생각해 왔음
- The thinking has been going on for some time

### Example 5: 변화의 과정

> **"My stamina has been improving."**
> 체력이 계속 좋아지고 있다.

- **has been improving** = 과거부터 지금까지 계속 향상 중
- The improvement is ongoing

## D. 활동의 현재 증거 (Present Evidence of Activity)

| Activity | Present Evidence |
|----------|------------------|
| I've been running. | I'm sweating. / I'm out of breath. |
| I've been walking. | My legs are sore. |
| I've been training. | I'm getting stronger. |
| I've been thinking. | I've made a decision. |

### 소설 속 예문:

> **"My legs are sore. I've been walking for over an hour."**
> 다리가 아파. 한 시간 넘게 걸어왔어.

> **"I'm tired. I've been running all morning."**
> 피곤해. 아침 내내 달렸어.

---

## E. 시간 표현 (Time Expressions)

### 현재완료진행형과 자주 쓰는 표현:

| Expression | Example |
|------------|---------|
| for + 기간 | I've been running **for six months**. |
| since + 시점 | I've been walking **since March**. |
| all morning/day | I've been training **all morning**. |
| lately/recently | I've been feeling great **lately**. |
| how long | **How long** have you been running? |

---

## F. 비교: Present Perfect Continuous vs Present Perfect Simple

| Present Perfect Continuous | Present Perfect Simple |
|---------------------------|----------------------|
| 활동/과정 강조 | 결과/완료 강조 |
| I've been running. (과정) | I've run 10km. (완료) |
| 진행 중/최근 활동 | 횟수/양 |
| I've been reading this book. | I've read three books. |

### 예문 비교:

> **"I've been running since March."** (계속 달려왔다 - 과정)
> **"I've run 500km since March."** (500km를 달렸다 - 총량)

> **"I've been training hard."** (열심히 훈련해 왔다 - 과정)
> **"I've trained five times this week."** (5번 훈련했다 - 횟수)

## Exercises

### A. 올바른 형태를 고르세요.

1. I (have run / have been running) for two hours. I'm tired!
2. She (has walked / has been walking) to school since March.
3. How long (have you run / have you been running)?
4. I (have read / have been reading) this book all week.
5. My legs hurt. I (have walked / have been walking) all day.

**Answers:**
1. have been running (과정 + 현재 결과)
2. has been walking (계속)
3. have you been running (기간)
4. have been reading (진행 중)
5. have been walking (현재 증거)

### B. 빈칸 채우기

1. I _______ (run) for six months. I feel much healthier now.
2. She _______ (train) hard lately.
3. How long _______ you _______ (walk) to school?
4. He _______ (think) about the race all week.
5. My stamina _______ (improve) since I started running.

**Answers:**
1. have been running
2. has been training
3. have ... been walking
4. has been thinking
5. has been improving

### C. 문장 완성하기 (증거 연결)

Match the result with the activity:

1. I'm sweating. → I _______. (run)
2. My legs are tired. → I _______. (walk)
3. I'm out of breath. → I _______. (run fast)
4. I feel confident. → I _______. (prepare well)

**Answers:**
1. I've been running.
2. I've been walking.
3. I've been running fast.
4. I've been preparing well.

### D. 문장 변환

Change to present perfect continuous:

1. I started running in March. (still running)
   → I _______ since March.
2. She started training last month. (still training)
   → She _______ since last month.
3. They started preparing yesterday. (still preparing)
   → They _______ since yesterday.

**Answers:**
1. have been running
2. has been training
3. have been preparing
