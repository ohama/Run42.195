# Unit 9: Present Perfect Continuous (I have been doing)

## A. 현재완료진행형의 형태 (Form)

**have/has + been + -ing**

| Subject | Have/Has | Been | -ing | Full Example |
|---------|----------|------|------|--------------|
| I/You/We/They | have | been | running | I have been running. |
| He/She/It | has | been | walking | She has been walking. |

### Negative and Questions:

| Type | Structure | Example |
|------|-----------|---------|
| Positive | have/has + been + -ing | I have been running. |
| Negative | have/has + not + been + -ing | I haven't been running. |
| Question | Have/Has + subject + been + -ing? | Have you been running? |

### Contractions:
- I have been → I've been
- He has been → He's been
- have not been → haven't been
- has not been → hasn't been

---

## B. 기본 용법 (Basic Usage)

**"I have been doing something" = I started doing it in the past and:**
1. **I'm still doing it** (아직 계속하고 있음), OR
2. **I've just stopped, but you can see the result** (방금 멈췄지만 결과가 보임)

### From the Original Book:
> "You're out of breath. **Have you been running**?"
> 숨이 차네. 달려왔어?

> "I'**ve been waiting** for an hour."
> 한 시간 동안 기다렸어.

---

## C. 현재완료진행형 사용 상황 (When to Use)

### 1. 과거부터 지금까지 계속 (Activity continuing to now)

> **"I've been running since 7 o'clock."**
> 7시부터 달리고 있다.

- Started at 7, still running now

### 2. 최근 멈췄지만 결과가 보임 (Recently stopped with visible results)

> **"I'm sweating. I've been running."**
> 땀이 나. 달려서 왔어.

- Stopped running, but sweating is evidence

### 3. 얼마나 오래? (How long?)

> **"How long have you been running?"**
> 얼마나 오래 달렸어?

- Asking about duration

---

## D. 소설 속 예문 (Examples from the Novel)

### Example 1: 과거부터 지금까지 계속

> **"I've been preparing for a whole year."**
> 1년 동안 준비해 왔다.

- **I've been preparing** = 1년 전에 시작해서 지금도 계속
- Activity started in the past and continues to the present

### Example 2: 얼마나 오래 (How long)

> **"I've been walking to school since March."**
> 3월부터 걸어서 학교에 다녔다.

- **I've been walking** = 3월에 시작, 지금도 계속
- Emphasizes the duration of the activity

### Example 3: 최근 멈췄지만 결과가 보임

> **"I'm sweating. I've been running."**
> 땀이 나. 달려서 왔어.

- **I've been running** = 방금 전까지 달렸음
- The sweating is evidence of the recent activity

### Example 4: 진행 중인 활동 강조

> **"I've been thinking about joining the track team."**
> 육상부에 들어갈까 생각 중이었다.

- **I've been thinking** = 한동안 계속 생각해 왔음
- The thinking has been going on for some time

### Example 5: 변화의 과정

> **"My stamina has been improving."**
> 체력이 계속 좋아지고 있다.

- **has been improving** = 과거부터 지금까지 계속 향상 중
- The improvement is ongoing

---

## E. 활동의 현재 증거 (Present Evidence of Activity)

| Recent Activity | Present Evidence |
|-----------------|------------------|
| I've been running. | I'm sweating. / I'm out of breath. |
| I've been walking. | My legs are sore. |
| I've been training. | I'm getting stronger. |
| I've been thinking. | I've made a decision. |
| It's been raining. | The ground is wet. |
| She's been crying. | Her eyes are red. |
| He's been working. | He looks tired. |

### From the Novel:

> **"My legs are sore. I've been walking for over an hour."**
> 다리가 아파. 한 시간 넘게 걸어왔어.

> **"I'm tired. I've been running all morning."**
> 피곤해. 아침 내내 달렸어.

---

## F. 시간 표현 (Time Expressions)

### 현재완료진행형과 자주 쓰는 표현:

| Expression | Korean | Example |
|------------|--------|---------|
| for + 기간 | ~동안 | I've been running **for six months**. |
| since + 시점 | ~부터 | I've been walking **since March**. |
| all morning/day | 아침/하루 내내 | I've been training **all morning**. |
| lately/recently | 요즘/최근에 | I've been feeling great **lately**. |
| how long | 얼마나 오래 | **How long** have you been running? |
| all week | 일주일 내내 | I've been training **all week**. |

## G. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I have been run.~~ | I have been **running**. | -ing 형태 필요 |
| ~~She has being walking.~~ | She has **been** walking. | been 사용 |
| ~~I've been knowing him.~~ | I've **known** him. | know = 상태동사 |
| ~~How long are you waiting?~~ | How long **have you been waiting**? | 현재완료진행형 필요 |
| ~~I've been walked since March.~~ | I've been **walking** since March. | -ing 형태 |
| ~~He's been have a car.~~ | He's **had** a car. | have (소유) = 상태동사 |

---

## H. 비교: Present Perfect Continuous vs Present Perfect Simple

| Present Perfect Continuous | Present Perfect Simple |
|---------------------------|----------------------|
| 활동/과정 강조 | 결과/완료 강조 |
| I've been running. (과정) | I've run 10km. (완료) |
| 진행 중/최근 활동 | 횟수/양 |
| I've been reading this book. | I've read three books. |
| How long...? | How many/much...? |

### 예문 비교:

> **"I've been running since March."** (계속 달려왔다 - 과정)
> **"I've run 500km since March."** (500km를 달렸다 - 총량)

> **"I've been training hard."** (열심히 훈련해 왔다 - 과정)
> **"I've trained five times this week."** (5번 훈련했다 - 횟수)

## Exercises

### A. 올바른 형태를 고르세요.

1. I (have run / have been running) for two hours. I'm tired!
2. She (has walked / has been walking) to school since March.
3. How long (have you run / have you been running)?
4. I (have read / have been reading) this book all week.
5. My legs hurt. I (have walked / have been walking) all day.
6. Look! Your eyes are red. (Have you cried / Have you been crying)?
7. It (has rained / has been raining) all morning.
8. I (have written / have been writing) emails since 9 o'clock.

**Answers:**
1. have been running (과정 + 현재 결과)
2. has been walking (계속)
3. have you been running (기간)
4. have been reading (진행 중)
5. have been walking (현재 증거)
6. Have you been crying (증거 - 빨간 눈)
7. has been raining (계속)
8. have been writing (계속)

### B. 빈칸 채우기

1. I _______ (run) for six months. I feel much healthier now.
2. She _______ (train) hard lately.
3. How long _______ you _______ (walk) to school?
4. He _______ (think) about the race all week.
5. My stamina _______ (improve) since I started running.
6. They _______ (wait) for over an hour.
7. It _______ (rain) since yesterday.
8. I _______ (not/sleep) well recently.

**Answers:**
1. have been running
2. has been training
3. have ... been walking
4. has been thinking
5. has been improving
6. have been waiting
7. has been raining
8. haven't been sleeping

### C. 증거와 활동 연결하기

Match the result with the activity:

1. I'm sweating. → I _______. (run)
2. My legs are tired. → I _______. (walk)
3. I'm out of breath. → I _______. (run fast)
4. I feel confident. → I _______. (prepare well)
5. The ground is wet. → It _______. (rain)

**Answers:**
1. I've been running.
2. I've been walking.
3. I've been running fast.
4. I've been preparing well.
5. It's been raining.

### D. 문장 변환

Change to present perfect continuous:

1. I started running in March. (still running)
   → I **have been running** since March.

2. She started training last month. (still training)
   → She **has been training** since last month.

3. They started preparing yesterday. (still preparing)
   → They **have been preparing** since yesterday.

4. It started raining this morning. (still raining)
   → It **has been raining** since this morning.

5. He started working at 8 AM. (still working)
   → He **has been working** since 8 AM.
