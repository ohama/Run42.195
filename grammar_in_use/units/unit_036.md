# Unit 36: Would

## A. 문법 설명 (Grammar Explanation)

### Would의 다양한 용법

1. **과거의 습관**: 과거에 반복적으로 했던 일
2. **가정법**: 조건절과 함께 (~라면 ...할 텐데)
3. **정중한 요청**: Would you...?
4. **의지/거부**: 하려고 했다/하지 않으려 했다
5. **미래의 과거**: 과거 시점에서 본 미래

---

## B. 소설 속 예문 (Examples from the Novel)

### Example 1: 가정법 (Hypothetical)

> **"If I took the bus, it would be more comfortable."**
> 버스를 타면 더 편할 텐데.

- 현재 사실과 반대되는 가정

### Example 2: 과거 습관 (Past habit)

> **"Every morning, I would feel nauseous on the bus."**
> 매일 아침 버스에서 속이 울렁거리곤 했어.

- 과거에 반복되던 일

### Example 3: 정중한 요청 (Polite request)

> **"Would you run with me tomorrow?"**
> 내일 나랑 같이 달릴래?

- 정중한 부탁

### Example 4: Would like (원하다)

> **"I would like to finish the race."**
> 대회를 완주하고 싶어.

- want의 정중한 표현

### Example 5: Wouldn't (거부)

> **"My legs wouldn't move anymore."**
> 다리가 더 이상 움직이지 않았어.

- 과거의 거부/불가능

---

## C. 추가 설명 (Additional Points)

### Would vs Used to

| Would | Used to |
|-------|---------|
| 과거 반복 행동만 | 행동 + 상태 둘 다 |
| I would run every day. | I used to run every day. |
| (X) I would be weak. | I used to be weak. (O) |

**상태 동사**는 would 사용 불가!

### Would in Conditionals (가정법)

| 가정법 현재 | 가정법 과거 |
|-------------|-------------|
| If I had time, I **would** run. | If I had had time, I **would have** run. |
| (시간 있으면 달릴 텐데) | (시간 있었으면 달렸을 텐데) |

### Would like to

| Want to | Would like to |
|---------|---------------|
| 직접적 | 정중함 |
| I want to go. | I would like to go. |
| 뭔가 원해 | ~하고 싶습니다 |

## D. 비교 (Comparison)

| Would 용법 | 예문 | 한국어 |
|-----------|------|--------|
| 가정법 | If I ran, I **would** feel better. | 달리면 기분이 나아질 텐데 |
| 과거 습관 | I **would** run every morning. | 매일 아침 달리곤 했어 |
| 정중한 요청 | **Would** you help me? | 도와주시겠어요? |
| Would like | I **would like** to finish. | 완주하고 싶어요 |
| 거부 | The door **wouldn't** open. | 문이 안 열렸어 |

### 가정법 시제

| 시제 | If절 | 주절 |
|------|------|------|
| 현재 | If + 과거 | would + 동사원형 |
| 과거 | If + had + p.p. | would have + p.p. |

## Exercises

### A. Would 용법 파악하기

다음 문장의 would 용법을 고르세요: (a) 가정법 (b) 과거 습관 (c) 요청 (d) would like

1. "Would you run with me?" → **(c) 요청**
2. "If I had time, I would run." → **(a) 가정법**
3. "I would run every morning back then." → **(b) 과거 습관**
4. "I would like to finish the race." → **(d) would like**

### B. 문장 완성

1. (가정법) 버스 타면 더 편할 텐데.
   → If I took the bus, it _______ more comfortable.

2. (과거 습관) 매일 아침 달리곤 했어.
   → I _______ every morning.

3. (요청) 물 좀 건네줄래?
   → _______ you pass me the water?

4. (would like) 완주하고 싶어요.
   → I _______ finish the race.

**Answers:**
1. would be
2. would run
3. Would
4. would like to

### C. Would vs Used to

1. 예전에 매일 달리곤 했어. (둘 다 가능)
   → I _______ run every day.
   → I _______ run every day.

2. 예전에 약했어. (used to만 가능)
   → I _______ be weak.

**Answers:**
1. would / used to
2. used to
