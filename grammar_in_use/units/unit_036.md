# Unit 36: Would

## A. 형태 (Form)

### Would의 다양한 용법:

| Usage | Structure | Example |
|-------|-----------|---------|
| 과거의 습관 | would + 동사원형 | I would run every morning. |
| 가정법 | if + 과거, would + 동사원형 | If I ran, I would feel better. |
| 정중한 요청 | Would you...? | Would you help me? |
| Would like | would like + to + 동사원형 | I would like to finish. |
| 거부/불가능 | wouldn't + 동사원형 | The door wouldn't open. |

### 축약형:

| Full form | Contraction |
|-----------|-------------|
| I would | I'd |
| You would | You'd |
| He would | He'd |
| would not | wouldn't |

---

## B. 과거의 습관 (Past Habit)

과거에 **반복적으로 했던 일**을 표현합니다.

### From the Original Book:

> "When I was a child, we**'d** go to the beach every summer."
> 어렸을 때 매년 여름 해변에 가곤 했어.

> "He**'d** always arrive late. That was typical of him."
> 그는 항상 늦게 오곤 했어. 그게 그 사람이야.

### Examples from the Novel:

> **"Every morning, I would feel nauseous on the bus."**
> 매일 아침 버스에서 속이 울렁거리곤 했어.

- 과거에 반복되던 일

> **"When I started, I would give up after 5 minutes."**
> 처음에는 5분 후에 포기하곤 했어.

### Would vs Used to:

| Would | Used to |
|-------|---------|
| 과거 반복 **행동**만 | 행동 + **상태** 둘 다 |
| I would run every day. | I used to run every day. |
| (X) I would be weak. | I used to be weak. (O) |

**중요**: **상태 동사**는 would 사용 불가!

| Wrong | Correct |
|-------|---------|
| ~~I would be weak.~~ | I **used to be** weak. |
| ~~I would have a dog.~~ | I **used to have** a dog. |
| ~~I would live in Jeju.~~ | I **used to live** in Jeju. |

---

## C. 가정법 (Hypothetical/Conditional)

**현재 사실과 반대**되는 가정:

### 가정법 현재 (Second Conditional):

If + 과거, would + 동사원형

> **"If I took the bus, it would be more comfortable."**
> 버스를 타면 더 편할 텐데. (안 탐)

> **"If I had more time, I would train harder."**
> 시간이 더 있으면 더 열심히 훈련할 텐데. (시간 없음)

### 가정법 과거 (Third Conditional):

If + had + p.p., would have + p.p.

> **"If I had trained harder, I would have finished faster."**
> 더 열심히 훈련했으면 더 빨리 끝냈을 텐데.

### From the Original Book:

> "If I were you, I **would** get a new job."
> 내가 너라면 새 직장을 구할 텐데.

> "I **wouldn't** do that if I were you."
> 내가 너라면 그렇게 안 할 텐데.

---

## D. 정중한 요청 (Polite Requests)

**Would you...?** = 정중한 부탁

> **"Would you run with me tomorrow?"**
> 내일 나랑 같이 달릴래?

> **"Would you pass me the water?"**
> 물 좀 건네줄래요?

### Would vs Will (요청):

| Will you...? | Would you...? |
|--------------|---------------|
| 일반적 | 더 정중 |
| Will you help? | Would you help? |
| 도와줄래? | 도와주시겠어요? |

---

## E. Would like (원하다)

**want**의 정중한 표현:

> **"I would like to finish the race."**
> 대회를 완주하고 싶어요.

> **"Would you like some water?"**
> 물 좀 드실래요?

### Want to vs Would like to:

| Want to | Would like to |
|---------|---------------|
| 직접적 | 정중함 |
| I want to go. | I would like to go. |
| 가고 싶어 | 가고 싶습니다 |

### Common Patterns:

| Pattern | Example |
|---------|---------|
| I'd like to... | I'd like to run tomorrow. |
| Would you like to...? | Would you like to join us? |
| Would you like + 명사? | Would you like some coffee? |

---

## F. Wouldn't: 거부/불가능 (Refusal/Inability)

**과거의 거부** 또는 **불가능**:

> **"My legs wouldn't move anymore."**
> 다리가 더 이상 움직이지 않았어.

> **"The door wouldn't open."**
> 문이 안 열렸어.

> **"He wouldn't listen to me."**
> 그는 내 말을 들으려고 안 했어.

### From the Original Book:

> "I tried to start the car, but it **wouldn't** start."
> 차를 켜려고 했는데 안 켜졌어.

> "I asked her to help, but she **wouldn't**."
> 도움을 청했는데 도와주려고 안 했어.

---

## G. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I would to go.~~ | I **would go**. / I'd **like to go**. | would + 동사원형 |
| ~~I would can run.~~ | I **could run**. | 조동사 하나만 |
| ~~I would be weak.~~ (상태) | I **used to be** weak. | 상태 = used to |
| ~~Would you like go?~~ | Would you like **to go**? | to 필요 |
| ~~I would liked to.~~ | I **would like to**. | would + 동사원형 |
| ~~He would runs every day.~~ | He **would run** every day. | would + 동사원형 |

### Would vs Used to 선택:

| 상황 | 선택 |
|------|------|
| 과거 반복 행동 | would 또는 used to |
| 과거 상태 | used to만 |
| 과거 습관 + 더 이상 안 함 | used to 선호 |

## Exercises

### A. Would 용법 파악하기 (Identify the usage)

Choose: (a) 과거 습관 (b) 가정법 (c) 요청 (d) would like (e) 거부/불가능

1. "Would you run with me?" → **(c) 요청**
2. "If I had time, I would run." → **(b) 가정법**
3. "I would run every morning back then." → **(a) 과거 습관**
4. "I would like to finish the race." → **(d) would like**
5. "The door wouldn't open." → **(e) 거부/불가능**
6. "He would always arrive late." → **(a) 과거 습관**
7. "Would you like some water?" → **(d) would like**
8. "My legs wouldn't move." → **(e) 거부/불가능**

### B. 문장 완성 (Complete the sentences)

1. (가정법) 버스 타면 더 편할 텐데.
   → If I took the bus, it _______ more comfortable.

2. (과거 습관) 매일 아침 달리곤 했어.
   → I _______ every morning.

3. (요청) 물 좀 건네줄래?
   → _______ you pass me the water?

4. (would like) 완주하고 싶어요.
   → I _______ finish the race.

5. (거부) 다리가 안 움직였어.
   → My legs _______ move.

**Answers:**
1. would be
2. would run
3. Would
4. would like to
5. wouldn't

### C. Would vs Used to (Choose the correct form)

1. 예전에 매일 달리곤 했어. (둘 다 가능)
   → I _______ run every day.
   → I _______ run every day.

2. 예전에 약했어. (used to만 가능)
   → I _______ be weak.

3. 예전에 제주에 살았어. (used to만 가능)
   → I _______ live in Jeju.

4. 항상 늦게 오곤 했어. (둘 다 가능)
   → He _______ always arrive late.

5. 강아지가 있었어. (used to만 가능)
   → I _______ have a dog.

**Answers:**
1. would / used to
2. used to
3. used to
4. would / used to
5. used to

### D. 가정법 문장 완성 (Complete the conditionals)

1. If I had more time, I _______ (train) harder.
2. If I were you, I _______ (rest) today.
3. If I _______ (be) taller, I would be better at basketball.
4. I _______ (help) you if I could.
5. If she _______ (run) faster, she would have won.

**Answers:**
1. would train
2. would rest
3. were
4. would help
5. had run / would have won

## Would vs Used to Comparison

| Aspect | Would | Used to |
|--------|-------|---------|
| 과거 행동 | O | O |
| 과거 상태 | X | O |
| 예: 달리곤 했다 | I would run. | I used to run. |
| 예: 약했다 | X | I used to be weak. |
