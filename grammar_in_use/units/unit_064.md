# Unit 64: To..., for..., and so that... (purpose)

## A. 목적을 나타내는 표현 (Form)

**목적(~하기 위해)**을 나타내는 세 가지 방법:

| Structure | Example | Korean |
|-----------|---------|--------|
| **to + 동사원형** | I ran to catch the bus. | 버스를 잡기 위해 달렸다. |
| **for + 명사** | I went out for a run. | 달리기하러 나갔다. |
| **so that + 주어 + 동사** | I woke up early so that I could run. | 달릴 수 있도록 일찍 일어났다. |

### From the Original Book:

> "I went out **to buy** some food."
> 음식 좀 사러 나갔어.

> "I went out **for** a walk."
> 산책하러 나갔어.

---

## B. to + 동사원형 (Primary Usage - Most Common)

가장 일반적인 목적 표현입니다.

### From the Original Book:

> "I'm going out **to get** a newspaper."
> 신문 사러 나가.

> "She came **to see** me."
> 그녀가 나를 만나러 왔어.

### Examples from the Novel:

> **"I stopped to rest."**
> 쉬기 위해 멈췄다.

- **to + 동사**: ~하기 위해

> **"I woke up early to run."**
> 달리기 위해 일찍 일어났다.

> **"I need to leave early to arrive on time."**
> 제시간에 도착하기 위해 일찍 떠나야 해.

> **"I trained hard to finish the marathon."**
> 마라톤을 완주하기 위해 열심히 훈련했다.

### Purpose vs Other Uses of 'to':

| 목적 (Purpose) | 다른 용법 |
|----------------|----------|
| I went **to buy** milk. | I like **to run**. (좋아함) |
| I stopped **to rest**. | I need **to sleep**. (필요) |
| I came **to see** you. | I want **to go**. (원함) |

---

## C. in order to / so as to (Formal Purpose)

**in order to**와 **so as to**는 to보다 더 격식적인 표현입니다.

### From the Original Book:

> "She's studying **in order to pass** the exam."
> 그녀는 시험에 합격하기 위해 공부 중이야.

> "I spoke slowly **so as to** make myself understood."
> 이해받기 위해 천천히 말했어.

### Examples from the Novel:

> **"I woke up at 5 AM in order to run before school."**
> 학교 가기 전에 달리기 위해 새벽 5시에 일어났다.

- **in order to**: ~하기 위해 (격식)

> **"I stretched so as to avoid injury."**
> 부상을 피하기 위해 스트레칭을 했다.

- **so as to**: ~하기 위해 (격식)

### 부정형 (Negative Purpose):

| Negative form | Example |
|---------------|---------|
| in order **not** to | I stretched **in order not to** get injured. |
| so as **not** to | I trained hard **so as not to** fail. |

> **"I stretched in order not to get injured."**
> 다치지 않기 위해 스트레칭을 했다.

**주의:** not to만 단독으로 쓰는 것은 어색합니다.
- ~~I woke up early not to be late.~~ (어색함)
- I woke up early **in order not to** be late. (자연스러움)

### 격식 수준:

| Expression | Formality |
|------------|-----------|
| to | ★☆☆☆☆ (일상) |
| in order to | ★★★☆☆ (격식) |
| so as to | ★★★★☆ (더 격식) |

## D. for + 명사 (Purpose with Noun)

**for** 뒤에는 명사가 옵니다.

### From the Original Book:

> "I went to the store **for** some bread."
> 빵 사러 가게에 갔어.

> "What did you do that **for**?"
> 왜 그랬어?

### Examples from the Novel:

> **"I went out for a run."**
> 달리기하러 나갔다.

- **for + 명사**: ~을 위해, ~하러

> **"I stopped for a break."**
> 쉬기 위해 멈췄다.

> **"I need new shoes for running."**
> 달리기용 새 신발이 필요해.

### for + 명사 vs to + 동사:

| for + 명사 | to + 동사 |
|------------|-----------|
| for a run | to run |
| for a break | to take a break |
| for exercise | to exercise |
| for a walk | to walk |
| for a swim | to swim |

### 주의: for + -ing (목적 vs 용도)

| 용도 (what something is for) | 목적 (why you do something) |
|------------------------------|----------------------------|
| a knife **for cutting** bread | I use it **to cut** bread |
| shoes **for running** | I bought them **to run** |

---

## E. so that + 주어 + can/could/will/would (Complex Purpose)

더 복잡한 목적을 표현할 때 사용합니다.

### From the Original Book:

> "I'll give you a map **so that** you can find the way."
> 길을 찾을 수 있도록 지도를 줄게.

> "I left early **so that** I wouldn't be late."
> 늦지 않도록 일찍 떠났어.

### Examples from the Novel:

> **"I woke up early so that I could run before school."**
> 학교 가기 전에 달릴 수 있도록 일찍 일어났다.

- **so that + can/could**: ~할 수 있도록

> **"I trained hard so that I would be ready for the race."**
> 레이스에 준비되도록 열심히 훈련했다.

> **"I ate well so that I'd have energy for the run."**
> 달리기에 에너지가 있도록 잘 먹었다.

### so that의 시제:

| Main clause | so that clause |
|-------------|----------------|
| 현재 | so that + can/will |
| 과거 | so that + could/would |

## F. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I went out **for run**.~~ | I went out **for a run**. | for + a/the + 명사 |
| ~~I woke up early **for to run**.~~ | I woke up early **to run**. | for + 명사 또는 to + 동사 |
| ~~I came here **for see** you.~~ | I came here **to see** you. | for + 명사, to + 동사 |
| ~~I studied not to fail.~~ | I studied **in order not to** fail. | 부정형은 in order not to |
| ~~I ran **so that catch** the bus.~~ | I ran **so that I could catch** the bus. | so that + 주어 + 동사 |
| ~~**For** to improve...~~ | **To** improve... | for + to 불가 |

## G. 목적 vs 결과 (Purpose vs Result)

**주의:** to가 목적이 아닌 결과를 나타낼 수도 있습니다.

| Purpose (목적) | Result (결과) |
|----------------|---------------|
| I ran **to catch** the bus. | I ran **only to miss** the bus. |
| (잡으려고 달렸다) | (달렸지만 놓쳤다) |

### From the Novel:

> **"I woke up early to run."** (목적: 달리기 위해)
> **"I arrived at school to find it was empty."** (결과: 도착하니 비어있었다)

### 결과를 나타내는 표현:

| Expression | Example |
|------------|---------|
| only to + 동사 | I ran, **only to miss** the bus. |
| to find (that) | I arrived **to find** it empty. |

## Exercises

### A. to, for, so that 중 알맞은 것을 고르세요.

1. I woke up early _______ run.
2. I went out _______ a walk.
3. I trained hard _______ I could finish the marathon.
4. I stopped _______ a break.
5. She ran fast _______ catch the bus.
6. I need new shoes _______ running.
7. I left early _______ I wouldn't be late.
8. I came _______ see you.

**Answers:**
1. to
2. for
3. so that
4. for
5. to
6. for
7. so that
8. to

### B. 문장을 완성하세요.

1. I trained hard _______ (finish) the marathon.
2. I need new shoes _______ (running).
3. I woke up early _______ (so that / can) run before school.
4. I stretched _______ (in order / not) get injured.
5. She went out _______ (for) a run.
6. I saved money _______ (buy) new running shoes.
7. I ate a big breakfast _______ (so that / would) have energy.
8. He speaks slowly _______ (so as / not) make mistakes.

**Answers:**
1. to finish
2. for
3. so that I could
4. in order not to
5. for
6. to buy
7. so that I would
8. so as not to

### C. 같은 의미로 바꾸세요.

1. I ran to catch the bus.
   → I ran **in order** _______.
   → I ran **so that** _______.

2. I woke up early for a morning run.
   → I woke up early _______ run in the morning.

3. I studied hard so that I wouldn't fail.
   → I studied hard **in order** _______.

**Answers:**
1. in order to catch the bus / so that I could catch the bus
2. to
3. in order not to fail

### D. 틀린 부분을 고치세요.

1. I went out for run.
   → _______

2. I came here for see you.
   → _______

3. I studied not to fail.
   → _______

4. I ran so that catch the bus.
   → _______

**Answers:**
1. I went out **for a run**.
2. I came here **to see** you.
3. I studied **in order not to** fail.
4. I ran **so that I could catch** the bus.

## Summary: 목적 표현 비교

| Expression | Structure | Formality | Example |
|------------|-----------|-----------|---------|
| **to** | + 동사원형 | 일상적 | I stopped to rest. |
| **in order to** | + 동사원형 | 격식 | I trained in order to win. |
| **so as to** | + 동사원형 | 더 격식 | I stretched so as to avoid injury. |
| **for** | + 명사 | 일상적 | I went out for a run. |
| **so that** | + 주어 + 동사 | 복잡한 목적 | I trained so that I could win. |

### 부정 목적:
| Positive | Negative |
|----------|----------|
| to win | **in order not to** fail |
| so that I can | so that I **won't** / **couldn't** |

### Quick Decision:
- 동사 필요? → **to** 또는 **in order to**
- 명사만? → **for**
- 복잡한 문장? → **so that + 주어 + 동사**
