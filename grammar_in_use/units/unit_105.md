# Unit 105: Comparison (1) - cheaper, more expensive, etc.

## A. 비교급 만들기 (Forming comparatives)

두 가지를 비교할 때 **비교급**을 사용합니다.

### Basic Rules:

| 형용사 유형 | 비교급 형태 | 예시 |
|-------------|-------------|------|
| 짧은 형용사 (1음절) | 형용사 + -er | fast → fast**er** |
| -e로 끝남 | + -r | large → larg**er** |
| -y로 끝남 | y → i + -er | easy → eas**ier** |
| 긴 형용사 (2음절+) | more + 형용사 | difficult → **more** difficult |

### From the Original Book:

> "It's **cheaper** to go by car than by train."
> 기차보다 차로 가는 게 더 싸다.

> "Going by train is **more expensive** than going by car."
> 기차로 가는 게 차로 가는 것보다 더 비싸다.

---

## B. 짧은 형용사 + -er (Short adjectives)

### Examples from the Novel:

> **"Running is faster than walking."**
> 달리기가 걷기보다 빠르다.

- fast → faster

> **"The morning bus is later than the afternoon bus."**
> 아침 버스가 오후 버스보다 늦다.

- late → later

> **"I got stronger over the months."**
> 몇 달이 지나면서 더 강해졌다.

- strong → stronger

> **"My legs felt heavier after 30km."**
> 30km 후에 다리가 더 무겁게 느껴졌다.

- heavy → heavier

> **"The second semester was harder than the first."**
> 2학기가 1학기보다 힘들었다.

- hard → harder

> **"I finished earlier than expected."**
> 예상보다 일찍 끝냈다.

- early → earlier

---

## C. more + 긴 형용사 (Long adjectives)

### Examples from the Novel:

> **"Walking was more interesting than taking the bus."**
> 걷는 게 버스 타는 것보다 더 재미있었다.

- interesting → more interesting

> **"The 10km race was more challenging than I expected."**
> 10km 대회는 예상보다 더 도전적이었다.

- challenging → more challenging

> **"Running became more enjoyable over time."**
> 달리기가 시간이 지나면서 더 즐거워졌다.

- enjoyable → more enjoyable

> **"The full marathon was more exhausting than the half."**
> 풀마라톤이 하프보다 더 지쳤다.

- exhausting → more exhausting

> **"The scenery was more beautiful than I imagined."**
> 경치가 상상보다 더 아름다웠다.

- beautiful → more beautiful

---

## D. 불규칙 비교급 (Irregular comparatives)

| 원급 | 비교급 | 의미 |
|------|--------|------|
| good | better | 좋은 → 더 좋은 |
| bad | worse | 나쁜 → 더 나쁜 |
| far | farther/further | 먼 → 더 먼 |
| little | less | 적은 → 더 적은 |
| much/many | more | 많은 → 더 많은 |
| old | older/elder | 나이 든 → 더 나이 든 |

### From the Original Book:

> "Her health is **worse** than it was last week."
> 그녀의 건강이 지난주보다 나빠졌다.

> "I can run **further** than you."
> 나는 너보다 더 멀리 달릴 수 있다.

### Examples from the Novel:

> **"I felt better after walking."**
> 걷고 나서 기분이 좋아졌다.

> **"The bus ride was worse than I expected."**
> 버스 타기가 예상보다 나빴다.

> **"I ran farther than last week."**
> 지난주보다 더 멀리 달렸다.

> **"I need more practice."**
> 더 많은 연습이 필요하다.

---

## E. 비교급 + than (Comparisons with than)

비교 대상을 나타낼 때 **than**을 사용합니다.

### From the Original Book:

> "He's two years **older than** me." / "He's two years **older than** I am."
> 그는 나보다 두 살 많다.

### Examples from the Novel:

> **"The afternoon bus is emptier than the morning bus."**
> 오후 버스가 아침 버스보다 비었다.

> **"Running feels better than sitting on the bus."**
> 달리기가 버스에 앉아 있는 것보다 기분이 좋다.

> **"I arrived earlier than expected."**
> 예상보다 일찍 도착했다.

> **"The 21km was harder than the 10km."**
> 21km가 10km보다 힘들었다.

> **"My stamina is better than before."**
> 내 체력이 예전보다 좋다.

## F. 철자 규칙 (Spelling rules)

### Detailed Rules:

| 규칙 | 원급 | 비교급 | 설명 |
|------|------|--------|------|
| 자음 중복 | big | bi**gg**er | 단모음 + 단자음 |
| | hot | ho**tt**er | |
| | thin | thi**nn**er | |
| | wet | we**tt**er | |
| -y → -ier | easy | eas**ier** | y 앞이 자음 |
| | early | earl**ier** | |
| | heavy | heav**ier** | |
| | happy | happ**ier** | |
| 기본 | fast | fast**er** | 그냥 -er |
| | long | long**er** | |
| -e → -r | large | larg**er** | 이미 e가 있음 |
| | late | lat**er** | |

### Examples from the Novel:

- big → bi**gg**er: The medal was bigger than I expected.
- easy → eas**ier**: The race got easier with practice.
- late → lat**er**: I arrived later than planned.

## G. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | 설명 |
|-----------|-------------|------|
| ~~more fast~~ | **faster** | 1음절 → -er |
| ~~more easy~~ | **easier** | -y로 끝나는 1-2음절 → -ier |
| ~~gooder~~ | **better** | 불규칙 |
| ~~badder~~ | **worse** | 불규칙 |
| ~~more better~~ | **better** | 불규칙에 more 불필요 |
| ~~difficultier~~ | **more difficult** | 긴 형용사 → more |
| ~~interestinger~~ | **more interesting** | 긴 형용사 → more |
| ~~biger~~ | **bigger** | 자음 중복 |

## Exercises

### A. 비교급을 쓰세요 (Write the comparative)

1. fast → _______
2. difficult → _______
3. good → _______
4. easy → _______
5. interesting → _______
6. hot → _______
7. bad → _______
8. far → _______

**Answers:**
1. faster
2. more difficult
3. better
4. easier
5. more interesting
6. hotter
7. worse
8. farther/further

---

### B. 문장을 완성하세요 (Complete the sentences)

1. Running is _______ (fast) than walking.
   → **faster**

2. The race was _______ (hard) than I expected.
   → **harder**

3. Walking is _______ (interesting) than taking the bus.
   → **more interesting**

4. I felt _______ (good) after the run.
   → **better**

5. The morning bus is _______ (crowded) than the afternoon bus.
   → **more crowded**

6. My legs felt _______ (heavy) after 30km.
   → **heavier**

7. The second semester was _______ (bad) than the first.
   → **worse**

8. I ran _______ (far) than last week.
   → **farther/further**

---

### C. than을 사용하여 문장을 만드세요 (Make sentences with than)

1. The 10km race / difficult / the 5km race
   → The 10km race is **more difficult than** the 5km race.

2. Running / enjoyable / taking the bus
   → Running is **more enjoyable than** taking the bus.

3. My stamina / good / before
   → My stamina is **better than** before.

4. The second semester / hard / the first
   → The second semester is **harder than** the first.

5. This route / long / that route
   → This route is **longer than** that route.

6. The marathon / exhausting / the half marathon
   → The marathon is **more exhausting than** the half marathon.

---

### D. 틀린 부분을 고치세요 (Error correction)

1. Running is more fast than walking.
   → Running is **faster** than walking.

2. I feel more good today.
   → I feel **better** today.

3. The race was difficultier than expected.
   → The race was **more difficult** than expected.

4. He is more tall than me.
   → He is **taller** than me.

5. The bus is more crowdeder than usual.
   → The bus is **more crowded** than usual.

6. This is more better than that.
   → This is **better** than that.

## Summary

### Formation Rules:

| 형용사 유형 | 비교급 규칙 | 예시 |
|-------------|-------------|------|
| 1음절 | + er | fast → faster |
| 1음절 (자음중복) | 자음 + er | big → bigger |
| -y로 끝남 | y → ier | easy → easier |
| 2음절+ | more + 형용사 | difficult → more difficult |
| 불규칙 | 외워야 함 | good → better |

### Irregular Comparatives:

| 원급 | 비교급 |
|------|--------|
| good | better |
| bad | worse |
| far | farther/further |
| much/many | more |
| little | less |

### Quick Decision:

```

 예    아니오
```
