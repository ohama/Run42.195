# Unit 51: Auxiliary Verbs (have/do/can, etc.)

## A. 조동사의 역할 (The Role of Auxiliary Verbs)

조동사(Auxiliary Verbs)는 본동사를 도와 **시제, 부정, 의문, 강조** 등을 표현합니다.

### From the Original Book:

| 조동사 | 역할 | 예문 |
|--------|------|------|
| **be** | 진행형/수동태 | I **am** running. / It **is** done. |
| **have** | 완료형 | I **have** finished. |
| **do** | 의문/부정/강조 | **Do** you run? / I **don't** know. |
| **will** | 미래 | I **will** run. |
| **can** | 능력/가능 | I **can** run. |
| **must** | 의무 | You **must** rest. |
| **should** | 조언/의무 | You **should** train. |
| **would** | 가정/정중 | I **would** go if I could. |
| **could** | 능력(과거)/가능 | I **could** run fast. |
| **might** | 가능성 | It **might** rain. |

---

## B. 대답에서 조동사 반복 (Short Answers)

짧은 대답에서 조동사만 사용하여 답할 수 있습니다.

### From the Original Book:

> "Are you tired?" "**Yes, I am.**" / "**No, I'm not.**"
> "Can you swim?" "**Yes, I can.**" / "**No, I can't.**"
> "Have you finished?" "**Yes, I have.**" / "**No, I haven't.**"

### 시제/조동사별 짧은 대답:

| 질문 | 긍정 대답 | 부정 대답 |
|------|-----------|-----------|
| Are you tired? | Yes, I **am**. | No, I'**m not**. |
| Do you run? | Yes, I **do**. | No, I **don't**. |
| Did you finish? | Yes, I **did**. | No, I **didn't**. |
| Have you eaten? | Yes, I **have**. | No, I **haven't**. |
| Can you help? | Yes, I **can**. | No, I **can't**. |
| Will you come? | Yes, I **will**. | No, I **won't**. |

### Examples from the Novel:

> **"Are you nervous?" "I am."** / **"Not really."**
> "긴장돼?" "응." / "별로."

> **"Can you run 10km?" "Yes, I can."** / **"No, I can't."**
> "10km 달릴 수 있어?" "응, 할 수 있어." / "아니, 못해."

> **"Did you finish the race?" "Yes, I did."**
> "대회 끝냈어?" "응, 끝냈어."

> **"Have you ever run a marathon?" "No, I haven't."**
> "마라톤 뛴 적 있어?" "아니, 없어."

## C. So do I / Neither do I (동의 표현)

상대방의 말에 **동의**할 때 조동사를 사용합니다.

### From the Original Book:

### 긍정에 동의: So + 조동사 + 주어

> "I'm tired." "**So am I.**" (= I am too)
> "나 피곤해." "나도."

> "I can swim." "**So can I.**" (= I can too)
> "나 수영할 수 있어." "나도."

> "I work hard." "**So do I.**" (= I do too)
> "나 열심히 일해." "나도."

### 부정에 동의: Neither + 조동사 + 주어

> "I'm not tired." "**Neither am I.**" (= I'm not either)
> "나 안 피곤해." "나도."

> "I can't swim." "**Neither can I.**" (= I can't either)
> "나 수영 못해." "나도."

> "I don't smoke." "**Neither do I.**" (= I don't either)
> "나 담배 안 펴." "나도."

### 조동사별 동의 표현:

| 긍정 동의 (So...) | 부정 동의 (Neither...) |
|-------------------|------------------------|
| So **am** I. | Neither **am** I. |
| So **do** I. | Neither **do** I. |
| So **did** I. | Neither **did** I. |
| So **have** I. | Neither **have** I. |
| So **can** I. | Neither **can** I. |
| So **will** I. | Neither **will** I. |
| So **would** I. | Neither **would** I. |

### Examples from the Novel:

> **"I'm tired." "So am I."**
> "나 피곤해." "나도."

> **"I can run 5km." "So can I."**
> "나 5km 달릴 수 있어." "나도."

> **"I ran this morning." "So did I."**
> "오늘 아침에 달렸어." "나도."

> **"I'm not tired." "Neither am I."**
> "나 안 피곤해." "나도."

> **"I can't run fast." "Neither can I."**
> "나 빨리 못 달려." "나도."

> **"I didn't finish." "Neither did I."**
> "나 못 끝냈어." "나도."

## D. 부가 의문문의 조동사 (Tag Questions)

부가 의문문에서 앞 문장의 조동사를 반복합니다.

### From the Original Book:

> "You **are** running, **aren't** you?"
> 달리고 있지, 그렇지?

> "He **can** run fast, **can't** he?"
> 그는 빨리 달릴 수 있지, 그렇지?

> "You **didn't** finish, **did** you?"
> 끝 못 냈지, 그렇지?

### 규칙:
- 긍정문 → 부정 태그
- 부정문 → 긍정 태그

## E. 강조의 do/does/did (Emphatic Do)

평서문에서 **do/does/did**를 사용하면 **강조** 효과가 있습니다.

### From the Original Book:

> "I **do** like him." (I really like him)
> 나는 정말 그를 좋아해.

> "She **does** work hard." (She really works hard)
> 그녀는 정말 열심히 일해.

> "I **did** tell you." (I really told you)
> 나는 정말 말했어.

### 비교:

| 일반 | 강조 |
|------|------|
| I like running. | I **do** like running. |
| She works hard. | She **does** work hard. |
| He finished. | He **did** finish. |

### Examples from the Novel:

> **"I do want to finish the marathon."**
> 나는 **정말** 마라톤을 완주하고 싶다.

> **"He did run 7km this morning."**
> 그는 오늘 아침에 **정말** 7km를 달렸다.

> **"The race does get harder after 30km."**
> 30km 이후에 대회가 **정말** 더 힘들어진다.

## F. Have / Have got (소유 표현)

**have**와 **have got**은 소유를 나타낼 때 같은 의미입니다.

### From the Original Book:

| Have (주로 미국식) | Have got (주로 영국식) |
|--------------------|------------------------|
| I **have** a car. | I**'ve got** a car. |
| I **don't have** a car. | I **haven't got** a car. |
| **Do** you **have** a car? | **Have** you **got** a car? |

### Examples from the Novel:

> **"I have a running watch."** = **"I've got a running watch."**
> 러닝 워치가 있다.

> **"Do you have running shoes?"** = **"Have you got running shoes?"**
> 러닝화 있어?

> **"I don't have much stamina."** = **"I haven't got much stamina."**
> 체력이 별로 없다.

### 주의사항:

| 용법 | Have | Have got |
|------|------|----------|
| 소유 | O | O |
| 습관 | O | X |
| 동작 | O | X |

```
소유: I have a car. = I've got a car.
습관: I have breakfast at 8. (have got X)
동작: I'm having lunch. (have got X)
```

**주의:** have got은 주로 영국 영어에서 사용되며, 소유 의미로만 사용됩니다.

## G. 흔한 실수 (Common Mistakes)

| 틀린 문장 (X) | 올바른 문장 (O) | 설명 |
|---------------|-----------------|------|
| "I'm tired." "**So I am.**" | "**So am I.**" | So + 조동사 + 주어 |
| "I don't like it." "**Neither I do.**" | "**Neither do I.**" | Neither + 조동사 + 주어 |
| "Do you have?" | "**Do you have** it?" / "**Have you got** it?" | 목적어 필요 |
| I **have got** breakfast. | I **have** breakfast. | have got은 소유만 |
| "Yes, I'm." | "Yes, I **am**." | 축약 불가 |
| "No, I amn't." | "No, I'**m not**." | amn't 없음 |

### So/Neither 어순 오류:

```
(X) "I'm tired." "So I am."
(O) "I'm tired." "So am I."

So/Neither + 조동사 + 주어 (도치)
```

### 짧은 대답 오류:

```
(X) "Are you coming?" "Yes, I'm."
(O) "Are you coming?" "Yes, I am."

긍정 짧은 대답에서는 축약 불가
```

## Summary: 조동사 주요 기능

### 1. 짧은 대답:
```
"Are you tired?" → "Yes, I am." / "No, I'm not."
"Did you go?" → "Yes, I did." / "No, I didn't."
"Can you help?" → "Yes, I can." / "No, I can't."
```

### 2. 동의 표현:
```
긍정 동의: So + 조동사 + 주어
"I'm tired." → "So am I."

부정 동의: Neither + 조동사 + 주어
"I can't swim." → "Neither can I."
```

### 3. 강조:
```
do/does/did + 동사원형
I do like running. (정말 좋아해)
He did finish. (정말 끝냈어)
```

### 4. Have / Have got (소유):
```
I have = I've got
Do you have? = Have you got?
I don't have = I haven't got
```
