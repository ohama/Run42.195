# Unit 51: Auxiliary Verbs (have/do/can, etc.)

## A. 조동사의 역할

조동사(Auxiliary Verbs)는 본동사를 도와 **시제, 부정, 의문, 강조** 등을 표현합니다.

| 조동사 | 역할 | 예문 |
|--------|------|------|
| be | 진행형/수동태 | I **am** running. |
| have | 완료형 | I **have** finished. |
| do | 의문/부정/강조 | **Do** you run? |
| will | 미래 | I **will** run. |
| can | 능력/가능 | I **can** run. |
| must | 의무 | You **must** rest. |
| should | 조언/의무 | You **should** train. |

---

## B. 대답에서 조동사 반복

짧은 대답에서 조동사만 사용하여 답할 수 있습니다.

### Examples from the Novel:

> **"Are you nervous?" "I am."** / **"Not really."**
> "긴장돼?" "응." / "별로."

> **"Can you run 10km?" "Yes, I can."** / **"No, I can't."**
> "10km 달릴 수 있어?" "응, 할 수 있어." / "아니, 못해."

> **"Did you finish the race?" "Yes, I did."**
> "대회 끝냈어?" "응, 끝냈어."

> **"Have you ever run a marathon?" "No, I haven't."**
> "마라톤 뛴 적 있어?" "아니, 없어."

---

## C. So do I / Neither do I

상대방의 말에 **동의**할 때 조동사를 사용합니다.

### 긍정에 동의: So + 조동사 + 주어

> **"I'm tired." "So am I."**
> "나 피곤해." "나도."

> **"I can run 5km." "So can I."**
> "나 5km 달릴 수 있어." "나도."

> **"I ran this morning." "So did I."**
> "오늘 아침에 달렸어." "나도."

### 부정에 동의: Neither + 조동사 + 주어

> **"I'm not tired." "Neither am I."**
> "나 안 피곤해." "나도."

> **"I can't run fast." "Neither can I."**
> "나 빨리 못 달려." "나도."

> **"I didn't finish." "Neither did I."**
> "나 못 끝냈어." "나도."

---

## D. 부가 의문문 (Tag Questions)의 조동사

부가 의문문에서 앞 문장의 조동사를 반복합니다.

> **"You are running, aren't you?"**
> 달리고 있지, 그렇지?

> **"He can run fast, can't he?"**
> 그는 빨리 달릴 수 있지, 그렇지?

> **"You didn't finish, did you?"**
> 끝 못 냈지, 그렇지?

(자세한 내용은 Unit 52에서)

---

## E. 강조의 do/does/did

평서문에서 **do/does/did**를 사용하면 **강조** 효과가 있습니다.

### Examples from the Novel:

> **"I do want to finish the marathon."**
> 나는 **정말** 마라톤을 완주하고 싶다.

> **"He did run 7km this morning."**
> 그는 오늘 아침에 **정말** 7km를 달렸다.

> **"The race does get harder after 30km."**
> 30km 이후에 대회가 **정말** 더 힘들어진다.

### 비교:
- I want to finish. (보통)
- I **do** want to finish. (강조)

---

## F. Have / Have got

**have**와 **have got**은 소유를 나타낼 때 같은 의미입니다.

| Have | Have got |
|------|----------|
| I **have** a car. | I**'ve got** a car. |
| I **don't have** a car. | I **haven't got** a car. |
| **Do** you **have** a car? | **Have** you **got** a car? |

### Examples from the Novel:

> **"I have a running watch."** = **"I've got a running watch."**
> 러닝 워치가 있다.

> **"Do you have running shoes?"** = **"Have you got running shoes?"**
> 러닝화 있어?

> **"I don't have much stamina."** = **"I haven't got much stamina."**
> 체력이 별로 없다.

**주의:** have got은 주로 영국 영어에서 사용됩니다.

## Exercises

### A. 짧은 대답을 완성하세요.

1. "Are you tired?" "Yes, I _______."
2. "Can you run 10km?" "No, I _______."
3. "Did he finish?" "Yes, he _______."
4. "Have you trained?" "No, I _______."
5. "Will you run tomorrow?" "Yes, I _______."

**Answers:**
1. am
2. can't
3. did
4. haven't
5. will

### B. So/Neither를 사용해 동의를 표현하세요.

1. "I'm tired." "_______ am I."
2. "I can run 5km." "_______ can I."
3. "I didn't sleep well." "_______ did I."
4. "I haven't finished." "_______ have I."
5. "I don't like the bus." "_______ do I."

**Answers:**
1. So
2. So
3. Neither
4. Neither
5. Neither

### C. 강조의 do/does/did를 사용해 문장을 다시 쓰세요.

1. I want to finish. → I _______ want to finish.
2. He runs every day. → He _______ run every day.
3. She finished the race. → She _______ finish the race.

**Answers:**
1. do
2. does
3. did

### D. Have 또는 Have got으로 바꿔 쓰세요.

1. I have a car. → I _______ got a car.
2. Do you have time? → _______ you got time?
3. She doesn't have a watch. → She _______ got a watch.

**Answers:**
1. 've
2. Have
3. hasn't

### E. 알맞은 조동사를 넣으세요.

1. I _______ running right now. (진행형)
2. He _______ finished the race. (완료형)
3. _______ you run every day? (의문문)
4. She _______ run very fast. (능력)
5. You _______ rest today. (조언)

**Answers:**
1. am
2. has
3. Do
4. can
5. should

## Summary

```
조동사의 주요 기능:

1. 짧은 대답: Yes, I am. / No, I can't.

2. 동의 표현:
   긍정 동의: So + 조동사 + I (So do I)
   부정 동의: Neither + 조동사 + I (Neither do I)

3. 강조: do/does/did + 동사원형
   I do want to finish.

4. Have / Have got (소유):
   I have = I've got
   Do you have? = Have you got?
```
