# Unit 34: Should (2)

## A. 형태 (Form)

### Should have done: 과거에 대한 후회/비판

"~했어야 했는데 (안 했다)" 또는 "~하지 말았어야 했는데 (했다)"

| Form | Structure | Meaning |
|------|-----------|---------|
| should have + p.p. | should have done | ~했어야 했다 (안 함) |
| shouldn't have + p.p. | shouldn't have done | ~하지 말았어야 했다 (함) |

### Examples:

| Form | Example | Reality |
|------|---------|---------|
| should have done | I should have trained harder. | 안 했음 |
| shouldn't have done | I shouldn't have eaten so much. | 먹었음 |

---

## B. Should have done: 후회 (Regret)

**과거에 했어야 했는데 안 한 것**에 대한 후회

### From the Original Book:

> "I should have studied harder for the exam."
> 시험 공부 더 열심히 했어야 했는데.

> "We missed the train. We **should have left** earlier."
> 기차 놓쳤어. 더 일찍 출발했어야 했는데.

> "I'm sorry I'm late. I **should have called** you."
> 늦어서 미안해. 전화했어야 했는데.

### Examples from the Novel:

> **"I should have listened to my coach."**
> 코치님 말씀을 들었어야 했는데.

- 듣지 않은 것에 대한 후회

> **"I should have trained more before the race."**
> 대회 전에 더 훈련했어야 했는데.

- 충분히 훈련하지 않은 것에 대한 후회

> **"I should have eaten breakfast."**
> 아침을 먹었어야 했는데.

---

## C. Shouldn't have done: 하지 말았어야 (Regret for doing)

**과거에 했는데 하지 말았어야 했던 것**

### From the Original Book:

> "I ate too much. I **shouldn't have eaten** so much."
> 너무 많이 먹었어. 그렇게 많이 먹지 말았어야 했는데.

> "You **shouldn't have said** that. She was very upset."
> 그 말 하지 말았어야 했어. 그녀가 많이 속상해했거든.

### Examples from the Novel:

> **"I shouldn't have run so fast at the beginning."**
> 처음에 그렇게 빨리 달리지 말았어야 했는데.

- 한 행동에 대한 후회

> **"I shouldn't have stayed up late last night."**
> 어젯밤에 늦게까지 안 깨어 있었어야 했는데.

---

## D. Should have: 예상했던 일 (Expected but didn't happen)

**당연히 일어났어야 하는데 안 일어난 것**

> **"The race should have started by now."**
> 대회가 지금쯤 시작했어야 하는데.

> "The letter **should have arrived** by now."
> 편지가 지금쯤 도착했어야 하는데.

> "He **should have been** here an hour ago."
> 한 시간 전에 여기 있었어야 하는데.

---

## E. Should have vs Could have vs Would have

| Expression | Meaning | Example |
|------------|---------|---------|
| should have | ~했어야 했다 (후회/비판) | I **should have** trained harder. |
| could have | ~할 수 있었다 (가능성) | I **could have** taken the bus. |
| would have | ~했을 것이다 (가정) | I **would have** helped you. |

### Comparison:

| Form | Focus | Example | Korean |
|------|-------|---------|--------|
| should have | 후회/비판 | You should have told me. | 말해줬어야지. |
| could have | 가능했던 선택 | You could have told me. | 말해줄 수 있었잖아. |
| would have | 가정 결과 | I would have told you. | 말해줬을 텐데. |

---

## F. Should do vs Should have done

| Should do | Should have done |
|-----------|------------------|
| 현재/미래 조언 | 과거에 대한 후회 |
| You should rest. | You should have rested. |
| (쉬어야 해) | (쉬었어야 했는데) |
| You should call her. | You should have called her. |
| (전화해야 해) | (전화했어야 했는데) |

---

## G. Ought to have done

**Should have**와 같은 의미:

> "I **ought to have** listened." = "I **should have** listened."
> 들었어야 했는데.

| Should have | Ought to have |
|-------------|---------------|
| 더 일반적 | 더 격식체 |
| I should have known. | I ought to have known. |

---

## H. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I should have train harder.~~ | I **should have trained** harder. | have + 과거분사 |
| ~~I should had trained.~~ | I **should have trained**. | should have + p.p. |
| ~~I shouldn't have ran.~~ | I **shouldn't have run**. | run의 과거분사 = run |
| ~~I should have to go.~~ | I **should have gone**. | have + p.p. |
| ~~He should have listen.~~ | He **should have listened**. | 과거분사 |
| ~~You should have not told.~~ | You **shouldn't have told**. | not 위치 |

### Common Past Participles:

| Base | Past Participle |
|------|-----------------|
| go | gone |
| eat | eaten |
| run | run |
| tell | told |
| listen | listened |
| train | trained |
| stay | stayed |

## Exercises

### A. Should have / Shouldn't have 완성 (Complete the sentences)

1. 더 열심히 훈련했어야 했는데. (안 함)
   → I _______ harder.

2. 그렇게 빨리 달리지 말았어야 했는데. (달림)
   → I _______ so fast.

3. 코치님 말씀을 들었어야 했는데. (안 들음)
   → I _______ to my coach.

4. 어젯밤에 늦게까지 안 깨어 있었어야 했는데. (깨어 있었음)
   → I _______ late last night.

5. 아침을 먹었어야 했는데. (안 먹음)
   → I _______ breakfast.

6. 그 말 하지 말았어야 했어. (말했음)
   → I _______ that.

**Answers:**
1. should have trained
2. shouldn't have run
3. should have listened
4. shouldn't have stayed up
5. should have eaten
6. shouldn't have said

### B. 상황에 맞게 작성 (Write based on context)

1. (후회) 아침을 먹었어야 했는데.
   → I _______ breakfast.

2. (후회) 그렇게 많이 먹지 말았어야 했는데.
   → I _______ so much.

3. (예상) 지금쯤 도착했어야 하는데.
   → He _______ by now.

4. (비판) 더 일찍 떠났어야 했어.
   → We _______ earlier.

5. (후회) 알았어야 했는데.
   → I _______.

**Answers:**
1. should have eaten
2. shouldn't have eaten
3. should have arrived
4. should have left
5. should have known

### C. 문장 변환 (Sentence Transformation)

Transform to express regret.

1. I didn't listen to my coach. (후회로 표현)
   → I _______ to my coach.

2. I ran too fast at the start. (후회로 표현)
   → I _______ so fast at the start.

3. I didn't rest before the race. (후회로 표현)
   → I _______ before the race.

4. I stayed up late. (후회로 표현)
   → I _______ late.

5. I didn't bring water. (후회로 표현)
   → I _______ water.

**Answers:**
1. should have listened
2. shouldn't have run
3. should have rested
4. shouldn't have stayed up
5. should have brought

### D. Should have vs Could have vs Would have (Choose the correct form)

1. I _______ helped you, but I didn't know you needed help. (가정)
   → **would have**

2. You _______ told me earlier! (비판/후회)
   → **should have**

3. I _______ taken the bus, but I chose to walk. (가능했던 선택)
   → **could have**

4. I _______ studied harder for the test. (후회)
   → **should have**

5. If I had known, I _______ come. (가정)
   → **would have**

| Pattern | Example | Korean |
|---------|---------|--------|
| I should have... | I should have listened. | 들었어야 했는데. |
