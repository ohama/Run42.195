# Unit 34: Should (2)

## A. 문법 설명 (Grammar Explanation)

### Should have done: 과거에 대한 후회/비판

"~했어야 했는데 (안 했다)" 또는 "~하지 말았어야 했는데 (했다)"

> "I **should have** trained harder."
> 더 열심히 훈련했어야 했는데. (안 했음)

> "I **shouldn't have** eaten so much."
> 그렇게 많이 먹지 말았어야 했는데. (먹었음)

---

## B. 소설 속 예문 (Examples from the Novel)

### Example 1: Should have done (후회)

> **"I should have listened to my coach."**
> 코치님 말씀을 들었어야 했는데.

- 듣지 않은 것에 대한 후회

### Example 2: Should have done (비판)

> **"I should have trained more before the race."**
> 대회 전에 더 훈련했어야 했는데.

- 충분히 훈련하지 않은 것에 대한 후회

### Example 3: Shouldn't have done

> **"I shouldn't have run so fast at the beginning."**
> 처음에 그렇게 빨리 달리지 말았어야 했는데.

- 한 행동에 대한 후회

### Example 4: Should have (예상했던 일)

> **"The race should have started by now."**
> 대회가 지금쯤 시작했어야 하는데.

- 예상했던 일이 안 일어남

### Example 5: Shouldn't have (하지 말았어야)

> **"I shouldn't have stayed up late last night."**
> 어젯밤에 늦게까지 안 깨어 있었어야 했는데.

- 한 행동에 대한 후회

---

## C. 추가 설명 (Additional Points)

### Should have done의 의미

| 형태 | 의미 | 현실 |
|------|------|------|
| should have done | ~했어야 했다 | 안 했음 |
| shouldn't have done | ~하지 말았어야 했다 | 했음 |

### Should have vs Could have vs Would have

| 표현 | 의미 |
|------|------|
| should have | ~했어야 했다 (후회/비판) |
| could have | ~할 수 있었다 (가능성) |
| would have | ~했을 것이다 (가정) |

### 일상적인 표현

후회할 때:
- "I **should have** known."
  (알았어야 했는데.)

- "I **shouldn't have** said that."
  (그 말 하지 말았어야 했는데.)

---

## D. 비교 (Comparison)

| Should do | Should have done |
|-----------|------------------|
| 현재/미래 조언 | 과거에 대한 후회 |
| You should rest. | You should have rested. |
| (쉬어야 해) | (쉬었어야 했는데) |

### Ought to have done

Should have와 같은 의미:
> "I **ought to have** listened." = "I **should have** listened."

## Exercises

### A. Should have / Shouldn't have 완성

1. 더 열심히 훈련했어야 했는데. (안 함)
   → I _______ harder.

2. 그렇게 빨리 달리지 말았어야 했는데. (달림)
   → I _______ so fast.

3. 코치님 말씀을 들었어야 했는데. (안 들음)
   → I _______ to my coach.

4. 어젯밤에 늦게까지 안 깨어 있었어야 했는데. (깨어 있었음)
   → I _______ late last night.

**Answers:**
1. should have trained
2. shouldn't have run
3. should have listened
4. shouldn't have stayed up

### B. 상황에 맞게 작성

1. (후회) 아침을 먹었어야 했는데.
   → I _______ breakfast.

2. (후회) 그렇게 많이 먹지 말았어야 했는데.
   → I _______ so much.

3. (예상) 지금쯤 도착했어야 하는데.
   → He _______ by now.

**Answers:**
1. should have eaten
2. shouldn't have eaten
3. should have arrived

### C. 문장 변환

1. I didn't listen to my coach. (후회로 표현)
   → I _______ to my coach.

2. I ran too fast at the start. (후회로 표현)
   → I _______ so fast at the start.

**Answers:**
1. should have listened
2. shouldn't have run
