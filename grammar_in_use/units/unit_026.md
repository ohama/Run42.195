# Unit 26: Can, could, and be able to

## A. 형태 (Form)

### Can: 현재의 능력/가능성

| Subject | can | Verb | Example |
|---------|-----|------|---------|
| I/You/He/She/It/We/They | can | run | I can run 10km. |
| I/You/He/She/It/We/They | can't (cannot) | run | I can't run anymore. |
| Can | I/you/he/etc. | run? | Can you run fast? |

### Could: 과거의 능력 또는 정중한 표현

| Subject | could | Verb | Example |
|---------|-------|------|---------|
| I/He/She/etc. | could | run | I could run fast. |
| I/He/She/etc. | couldn't | run | I couldn't run 3 laps. |

### Be able to: can의 대체 표현

| Tense | Form | Example |
|-------|------|---------|
| 현재 | am/is/are able to | I am able to run. |
| 과거 | was/were able to | I was able to finish. |
| 미래 | will be able to | I will be able to run. |
| 현재완료 | have/has been able to | I have been able to improve. |

## B. Can (현재 능력/가능성)

### 용법 1: 능력 (Ability)

> **"I can run 10km now."**
> 이제 10km를 달릴 수 있어.

### 용법 2: 가능성 (Possibility)

> **"You can see the ocean from here."**
> 여기서 바다가 보여.

### 용법 3: 허락 (Permission)

> **"Can I sit here?"**
> 여기 앉아도 돼?

### From the Original Book:

> "**Can** you speak any foreign languages?"
> 외국어 할 수 있어?

> "I **can** see a ship in the distance."
> 멀리 배가 보여.

> "**Can** I use your phone?"
> 전화기 써도 돼?

### Examples from the Novel:

> **"I can run 10km now."**
> 이제 10km를 달릴 수 있어.

> **"I can't breathe. I can't move my legs."**
> 숨을 못 쉬겠어. 다리를 못 움직이겠어.

---

## C. Could (과거 능력)

### From the Original Book:

> "When I was young, I **could** run very fast."
> 어렸을 때 아주 빨리 달릴 수 있었어.

> "She **couldn't** swim until she was seven."
> 그녀는 7살까지 수영을 못 했어.

### Examples from the Novel:

> **"In elementary school, I couldn't even run three laps."**
> 초등학교 때는 세 바퀴도 못 뛰었어.

> **"Before, I couldn't walk 7km."**
> 예전에는 7km도 못 걸었어.

### Could의 제한:

**Could**는 **일반적인 과거 능력**에 사용합니다.
**특정 상황에서의 성취**에는 **was/were able to**를 사용합니다.

---

## D. Could vs Was/Were able to

### 핵심 차이:

| Could | Was/Were able to |
|-------|------------------|
| 일반적인 과거 능력 | 특정 상황에서의 성취 |
| General ability | Specific achievement |
| I could run fast (when I was young). | I was able to finish the race. |
| (어렸을 때 빨리 달릴 수 있었다) | (그 대회를 완주할 수 있었다) |

### From the Original Book:

> "The fire spread quickly, but everyone **was able to** escape." (O)
> "The fire spread quickly, but everyone **could** escape." (less natural)
> 불이 빨리 퍼졌지만 모두 탈출할 수 있었다.

> "She **could** play piano when she was six." (O)
> 6살 때 피아노를 칠 수 있었다. (일반 능력)

### Examples from the Novel:

> **"I was able to finish the race."**
> 대회를 완주할 수 있었어.

- 특정 상황에서 성공한 것

> **"After months of training, I was able to run 10km."**
> 몇 달간 훈련 후 10km를 달릴 수 있었어.

- 특정 성취

### 부정문에서는 둘 다 사용 가능:

> "I **couldn't** finish." = "I **wasn't able to** finish."
> 끝내지 못했어.

---

## E. Will be able to (미래 능력)

**Can**은 미래 시제가 없으므로 **will be able to**를 사용합니다.

### Examples:

> **"Soon I'll be able to run a marathon."**
> 곧 마라톤을 달릴 수 있을 거야.

> **"After more training, you'll be able to run faster."**
> 더 훈련하면 더 빨리 달릴 수 있을 거야.

### From the Original Book:

> "I **won't be able to** come to the party."
> 파티에 못 갈 거야.

> "**Will** you **be able to** help me tomorrow?"
> 내일 도와줄 수 있어?

---

## F. Can의 시제별 표현 (Tense Chart)

| Tense | Expression | Example |
|-------|------------|---------|
| 현재 | can | I **can** run 10km. |
| 과거 (일반) | could | I **could** run fast before. |
| 과거 (특정 성취) | was/were able to | I **was able to** finish the race. |
| 미래 | will be able to | I**'ll be able to** run a marathon. |
| 현재완료 | have/has been able to | I**'ve been able to** improve. |

---

## G. Could for Politeness (정중한 요청)

**Could**는 정중한 요청에도 사용됩니다.

| Can (일상적) | Could (더 정중) |
|-------------|-----------------|
| Can you help me? | Could you help me? |
| Can I sit here? | Could I sit here? |
| Can I ask a question? | Could I ask a question? |

### Examples from the Novel:

> **"Could you help me with my stretching?"**
> 스트레칭 좀 도와주시겠어요?

---

## H. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~I can to run.~~ | I **can run**. | can + 동사원형 (to 없음) |
| ~~He cans run.~~ | He **can run**. | can은 변하지 않음 |
| ~~I will can run.~~ | I **will be able to** run. | 미래 = will be able to |
| ~~I could finish the race.~~ (특정 성취) | I **was able to** finish the race. | 특정 성취 = was able to |
| ~~I could not able to.~~ | I **couldn't** / I **wasn't able to**. | 둘 중 하나만 사용 |
| ~~Do you can swim?~~ | **Can** you swim? | Can이 맨 앞으로 |

### Could vs Was able to 선택 가이드:

| 상황 | 선택 | 예문 |
|------|------|------|
| 어렸을 때 빨리 달렸다 (일반) | could | I could run fast. |
| 어제 대회를 완주했다 (특정) | was able to | I was able to finish. |
| 예전에 피아노 쳤다 (일반) | could | I could play piano. |
| 그 시험에 붙었다 (특정) | was able to | I was able to pass. |

## Exercises

### A. 빈칸 채우기 (Fill in the blanks)

Use can, can't, could, couldn't, or be able to.

1. Now I _______ (can/run) 10km.
2. In elementary school, I _______ (not/run) 3 laps.
3. Yesterday, I _______ (be able to/finish) the race.
4. Next year, I _______ (be able to/run) a marathon.
5. _______ you help me with my stretching?
6. When she was young, she _______ (not/swim).
7. After months of training, I _______ (be able to/complete) 10km.
8. I _______ (can/see) the finish line from here!

**Answers:**
1. can run
2. couldn't run
3. was able to finish
4. will be able to run
5. Could / Can
6. couldn't swim
7. was able to complete
8. can see

### B. Could vs Was able to 선택 (Choose the correct form)

1. When I was young, I _______ run very fast. (일반 능력)
2. The race was hard, but I _______ finish. (특정 성취)
3. She _______ play piano when she was five. (일반 능력)
4. After three hours, I _______ complete the marathon. (특정 성취)
5. He _______ speak three languages when he was a child. (일반 능력)
6. The door was stuck, but I _______ open it. (특정 성취)
7. My grandmother _______ cook very well. (일반 능력)
8. Despite the rain, we _______ finish the race. (특정 성취)

**Answers:**
1. could
2. was able to
3. could
4. was able to
5. could
6. was able to
7. could
8. were able to

### C. 문장 완성 (Complete the sentences)

1. 지금은 10km 달릴 수 있어.
   → Now I _______ 10km.

2. 초등학교 때는 달리기를 못했어.
   → In elementary school, I _______ run.

3. 곧 마라톤을 달릴 수 있을 거야.
   → Soon I _______ run a marathon.

4. 어제 대회를 완주할 수 있었어.
   → Yesterday, I _______ finish the race.

5. 도와주시겠어요? (정중하게)
   → _______ you help me?

**Answers:**
1. can run
2. couldn't
3. will be able to
4. was able to
5. Could

### D. 틀린 부분 고치기 (Error Correction)

1. I can to run 10km now.
   → I **can run** 10km now.

2. He cans swim very well.
   → He **can swim** very well.

3. I will can finish the race.
   → I **will be able to** finish the race.

4. The exam was hard, but I could pass. (특정 성취)
   → The exam was hard, but I **was able to** pass.

5. Do you can help me?
   → **Can** you help me?
