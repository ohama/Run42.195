# Unit 26: Can, could, and be able to

## A. 문법 설명 (Grammar Explanation)

### Can: 현재의 능력/가능성
- 할 수 있다 (능력)
- ~해도 된다 (허락)
- ~일 수 있다 (가능성)

### Could: 과거의 능력 또는 정중한 표현
- 할 수 있었다 (과거 능력)
- ~해 주시겠어요? (정중한 요청)

### Be able to: can의 대체 표현
- Can이 쓰일 수 없는 시제에 사용 (미래, 완료형 등)

---

## B. 소설 속 예문 (Examples from the Novel)

### Example 1: Can (현재 능력)

> **"I can run 10km now."**
> 이제 10km를 달릴 수 있어.

- 현재 할 수 있는 능력

### Example 2: Can't (불가능)

> **"I can't breathe. I can't move my legs."**
> 숨을 못 쉬겠어. 다리를 못 움직이겠어.

- 현재 할 수 없음

### Example 3: Could (과거 능력)

> **"In elementary school, I couldn't even run three laps."**
> 초등학교 때는 세 바퀴도 못 뛰었어.

- 과거에 할 수 없었던 것

### Example 4: Was able to (특정 과거 성취)

> **"I was able to finish the race."**
> 대회를 완주할 수 있었어.

- 특정 상황에서 성공한 것

### Example 5: Will be able to (미래 능력)

> **"Soon I'll be able to run a marathon."**
> 곧 마라톤을 달릴 수 있을 거야.

- 미래에 할 수 있게 될 것

---

## C. 추가 설명 (Additional Points)

### Could vs Was/Were able to

| Could | Was/Were able to |
|-------|------------------|
| 일반적인 과거 능력 | 특정 상황에서의 성취 |
| I could run fast (when I was young). | I was able to finish the race. |
| 일반적으로 할 수 있었다 | 그 상황에서 성공했다 |

**주의**: 부정문에서는 둘 다 사용 가능
> "I couldn't finish." = "I wasn't able to finish."

### Can의 시제별 표현

| 시제 | Can 대신 |
|------|----------|
| 현재 | can |
| 과거 | could / was able to |
| 미래 | will be able to |
| 현재완료 | have been able to |

### 소설 속 도윤이의 변화

| Before | After |
|--------|-------|
| I couldn't run 3 laps. | I can run 6 laps now. |
| I couldn't walk 7km. | I'm able to run 7km. |

---

## D. 비교 (Comparison)

| 표현 | 의미 | 예문 |
|------|------|------|
| can | 현재 능력 | I **can** run 10km. |
| can't | 현재 불가능 | I **can't** breathe. |
| could | 과거 일반 능력 | I **could** run fast before. |
| couldn't | 과거 불가능 | I **couldn't** even run 3 laps. |
| was able to | 특정 성취 | I **was able to** finish. |
| will be able to | 미래 능력 | I**'ll be able to** run a marathon. |

### Could for Politeness (정중함)

| Can | Could (더 정중) |
|-----|-----------------|
| Can you help me? | Could you help me? |
| Can I sit here? | Could I sit here? |

## Exercises

### A. 빈칸 채우기

1. Now I _______ (can/run) 10km.
2. In elementary school, I _______ (not/run) 3 laps.
3. Yesterday, I _______ (be able to/finish) the race.
4. Next year, I _______ (be able to/run) a marathon.
5. _______ you help me with my stretching?

**Answers:**
1. can run
2. couldn't run
3. was able to finish
4. will be able to run
5. Could / Can

### B. Could vs Was able to 선택

1. When I was young, I _______ run very fast. (일반 능력)
2. The race was hard, but I _______ finish. (특정 성취)
3. She _______ play piano when she was five. (일반 능력)
4. After three hours, I _______ complete the marathon. (특정 성취)

**Answers:**
1. could
2. was able to
3. could
4. was able to

### C. 문장 완성

1. 지금은 10km 달릴 수 있어.
   → Now I _______ 10km.

2. 초등학교 때는 달리기를 못했어.
   → In elementary school, I _______ run.

3. 곧 마라톤을 달릴 수 있을 거야.
   → Soon I _______ run a marathon.

**Answers:**
1. can run
2. couldn't
3. will be able to
