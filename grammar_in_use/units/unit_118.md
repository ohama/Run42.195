# Unit 118: Like / as if / as though

## A. Like / as if / as though의 형태와 의미 (Form and Meaning)

**Like, as if, as though**는 모두 "마치 ~처럼"의 의미로, **실제가 아닌 상황** 또는 **추측**을 표현합니다.

| Expression | Formality | Usage |
|------------|-----------|-------|
| like | 구어체 (informal) | 일상 회화 |
| as if | 중립적 | 모든 상황 |
| as though | 약간 격식체 | 격식적/문어체 |

### From the Original Book:
> "That house looks **as if** nobody lives there."
> 그 집은 마치 아무도 살지 않는 것처럼 보인다.

> "You look tired. **As if** you didn't sleep last night."
> 피곤해 보여. 마치 어젯밤에 잠을 못 잔 것처럼.

### Examples from the Novel:

> **"My legs felt like they wouldn't move anymore."**
> 다리가 마치 움직이지 않을 것처럼 느껴졌다.

> **"I felt as if I was going to collapse."**
> 마치 쓰러질 것 같은 느낌이었다.

> **"It looked as though it was going to rain."**
> 마치 비가 올 것 같았다.

---

## B. Like / as if / as though + 과거시제 (가정법)

**실제와 다른 상황**을 표현할 때 **과거시제**를 사용합니다 (가정법 과거).

### 구조:
> 주어 + look/feel/act + as if + 주어 + **과거시제**

### From the Original Book:
> "He talks **as if** he **knew** everything." (실제로는 모름)
> 그는 마치 모든 것을 아는 것처럼 말한다.

> "She acts **as if** she **were** the boss." (실제로는 보스가 아님)
> 그녀는 마치 보스인 것처럼 행동한다.

### Examples from the Novel:

> **"My heart felt like it would explode."**
> 심장이 터질 것 같았다. (실제로 터지진 않음)

> **"I felt as if I were dying."**
> 마치 죽을 것 같았다. (실제로 죽지 않음)

> **"He ran as though his life depended on it."**
> 마치 목숨이 달린 것처럼 달렸다. (실제로 목숨이 걸린 건 아님)

### 참고: 가정법에서의 were

be동사는 주어에 관계없이 **were**를 사용합니다 (격식체).

| 주어 | 가정법 | 예시 |
|------|--------|------|
| I | were | as if I **were** |
| he/she | were | as if he **were** |
| it | were | as if it **were** |
| they | were | as if they **were** |

> "He acts as if he **were** the coach." (격식체)
> "He acts as if he **was** the coach." (구어체 - 허용됨)

---

## C. Like / as if / as though + 현재시제 (사실 가능성)

**사실일 수도 있는 상황**을 표현할 때 **현재시제**를 사용합니다.

### From the Original Book:
> "You look **as if** you'**ve** just seen a ghost."
> 방금 유령을 본 것처럼 보여. (실제로 봤을 수도 있음)

> "It looks **like** it'**s** going to rain."
> 비가 올 것 같아. (실제로 올 수도 있음)

### Examples:

> **"It looks like it's going to rain."** (실제로 비가 올 수도 있음)
> 비가 올 것 같다.

> **"He looks as if he's tired."** (실제로 피곤할 수도 있음)
> 그는 피곤해 보인다.

### From the Novel:

> **"It feels like spring is coming."**
> 봄이 오는 것 같다.

> **"You look as if you've been running."**
> 달리기 한 것 같아 보여.

---

## D. It looks like / It sounds like / It seems like

**It looks/sounds/seems like**는 "~인 것 같다"의 의미로 많이 쓰입니다.

| Expression | Meaning | Example |
|------------|---------|---------|
| It looks like | ~처럼 보인다 | It looks like rain. |
| It sounds like | ~인 것 같다 (들리기로) | It sounds like fun. |
| It seems like | ~인 것 같다 | It seems like a good idea. |
| It feels like | ~인 것 같다 (느낌) | It feels like home. |

### Examples from the Novel:

> **"It looks like you've improved a lot."**
> 많이 나아진 것 같아 보여.

> **"It sounds like you had a good run."**
> 잘 달린 것 같네.

> **"It seems like the weather will be good."**
> 날씨가 좋을 것 같다.

---

## E. As if / As though + to부정사 (동작 직전)

**As if / as though + to부정사**는 "마치 ~하려는 것처럼"의 의미입니다.

### From the Original Book:
> "She stood up **as if to go**."
> 마치 가려는 듯이 일어섰다.

### Examples:

> **"He stood up as if to leave."**
> 마치 떠나려는 듯 일어섰다.

> **"She raised her hand as if to wave."**
> 마치 손을 흔들려는 듯 손을 들었다.

> **"He opened his mouth as if to speak."**
> 마치 말하려는 듯 입을 열었다.

## F. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | 설명 |
|-----------|-------------|------|
| ~~He talks like he is the boss.~~ (격식체) | He talks **as if he were** the boss. | 가정법 사용 |
| ~~She acts as if she was knowing.~~ | She acts **as if she knew**. | 상태동사 진행형 불가 |
| ~~It seems like he will come.~~ | It **seems like** he'll come. / It **seems** he'll come. | 구어체 허용 |
| ~~As if I would do that!~~ | (맞음 - 수사적 표현) | 강한 부정 표현 |
| ~~He looks as if tired.~~ | He looks **as if he were tired**. | 절 완성 필요 |

### 가정법 vs 사실 구분:

```
실제와 다른 상황 (가정) → 과거시제
He acts as if he were rich. (실제로 부자 아님)

사실일 수도 있는 상황 → 현재시제
He looks as if he's tired. (실제로 피곤할 수도)
```

### B. 괄호 안의 동사를 알맞은 형태로 바꾸세요.

1. He acts as if he (be) _______ the best runner.
2. It looks like it (rain) _______ soon.
3. She felt as though she (be) _______ floating.
4. They talked as if they (know) _______ each other for years.
5. It seems like he (be) _______ telling the truth.
6. She looked as if she (see) _______ a ghost.
7. He speaks as though he (be) _______ an expert.
8. It sounds like you (have) _______ a good time.

**Answers:**
1. were (가정법)
2. is going to rain / will rain (사실 가능성)
3. were (가정법)
4. had known / knew (과거 가정)
5. is (사실 가능성)
6. had seen (과거 완료 가정)
7. were (가정법)
8. had (사실)

### D. 틀린 부분을 고치세요.

1. He talks like he is the boss. (격식체로)
   → He talks **as if he were** the boss.

2. She acts as if she was knowing everything.
   → She acts **as if she knew** everything.

3. He looks as if tired.
   → He looks **as if he were tired**.

4. It seems like it will to rain.
   → It seems like **it will rain**.

5. My heart felt like it will explode.
   → My heart felt like it **would** explode.

## Summary: Like / as if / as though 비교표

| Structure | Example | Meaning |
|-----------|---------|---------|
| like + 명사 | run like the wind | 바람처럼 달리다 |
| like + 절 | feel like I'm dying | 죽을 것 같다 (구어체) |
| as if + 과거 | as if he were tired | 마치 피곤한 것처럼 (실제 아님) |
| as if + 현재 | as if he is tired | 피곤한 것 같다 (사실일 수도) |
| as though + 과거 | as though she knew | 마치 아는 것처럼 |
| as if + to부정사 | as if to leave | 마치 떠나려는 듯 |

## 가정법 vs 사실 비교

| 가정법 (실제 아님) | 사실 (실제일 수 있음) |
|------------------|---------------------|
| He acts as if he **were** rich. | He looks as if he **is** tired. |
| 마치 부자인 것처럼 행동한다. (실제 아님) | 피곤해 보인다. (실제로 피곤할 수도) |
| I felt as if I **were** flying. | It seems like the weather **is** improving. |
| 마치 나는 것 같았다. (실제 아님) | 날씨가 좋아지는 것 같다. (사실) |
