# Unit 8: Present Perfect (2) (I have done)

## A. 추가 용법 (Additional Uses)

### English
We also use **Present Perfect** for:
1. **Recent events** - news, announcements, recent changes
2. **Actions in a period up to now** - today, this week, this year
3. **"It's the first time..."** - experiences happening for the first time
4. **Superlatives** - the best/worst I've ever...

### Korean
**현재완료**는 다음 상황에서도 사용합니다:
1. **최근 소식** - 뉴스, 발표, 최근의 변화
2. **현재를 포함하는 기간의 일** - 오늘, 이번 주, 올해
3. **"처음으로..."** - 처음 일어나는 경험
4. **최상급** - 내가 해본 것 중 최고/최악

---

## B. 최근 소식/변화 (Recent News/Changes)

### From the Original Book:
> "**Ow!** I'**ve burnt** my finger!"
> 아야! 손가락을 데었어! (지금 아픔)

> "The road is closed. There'**s been** an accident."
> 도로가 폐쇄됐어. 사고가 났어. (지금 영향)

### Examples from the Novel:

> **"I've decided to walk to school."**
> 학교에 걸어가기로 결심했다.

- The decision was made recently and affects the present

> **"My stamina has improved."**
> 체력이 좋아졌다.

- The improvement happened and the result is visible now

---

## C. 현재를 포함하는 기간 (Periods Including Now)

### Time Expressions:

| Expression | Korean | Example |
|------------|--------|---------|
| today | 오늘 | I've run 7km today. |
| this week | 이번 주 | I've walked twice this week. |
| this month | 이번 달 | She's run 100km this month. |
| this year | 올해 | I've improved a lot this year. |
| recently | 최근에 | I haven't taken the bus recently. |
| lately | 요즘 | I've been tired lately. |
| in the last few days | 지난 며칠간 | I've trained hard in the last few days. |

### From the Novel:

> **"I've walked more than ten times."**
> 열 번 이상 걸었다.

- **I've walked** = 지금까지 누적된 횟수

> **"It's been three months since I started."**
> 시작한 지 3개월이 됐다.

- **It's been** = It has been

> **"I've run 7km today."**
> 오늘 7km를 뛰었다.

- Today isn't finished, so present perfect

## D. "It's the first/second/best time..." 구조

### From the Original Book:
> "**This is the first time** I'**ve driven** a car."
> 차를 운전해 보는 건 이번이 처음이야.

> "**It's the best film** I'**ve ever seen**."
> 내가 본 영화 중 최고야.

### Structure:

| Pattern | Structure | Example |
|---------|-----------|---------|
| 처음 | This is the first time + have + p.p. | This is the first time I've walked to school. |
| 두 번째 | It's the second time + have + p.p. | It's the second time I've met Grandpa. |
| 최고 | It's the best + have ever + p.p. | It's the best race I've ever run. |
| 최악 | It's the worst + have ever + p.p. | It's the worst weather I've ever seen. |

### Examples from the Novel:

> **"This is the first time I've run so far."**
> 이렇게 멀리 뛰어본 건 처음이다.

- **This is the first time + have + p.p.** 구조

> **"It's the longest I've ever run."**
> 내가 뛰어본 것 중 가장 길어.

> **"This is the first time I've felt this good."**
> 이렇게 기분 좋은 건 처음이야.

## E. 흔한 실수 (Common Mistakes)

| Wrong (X) | Correct (O) | Explanation |
|-----------|-------------|-------------|
| ~~This is the first time I run.~~ | This is the first time I **have run**. | 현재완료 필요 |
| ~~It's the best I ever saw.~~ | It's the best I **have ever seen**. | 현재완료 필요 |
| ~~I've finished yesterday.~~ | I **finished** yesterday. | 특정 과거 → 과거시제 |
| ~~I've been there last week.~~ | I **went** there last week. | last week → 과거시제 |
| ~~This week I walked 3 times.~~ | This week I **have walked** 3 times. | 현재 포함 기간 → 현재완료 |
| ~~It's the first time I saw this.~~ | It's the first time I **have seen** this. | 현재완료 필요 |

---

## F. 비교: "This is the first time..." vs "Last time..."

| Present Perfect | Past Simple |
|-----------------|-------------|
| This is the first time I've run. | The **last time** I ran was in March. |
| (지금까지 처음) | (과거 특정 시점) |
| It's the best I've ever felt. | I felt great yesterday. |
| (지금까지 최고) | (어제의 기분) |
| I've finished three books this month. | I finished a book last week. |
| (이번 달 - 진행 중) | (지난주 - 끝남) |

## Exercises

### A. 올바른 형태를 고르세요.

1. This is the first time I (have run / ran) a marathon.
2. I (have decided / decided) to start training yesterday.
3. It's the best race I (have ever run / ever ran).
4. She (has worked / worked) very hard this week.
5. The weather (has been / was) good today.
6. It's the longest distance I (have ever walked / ever walked).
7. I (have seen / saw) that movie three times.
8. This is the worst meal I (have ever had / ever had).

**Answers:**
1. have run (first time + 현재완료)
2. decided (yesterday = 과거)
3. have ever run (ever + 현재완료)
4. has worked (this week = 현재 포함)
5. has been (today = 현재 포함)
6. have ever walked
7. have seen (경험)
8. have ever had (최상급 + 현재완료)

### B. 문장 완성하기

1. This is the first time I _______ (walk) 7km.
2. It's the longest race I _______ ever _______ (run).
3. My stamina _______ (improve) a lot recently.
4. She _______ (not take) the bus this week.
5. It _______ (be) three months since I started walking.
6. This is the best decision I _______ ever _______ (make).
7. I _______ (train) every day this month.
8. It's the first time she _______ (feel) this confident.

**Answers:**
1. have walked
2. have ... run
3. has improved
4. hasn't taken
5. has been
6. have ... made
7. have trained
8. has felt

### C. "This is the first time..." 문장으로 변환

Change to "This is the first time..." structure:

1. I ran 10km for the first time.
   → This is the first time I **have run** 10km.

2. He completed a marathon for the first time.
   → This is the first time he **has completed** a marathon.

3. I felt this good for the first time.
   → This is the first time I **have felt** this good.

4. She visited Jeju for the first time.
   → This is the first time she **has visited** Jeju.

5. We met for the first time.
   → This is the first time we **have met**.

### D. Today/This week 문장 만들기

Make sentences with present perfect:

1. (walk / 5km / today)
   → I **have walked** 5km today.

2. (run / twice / this week)
   → He **has run** twice this week.

3. (not miss / the bus / this month)
   → I **haven't missed** the bus this month.

4. (improve / a lot / this year)
   → My running **has improved** a lot this year.

5. (not rain / this week)
   → It **hasn't rained** this week.

## Present Perfect 핵심 정리

### 언제 사용하나요?

| 상황 | 예문 | Signal Words |
|------|------|--------------|
| 최근 소식/변화 | I've finished the race! | - |
| 현재 포함 기간 | I've run 10km today. | today, this week, this year |
| 처음/최고 경험 | This is the first time I've done this. | first time, best, worst |
| 지금까지의 경험 | I've never felt this good. | ever, never, before |
| 과거-현재 연결 | I've always loved running. | always |

## Summary: Time Expressions

| Present Perfect | Past Simple |
|-----------------|-------------|
| today (아직 끝나지 않음) | yesterday |
| this week/month/year | last week/month/year |
| recently, lately | two days ago |
| already, yet, just | at 3 o'clock |
| ever, never | when I was young |
| for, since | in 2023 |
